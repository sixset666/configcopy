Общая форма "Общие настройки СофтФон"

На форме устанавливается использование подсистемы работы с телефонией "СофтФон", выполняется подключение к серверу СофтФон и заполняются параметры и настройки работы с данной подсистемой.

Настройки: 


  • Перезаполнить регистр поиска по номерам - перезаполняет регистр, в котором сопоставляются номера телефонов и абонентов. Данный регистр используется для поиска абонента по номеру его телефона при входящем звонке.
  • Общие настройки - настройки, необходимые для корректной работы СофтФона. Устанавливаются автоматически.
  • Маршрутизация - для отображения настроек маршрутизации необходимо на сервере СофтФона установить настройку "Использовать маршрутизацию звонков сервером по переданной таблице".
  • Переводить входящий звонок после - количество документов "Телефонный звонок", по достижении которого включается автоматический перевод входящих звонков данного клиента на текущий номер пользователя СофтФона или на номер из контактной информации пользователя.
  • Запись телефонных переговоров - использовать систему записи переговоров, настроенную на сервере СофтФон. Поддерживаются системы записи АТС Астериск и FreeSwitch, а также система записи Спрут для АТС серии Агат UX.
  • История звонков - для загрузки всей информации по звонкам от АТС необходимо установить галку «Использовать регламентное задание для загрузки истории звонков» и указать дату, начиная с которой необходимо загрузить историю звонков в программу. История будет загружаться в документы Телефонный звонок. Также необходимо указать расписание регламентного задания для загрузки истории.
  • Общие телефонные книги - в книге указываются сотрудники компании, отобранные по определенному условию, например, по офису или подразделению. Общие телефонные книги автоматически доступны всем пользователям в панели СофтФон.

Способ вызова

Вызывается из меню "Сервис --> Настройка параметров --> СофтФон  --> Настройки СофтФон".
