Отчет "Анализ неликвидов запчастей"

Назначение

Отчет служит для анализа неликвидных товаров по складам и контрагентам.

Относится к подсистеме: «Складской учет».

Параметры

  • Неликвид после. Количество дней без продаж и выдач в производство, чтобы признать товар неликвидом.
  • Претендент после. Количество дней без продаж и выдач в производство, чтобы анализировать товар как претендента на неликвидность.
  • Критичный запас. Количество месяцев, на которое хватит товара имеющегося на складе(при текущем расходе - количество в остатке деленое на среднемесячный расход N месяцев), чтобы товар, рассматриваемый как претендент на неликвидность по сроку без продаж, был признан таковым.
  • Тип цен. Тип цен, для показателя себестоимость по типу цен.
  • Интервал расхода. Количество дней для расчета среднемесячного(текущего) расхода. Текущий расход считается, как количество проданного / количество дней для расчета текущего расхода деленое на 30 с округлением в большую сторону. 
  • Подразделение. Подразделение по которому будет выбираться цены для расчета показателя стоимость по типу цен.

Формы

  • Анализ неликвидов запчастей. Позволяет получить информацию об остатках неликвидных товарах на складах на указанную дату. Данная форма предоставляет возможность гибкой настройки группировок строк и колонок, показателей, дополнительных полей и т.д.

Способ вызова

Вызывается из меню "Отчеты-->Складской учет-->Анализ неликвидов запчастей"

Дополнительная информация

Отчет может формироваться в только виде табличного документа.

Отчет может быть сформирован в разрезе дополнительных свойств.
