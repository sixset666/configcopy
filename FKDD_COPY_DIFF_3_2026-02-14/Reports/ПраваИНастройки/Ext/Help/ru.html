Отчет "Права и настройки"

Назначение

Отчет служит для получения упрощенной информации о правах и настройках текущей базы. Отчет выводится в виде списка с пиктограммами.
Пиктограмма жёлтого цвета означает, что данная строка является группой, зеленого - настройкой (может быть изменена пользователем), красного - правом (изменяется только администратором).
Также, в зависимости от параметров отчета может быть выведена следующая информация:
- описание
- значение по умолчанию
- способ назначения


Относится к подсистеме: «Сервисные».

Форма настройки

  позволяет настраивать получаемую информацию о правах и настройках с различной детализацией для чего используются флажки и отборы.

Флажки

  • Формировать права, Формировать настройки - означают, что отчет может формироваться как с правами и настройками, так и по отдельности. Если не выбран ни один из вариантов, отчет не будет сформирован.
  • Выводить описание права - разрешает, запрещает показывать описания прав и настроек.
  • Показывать значение по умолчанию - разрешает, запрещает показывать значение по умолчанию.
  • Только по выбранному типу - разрешает, запрещает выбор типа, для которого будут выводиться права и настройки. Существует четыре типа: компания, организация, подразделение, пользователь.

Отборы

  • Организация - Права и настройки будут выводиться только для пользователей выбранной организации.
  • Подразделение - Права и настройки будут выводиться только для пользователей выбранного подразделения.
  • Пользователь - Права и настройки будут выводиться только для выбранных пользователей.

Отборы могут использоваться как раздельно, так и вместе.

Способ вызова

Вызывается из справочника  «Пользователи»  (Справочники-->Структура компании-->Пользователи-->Печать-->Права и настройки), а также из «АРМ Администратора» (Закладка "Права и настройки", кнопка "Печать")

Дополнительная информация

Отчет может формироваться только в виде табличного документа.
