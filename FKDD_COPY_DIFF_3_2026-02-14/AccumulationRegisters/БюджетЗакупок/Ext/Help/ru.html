Регистр накопления "Бюджет закупок"

Назначение

Оборотный регистр "Бюджет закупок" предназначен для накопления информации по спланированным данным закупок в разрезе номенклатуры, либо категорий номенклатуры.

Относится к подсистеме: «Планирование».

Измерения

  • Подразделение компании. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, для которого спрогнозированы данные закупок. 
  • Сценарий планирования. Реквизит является обязательным для заполнения. Ссылается на справочник «Сценарии планирования». Сценарий планирования, в разрезе которого было произведено планирование.
  • Номенклатура. Реквизит является обязательным для заполнения. Имеет составной тип. Ссылается на справочник «Номенклатура», либо на справочник «Типы номенклатуры». Спрогнозированный товар или категория номенклатуры. 

Ресурсы

  • Количество. Тип значения: «Число». Количество спрогнозированного товара. Имеет смысл только для номенклатурного планирования. 
  • Сумма рег.. Тип значения: «Число». Спрогнозированная сумма закупок в валюте регламентированного учета.
  • Сумма упр.. Тип значения: «Число». Спрогнозированная сумма закупок в валюте управленческого учета.
  • Сумма НДС. Тип значения: «Число». Спрогнозированная сумма НДС закупок в валюте регламентированного учета.

Дополнительная информация

Из меню "Перейти" формы документа возможен просмотр движений, сформированных этим документом.

Документы, порождающие движения: «Бюджет закупок».
