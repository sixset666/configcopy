Справочник "Валюты"

Назначение

Данный справочник используется во всех видах учета (управленческом, регламентированном, бухгалтерском, налоговом и международном), а также в подсистеме бюджетирования для хранения перечня используемых валют, а также их курсов относительно базовой валюты. В справочнике валют с самого начала имеется две валюты: российский рубль и доллар США. По умолчанию в качестве валюты регламентированного учета (базовой валюты) в конфигурации установлен российский рубль.

Динамику изменения курса можно посмотреть, открыв форму элемента валюты для в списке. Данные о курсах валют хранятся в регистре сведений «Курсы валют».

С помощью стандартных клавиш сортировки и отбора можно отсортировать список курсов валют или установить период отображения истории курсов валют.

Относится к подсистеме: «Классификаторы».

Параметры

Длина кода: 3, тип кода: Строка.

Длина наименования: 10.

Не иерархический.

Реквизиты

  • Полное наименование. Тип значения: «Строка». Полное наименование валюты.
  • Параметры прописи на русском. Тип значения: «Строка». Параметры записи наименования целой и дробной части валюты на русском языке.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения реквизитов, просмотра и редактирования курсов валюты. На форме расположено табличное поле с историей изменения курсов данной валюты.
  • Форма подбора из классификатора. Дополнительная форма для печати и подбора валюты из общероссийского классификатора валют. Форма используется для быстрого подбора валют в справочник.

Печатные формы

  • Справочник "Валюты". Используется для печати справочника "Валюты".

Способ вызова

Вызывается из меню "Справочники --> Классификаторы --> Валюты".
