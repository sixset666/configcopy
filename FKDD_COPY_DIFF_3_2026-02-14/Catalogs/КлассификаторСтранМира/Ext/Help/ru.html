Справочник "Классификатор стран мира"

Назначение

В этом справочнике хранится список стран, с товарами которых наша компания когда-либо работала.

Справочник используется при заполнении справочника «Номенклатура», а также при заполнении спецификации документов, сопровождающих поступление импортных товаров.

Имеется возможность заполнить информацию в справочнике с помощью общероссийского классификатора стран мира (кнопка "Подбор из ОКСМ"). Подробности описаны в "Дополнительной информации"

Относится к подсистеме: «Классификаторы».

Параметры

Длина кода: 3, тип кода: Строка.

Длина наименования: 60.

Не иерархический.

Реквизиты

  • Полное наименование. Тип значения: «Строка». Полное наименование страны. По умолчанию копируется наименование. 
  • Нужна ГТД. Тип значения: «Булево». Признак необходимости учета ГТД из данной страны. Обычно они ставятся только для страны в которой ведется учет, например, Россия.

Табличная часть «Префиксы штрихкодов»

  • Префикс штрихкода. Тип значения: «Строка». Основной префикс штрихкодов товаров, поступающих из данной страны, в соответствии с таблицей разграничения EAN.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.

    Имеется возможность заполнить информацию в справочнике с помощью общероссийского классификатора стран мира (кнопка "Подбор из ОКСМ"). Подробности описаны в дополнительной информации.

  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов. Содержит табличную часть "Префиксы штрихкодов".
  • Форма подбора из классификатора. Дополнительная форма выбора. Используется для подбора элементов из общероссийского классификатора стран мира (ОКСМ). 

Печатные формы

  • Классификатор стран мира. Используется не для печати, а для подбора с новых стран мира из общего классификатора в этой форме.

Способ вызова

Вызывается из меню "Справочники --> Классификаторы --> Классификатор стран мира".

Дополнительная информация

Имеется возможность заполнить информацию в справочнике с помощью общероссийского классификатора стран мира (кнопка "Подбор из ОКСМ").

Откроется таблица общероссийского классификатора стран мира. В этой таблице можно осуществлять поиск стандартными средствами, предоставляемыми программой «1С:Предприятие». Непосредственно из таблицы можно добавить новую страну в справочник «Классификатор стран мира».

Для этого надо установить курсор на нужной записи и нажать клавишу «ENTER» или два раза нажать левую клавишу мыши.

Если такой элемент уже есть в справочнике, то конфигурация выдаст предупреждающее сообщение, если такого элемента нет, то будет сформирована новая форма элемента с заполненными параметрами. При нажатии на кнопку «ОК» эта информация добавится в справочник.
