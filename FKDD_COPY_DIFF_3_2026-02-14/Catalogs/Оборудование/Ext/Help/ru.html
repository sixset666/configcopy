Справочник "Оборудование"

Назначение

Справочник содержит список оборудования, подключенного к системе с настройками каждого конкретного экземпляра оборудования различных классов. Конфигурация поддерживает работу с различными классами торгового оборудования. 
При запуске система автоматически пытается проинициализировать все оборудование текущего компьютера.

Относится к подсистеме «Сервис».

Параметры

Длина кода: 5, тип кода: Число.

Длина наименования: 70.

Не иерархический.

Реквизиты

  • Идентификатор оборудования. Тип значения: «Строка». Идентификатор, позволяющий однозначно привязать элемент справочника "оборудование" к логическому устройству. Если идентификатор оборудования задан, то он должен быть уникальным
  • Класс оборудования. Реквизит является обязательным для заполнения. Ссылается на перечисление «Классы оборудования». Класс оборудования (Весы, ТСД, POS терминал, Сканер, ФР...).
  • Модель оборудования. Реквизит является обязательным для заполнения. Тип значения: «Строка».  Конкретная модель оборудования для выбранного класса.
  • Компьютер. Реквизит является обязательным для заполнения. Ссылается на справочник «Компьютеры». Компьютер, к которому подключено данное оборудование.
  • Группа товаров оборудования. Ссылается на справочник «Группы товаров оборудования». Группа, объединяющая товары, которые загружаются в данное оборудование. Для классов оборудования POS и ТСД данный реквизит является обязательным.
  • Основной отдел. Ссылается на справочник «Склады (места хранения) компании». Склад компании в котором расположено данное оборудование. Для классов оборудования POS и ТСД данный реквизит является обязательным.
  • Таймаут. Тип значения: «Число». Время ожидания ответа от удаленного оборудования, сек.
  • Момент последней загрузки. Тип значения: «Дата». Дата и время когда в последний раз были обновлены данные в оборудовании.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Справочники --> Розница и оборудование --> Оборудование".
