Справочник "Сотрудники"

Назначение

Справочник хранит список сотрудников предприятия. Используется для подстановки в реквизиты ряда справочников и документов.

Относится к подсистеме: «Структура компании».

Параметры

Длина кода: 10, тип кода: Строка.

Длина наименования: 50.

Иерархический.

Реквизиты

  • ФИО. Реквизит является обязательным для заполнения. Тип значения: «Строка». Фамилия, имя и отчество сотрудника компании.
  • Должность. Реквизит является обязательным для заполнения. Ссылается на справочник «Должности». Должность сотрудника компании.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, к которому относится сотрудник.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Организация компании, к которой относится сотрудник.
  • Дата рождения. Тип значения: «Дата». Дата рождения сотрудника компании.  
  • Дата приема. Тип значения: «Дата». Дата приема сотрудника на работу.
  • Сотрудник уволен. Тип значения: «Булево». Истина, если сотрудник уволен.
  • Дата увольнения. Тип значения: «Дата». Дата увольнения сотрудника.
  • График работы. Ссылается на справочник «Графики работы». График работы сотрудника.
  • Псевдоним. Тип значения: «Строка». Используется для группировки исполнителей для отображения в АРМ Запись на ремонт.  
  • Табельный номер. Тип значения: «Строка». Табельный номер сотрудника.
  • День рождения. Тип значения: «Строка». Месяц и число дня рождения сотрудника для сортировки списка по дням рождения. Формируется автоматически от даты рождения.
  • Показывать уволенных. Тип значения: «Булево». Если флаг установлен, то в списке также будут выводиться уволенные сотрудники.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма группы. Основная форма группы. Используется для заполнения обязательных реквизитов группы.

Способ вызова

Вызывается из меню "Справочники --> Структура компании --> Сотрудники".

Дополнительная информация

Во время работы с формой элемента доступны следующие сочетания клавиш для быстрого ввода:

  • [Ctrl+K] - Добавить новое контактное лицо;
  • [Ctrl+T] - Добавить новый телефон;
  • [Ctrl+W] - Добавить новый адрес Web-страницы;
  • [Ctrl+E] - Добавить новый адрес электронной почты;
  • [Ctrl+R] - Добавить новый адрес.

Если выбрана закладка "Контактные лица" и курсор находится на одной из строк списка контактных лиц, то при нажатии перечисленных выше сочетаний клавиш добавляется контактная информация для выбранного контактного лица, иначе добавляется контактная информация для сотрудника.
