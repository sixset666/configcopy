Справочник "Виды контактной информации"

Назначение

Справочник служит для классификации контактной информации различных объектов системы.

Для каждого вида контактной информации определен тип контактной информации из заложенного в системе фиксированного списка.

Относится к подсистеме: «Идентификационные».

Предопределенные элементы

  • Фактический адрес.
  • Юридический адрес.
  • Почтовый адрес.
  • Адрес за пределами РФ.
  • Рабочий телефон.
  • Домашний телефон.
  • Сотовый телефон.
  • Факс.
  • Адрес электронной почты (рабочий).
  • Адрес электронной почты (домашний).
  • Веб-страница организации.
  • Домашняя веб-страница.
  • Номер ICQ.
  • Прочая  информация.
  • Контактный телефоны.

Параметры

Длина кода: 8, тип кода: Строка.

Длина наименования: 50.

Не иерархический.

Реквизиты

  • Тип. Ссылается на перечисление «Типы контактной информации». Тип контактной информации. 

    Для просмотра списка сохраненных значений контактной информации используется список регистра сведений «Контактная информация».
    Вызов: подменю формы "Перейти --> Контактная информация.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для ввода и редактирования элементов, отображения списка или выбора элементов.

Способ вызова

Вызывается из меню "Справочники  --> Идентификационные --> Виды контактной информации".

Дополнительная информация

Некоторые элементы справочника «Виды контактной информации» являются предопределенными и не могут быть изменены пользователями.
