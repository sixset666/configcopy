Справочник "Значения свойств"

Назначение

Предназначен для хранения значений дополнительных свойств различных объектов, физический смысл которых не отражен в других справочниках конфигурации. В справочник "Значения свойств" вводятся подчиненные значения для различных свойств определенных в плане видов характеристик «Виды свойств». Таким образом, получается набор необходимых дополнительных справочников - расширений для каждого вида свойств, использующего этот справочник в качестве типа значения.

Относится к подсистеме: «Статьи и свойства».

Параметры

Длина кода: 8, тип кода:Строка.

Длина наименования: 50.

Не иерархический

Владельцы справочника: «Свойства объектов».

Формы

  • Форма списка. Основная форма списка и выбора. Используется для редактирования, отображения списка или выбора элементов.

Способ вызова

Вызывается из плана видов характеристик «Свойства объектов» и обработки «Значения свойств объекта».
