Справочник "Смены"

Назначение

Используется для разбиения интервала одного дня на множество интервалов. Применяется для описания графика работы сложной структуры.

Относится к подсистеме: «Планирование».

Параметры

Длина кода: 8, тип кода: Строка.

Длина наименования: 50.

Табличная часть «Интервалы»

  • Начало. Тип значения: «Дата». Время суток, от которого начинается действие смены.
  • Окончание. Тип значения: «Дата». Время суток, в котором заканчивается действие смены.
  • Продолжительность. Тип значения: «Дата». Время действия смены.
  • Вид интервала. Ссылается на перечисление «Вид интервала». Может принимать следующие значения: рабочее время, обеденный перерыв, технический перерыв.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

 
Вызывается из меню "Справочники --> Календари--> Смены".
