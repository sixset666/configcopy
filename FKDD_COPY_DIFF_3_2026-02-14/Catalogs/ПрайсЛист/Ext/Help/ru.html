Справочник "Прайс-Лист"

Назначение

Предназначен для заполнения и печати произвольных прайс-листов. Прайс-лист - это список товаров, создаваемый для покупателя. Товары в прайс-лист можно свободно добавлять и удалять, поддерживая, таким образом, актуальный ассортимент товаров или услуг для покупателя. Набор используемых в информационной базе Прайс-листов определяется в справочнике «Виды прайс-листов».

Относится к подсистеме: «Сервис».

Параметры

Длина кода: 8, тип кода: Строка.

Длина наименования: 150.

Иерархический.

Владелец справочника: «Виды прайс-листов».

Реквизиты

  • Номенклатура. Ссылается на справочник «Номенклатура». Номенклатура, соответствующая позиции прайс-листа, по которой определяются цены, штрихкоды и другие параметры при печати прайс-листа.
  • Штрихкод. Тип значения: «Строка». Уникальный штрихкод номенклатуры.
  • Владелец. Ссылается на справочник «Виды прайс-листов». Вид прайс-листа, который является владельцем данного.
  • Способ заполнения.Тип значения: «Число». Способ заполнения прайс-листа.Возможны следующие значения: по складским остаткам товара, по товарам поставщика, по всей номенклатуре и по существующему прайс-листу.
  • Склад. Ссылается на справочник «Склады компании». Склад компании по остаткам которого будет заполнятся прайс-лист. Поле активно при выбранном способе заполнения:"По складским остаткам".
  • На дату. Тип значения: «Дата». Прайс-лист будет формироваться по остаткам на указанную дату. Поле активно при выбранном способе заполнения:"По складским остаткам".
  • Поставщик. Ссылается на справочник «Контрагенты и контакты». Посатвщик по товарам которого будет заполняться прайс-лист. Поле активно при выбранном способе заполнения:"По товарам поставщика".
  • Группа. Ссылается на справочник «Номенклатура». Группа номенклатуры, по которой создается прайс-лист.
  • Прайс-лист. Ссылается на справочник «Виды прайс-листов».
  • Способ создания групп. Тип значения: «Число». Способ создания групп прайс-листов. Может принимать следующие значения: не создавать, создавать по видам товаров и создавать по группам товаров.

Табличная часть «Цены»  

В табличной части отображается информация по выделенной номенклатуре.

  • Тип цен. Ссылается на справочник «Типы цен». Типы цен которые установлены для выделенной номенклатуры.
  • Валюта. Ссылается на справочник «Валюты». Валюта, в которой указана цена выделенной номенклатуры.
  • Характеристика. Ссылается на справочник «Характеристики номенклатуры». Характеристика выделенной номенклатуры по которой указана цена.
  • Единица измерения. Ссылается на справочник «Единицы измерения». Единица измерения выделенной номенклатуры.
  • Подразделение компании. Ссылается на справочник «Подразделения компании». Подразделение компании в котором используется указанная цена на выбранную номенклатуру.
  • Цена. Тип значения: «Число». Числовое выражения цены выделенной номенклатуры.

Табличная часть «Партии»  

В табличной части отображается информация по выделенной номенклатуре.

  • Партия. Ссылается на документы, по которым были движения выделенной номенклатуры.
  • Остаток партии. Тип значения: «Число». Количество остатка выделенной номенклатуры на складе.
  • Себестоимость. Тип значения: «Число». Числовое выражение себестоимости выделенной номенклатуры.

Табличная часть «Остатки»  

В табличной части отображается информация по выделенной номенклатуре.

  • Склад компании. Ссылается на справочник «Склады компании». Склад компании, на который были поступление выделенной номенклатуры. 
  • Остаток на складе. Тип значения: «Число». Количество товара, которое есть на указанном складе.
  • Из них в резерве. Тип значения: «Число». Число зарезервированных товаров из оставшихся на указанном складе.
  • Сумма розничная. Тип значения: «Число». Розничная сумма товаров оставшихся на указанном складе.

Табличная часть «Цены поставщиков»  

В табличной части отображается информация по выделенной номенклатуре.

  • Тип цен. Ссылается на справочник «Типы цен». Типы цен которые установлены для выделенной номенклатуры у поставщика.
  • Контрагент. Ссылается на справочник «Контрагенты и контакты». Поставщик у которого приобреталась выделенная номенклатура с указанной ценой.
  • Валюта. Ссылается на справочник «Валюты». Валюта, в которой указана цена выделенной номенклатуры у поставщика.
  • Цена. Тип значения: «Число». Числовое выражения цены выделенной номенклатуры у поставщика.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для просмотра, печати, редактирования и заполнения
    прайс-листов.  отображают соответствующую информацию по номенклатуре прайс-листа. В левом нижнем углу формы есть ссылка на элемент справочника «Виды прайс-листов», который по необходимости можно редактировать (если у пользователя есть соответствующие права). С помощью кнопки "Обработка" можно выполнить две обработки: загрузить прайс-лит и выгрузить прайс-лист. Для этого открываются формы "Форма автозаполнения" и "Форма выгрузки", которые описаны ниже.  

    На верхней панели формы есть кнопки "Отображать панель информации" и "Отображать дерево", с помощью которых можно отобразить или скрыть информацию или дерево. На панели информации отображаются табличные части "Цены", "Партии", "Остатки" и "Цены поставщиков", которые описаны выше и  отображают соответствующую информацию по выделенной номенклатуре прайс-листа. .

  • Форма выбора группы. Форма выбора группы прайс-листов.  
  • Форма автозаполнения. Форма заполнения списка справочника. Используется для заполнения прайс-листа по складским остаткам, товарам поставщика и по справочнику «Номенклатура», формирования групп товаров.  Перед автозаполнением на форме следует заполнить"Способ заполнения", "Параметры заполнения" и "Способ создания группы" .
  • Форма выгрузки. Форма настройки выгрузки прайс-листа. Используется для задания параметров выгрузки прайс-листа. Имеется возможность выгрузки в различных форматах, задания полей выгрузки, сохранения настроек в файл. 

Способ вызова

Вызывается из меню "Справочники --> Номенклатурные --> Прайс-лист".

Дополнительная информация

В форме выгрузки имеется возможность выгрузки прайс-листа в файл заданного формата.

Форматы данных выгрузки прайс-листа следующие:

  • Для случая выгрузки в файл MS Excel (.XLS): в колонке "Поле таблицы" цифрами указываются соответствующие реквизитам колонки документа MS Excel, в поле "Со строки" указывается номер строки таблицы MS Excel, с которой начинаются данные, в поле "По строку" указывается номер последней строки с данными. Для приведенного примера:

      1 2 3 4 5 6 7
    1              
    2   6475647364 DVD плеер "Sony" 12000.00 100    
    3   1375647344 Плазменный телевизор "Самсунг" 21' 18000.00 10    
    4   5474644364 Акустическая система 5.1 "Sven" 9000.00 5    
    5              
     
    значения будут такими:
    - "Артикул номенклатуры" = 2;
    - "Наименование номенклатуры" = 3;
    - "Цена номенклатурной позиции" = 4;
    - "Количество номенклатуры" = 5;
    В поле ввода "Со строки" должно быть введено "2", в поле "По строку" - "4".
  • Для случая выгрузки в текстовый файл (.TXT): данные располагаются в файле через разделитель, указанный в поле ввода "Разделитель". В колонке "Поле таблицы" цифрами указываются номера значений (т.е. количество разделителей с начала строки до требуемого значения плюс 1). В поле "Со строки" указывается начальная строка, а в поле "По строку" - конечная строка текстового файла, которую нужно обработать. Например, для файла, содержащего:

    3647564736,DVD плеер "Sony",12000.00,10
    1375647344,Плазменный телевизор "Самсунг" 21',18000.00,10
    5474644364,Акустическая система 5.1 "Sven",9000.00,5

    значения будут такими:
    - "Артикул номенклатуры" = 1;
    - "Наименование номенклатуры" = 2;
    - "Цена номенклатурной позиции" = 3;
    - "Количество номенклатуры" = 4;
    В поле ввода "Со строки" должно быть введено "1", в поле "По строку" - "3". Поле "Разделитель" должно содержать ",".
  • Для случая выгрузки через источник данных ADO или в файл формата dBase (.DBF) в колонке "Поле таблицы" указываются названия полей таблицы базы данных соответствующие реквизитам. В колонке "Тип поля" указывается необходимый тип для указанного поля таблицы базы данных: Строка, Число, Дата, Булево. В колонке "Длина поля" указывается размерность поля таблицы базы данных, а в колонке "Точность поля" - точность поля типа Число.
    Примеры строки подключения через ADO:
    "Provider=Microsoft.Jet.OLEDB.4.0;Data Source=C:\Price.mdb;Persist Security Info=False"  (подключение к базе MS Access)
    "Provider=Microsoft.Jet.OLEDB.4.0;Data Source=C:\Price.xls;Persist Security Info=False;Extended Properties="Excel 8.0;HDR=0""  (подключение к книге MS Excel).

Если поле "Со строки" не заполнено, подразумевается, что там введена цифра 1, если поле "По строку" не заполнено, считываются данные с позиции, указанной в поле "Со строки" и до конца.
