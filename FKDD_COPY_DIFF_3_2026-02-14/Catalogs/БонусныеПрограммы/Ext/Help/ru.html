Справочник "Бонусные программы"

Назначение

Справочник хранит список бонусных программ предприятия. Используется для подстановки в реквизиты ряда справочников и документов.

Относится к подсистеме: «Бонусные программы».

Параметры

Длина кода: 12, тип кода: Строка.

Длина наименования: 100.

Иерархический.

Реквизиты

  • Кратность бонусов. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество денежных единиц в одном бонусе.
  • Валюта бонуса. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта, в которой начисляются бонусы.
  • Отсрочка использования бонусных баллов. Тип значения: «Число». Количество дней, через которое бонусы будет переведены в активное состояние после начисления.  
  • Срок сгорания бонусов. Тип значения: «Число». Количество дней существования бонусов после активации.
  • Запрет начисления при оплате бонусами. Тип значения: «Булево». Если "Истина" - запрещено начислять бонусы при оплате бонусами.
  • Срок напоминания перед сгоранием бонусов. Тип значения: «Число». Количество дней, за которые клиенту будет отправлено напоминание о скором сгорании бонусов.
  • Использовать SMS-рассылку. Тип значения: «Булево». Признак отправки оповещений клиенту по SMS.
  • Использовать Email-рассылку. Тип значения: «Булево». Признак отправки оповещений клиенту по электронной почте.
  • Процент начисления бонусов. Тип значения: «Число». Процент от суммы документа, начисляемый как бонус.  
  • Максимальный процент оплаты бонусами. Тип значения: «Число». Процент от суммы документа, который может быть оплачен бонусами.
  • Метод округления. Тип значения: «Булево». Метод округления для получения количества бонусов.
  • Точность. Тип значения: «Число». Точность округления количества бонусов.

Табличная часть «Проценты начислений»

  • Группа/Тип. Ссылается на справочники «Автоработы», «Номенклатура», «Типы номенклатуры». Аналитика расчета.
  • Процент. Тип значения: «Число». Процент для группы.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.  

Способ вызова

Вызывается из меню "Все действия--> Справочники --> Бонусные программы".
