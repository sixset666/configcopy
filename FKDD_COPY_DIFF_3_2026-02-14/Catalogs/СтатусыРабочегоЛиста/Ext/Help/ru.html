Справочник «Статус рабочего листа»

Назначение

Справочник описывает возможные статусы рабочего листа продавца автомобилей, предлагаемых страхователю или застрахованному при наступлении страхового случая. Используется в качестве статуса в документах "Рабочий лист", "Рабочий лист кредитного отедла", "Рабочий лист отдела страхования".

Параметры

Длина кода: 8, тип кода: Строка.

Длина наименования: 50.

Не иерархический.

Реквизиты

  • Вероятность продажи. Реквизит является обязательным для заполнения. Тип значения: «Число». Указывает на вероятность продажи (%).
  • Использовать в рабочем листе. Тип значения: «Булево». Признак использования статуса в Рабочем листе
  • Использовать в рабочем листе отдела страхования. Тип значения: «Булево». Признак использования статуса в Рабочем листе отдела.
  • Использовать в рабочем листе кредитного отдела. Тип значения: «Булево». Признак использования статуса в Рабочем листе кредитного отдела.
  • Порядок в отчете. Реквизит является обязательным для заполнения. Тип значения: «Число». Порядок для сортировки в отчетах.
  • Комментарий. Комментарий к статусу рабочего листа.

Табличная часть "Шаблоны событий"

  • Сдвиг. Сдвиг даты создаваемого события относительно текущей в днях. Используется для документов вида: Событие, SMS, Тест-драйв. Для остальных документов заполнять не обязательно.
  • Шаблон события. Идентификатор шаблона документа (из регистра сведений "Картинки и файлы"), используемый для создания нового документа при переходе рабочего лита в данный статус. Открыть обработку "Шаблоны событий" можно через подменю "Перейти"-->"Открыть обработку шаблонов".

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка. Реквизит табличной части "Цвет" позволяет задавать пользователю свои цвета статусов рабочего листа.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма выбора шаблона. Форма выбора шаблона заполнения документа при переходе документа "Рабочий лист" в данный статус. 

Способ вызова

Вызывается из меню "Справочники --> Взаимоотношения с клиентами --> Статусы рабочего листа".
