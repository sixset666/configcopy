Справочник "Описание последовательностей событий"

Назначение

Справочник хранит информацию настроек вида событий при планировании нового события.

Параметры

Длина кода: 8, тип кода:Строка.

Длина наименования: 100.

Не иерархический.

Реквизиты

  • Вид события. Ссылается на перечисление «Виды событий». Вид события, для которого устанавливаются параметры.
  • Последнее. Тип значения: «Булево». Признак того, что данный вид события не может быть использован для указания следующего события.
  • Может повторяться. Тип значения: «Булево». Признак того, что данный вид события может повторяться в запланированном событии.
  • Может быть первым. Тип значения: «Булево». Признак того, что данный вид события может использоваться для события, к которому вводятся запланированные.

Табличная часть «Очередные события»

  • Вид события. Ссылается на перечисление «Виды событий». Вид события, которое выводится для выбора в планируемом следующем событии.
  • При наличии. Ссылается на перечисление «Виды событий». Для выбора в следующем событии вида события строки необходимо наличие в предыдущих событиях данного вида события.
  • Планируется. Тип значения: «Булево». Устанавливается при заполненом реквизите «При наличии». Признак того, что если имеется событие с видом события, который указан в реквизите  "При наличии", то на этапе его планирования доступна возможность запланировать следующее событие.
  • При отсутствии. Ссылается на перечисление «Виды событий». Для выбора в следующем событии вида события строки при условии отсутствия данного вида события.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Справочники --> Взаимоотношения с клиентами --> Описание последовательностей событий".
