Справочник "Контактные лица"

Назначение

Справочник служит для хранения контактных лиц и данных о них.

Для каждого контактного лица хранится его контактная информация. Также на соответствующей закладке можно просмотреть список событий и электронных писем по данному контактному лицу.

Относится к подсистеме: «Идентификационные».

Параметры

Длина кода: 8, тип кода: Строка.

Длина наименования: 150.

Не иерархический.

Владельцы справочника: «Контрагенты», «Организации», «Сотрудники».

Реквизиты

  • Фамилия. Тип значения: «Строка». Фамилия контактного лица.
  • Имя. Тип значения: «Строка». Полное имя контактного лица.
  • Отчество. Тип значения: «Строка». Отчество контактного лица.
  • Должность. Ссылается на справочник «Должности». Должность контактного лица.
  • Пол. Ссылается на перечисление «Пол физических лиц». Пол контактного лица. Может принимать следующие значения: не указан, мужской, женский.
  • Дата рождения. Тип значения: «Дата». Дата рождения контактного лица.
  • Кураторы. Ссылается на регистр сведений «Кураторы». Сотрудник курирующий данное контактное лицо. Ввод куратора возможен только для контактных лиц контрагентов. Информация о кураторах сохраняется с учетом истории изменения. Для просмотра ранее установленных значений используется список регистра сведений «Сведения компании».
    Вызов: подменю формы "Перейти --> Сведения компании".
  • График работы. Ссылается на справочник «Графики работы». График работы контактного лица.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий.

Табличная часть «События и электронные письма»

  • Операция. Ссылается на справочник «Хозяйственная операция». Выполняемая хозяйственная операция.
  • Дата. Тип значения: «Дата». Дата выполнения хозяйственной операции.
  • Время. Тип значения: «Дата». Время выполнения хозяйственной операции.
  • Документ. Ссылается на документы. Документ, подтверждающий выполнение хозяйственной операции.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов. Над списком контактных лиц помещено поле для установки отбора по владельцу. На форме также имеется кнопка "Найти клиента", при нажатии на которую запускается обработка «Полнотекстовый поиск данных», осуществляющая интеллектуальный поиск по базе данных.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов. На форме можно создать напоминание с помощью ссылки «Создать напоминание» Форма содержит две закладки, которые устанавливают соответствующие параметры подразделения.
    • Закладка "Контактная информация". Предоставляет возможность просмотра и редактирования регистра сведений «Контактная информация» связанного с контактным лицом.

      Вызов: подменю формы "Перейти --> Контактная информация".

    • Закладка "События и электронные письма". Предоставляет возможность просмотра и редактирования списка «Электронных писем» и «Событий» связанных с контактным лицом.

Можно посмотреть «Подтверждающие документы» контактного лица с помощью подменю формы "Перейти --> Подтверждающие документы".

Способ вызова

Вызывается из меню "Справочники  --> Идентификационные --> Контактные лица".

Дополнительная информация

Во время работы с формой элемента доступны следующие сочетания клавиш для быстрого ввода:

  • [Ctrl+T] - Добавить новый телефон;
  • [Ctrl+W] - Добавить новый адрес Web-страницы;
  • [Ctrl+E] - Добавить новый адрес электронной почты;
  • [Ctrl+R] - Добавить новый адрес.
