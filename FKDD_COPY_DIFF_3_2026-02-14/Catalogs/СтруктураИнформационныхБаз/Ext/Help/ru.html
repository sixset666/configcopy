Справочник "Структура информационных баз"

Назначение

В данном справочнике хранится схема распределенной БД.

Относится к подсистеме: «Сервис».

Параметры

Длина кода: 8, тип кода:Строка.

Длина наименования: 70.

Иерархический. Иерархия элементов.

Реквизиты

  • Префикс. Реквизит является уникальным и обязательным для заполнения. Тип значения "Строка". Хранит префикс информационной базы, совпадающий с префикс.ом узла плана обмена.
  • Настройка доставки по умолчанию. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Настройки доставки сообщений». Определяет способ доставки (транспорта) сообщений по умолчанию для текущей информационной базы.
  • Режим проведения по партиям. Реквизит является обязательным для заполнения. Ссылается на перечисление "РежимыПроведенияПартий". Отложенное проведение по партиям.

    При выбранном режиме "Партионные движения есть и выполняется допроведение" будут учитываться партионные движения и можно выполнять допроведения документов по партиям. Список документов которые необходимо допровести по партиям хранится в регистре сведений «Допроведения по партиям».
    Вызов: подменю формы "Перейти --> Допроведения по партиям".

    Режим "Партионные движения отсутствуют" означает, что учет партионных движений на предприятии не ведется.

    Режим "Партионные движения есть (допроведения НЕ выполняются)" означает, что на предприятии ведется партионный учет, но допроводить документы по партиям нельзя.

Табличная часть «Подразделения»

  • Подразделение. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, соответствующее текущей информационной базе.  

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Справочники--> Структура компании --> Структура информационных баз".
