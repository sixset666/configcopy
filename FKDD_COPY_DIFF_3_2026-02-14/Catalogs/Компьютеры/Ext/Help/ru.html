Справочник "Компьютеры"

Назначение

Содержит список всех компьютеров локальной сети, с которых осуществляется доступ к информационной базе. При первом запуске с нового рабочего места, автоматически создается новый элемент справочника.

Относится к подсистеме: «Розница».

Параметры

Длина кода: 24, тип кода: Число.

Длина наименования: 30.

Не иерархический.

Реквизиты

  • Касса ККМ. Ссылается на справочник «Кассы ККМ». Касса ККМ, к которой по умолчанию относиться данный компьютер.
  • Основной отдел. Ссылается на справочник «Склады (места хранения)». Склад компании (отдел), с которого по умолчанию осуществляется продажа товаров в розницу («Фронт кассира»).
  • ФР. Ссылается на справочник «Оборудование». Фискальный регистратор, с которым данный компьютер работает по умолчанию.
  • POS. Ссылается на справочник «Оборудование». POS, с которым данный компьютер работает по умолчанию.
  • Дисплей. Ссылается на справочник «Оборудование». Дисплей покупателя, с которым данный компьютер работает по умолчанию.
  • Авторизатор. Ссылается на справочник «Оборудование». Авторизатор безналичных платежей, с которым данный компьютер работает по умолчанию.
  • Весы. Ссылается на справочник «Оборудование». Электронные весы, с которыми данный компьютер работает по умолчанию.
  • Сканер. Ссылается на справочник «Оборудование». Сканер штрихкодов, с которым данный компьютер работает по умолчанию.
  • ТСД. Ссылается на справочник «Оборудование». Терминал сбора данных, с которым данный компьютер работает по умолчанию.
  • Принтер. Ссылается на справочник «Оборудование». Принтер этикеток и ценников, с которым данный компьютер работает по умолчанию.
  • Принтер чеков. Ссылается на справочник «Оборудование». Принтер чеков, с которым данный компьютер работает по умолчанию.
  • Клавиатура. Ссылается на справочник «Оборудование». Клавиатура кассира, с которой данный компьютер работает по умолчанию.
  • Включать оборудование только при использовании. Тип значения: «Число». При установке данного флажка не будет происходить автоматическое подключение оборудования, подключенного к данному компьютеру. В таком случае любое оборудование перед началом использования должно быть предварительно включено. Используется при работе в режиме терминальной сессии и устанавливается для компьютера, соответствующего серверу терминального доступа. Значение по умолчанию - флажок не установлен.
  • Вести архив чеков. Тип значения: «Булево». При установке данного флажка при формировании чеков через фронт кассира дополнительно будет вестись архив чеков. 
  • Локальный каталог системы защиты. Тип значения: «Строка». Каталог в который копируются компоненты и настройки системы защиты, для ускорения запуска информационной базы и возможности работы распределенной информационной базы в отрыве от сети.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Справочники --> Розница и оборудование --> Компьютеры".
