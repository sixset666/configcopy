Справочник "Карточки"

Назначение

В справочнике хранятся идентификационные карточки следующих видов:
 - Дисконтные карты, использующиеся для назначения скидок на чек (групповые скидки на сумму чека).
 - Карточки пользователей программы, использующиеся для смены текущего пользователя во время работы системы.
 - Карточки служебных (системных) команд, использующиеся для упрощения работы с базой данных при помощи сканеров штрихкодов или магнитных карт.
 - Чистые. Заранее заказанные и изготовленные карточки (обычно магнитные), но ещё не получившие в базе данных определенного назначения.
 - Клубная карта используется для оплаты покупок по предоплате и в кредит.
 - Талон. Карта с фиксированным номиналом. Используется для оплаты покупок по предоплате.

Все карточки однозначно идентифицируются уникальным 50-ти значным буквенно-цифровым кодом (Наименование элемента). При срабатывании сканера ШК или МК, программа пытается найти карточку по полученному коду и выполнить соответствующие ей и текущему режиму работы действия.

Относится к подсистеме: «Розница».

Предопределенные элементы

  • Дисконтная карта. Группа по умолчанию для дисконтных карточек.
  • Карточка пользователя. Группа по умолчанию для пользовательских карточек.
  • Служебная операция. Группа по умолчанию для карточек служебных (системных) команд.
  • Чистая. Группа по умолчанию для чистых карточек (заготовок). 
  • Клубная карта. Группа по умолчанию для клубных карт.
  • Талон. Группа по умолчанию для талонов.
  • Не найденная в системе карта. Дисконтная карта по умолчанию при загрузке из ККМ (если указанная карточка отсутствует в ИБ). 

Параметры

Длина кода: 8, тип кода:Строка.

Длина наименования: 50.

Иерархический.

Реквизиты

  • Вид карточки. Реквизит является обязательным для заполнения. Ссылается на перечисление «Виды карточек». Вид карточки. См. назначение. 
  • Объект. Типы значений: «Строка», ссылка на справочники: «Контрагенты и контакты», «Пользователи», «Договоры взаиморасчетов». Объект, идентифицируемый карточкой в зависимости от вида.
  • Полное наименование. Тип значения: «Строка». Печатное наименование карточки.
  • Штрихкод. Тип значения: «Строка». Штрихкод карточки. Если данный реквизит не заполнен, то карточка не сможет быть считана сканером штрих-кодов. 
  • Комментарий. Тип значения: «Строка». Произвольный комментарий по карточке.
  • Карточка заблокирована. Тип значения: «Булево». Флаг блокировки карточки.
  • Номинал талона. Тип значения: «Число». Реквизит активный при создании талона. Определяет номинал талона.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма группы. Основная форма группы. Используется для заполнения обязательных реквизитов группы.
  • Форма выбора группы. Основная форма выбора группы. Используется для выбора групп элементов.
  • Форма серии. Дополнительная форма объекта. Используется для автоматического ввода большого количества  карточек.
  • Форма выбора фронта. Дополнительная форма объекта. Используется для выбора карточки в обработке «Фронт кассира».

Способ вызова

Вызывается из меню "Справочники --> Розница и оборудование --> Карточки".
