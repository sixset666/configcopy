Справочник "Патенты"

Назначение

В справочник заносится информация о патентах, оформленных на организацию.

Параметры

Длина кода: 9, тип кода: Строка.

Длина наименования: 100.

Не иерархический.

Реквизиты

  • Номер патента.  Реквизит является обязательным для заполнения. Регистрационный номер патента.
  • Дата начала. Реквизит является обязательным для заполнения. Дата начала действия патента. 
  • Дата окончания. Реквизит является обязательным для заполнения. Дата окончания действия патента. 
  • Налоговая база. Реквизит является обязательным для заполнения. Потенциально возможный годовой доход. 
  • КБК. Ссылается на справочник «Налоги и взносы». Заполняется только код БК. 

Способ вызова

Вызывается из меню "Организация --> Патенты".
