Справочник "Прочие активы"

Назначение

В справочник "Прочие активы" заносится информация обо всех нетоварных активах, включаемых в оформляемые в компании документы.

Относится к подсистеме: «Нетоварные активы».

Параметры

Длина кода: 8, тип кода: Строка.

Длина наименования: 50.

Иерархический.

Реквизиты

  • Родитель. Ссылается на справочник «Прочие активы». Актив, который является родителем данному.
  • Вид прочего актива. Реквизит является обязательным для заполнения. Ссылается на перечисление «Виды прочих активов». Вид нетоварного актива. Может принимать следующие значения: основное средство (ОС), нематериальный актив (НМА), объект строительства, оборудование, спецоснастка, инструменты, инвестиционный актив.
  • Тип номенклатуры. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы номенклатуры». Тип номенклатуры.
  • Номенклатура. Реквизит является обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура.
  • Основной тип эксплуатации. Ссылается на справочник «Типы эксплуатации». Основной, используемый по умолчанию, тип эксплуатации. 
  • Серийный номер. Тип значения: «Строка». Серийный номер. 
  • Первоначальная стоимость. Тип значения: «Число». Первоначальная (начальная) стоимость. 
  • Инвентарный номер. Тип значения: «Строка». Инвентарный номер.
  • Балансов. стоимость. Тип значения: «Число». Текущая балансовая стоимость актива.
  • Начисл. стоимость. Тип значения: «Число». Текущая начисленная стоимость актива.
  • Остаточн. стоимость. Тип значения: «Число». Текущая остаточная стоимость актива.
  • Штирхкод. Тип значения: «Строка». Уникальный штрихкод актива.
  • Дата ввода. Тип значения: «Дата». Дата ввода актива в эксплуатацию, заполняется при проведении документа «Ввод в эксплуатацию».
  • Срок полезного использования. Тип значения: «Число». Реквизит является обязательным для заполнения. Срок полезного использования в месяцах.
  • Дата выбытия. Тип значения: «Дата». Дата выбытия актива из эксплуатации, заполняется при проведении документов «Списание активов», «Реализация активов».
  • Коэффициент ускорения. Тип значения: «Число». Коэффициент ускорения амортизации. Имеет смысл только при нелинейном способе начисления.
  • Ресурс выработки. Тип значения: «Число». Ресурс выработки.
  • Единица выработки. Тип значения: «Строка». Единица выработки.
  • Комментарий. Тип значения: «Строка». Дополнительная информация.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма группы. Основная форма группы. Используется для заполнения обязательных реквизитов группы.

Способ вызова

Вызывается из меню "Справочники --> Прочие активы --> Прочие активы".
