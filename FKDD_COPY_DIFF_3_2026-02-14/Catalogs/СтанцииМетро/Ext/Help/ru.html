Справочник "Станции метро"

Назначение

Справочник станций метрополитена (используется как дополнение адреса и для сводной отчетности по станциям метро).

Можно автоматически заполнить станциями метрополитена при помощи кнопки "Заполнить". Подробности в дополнительной информации.

Относится к подсистеме: «Классификаторы».

Параметры

Длина кода: 8, тип кода: Строка.

Длина наименования: 50.

Иерархический.

Реквизиты 

  • Комментарий. Тип значения: «Строка». Произвольный комментарий к элементу справочника.
  • Раскрашивать станции метро. Тип значения: «Булево». При установленном флаге станции метро раскрашиваются в разные цвета.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов. Редактирование и ввод новых элементов производится также в форме списка справочника.

Способ вызова

Вызывается из меню "Справочники --> Классификаторы --> Станции метро".

Дополнительная информация

Загрузка станций метрополитена из макета выполняется при нажатии кнопки "Загрузить" на форме списка справочника и при первом старте системы. Перед загрузкой пользователю будет предложено выбрать из списка макет, из которого будет загружен список станций.
