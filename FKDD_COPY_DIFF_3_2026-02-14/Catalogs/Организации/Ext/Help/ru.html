Справочник "Организации"

Назначение

Справочник "Организации" предназначен для описания формальной структуры компании, которая может состоять из нескольких хозяйствующих субъектов, фактически принадлежащих одному и тому же собственнику или группе собственников, действующих согласованно в целях получения прибыли.

Организациями могут быть:

  • юридические лица;
  • обособленные подразделения юридических лиц;
  • ПБОЮЛ.

В справочнике хранится статичная информация, идентифицирующая организации, и дополнительные сведения, необходимые для ведения учета по ним в единой информационной базе. Также, в реквизитах справочника хранятся статические данные, характеризующие организацию с точки зрения управленческого учета компании.

Для каждой организации может быть назначен префикс номера документов. Это позволяет вести раздельную нумерацию для документов, оформленных от имени разных организаций.

Относится к подсистеме: «Структура компании».

Предопределенные элементы

  • Переименуйте этот элемент на вашу организацию. Элемент, зарезервированный под вашу организацию. Перед началом работы измените наименование и заполните необходимые реквизиты.

Параметры

Длина кода: 8, тип кода: Строка.

Длина наименования: 50.

Не иерархический.

Реквизиты

  • Полное наименование. Тип значения: «Строка». Полное наименование организации.
  • Префикс. Тип значения: «Строка». Префикс автонумерации для документов (до 3-х символов). Подстановка префикса в документ зависит от параметров префиксации, указанных в правах и настройках.
    Доступ осуществляется через «АРМ администратора».
  • Форма собственности. Ссылается на перечисление «Формы собственности». Форма собственности.
  • Фамилия. Тип значения: «Строка». Обязательно для заполнения для форм собственности "Частный предприниматель" и "Частное лицо".
  • Имя. Тип значения: «Строка». Обязательно для заполнения для форм собственности "Частный предприниматель" и "Частное лицо".
  • Отчество. Тип значения: «Строка». Необязательно к заполнению.
  • НЕ является плательщиком НДС. Тип значения: «Булево». Если флажок установлен, то при заполнении документов учет НДС производится не будет.
  • Дата регистрации. Тип значения: «Дата». Дата регистрации организации.
  • ИНН. Тип значения: «Число». Идентификационный номер налогоплательщика.
  • КПП. Тип значения: «Число». Код причины постановки на учет.
  • Код по ОКПО. Тип значения: «Строка». Код по ОКПО.
  • ГНИ. Тип значения: «Строка». Государственная налоговая инспекция.
  • Код по КОПФ. Тип значения: «Строка». Код организационно-правовой формы организации по КОПФ.
  • Код по КФС. Тип значения: «Строка». Код формы собственности по КФС.
  • Код по ОКВЭД. Тип значения: «Строка». Код вида экономической деятельности по ОКВЭД.
  • Код по ОКДП. Тип значения: «Строка». Код по ОКДП основного вида деятельности.
  • Код по ОКОНХ. Тип значения: «Строка». Код отрасли по ОКОНХ.
  • ОГРН. Тип значения: «Строка». Основной государственный регистрационный номер.  
  • Код ИМНС. Тип значения: «Строка». Код налоговой инспекции организации. 
  • Руководитель. Ссылается на справочник «Сотрудники», по умолчанию выводятся сотрудники Организации.  Директор организации.
  • Главный бухгалтер. Ссылается на справочник «Сотрудники», по умолчанию выводятся сотрудники Организации.  Главный бухгалтер организации. 
  • Кассир. Ссылается на справочник «Сотрудники», по умолчанию выводятся сотрудники Организации.  Главный кассир организации.
  • МОЛ. Ссылается на справочник «Сотрудники», по умолчанию выводятся сотрудники Организации.  Материально ответственное лицо организации.

Информация о ответственных лицах организации сохраняется с учетом истории изменения. Для просмотра ранее установленных значений используется список регистра сведений «Сведения компании».
Вызов: подменю формы "Перейти --> Сведения компании".

  • Основной расчетный счет. Ссылается на справочник «Банковские счета». Основной банковский счет организации.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов. Также форма элемента используется для заполнения параметров организации. Форма содержит несколько закладок, каждая из которых устанавливает соответствующие параметры организации.
    • Закладка "Основные данные". Устанавливаются основные данные о предприятии: форма собственности, дата регистрации, плательщик НДС или нет, коды регистрации и ответственные лица организации. 
    • Закладка "Контактная информация". Предоставляет возможность просмотра и редактирования регистра сведений «Контактная информация» связанного с организацией.
    • Закладка "Счета и подтверждающие документы". Предоставляет возможность просмотра и редактирования элемента справочника «Банковские счета» и элемента справочника «Подтверждающие документы» связанных с организацией.

Способ вызова

Вызывается из меню "Справочники --> Структура компании --> Организации".

  • [Ctrl+R] - Добавить новый адрес.
