Обработка "Обращение контрагента"

Назначение

Предназначена для приема обращения контрагента на основании документа «Событие».

Формы

  • Форма. Основная форма. Используется для поиска контрагента по ключевым полям: наименованию контрагента, телефона, электронной почты. Результат поиска выводится в список "Выбор клинета события". В таблице "Операции клиента" выводятся документы контрагента по текущей строке список "Выбор клинета события". Состав документов настраиваемый. Информация о событии выводится в поле "Информация обращения". Для формирования следующего события необходимо создать новый документ с помощью кнопок на нижней командной панели или выбрать из существующих документов данного контрагента. Если контрагент не был найден или в списке контарагентов выделена строка "Новый контрагент", то перед обработкой обращения будет открыта форма быстрого ввода контрагента. На данной форме можно отменить обращение, указав при этом причину отмены.
  •  Форма выбора видов документа. Вызывается из основной формы обработки для установки состава документов, которые будут выведены на форму для контрагента.

Способ вызова

Вызывается из формы документа или формы списка документа "Событие" по кнопке "Принять обращение"
