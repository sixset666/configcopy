Обработка "Обработка документов"

Назначение

Предназначена для групповой обработки документов.

Относится к подсистеме: «Сервис».

Формы

  • Форма. Основная форма. В списке "Виды документов" перечислены виды документов, которые присутствуют в системе (по умолчанию заполняется всеми видами документов). Флажками отмечаются необходимые для обработки виды документов. 
    В списке "Отбор по реквизитам" перечисляется набор фильтров, необходимых для обработки (имеется определенный набор значений, по умолчанию он выводится список общий для всех видов документов, но со снятыми флажками). По мере выделения документов в списке список доступных отборов изменяется. Список доступных отоборов всегда соответствует списку общих реквизитов выделенных видов документов.
    По кнопке "Заполнить" осуществляется перезаполнение списка документов (он находится на другой закладке и во время настройки недоступен) и активизируется страница со списком документов. Этот список ("Выбранные документы") можно заполнять и "вручную". Флажками отмечаются документы, участвующие в обработке (по умолчанию - все документы). По кнопке "Настройка" можно вернуться на первую страницу обработки и поменять настройки фильтров выбираемых документов. Нажатие на кнопку "Настройка параметров" открывает форму, где можно задать выполнение обработки в транзакции и индикацию выполнения обработки.
    Кнопка "Выполнить" осуществляет выполнение действия, установленного из выпадающего списка "Действие", со списком выбранных документов. Вывести на печать текущий документ в списке выбранных можно, нажав на кнопку "Печать". Имеется возможность останова по ошибке во время выполнения действий обработки - флажок "Не прерывать выполнение при ошибках" (по умолчанию флажок установлен).

Способ вызова

Вызывается из меню "Обработки-->Обработка документов"
