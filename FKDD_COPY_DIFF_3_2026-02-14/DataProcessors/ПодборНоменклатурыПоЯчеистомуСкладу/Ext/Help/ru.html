Обработки "Подбор номенклатуры по ячеистому складу"

Назначение

Позволяет подбирать товары в документ, а не вводить эти товары построчно.

Относится к подсистеме: «Сервис».

Формы

  • Форма для размещения. Основная форма. В верхней части формы расположен список подбираемых товаров. При выборе товара в этом списке он переносится в документ. В нижней части расположен список ячеек текущего склада (склад устанавливается из документа, в котором открыта форма подбора).
    Над списком выбираемых товаров расположен переключатель с вариантами отбора списка ячеек:
    - Выводить все ячейки на складе - отбор отключен, выводятся все ячейки;
    - Выводить все свободные ячейки на складе - выводятся все непустые ячейки;
    - Выводить ячейки, в которые товар уже помещался - выводятся только те ячейки, в которые когда-либо помещался товар, выбранный в списке подбираемых товаров;
    - Выводить ячейку оптимального размещения товара - выводится свободная ячейка, наиболее оптимально расположенная для размещения товара по отношению к ячейке, введенной в последней строке документа.
    - В разделе Список номенклатуры есть два флажка: Запрашивать количество и Выводить вопрос при замене ячеек. Если флажок Запрашивать количество снят, то при выборе товара, он (товар) добавится в документ с количеством=1. Иначе при выборе товара будет открыта форма Ввод количества, в которой можно ввести количество товара. При установленном флажке Выводить вопрос при замене ячеек вопрос будет выводиться в том случае, если выбираемый товар уже есть в табличной части и значение поля Ячейка отличается от выбираемого. 
  • Форма подбора с остатками.  В верхней части формы расположен список подбираемых товаров. При выборе товара в этом списке он переносится в документ. В нижней части расположен список ячеек текущего склада (склад устанавливается из документа, в котором открыта форма подбора).
    Над списком выбираемых товаров расположен переключатель с вариантами отбора списка ячеек:
    - Выводить все ячейки на складе - отбор отключен, выводятся все ячейки;
    - Выводить ячейки, в которых есть остатки текущего товара  - выводятся только те ячейки, в которых есть остатки товара, выбранного в списке подбираемых товаров;
    - В разделе Список номенклатуры находится флажок Запрашивать количество. Если флажок снят, то при выборе товара, он (товар) добавится в документ с количеством=1. Иначе при выборе товара будет открыта форма Ввод количества, в которой можно ввести количество товара.  
  • Ввод количества. Стандартная дополнительная форма для ввода параметров. Форма разделена на 2 части: в левой части расположены поля для ввода новых параметров (количество...) подбора, в правой части показаны предыдущие значения параметров подбора.

Способ вызова

Вызывается из форм складских документов по кнопке "Подбор номенклатуры по ячеистому складу"
