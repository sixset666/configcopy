Форма "Операции с заказами покупателей"

Назначение

Форма предназначена для ввода параметров при выполнения различных операций с заказом покупателя.  В результате ввода параметров на форме, в конфигурации в фоновом режиме (без показа формы на экране) создаются и проводятся новые документы.

Операции с заказми:

1. Резервировать. Резервирование номенклатуры по выбранному заказу на склад, который указан в заказе. Создается документ "Резервирование заказов покупателя". В колонке "Доступно" указывается общее количество, доступное для всех строк в сумме.

2. Снять резерв. Снятие резервов номенклатуры по выбранному заказу. Создается документ "Снятие резервов заказов покупателя". В колонке "Доступно" указывается количество, доступное для текущей строки.

3. Отменить заказ. Снятие резервов номенклатуры по выбранному заказу и корректировка заказанного количества. Создается документ "Снятие резервов заказов покупателя" на заданное количество номенклатуры, находящееся в резерве по различным складах. Оставшееся количество, которое не было в резерве, корректируется документом "Корректировка заказа покупателя".

4. Отдать заказ. Снятие резервов номенклатуры по выбранному заказу, корректировка заказанного количества и ввод заказа и резервирования для другого контрагента. Выполняется создание документов, аналогично отмене заказа. После этого вводится документ "Заказ покупателя" с новым контрагентом, хозяйственная операция "Заказ и резервирование покупателя". Операция возможна в случае, если количества товара в наличие достаточно для постановки в резерв.

Форма состоит из следующих элементов:

  • Товары.  В таблице в каждой строке указывается количество товара (не должно превышать значение в колонках "доступно" или "заказано" в зависимости от вида операции). Серым цветом выделяются строки, где нет доступного количества для операции, красным цветом - строки, где указанное количество превышает доступное.
  • Новый клиент. Только для операции "Отдать заказ".  Клиент, которому передается заказанная номенклатура из заказа покупателя.
  • Комментарий. Произвольный комментарий, который будет записан во все новые документы  в результате выполнения операции.

Способ вызова

Открывается из «АРМ Корзина» на странице "Заказы покупателей" при выполнении операций с заказом. Отдельно не используется.
