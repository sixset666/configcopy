Обработка "Упущенный спрос"

Назначение

Обработка предназначена для регистрации случая потери сделки с клиентом из-за отсутствия в данный момент нужного товара или для регистрации пожелания клиента добавить товар в ассортимент. По накопившимся записям можно сделать дополнительные выводы в стратегии закупок: расширение ассортимента новым товаром, увеличение объема закупки востребованного товара. Данные по упущенному спросу анализируются при помощи отчета «Упущенный спрос».

Относится к подсистеме: «Отгрузка ТМЦ».

Реквизиты

  • № по каталогу. Тип значения: «Строка». Код производителя.
  • Производитель. Тип значения: «СправочникСсылка.Производители». Производитель товара, за который не получили прибыль.
  • Количество. Тип значения: «Число». Количество товара, за которое не получили прибыль.
  • Причина. Тип значения: «СправочникСсылка.ПричиныУпущенногоСпроса». Причина по которой был упущен спрос.
  • Источник запроса. Тип значения: «СправочникСсылка.ИсточникиЗапроса». Источник запроса на товар.
  • Комментарий. Тип значения: «Строка». Любая дополнительная информация.

Формы

  • Упущенный спрос. Основная форма. Предназначена для создания записи упущенного спроса. 

Способ вызова

Вызывается из меню "Документы --> Отгрузка ТМЦ --> Упущенный спрос.

Вызывается из меню формы списка справочника «Номенклатура».
