Обработка "АРМ Обмен с ЕАИС ТО"

Назначение

Обработка предназначена для выполнения операций обмена документами технического осмотра с ЕАИС ТО.

Относится к подсистеме: «Технический осмотр».

Формы

Интерфейс "Обмен"

Является основной формой АРМ. Форма состоит из следующих элементов:

  • Закладка "Обмен диагностических карт"

На этой закладке осуществляется основная работа по отправке запросов на создание карточки в ЕАИС ТО. Из элементов управления имеются табличная часть с набором уже отправленных и готовых к отправке документов, состав которых регулируется отбором. При активизации строки табличной части в информационном поле отображается список запросов к сервису и полученных на них ответов. Имеется два режима отправки создание новой карточки и изменение уже созданной, возможно только в том случае если карточка была создана обработкой.

 

Раскраска строк в табличной части с документа для отправки в ЕАИС ТО:

  • Зеленый цвет - документ готов к отправке в ЕАИС ТО.
  • Серый цвет    - документ отправлен в ЕАИС ТО.

Элементы управления составом списка документов:

    • Строка поиска - Поиск осуществляется по документам "Технического диагностирования" по следующим полям: наименование клиента, наименование представителя контрагента, модель автомобиля, VIN-код автомобиля, контактная информация клиента и представителя.
    • Флаги управления отбором. Используются для наложения дополнительных отборов на список документов. Дополнительные отборы применяются и для списка, полученного в результате поиска.  
      • Показывать отправленные - будут показаны документы со статусом "Отправлено".
      • Показывать готовые к отправке - будут показаны документы со статусом "Отправить".
    • Кнопка "Отключить отбор". Используется для отключения отбора в списке, полученного в результате поиска или наложения фильтров.
    • Кнопка "Настройка полей поиска". Используется для настройки списка полей, по которым производится поиск. В форме настройки полей поиска обязательно должно быть выбрано хотя бы одно поле.
    • Выбор периода - отбор по дате документа.

  • Закладка "Поиск диагностических карт"

Закладка предназначена для получения информации по карточке в базе ЕАИС ТО по указанным параметрам (хотя бы один из параметров должен быть заполнен):

  • Автомобиль - в поиске не участвует, служит подбором для параметров гос. номер, vin, номер кузова.
  • Вид проверки - первичный или вторичный вид проверки имеет карточка. Обязательный параметр.
  • Гос. номер - гос. номер автомобиля указанный в карточке ТО.
  • Номер карточки - номер карточки в системе ЕАИС ТО.
  • Номер кузова - номер кузова автомобиля указанный в карточке ТО.
  • VIN - vin номер автомобиля указанный в карточке ТО.

результат выводится в поле табличного документа. Так же доступна печать полученнай диагностичекой карты и создания дубликата на ее основании.

  • Закладка "Журнал отправок"

Закладка предназначена для просмотра событий обращения к сервису и получения ответов от него и представлена в виде табличной части с записями и группой отборов, для регулирования состава таб. части.

 

Управляющие кнопки АРМ-а:

    • Создать карты - отправка запроса на создание диагностических карт по документам, отмеченным пометками, веб-сервису.
    • Изменить карты - отправка запроса на изменение диагностических карт по документам, отмеченным пометками, веб-сервису. 
    • Поиск карты - отправка запроса на получение диагностической карты веб сервису
    • Закрыть - закрытие АРМ.
