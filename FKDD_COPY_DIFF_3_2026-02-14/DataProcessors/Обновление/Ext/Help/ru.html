Обработка "Обновление" (Мастер обновления конфигурации)

Назначение

Обработка "Обновление" предназначена для получения актуальных релизов текущего типового решения, а также для выполнения динамического обновления конфигурации, находящейся на поддержке. Для работы с обработкой необходимо зарегистрироваться на сервере поддержки 1С-Раруса Альфа-Авто.

Работа с обработкой происходит в режиме мастера и разбита по шагам.

  • Шаг 1. Ввод регистрационных данных. Необходимо указать ваш логин и пароль, который вы получили при регистрации на сервере поддержки. Дальнейшая работа с мастером будет возможна, только при условии корректности введенных данных.
  • Шаг 2. Выбор релиза и варианта обновления. На данном этапе вам предлагается ознакомится с доступными релизами текущего типового решения. В зависимости от назначения релиза, он может содержать несколько вариантом для обновления. Например, обновление только внешних обработок,  файл обновления конфигурации или полная установка. Вам необходимо выбрать нужный релиз и вариант обновления (обязательно). Для этого выделите соответствующую строку в дереве релизов.
  • Шаг 3. Выбор каталога для загрузки. Предлагается выбрать каталог, в который будет произведена загрузка выбранного на предыдущем этапе варианта обновления. При нажатии на кнопку "Далее" начнется загрузка. В зависимости от размера обновления и скорости вашего интернет-соединения этот процесс может занять значительное время.
  • Шаг 4. Обновление. На данном этапе выбранное вами обновление успешно загружено и распаковано. Вам необходимо выбрать вариант дальнейшего действия. Если вы не хотите производить обновление в данный момент, или хотите произвести обновление самостоятельно, то работа с мастером на данном этапе окончена. Если вы хотите произвести динамическое обновление конфигурации, то необходимо выбрать "Выполнить обновление" (эта опция будет доступна, только в том случае, если вариант загружаемого обновления "Файл обновления конфигурации"). Перед началом обновления рекомендуется сохранить конфигурацию файл (*.cf), для этого установите флаг "Сохранить конфигурацию" и укажите путь для выгрузки. После нажатия на кнопку "Далее" будет произведена попытка обновления.

Относится к подсистеме: «Сервис».

Формы

  • Обновление. Основная форма обработки (мастера)

Способ вызова

Вызывается из меню "Сервис-->Все операции-->Обработки-->Обновление"
