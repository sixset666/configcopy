Обработка "Автоматическое снятие резервов автомобилей"

Назначение

Позволяет формировать документы снятия резервов автомобилей и отмена заказа по заказам на автомобили и резервированию без заказа с истекшим сроком резервирования. В результате выполнения обработки по выбранным пользователем документам: «Заказ на автомобиль» и «Резервирование автомобилей», формируется набор документов «Заказ на автомобиль» с хоз.операций "Отмена заказа на автомобиль" и «Снятие резервов автомобилей».

Относится к подсистеме: «Сервис».

Формы

  • Форма. Основная форма. При открытии заполняется деревом документов «Заказ на автомобиль» и «Резервирование автомобилей», у которых просрочен срок снятия резерва (меньше текущей даты), так же есть возможность отобразить документы с незаполненным сроком снятия резерва.

Документы, у которых не заполнен срок снятия резервов выделяются в списке темно-серым цветом, по умолчанию не выбраны для снятия резервов.
Документы, по которым имеется предоплата от клиента, выделаются в списке темно-красным цветом, по умолчанию не выбраны.
Документы, по которым нет предоплаты и срок снятия резерва просрочен, выделяются в списке черным цветом, по умолчанию выбраны.

Группы обозначены жирным шрифтом.

По нажатию на кнопку "Выполнить" выполняется создание новых документов «Заказ на автомобиль» с хоз.операций "Отмена заказа на автомобиль" и «Снятие резервов автомобилей» для выбранных в списке пунктов. Позиции с одним и тем же складом, группируются в один документ. Если установлен флаг "Создавать напоминание", то для каждого из документов формируется напоминание о списании резерва менеджеру, указанному в документе, а если менеджер не задан - то автору документа. Документы, сформированные в результате работы обработки, будут отображены на форме обработки.

Способ вызова

Вызывается из меню "Обработки --> Автоматическое снятие резервов автомобилей".

 

Дополнительная информация

Для автоматического формирования напоминаний имеется регламентное задание "Напоминание про снятие резервов". На форме регламентного задания должен быть установлен флаг "Заказы на автомобиль". Задание формирует напоминания автору (менеджеру), указанному в заказе, о необходимости снятия просроченных резервов. Напоминания формируются только для заказов, в которых указан срок снятия резерва, в том числе и для предоплаченных заказов.

Для автоматического снятия резервов имеется регламентное задание "Автоматическое снятие резервов". На форме регламентного задания должен быть установлен флаг "Заказы на автомобиль".  Задание формирует документы снятия резервов в фоновом режиме по алгоритму данной обработки. В настройках регламентного задания может быть задано, будут ли создаваться напоминания автору(менеджеру) о списании резерва, необходимость снятия резервов по предоплаченным заказам. Действие настроек распространяется на все документы, формируемые в результате выполнения задания.
