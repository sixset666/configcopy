Форма настройки обработки "СофтФон"

Назначение

Форма предназначена для настройки и отображения параметров подключения к серверу "СофтФона".

Сначала необходимо подключиться к серверу "СофтФона". Для этого нужно указать имя компьютера и порт, на котором установлен сервер "СофтФона". Как правило, это тот компьютер, который подключен непосредственно к телефонной станции.

Для правильной работы "СофтФона" указать настройки для подключения к серверу и префиксы, используемые в телефонной станции. А именно, префикс выхода в город и межгород. Так же указать код страны и код города. Это поможет в дальнейшем автоматически заполнять соответствующие поля при вводе нового номера, и автоматического набора префикса при исходящем звонке.

Настройка "СофтФона" разделяется на два этапа. Настройка основных значений, определенных для всей системы в целом. И настройка локальных значений для отдельно взятого пользователя (настройка значений для пользователя используется по желанию).

Для указания основных значений используется форма "Настройка параметров учета" на закладке "СофтФон". Эти значения будут использоваться для любого оператора системы, при условии, что оператор не настраивал локальные значения специально для себя. Обычно, эти значения устанавливает администратор, и изменять их уже не следует.

Следующие реквизиты на форме не изменяются, а только отображают оператору - какие именно значения используются в данный момент системой. Это либо основные значения всей системы в целом, либо локальные значения, используемые только для данного оператора.

Если оператору необходимо изменить любое из перечисленных значений, сделать он это может на форме справочника "Пользователи" перейдя на закладку "Настройка".

Реквизиты

  • Код страны. Код нашей страны. Например, +7.
  • Код города.  Код нашего города. Например, 495.
  • Префикс выхода на межгород. Префикс, установленный в АТС, для выхода на межгород. Например, 8.
  • Префикс выхода в город. Префикс, установленный в АТС, для выхода в город. Например, 9.
  • Максимальная длина внутреннего номера. Максимальная длина номера телефона, которая будет интерпретироваться "СофтФоном" как внутренний номер телефона.
  • Последние цифры телефонного номера. Последние цифры телефонного номера для ассоциации входящего телефонного номера и номером в базе данных.

ВНИМАНИЕ!!! Если не заполнить выше перечисленные реквизиты, то соответственно "СофтФон" может работать не корректно.

  • Имя сервера. Имя компьютера, на котором установлена телефонная станция и запущен сервер. Поле может быть недоступно, определяется настройкой для текущего пользователя.
  • Порт. Номер порта, выбранный на сервере, к которому будет выполняться подключение. Поле может быть недоступно, определяется настройкой для текущего пользователя.

ВНИМАНИЕ!!! Если не заполнить имя компьютера и порт, то "СофтФон" работать не будет.

Кнопки управления подключением к серверу:

  • Подключиться. Выполняет попытку подключения к серверу "СофтФона", по имени компьютера и порту. После того, как соединение будет выполнено, поля имя компьютера и порт станут недоступными.
  • Отключиться.  Выполняет отключение от сервера "СофтФона". После отключения поля имя компьютера и порт становятся доступными.
  • Подключиться к линии. Выполняет подключение к линии, которую оператор отметил флажком или установил курсор.
  • Отключение от линии. Выполняет отключение от линии.
  • Загрузить список линий. Выполняет загрузку всех линий с сервера.
