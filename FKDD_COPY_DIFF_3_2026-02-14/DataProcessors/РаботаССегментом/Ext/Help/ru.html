Обработки "Работа с сегментом"

Назначение

Обработка предназначена для работы со списком клиентов, относящихся к выбранному сегменту в заданный момент времени (по умолчанию - текущий). Присутствует возможность установки быстрых отборов для выбора нужного сегмента с информацией по нему. С помощью групповых операцией с выделенными строками сегмента можно осуществить такие операции как: перемещение выделенных строк в документ "Рассылка", перемещение в другой сегмент, удаление выбранных строк.

Относится к подсистеме: «Учет взаимоотношений с клиентами»

Формы

  • Форма. Основная форма обработки.

Способ вызова

Вызывается из главного меню "Обработки --> Взаимоотношения с клиентами --> Работа с сегментом".
