Метод экспоненциального сглаживания

 

Метод экспоненциального сглаживания дает возможность оценить степень воздействия трендовой и/или циклической компоненты на отклик системы и может быть использован для краткосрочных прогнозов будущей тенденции на один период вперед.
Коэффициент (К.) может принимать любые значения из диапазона 0 < К. < 1, обычно ограничиваются интервалом от 0.2 до 0.5. При высоких значениях К. в большей степени учитываются мгновенные текущие наблюдения отклика и наоборот, при низких его значениях сглаженная величина определяется в большей степени прошлой тенденцией развития, нежели текущим состоянием отклика системы.
