Обработки "Загрузка номенклатуры из автокаталога"

Назначение

Эта обработка позволяет производить загрузку номенклатуры деталей из "Автокаталога" с использованием файла обмена. Загрузка может производиться как из предопределенного файла обмена, так и выбираться пользователем. При загрузке могут учитываться каталожные номера производителей для исключения дублирования номенклатурных позиций, а так же режим формирования файла обмена автокаталогом, а именно наличие количества в файле обмена и задание разделителя элементов. При загрузки номенклатурных позиций новым элементам можно указать основные реквизиты для записи их в реквизиты номенклатуры.

Относится к подсистеме: «Обмен данными».

Формы

  • Форма. Основная форма. Позволяет произвести загрузку номенклатурного справочника из файла обмена с автокаталогом с заданием основных реквизитов вновь добавляемых элементов номенклатуры. При отображении дерева номенклатуры, считанного из файла обмена зеленым цветом отображаются элементы, уже имеющиеся в справочнике, синим - помеченные на удаление и красным - отсутствующие в справочнике.

Способ вызова

Вызывается по кнопке "Загрузить" из формы списка справочника номенклатуры или через меню "Сервис --> Обработки --> Загрузка номенклатуры из автокаталога".

При вызове обработки из справочника номенклатуры кнопкой "Загрузка из автокаталога "Авто-Дилер"" автоматически в качестве файла обмена подставляется файл, заданный в качестве файла обмена с автокаталогом "Авто-Дилер" по умолчанию в настройках пользователя. После открытия обработки файл обмена автоматически считывается и пользователю достаточно только нажать кнопку "Выполнить" для загрузки номенклатурных позиций в справочник.

При других способах вызова имеется возможность самостоятельно выбрать файл обмена, его структуру и загрузить его путем нажатия кнопки "Считать", расположенной правее имени файла обмена.
