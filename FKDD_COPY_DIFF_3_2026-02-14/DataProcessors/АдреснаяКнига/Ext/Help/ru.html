Обработка "Адресная книга"

Назначение

Обработка "Адресная книга" предназначена для просмотра, редактирования и поиска контактной информации (телефон, адрес, адрес эл.почты и пр.) по объектам информационной базы

«Контрагенты и контакты», «Организации», «Сотрудники», «Подразделения компании».

Относится к подсистеме: «Идентификационные».

 

Форма "Адресная книга".


Основная форма "Адресная книга" разделена на несколько областей:

  • Дерево объектов.
    В дереве представлены следующие разделы: контрагенты, организации, сотрудники, подразделения, поиск. Для "Контрагентов" и "Сотрудников" выводятся подразделы, которые соответствуют группам соответствующих справочников. При выборе раздела панель списка объектов переключается на соответствующую страницу. При выборе подраздела - для справочника также выбирается соответствующая группа.
  • Панель списка объектов.
    Панель имеет несколько страниц, переключение между которыми происходит при выборе соответствующего раздела в "Дереве объектов". При выборе объекта на странице в панеле контактной информации отображается информация о нем.
    • Контрагенты. Элементы управления на странице:
      • Список "Контрагенты". Представляет собой аналог формы списка справочника "Контрагенты и контакты". При выборе объекта в панеле контактной информации отображается связанная с ним информация.
      • Кнопка "Конт. лица". Выполняет переключение на страницу "Контактные лица" и устанавливает на ней фильтр по выбранному контрагенту.
      • Кнопка "Карточка". Открывает карточку контрагента.
    • Организации. Элементы управления на странице:
      • Список "Организации". Представляет собой аналог формы списка справочника "Организации". При выборе объекта в панели контактной информации отображается связанная с ней информация.
    • Сотрудники. Элементы управления на странице:
      • Список "Сотрудники". Представляет собой аналог формы списка справочника "Сотрудники".
      • Кнопка "Конт. лица". Выполняет переключение на страницу "Контактные лица" и устанавливает на ней фильтр по выбранному сотруднику.
    • Подразделения. Элементы управления на странице:
      • Список "Подразделения компании". Представляет собой аналог формы списка справочника "Подразделения компании".
    • Контактные лица. Элементы управления на странице:
      • Список "Контактных лиц". Представляет собой аналог формы списка справочника  "Контрагенты и контакты".
      • Поле "Владелец". Устанавливает фильтр для списка контактных лиц. Владельцем является объект справочников: контрагенты и контакты, сотрудники.
    • Поиск. Элементы управления на странице:
      • Поле "Поисковое выражение". Поле предназначено для ввода поискового выражения. В выпадающем списке сохраняется история последних 10-ти выражений.
      • Кнопка "Найти". Выполняет полнотекстовый поиск объектов по введенному поисковому выражению.
      • Кнопка "Открыть". Открывает на редактирование выбранные элемент в списке "Результаты поиска".
      • Кнопка "Найти в списке". Переключает форму "Адресная книга" на соответствующий справочник и переходит на выбранный объект в этом справочнике.
      • Список "Результаты поиска". В списке выводятся найденные объекты.
  • Панель контактной информации.
    Предназначена для просмотра и редактирования контактной информации.
      • Кнопки "Показывать телефоны, адреса и т.п". Предназначены для наложения фильтра на выводимую контактную информацию.
      • Кнопка "Основная". Устанавливает выбранную контактную информацию в качестве основной для данного типа.
      • Кнопка "Открыть". Открывает выделенный в списке Web-сайт или создает электронное письмо.
      • Список "Контактная информация". В списке выводится контактная информация связанная с выбранным объектом.
  • Панель выбранных электронных адресов.
    Панель появляется когда "Адресная книга" вызывается из документа "Электронное письмо". Предназначена для выбора электронных адресов из контактной информации, которые по кнопке "Выбрать" вставятся в "Электронное письмо".
      • Кнопка "Добавить адрес электронной почты". Добавляет e-mail из выводимуюконтактной информации в выбранные адреса.
      • Кнопка "Добавить все адреса электронной почты". Добавляет все e-mail-ы из контактной информации в выбранные адреса.
      • Кнопка "Удалить адрес электронной почты". Удаляет e-mail из выбранных адресов.
      • Кнопка "Удалить все адреса электронной почты". Удаляет все e-mail-ы из выбранных адресов.

Способ вызова

Вызывается из меню "Справочники  --> Идентификационные --> Адресная книга", из документа "Электронное письмо" и с командной панели.

Дополнительная информация

Контактная информация хранится по типам, которые жестко задаются в конфигурации и не могут быть изменены пользователем. Пользователь может ввести произвольное количество видов контактной информации, принадлежащих одному из типов. Для этого служит справочник «Виды контактной информации».
