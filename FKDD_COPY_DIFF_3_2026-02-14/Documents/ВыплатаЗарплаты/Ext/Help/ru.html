Документ "Выплата зарплаты"

Назначение

Документ "Выплата зарплаты" предназначен для учета выплаты наличных денежных средств из операционной кассы компании. С помощью данного документа может быть зафиксирован расход наличных денежных средств по различным операциям:

  • Выдача денежных средств сотруднику.
  • Прочий расход денежных средств.

В конфигурации предусмотрен учет денежных средств в нескольких «Кассах компании». Поэтому при оформлении документа необходимо выбрать кассу, из которой будет производиться выплата наличных денежных средств. В кассе денежные средства могут храниться в одной валюте. Поэтому валюта, указанная в кассе определяет валюту оформляемого документа.

При проведении документ формирует хозяйственную операцию:

  • Выплата заработной платы. Хозяйственная операция погашает кредиторские задолженности по всем сделкам, если таковые были. Если после погашения остается нераспределенная сумма, то операция начисляет дебиторскую задолженность сотрудника на регистре «Взаиморасчеты компании», списывает денежные средства компании в выбранной операционной кассе на регистре «Денежные средства компании», начисляет или списывает курсовую разницу, если таковая была выявлена, на регистре «Доходы и расходы».

Относится к подсистемам: «Движения денежных средств», «Взаиморасчеты».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «Начисление зарплаты».

Объекты, вводимые на основании: «Корректировка».

Реквизиты

  • Касса компании. Реквизит является обязательным для заполнения. Ссылается на справочник «Кассы компании». Касса компании, из которой выдаются денежные средства.
  • Статья ДДС. Реквизит является обязательным для заполнения. Ссылается на справочник «Статьи ДДС». Статья движения денежных средств.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете. 
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Документ основание. Тип значения: «Строка». Основание для выдачи денежных средств (например, документ, подтверждающий личность).  
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа, на дату документа.
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.  
  • Сумма. Тип значения: «Число». Сумма документа.

Табличная часть «Сотрудники»

  • Сотрудник. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Сотрудник, которому выдаются денежные средства.
  • Договор взаиморасчетов. Реквизит является обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор, по которому ведутся взаиморасчеты с сотрудником.
  • Документ планирования. Ссылается на документ «Заявка на Расход ДС», «План поступления ДС». Документ планирования поступления или расхода ДС.
  • Задолженность перед сотрудником. Тип значения: «Число». Задолженность перед сотрудником.
  • Сумма. Реквизит является обязательным для заполнения. Тип значения: «Число». Сумма для выдачи.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Выплата зарплаты. Простая печатная форма, содержит список сотрудников и информацию о выплате.
  • Платежная ведомость. Унифицированная печатная форма №Т-53, утвержденная постановлением Госкомстата России.

Способ вызова

Вызывается из меню "Документы --> Расчеты по зарплате --> Выплата зарплаты".

Журналы, отображаемые документ: «Расчеты по зарплате».

Движения

  • Остаточный регистр «Взаиморасчеты компании».
  • Остаточный регистр «Денежные средства компании».  
  • Остаточный регистр «Доходы и расходы».
  • Оборотный регистр «Платежный календарь».

Последовательности, изменяемые документом: "Взаиморасчеты".
