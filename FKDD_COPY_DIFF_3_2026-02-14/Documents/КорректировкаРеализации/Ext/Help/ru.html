Документ "Корректировка реализации"

Назначение

Документ предназначен для регистрации исправленных первичных документов, переданных покупателю. Например, при выявлении продавцом ошибки при оформлении первичных документов или в случае согласованного между продавцом и покупателем изменения стоимости ранее реализованных товаров, работ и услуг. Имеет две хоз. операции "Исправление в первичных документах" и "Корректировка по согласованию сторон", используются при перегрузке в бухгалтерию.

 

Относится к подсистеме: «Отгрузка ТМЦ».

Параметры

Длина номера: 10

Периодичность номера: Год

Тип номера: Строка

Вводится на основании: Корректировка реализации, «Заказ-наряд», «Реализация товаров».

Объекты, вводимые на основании: Корректировка реализации, «Счет фактура выданный».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Менеджер. Ссылается на справочник «Сотрудники». Сотрудник, заключивший сделку с контрагентом.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  • Комментарий. Произвольный комментарий к документу.
  • Документ основание. Ссылается на документ Корректировка реализации, «Заказ-наряд», «Реализация товаров». Документ, на основании которого введен данный , необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Курс валюты документа.
  • Курс валюты упр. Реквизит является обязательным для заполнения. Курс валюты управленческого учета.
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип Цен, в которых отгружаются детали для выполнения работ.
  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты». Плательщик по документу реализации.
  • Договор. Реквизит является обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор по которому ведутся взаиморасчеты с контрагентом.
  • Автоматическое закрытие сделок. Если флажок установлен, то при проведении документа сначала будут оплачены все незакрытые сделки.
  • Сумма. Сумма документа.

Табличная часть «Товары»

  • Подтверждение. Флаг подтверждения оплаты позиции плательщиком.
  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Реализованный товар.
  • Количество. Количество деталей, подтвержденных плательщиком к оплате.
  • Количество по документу реализации. Количество деталей согласно документа реализации.
  • Количество разница. Разница количества деталей, подтвержденных плательщиком к оплате и количеством согласно документа реализации.
  • Единица измерения. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения деталей, подтвержденных плательщиком к оплате.
  • Единица измерения по документу реализации. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения деталей, согласно документа реализации.
  • Коэффициент. Реквизит является обязательным для заполнения. Коэффициент пересчета по отношению к базовой единице номенклатуры, подтвержденных плательщиком к оплате.
  • Коэффициент по документу реализации. Реквизит является обязательным для заполнения. Коэффициент пересчета по отношению к базовой единице номенклатуры согласно документа реализации.
  • Цена. Цена детали в валюте документа, подтвержденная плательщиком к оплате.
  • Сумма. Сумма деталей, подтвержденная плательщиком к оплате. Сумма всегда без скидки. НДС входит в сумму если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Ставка НДС. Ставка НДС, подтвержденная плательщиком к оплате. Ссылается на справочник «Ставки НДС».
  • Ставка НДС по документу реализации. Ставка НДС, согласно документа реализации. Ссылается на справочник «Ставки НДС».
  • Сумма НДС. Сумма НДС по номенклатурной позиции, подтвержденная плательщиком к оплате.
  • Сумма НДС по документу реализации. Сумма НДС по номенклатурной позиции, согласно документа реализации.
  • Разница сумм НДС. Разница сумм НДС, подтвержденной плательщиком к оплате и суммы НДС, согласно документа реализации.
  • Процент скидки. Процент скидки на деталь, подтвержденная плательщиком к оплате.
  • Сумма скидки. Сумма скидки на деталь, подтвержденная плательщиком к оплате.
  • Сумма скидки по документу реализации. Сумма скидки на деталь, согласно документа реализации.
  • Разница сумм скидки. Разница сумм скидки на деталь, подтвержденной плательщиком к оплате и суммы скидки, согласно документа реализации. 
  • Характеристика номенклатуры. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры.
  • Всего. Итоговая сумма по строке документа со всеми налогами, подтвержденная плательщиком к оплате.
  • Всего по документу реализации. Итоговая сумма по строке документа со всеми налогами, согласно документа реализации.
  • Разница всего. Разница итоговых сумм по строке документа со всеми налогами, подтвержденная плательщиком к оплате и суммы, согласно документа реализации. 

Табличная часть «Работы»

  • Сумма НДС по документу реализации. Сумма НДС по работе, согласно документа реализации.

  • Процент скидки. Процент скидки на работу, подтвержденный плательщиком к оплате.
  • Сумма скидки. Сумма скидки на работу, подтвержденная плательщиком к оплате.
  • Сумма скидки по документу реализации. Сумма скидки на работу, согласно документа реализации.

  • Всего по документу реализации. Итоговая сумма по строке документа со всеми налогами, согласно документа реализации.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.

Раскраска строк и ячеек табличной части товары

  • Строки:
  • Текст, позиции которых подтверждены плательщиком к оплате(флаг "Подтверждение" установлен) выделяется обычным цветом, без видоизменения.
  • Текст, позиции которых не подтверждены плательщиком к оплате(флаг "Подтверждение" снят) выделяется красным цветом с видоизменением - зачеркнутый.
  • Текст, позиции которых были добавлены в документ не на основании документа реализации(добавлены в ручную в табличную часть)  имеют начертание "Полужирный". Подтверждения плательщика к оплате устанавливается автоматически, возможности изменить состояние флага не имеется.
  • Ячейки:
  • Текст в ячейках также выделяется цветом. Для положительных значений - зелёный, отрицательных значений - красный. Значение соответствующих ячеек устанавливается как разница между значением подтверждённым плательщиком к оплате и значением из документа реализации.
