Документ "Заказ поставщику на автомобиль"

Назначение

Документ служит для отражения в учете заказов поставщикам на автомобили.

При проведении документ формирует хозяйственные операции:

  • Заказ поставщику на автомобиль. Хозяйственная операция приходует заказы поставщику на автомобили по списку заказанных автомобилей на регистр «Заказы поставщикам на автомобили».
  • Заказ поставщику на автомобиль под тест-драйв. Хозяйственная операция приходует заказы поставщику на автомобили по списку заказанных автомобилей для тест-драйва на регистр «Заказы поставщикам на автомобили». 
  • Изменение автомобиля.  Хозяйственная операция позволяет оформить заказ поставщику на автомобиль для изменения заказываемого автомобиля.
  • Отмена заказа поставщику на автомобиль.  Хозяйственная операция выполняется отмену заказа поставщику на автомобиль, на основании которого введен на регистре «Заказы автомобилей». Если автомобиль был зарезервированы под меняемый заказ на автомобиль, то происходит автоматическое снятие резерва.

Относится к подсистеме: «Автосалон».

Параметры

Длина номера: 10

Периодичность номера: Год

Тип номера: Строка

Объекты, вводимые на основании: «Поступление автомобилей», «Событие», «Расходный кассовый ордер», «Заказ поставщику на автомобиль», «Счет от поставщика за автомобили», «Взаимозачет», «Платежное поручение».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Комментарий. Произвольный комментарий к документу.
  • Документ основание. Ссылается на документы «Заказ на автомобиль» и "Заказ поставщику на автомобиль". Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).
  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Поставщик автомобиля.
  • Договор. Реквизит является обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор по которому ведутся взаиморасчеты с поставщиком.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Курс валюты документа, на дату документа. 
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип Цен, по которому происходит заказ автомобиля.
  • Сумма. Сумма документа.
  • Хоз операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Срок поставки. Реквизит является обязательным для заполнения. Срок отгрузки автомобиля по заказу контрагенту.
  • Статус автомобиля. Ссылается на справочник «Статусы автомобилей». Текущее состояние выполнение заказа.
  • Модель автомобиля. Реквизит является обязательным для заполнения. Ссылается на справочник «Модели автомобилей». Модель заказываемого автомобиля.
  • Вариант комплектации автомобиля. Ссылается на справочник «Варианты комплектации». Вариант комплектации модели автомобиля.
  • Цвет автомобиля. Ссылается на справочник «Цвета». Цвет заказываемого автомобиля.
  • Код лакокрасочного покрытия. Код лакокрасочного покрытия согласно спецификации производителя.
  • Тип салона. Ссылается на справочник «Типы салона». Тип салона заказываемого автомобиля.  
  • Автомобиль. Реквизит является обязательным для заполнения. Ссылается на справочник «Автомобили». Автомобиль, на который оформляется заказ. Если автомобиль не выбран, то будет создан новый элемент справочника автомобилей с параметрами, указанными в заказе.
  • Внешний номер заказа. Внешний номер заказа поставщику.
  • Цена. Цена закупки автомобиля.
  • Ставка НДС на автомобиль. Ссылается на справочник «Ставки НДС». Ставка НДС на автомобиль.
  • Сумма НДС на автомобиль. Сумма НДС на автомобиль.
  • Сумма всего на автомобиль. Сумма автомобиля с налогами.

Табличная часть «Опции»

  • Опция. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Опции автомобилей». Опции, которые будут установлены на автомобиль поставщиком.
  • Количество. Реквизит является обязательным для заполнения. Количество устанавливаемых опций.
  • Цена. Цена единицы опции.
  • Сумма. Сумма всегда без скидки. НДС входит в сумму если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Ставка НДС. Ссылается на справочник «Ставки НДС». Ставка НДС на опцию.
  • Сумма НДС. Сумма НДС на опцию.
  • Сумма всего. Итоговая сумма по строке документа с налогами.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Заказ поставщику на автомобиль. Простая печатная форма. Выводит информацию о заказаном автомобиле и его оборудовании.

Способ вызова

Вызывается из меню "Документы --> Автосалон --> Заказы на автомобили --> Заказ поставщику на автомобиль".

Движения

  • Остаточный регистр «Заказы поставщикам на автомобили».
