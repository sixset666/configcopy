Документ "Инвентаризация ордерного склада"

Назначение

Документ предназначен для отражения в учете факта проведения инвентаризации товаров по выбранному ордерному складу. В табличной части документа не могут фигурировать услуги, работы и т.п.  При проведении если в строке фактическое количество больше учетного, то документ приходует излишек (разницу между фактическим и учетным количеством) на склад по регистру «Остатки товаров на ордерном складе», в противном случае, если фактическое количество меньше учетного, то документ списывает недостачу со склада по регистру «Остатки товаров на ордерном складе». Новый (не проведенный) документ можно заполнить автоматически учетными остатками по выбранному ордерному складу (меню по кнопке  "Заполнение"). При вводе новых строк, автоматически подставляются учетные количество и стоимость.

При проведении документ формирует хозяйственные операции:

  • Инвентаризация ордерного склада. Хозяйственная операция в зависимости от реальных остатков приходует или списывает материальные активы, хранимые на ордерных или ордерно-ячеистых складах компании, на регистр «Остатки товаров на ордерном складе».

Относится к подсистеме «Ордерный склад».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «Инвентаризация».

Объекты, вводимые на основании: «Инвентаризация», «Корректировка», «Событие».

Реквизиты

  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад компании, на котором проводим инвентаризацию.
  • Документ основание. Ссылается на документ «Инвентаризация». Документ, на основании которого введен данный.
  • Основание. Тип значения: «Строка». Основание проведения инвентаризации.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Юридическое лицо, от которого выписан документ.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, создавший документ.
  • Операция. Ссылается на справочник «Хоз. операции». Хозяйственная операция.

Табличная часть «Товары»

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура, подлежащая инвентаризации.
  • Количество расчетное. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество номенклатуры, которое по документам находится на складе.
  • Количество фактическое. Тип значения: «Число». Фактическое количество номенклатуры, которое фактически имеется на складе.
  • Разница. Тип значения: «Число». Разница между расчетным количеством и фактическим количеством.
  • Единица. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Ячейка. Ссылается на справочник «Ячейки хранения». Место хранения ячеистого склада.
  • Характеристика. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Инвентаризационная опись (ИНВ-3) - - Унифицированная печатная форма ИНВ-3, утвержденная постановлением Госкомстата России.
  • Сличительная ведомость (ИНВ-19) - Унифицированная печатная форма ИНВ-19, утвержденная постановлением Госкомстата России.
  • Пустографка - Простая печатная форма, содержит список инвентаризируемых товаров без их фактического количества. 

Способ вызова

Вызывается из меню "Документы --> Складские документы --> Инвентаризация ордерного склада"

Журналы, отображающие документ: «Складские ордера».

Движения

  • Остаточный регистр «Остатки товаров на ордерном складе».
