Документ "Объявление о продаже автомобиля"

Назначение

Используется для формирования в системе объявлений о продаже автомобилей для выгрузки в сервисы объявлений («Автоклауд»).

 

Может быть введен на основании документов «Ввод остатков автомобилей» и «Поступление автомобилей» (выбирается первый в табличной части основания автомобиль). Также может быть введен на основании элемента справочника «Автомобили».

 

Стоимость автомобиля для объявления по-умолчанию заполняется из установленной цены в регистре цен автомобилей. Стоимость может изменяться вручную без каких-либо ограничения прав доступа.

 

Для хранения картинок объявления в информационной базе, а также для их отображения на форме документа используется типовой механизм картинок и файлов. Картинки объявления хранятся в регистре картинок и файлов, а не в документе.

 

Список хозяйственных операций:

  • Объявление о продаже автомобиля. 

Относится к подсистеме: «?».

Параметры

Длина номера: 10

Периодичность номера: Год

Тип номера: Строка

Вводится на основании: «Автомобиль», «Ввод остатков автомобилей», «Поступление автомобилей».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, создавший объявление.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании в котором был выписан документ.
  • Комментарий. Произвольный комментарий к документу.
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Сервис. Реквизит является обязательным для заполнения. Ссылается на перечисление «Сервисы объявлений о продаже автомобилей». Сервис объявлений о продаже автомобиля, в который будет выгружено объявление.
  • Автомобиль. Реквизит является обязательным для заполнения. Ссылается на справочник «Автомобили». Продаваемый по объявлению автомобиль.
  • Состояние. Реквизит является обязательным для заполнения. Ссылается на перечисление «Состояния объявлений о продаже автомобилей». Текущее состояние объявления.
  • Стоимость. Реквизит является обязательным для заполнения. Стоимость автомобиля по объявлению. По-умолчанию заполняется установленной в программе ценой автомобиля. Может редактироваться и отличаться от установленной цены автомобиля.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.
  • Форма документа. Основная форма документа. Используется для ввода и редактирования документов данного типа.

Печатные формы

  • Объявление о продаже автомобиля.

Способ вызова

Вызывается из меню "Документы --> Автосалон--> Объявление о продаже автомобиля".

Дополнительная информация

Объявление может находится в различных состояниях:

1. «В работе». Статус устанавливается по-умолчанию при создании нового документа. Означает, что с документом (объявлением) ведется работа и выгружать в сервис его нельзя. Документы с данным статусом не попадают в обработку выгрузки объявлений.

2. «Ожидает отправки». Статус устанавливается вручную пользователем. Означает, то с документом (объявлением) завершена подготовительная работа и он готов для выгрузки в сервис. Документы с данным статусом попадают в обработку выгрузки объявлений.

3. «Отправлено». Статус устанавливается автоматически после успешной выгрузки документа (объявления) в сервис.
