Документ «Рассылка»

Назначение

Документ предназначен для осуществления рассылки указанным клиентам с помощью электронных писем, SMS, телемаркетинга.

Параметры

Длина номера: 10.

Периодичность номера: Непериодический.

Тип номера: Строка.

Вводится на основании: «Рассылка», «Событие».

Объекты, вводимые на основании: «Жалоба клиента», «Опрос», «Рассылка».

Реквизиты

  • Анкета. Ссылается на справочник "Типовые анкеты". Анкета получателей.
  • Автор. Ссылка на справочник "Пользователи". Автор документа - пользователь, породивший документ.
  • Документ основание. Документ, на основании которого был введен рабочий лист отдела страхования.
  • Комментарий. Тип значения: Строка. Произвольный комментарий к документу.
  • Менеджер. Ссылается на справочник "Сотрудники". Менеджер.
  • Номер отправителя SMS. Тип значения: Строка. Номер отправителя SMS.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник "Организации". Идентификатор юридического лица, от которого выписан документ.
  • Отправитель представление. Тип значения: Строка. представление отправителя.
  • Подразделение компании. Реквизит является обязательным для заполнения. Ссылается на справочник "Подразделения компании". Подразделение компании, в котором был выписан документ.
  • Проект. Ссылается на справочник "Проекты". Проект документа.
  • Реклама. Реквизит является обязательным для заполнения. Ссылается на справочник "Источники информации". Источник информации (реклама).
  • Состояние. Ссылается на перечисление "Состояния рассылок". Состояние рассылок.
  • Текст письма. Тип значения: Строка. Текст письма.
  • Тема. Реквизит является обязательным для заполнения. Тип значения: Строка. Тема письма.
  • Учетная запись. Ссылается на справочник "Учетные записи электронной почты". Учетная запись электронной почты.
  • Уведомить о прочтении. Тип значения: Булево.  Признак уведомить о прочтении сообщения.
  • Формат текста. Ссылка на перечисление "Формат текста". Формат текста.
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник "Хоз. операции". Выполняемая документом операция.

Табличная часть «Получатели»

  • SMS. Ссылается на документ "SMS". Ссылка на SMS.
  • Адрес. Тип значения: "Строка". Адрес для отправки. 
  • Адрес электронной почты. Тип значения: "Строка". Адрес электронной почты. 
  • Код группы адреса. Ссылается на перечисление "Коды группы адреса электронного письма". Код группы адреса.
  • Опрос. Ссылается на документ "Опрос". Ссылка на опрос. 
  • Отказ. Тип значения: "Булево". Прзнак отказа.
  • Получатель. Ссылается на справочник "Контрагенты". Получатель сообщения рассылки. 
  • Представление. Тип значения: "Строка". Представление получателя сообщения. 
  • Результат обработки звонка. Ссылается на перечисление "Результаты обработки звонка". Результат обработки звонка. 
  • Событие. Ссылается на документ "Событие". Ссылка на событие.
  • Телефон. Тип значения: "Строка". Телефон для осуществления рассылки. 
  • Электронное письмо. Ссылка на документ "Электронное письмо". Ссылка на электронное письмо. 
  • Жалоба клиента. Ссылается на документ "Жалоба клиента". Ссылка на жалобу клиента.  

Табличная часть «Прикрепленные файлы»

  • Имя. Тип значения: "Строка". Имя файла. 
  • Файл. Тип значения: "Хранилище значений". Сохраненный файл.
  • Размер. Тип значения: "Число". Размер файла в байтах.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Документы --> Взаимоотношения с клиентами --> Рассылка".
