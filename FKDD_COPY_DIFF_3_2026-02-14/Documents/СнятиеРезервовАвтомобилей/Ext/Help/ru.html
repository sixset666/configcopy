Документ "Снятие резервов автомобилей"

Назначение

Документ предназначен для оформления операций по снятию (отмене) резервирования автомобилей под заказы покупателей. Основным режимом ввода, является ввод на основании документа «Заказ на автомобиль», однако, при необходимости, документ можно заполнить вручную.

При проведении документ формирует хозяйственные операции:

  • Снятие резервов автомобилей. Хозяйственная операция списывает зарезервированный товар на регистре «Остатки автомобилей».

Относится к подсистеме: «Автосалон».

Параметры

Длина номера: 10

Периодичность номера: Год

Тип номера: Строка

Вводится на основании: «Заказ на автомобиль».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Комментарий. Произвольный комментарий к документу.
  • Документ основание. Ссылается на документ «Заказ на автомобиль». Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).
  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад компании на котором отменяется резервирование.
  • Хозяйственная операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.

Табличная часть «Автомобили»

  • Автомобиль. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Автомобили». Автомобиль, который мы снимаем с резерва.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Снятие резервов автомобилей. Простая печатная форма, содержит список автомобилей, снимаемых с резерва.

Способ вызова

Вызывается из меню "Документы --> Автосалон --> Заказы на автомобили --> Снятие резервов автомобилей".

Движения

  • Остаточный регистр «Остатки автомобилей».
