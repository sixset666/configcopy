Документ "Изменение цен"

Назначение

Документ предназначен для документального фиксирования изменения отпускных цен компании или контрагента. При проведении документа информация о ценах записывается в регистр сведений  «Цены» по каждой позиции номенклатуры. Также, если для организации установлено право "Переоценивать розничные остатки при оприходовании товаров", данным документом может производиться переоценка товаров на розничном складе (регистр «Остатки товаров компании»).

При проведении документ формирует хозяйственные операции:

  • Изменение цен компании. Хозяйственная операция сохраняет цены компании, указанные для каждого товара на регистре сведений «Цены».
  • Изменение цен контрагента. Хозяйственная операция аналогична "Изменению цен компании" за тем исключением, что сохраняются цены поставщика. Используется в тех случаях, когда мы хотим сохранить цены поставщиков или конкурентов и в дальнейшем их анализировать.

Относится к подсистеме: «Ценообразование».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «Поступление товаров», «Ввод остатков товаров», «Ввод остатков товаров в производстве».

Объекты, вводимые на основании: «Переоценка», «Корректировка».

Реквизиты

  • Контрагент. Ссылается на справочник «Контрагенты и контакты». Поставщик товара, для которого регистрируются цены.
  • Действует с. Тип значения: «Дата». Дата начала действия цены.
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип Цен.
  • Расчет от. Служебный реквизит. Тип значения: «Число». Признак того, относительно чего производится расчет цен. 
  • Базовый тип цен. Ссылается на справочник «Типы цен». Тип цен, относительно которого производим расчет.
  • Процент наценки. Тип значения: «Число». Процент наценки на базовую цену.
  • Процент наценки из номенклатуры. Тип значения: «Булево». При установке данного реквизита в значение ИСТИНА, проценты наценки от базовой цены будут подставляться из карточки номенклатуры.
  • Округлять до. Тип значения: «Число». Метод округления расчетной цены.
  • Комментарий. Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.   
  • Документ основание. Ссылается на документ «Поступление товаров». Документ, на основании которого введен данный.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа, на дату документа.

Табличная часть «Товары»

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар, услуга и т.п.).
  • Единица измерения. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Старая цена. Реквизит является обязательным для заполнения. Тип значения: «Число». Старая цена номенклатурной позиции.
  • Цена базовая. Тип значения: «Число». Цена, относительно которой производится расчет новой цены.
  • Процент наценки. Тип значения: «Число». Наценка на базовую цену для вычисления устанавливаемой цены.
  • Цена новая. Тип значения: «Число». Новая цена номенклатурной позиции.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Печать этикеток и ценников. Обработка, позволяющая получить печатную форму этикеток и ценников товаров, у которых изменяют цены.
  • Изменение цен. Простая печатная форма, содержит список товаров с измененными ценами.

Способ вызова

Вызывается из меню "Документы --> Ценообразование --> Изменение цен".

Движения

  • Регистр сведений «Цены».
  • Остаточный регистр «Остатки товаров компании»
