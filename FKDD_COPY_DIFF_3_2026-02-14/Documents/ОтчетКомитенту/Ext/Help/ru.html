Документ "Отчет комитенту"

Назначение

Документ "Отчет комитенту" фиксирует факт возникновения задолженности перед комитентом за товары, принятые от него на комиссию и реализованные. До оформления данного документа считается, что компания ничего не должна комитенту, независимо от того продан товар данного комитента или нет. Размер комиссионного вознаграждения рассчитывается как процент от суммы продажи. Процент вознаграждения задается в «Договоре» с комитентом. Под суммой продажи понимается та сумма, на которую был продан комиссионный товар (Сумма реализации).

При проведении документ формирует хозяйственную операцию:

  • Отчет комитенту. Хозяйственная операция списывает дебиторскую задолженность покупателя по другим сделкам, если таковые были и начисляет кредиторскую задолженность компании перед комитентом, за вычетом суммы комиссионного вознаграждения, по регистру «Взаиморасчетов компании», списывает товары, которые были проданы, но за которые наша компания еще не отчиталась перед комитентом c регистра «Реализованные товары», приходует сумму комиссионного вознаграждения на регистр «Доходы и расходы» компании.
  • Отчет агента. Хозяйственная операция предназначен для подготовки отчета комитенту об агентских услугах, реализованных в рамках агентского договора и выставления комитенту агентского вознаграждения.

Относится к подсистеме: «Поступление ТМЦ».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «Поступление товаров», «Ввод остатков товаров».

Объекты, вводимые на основании: «Платежное поручение», «Счет фактура выданный», «Счет фактура полученный», «Корректировка», «Расходный кассовый ордер», «Событие».

Реквизиты

  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Комитент, перед которым мы отчитываемся за проданный товар.
  • Договор взаиморасчетов. Реквизит является обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор, по которому ведутся взаиморасчеты с контрагентом.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.  
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.  
  • Документ основание. Ссылается на документы «Поступление товаров», «Ввод остатков товаров». Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ). 
  • Тип цен. Ссылается на справочник «Типы цен». Тип цен, в которых наша компания отчитывается перед комитентом.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа, на дату документа.
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Сумма. Тип значения: «Число». Сумма документа.
  • Проект. Ссылается на справочник «Проекты». Проект, к которому относится документ.

Табличная часть «Товары»

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар).
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество единиц измерения номенклатуры.
  • Единица. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Цена. Тип значения: «Число». Цена номенклатурной позиции.
  • Сумма поступления. Тип значения: «Число». Сумма всегда без скидки. НДС входит в сумму, если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Сумма продажи. Тип значения: «Число». Сумма продажи комиссионных товаров и агентских услуг.
  • Всего. Тип значения: «Число». Итоговая сумма по строке документа со всеми налогами.
  • Вознаграждение. Тип значения: «Число». Сумма комиссионного вознаграждения.
  • Документ поступления. Реквизит является уникальным и обязательным для заполнения. Ссылается на документы  «Поступление товаров», «Ввод остатков товаров».  Документ поступления комиссионных товаров.
  • Документ реализации. Реквизит является уникальным и обязательным для заполнения. Ссылается на документ  «Поступление товаров»,.  Документ реализации агентских услуг.
  • Процент НДС. Ссылается на справочник «Ставки НДС». Ставка НДС по номенклатурной позиции.
  • НДС. Тип значения: «Число». Сумма НДС по номенклатурной позиции.
  • Характеристика номенклатуры. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Отчет комитенту. Простая печатная форма, содержит список товаров, которые были проданы.
  • Акт об указании услуг. Простая печатная форма, содержит услуги, которые были оказаны комитенту.
  • Печать этикеток и ценников. Обработка, позволяющая получить печатную форму этикеток и ценников приходуемых товаров.

Способ вызова

Вызывается из меню "Документы --> Поступление ТМЦ --> Отчет комитенту".

Движения

  • Регистр сведений «Допроведение по партиям».
  • Остаточный регистр «Взаиморасчеты компании».
  • Остаточный регистр «Доходы и расходы».
  • Остаточный регистр «Реализованные товары».

Последовательности, изменяемые документом: "Партии", "Взаиморасчеты".
