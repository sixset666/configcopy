Документ "Корректировка поступления"

Назначение

Документ предназначен для регистрации исправленных первичных документов, полученных от поставщика. Например, при выявлении ошибки при оформлении первичных документов или в случае согласованного между продавцом и покупателем изменения стоимости ранее реализованных товаров и услуг. Имеет две хоз. операции "Исправление в первичных документах" и "Корректировка по согласованию сторон", используются при перегрузке в бухгалтерию.

 

Относится к подсистеме: «Поступление ТМЦ».

Параметры

Длина номера: 10

Периодичность номера: Год

Тип номера: Строка

Вводится на основании: Корректировка поступления, «Поступление товаров».

Объекты, вводимые на основании: Корректировка поступления, «Счет-фактура полученный».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Менеджер. Ссылается на справочник «Сотрудники». Сотрудник, заключивший сделку с контрагентом.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Основание. Реквизит является обязательным для заполнения. Документ на основании которого введена корректировка.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  • Комментарий. Произвольный комментарий к документу.
  • Документ основание. Ссылается на документ Корректировка поступления, «Поступление товаров». Документ, на основании которого введен данный , необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Курс валюты документа.
  • Курс валюты упр. Реквизит является обязательным для заполнения. Курс валюты управленческого учета.
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип Цен, в которых отгружаются детали для выполнения работ.
  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты». Поставщик по документу поступления.
  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад по документу поступления.
  • Договор. Реквизит является обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор по которому ведутся взаиморасчеты с контрагентом.
  • Проект.  Ссылается на справочник «Проекты». Проект к которму относится документ.
  • Автоматическое закрытие сделок. Если флажок установлен, то при проведении документа сначала будут оплачены все незакрытые сделки.
  • Сумма. Сумма документа.

Табличная часть «Товары»

  • Подтверждение. Флаг подтверждения получения товара от поставщика.
  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Поступивший товар.  
  • Количество. Количество деталей, подтвержденных плательщиком к оплате.
  • Количество по документу поступления. Количество деталей согласно документа поступления.
  • Количество разница. Разница количества деталей, подтвержденных плательщиком к оплате и количеством согласно документа поступления.
  • Единица измерения. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения деталей, подтвержденных плательщиком к оплате.
  • Единица измерения по документу поступления. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения деталей, согласно документа поступления.
  • Коэффициент. Реквизит является обязательным для заполнения. Коэффициент пересчета по отношению к базовой единице номенклатуры, подтвержденных плательщиком к оплате.
  • Коэффициент по документу поступления. Реквизит является обязательным для заполнения. Коэффициент пересчета по отношению к базовой единице номенклатуры согласно документа поступления.
  • Цена. Цена детали в валюте документа, подтвержденная плательщиком к оплате.
  • Цена розничная. Розничная цена детали в валюте регл. учета. 
  • Сумма. Сумма деталей, подтвержденная плательщиком к оплате. Сумма всегда без скидки. НДС входит в сумму если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Сумма розничная. Розничная сумма детали в валюте регл. учета. НДС входит в сумму если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Сумма розничная по документу поступления. Розничная сумма детали в валюте регл. учета по документу поступления.
  • Сумма розничная разница. Разница розничных сумм, подтвержденной плательщиком к оплате и розничной суммы, согласно документа поступления.
  • % НДС. Ставка НДС, подтвержденная плательщиком к оплате. Ссылается на справочник «Ставки НДС».
  • % НДС по документу поступления. Ставка НДС, согласно документа поступления. Ссылается на справочник «Ставки НДС».
  • НДС. Сумма НДС по номенклатурной позиции, подтвержденная плательщиком к оплате.
  • НДС по документу поступления. Сумма НДС по номенклатурной позиции, согласно документа поступления.
  • Разница сумм НДС. Разница сумм НДС, подтвержденной плательщиком к оплате и суммы НДС, согласно документа поступления.
  • Характеристика номенклатуры. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры.
  • Всего. Итоговая сумма по строке документа со всеми налогами, подтвержденная плательщиком к оплате.
  • Всего по документу поступления. Итоговая сумма по строке документа со всеми налогами, согласно документа поступления.
  • Разница всего. Разница итоговых сумм по строке документа со всеми налогами, подтвержденная плательщиком к оплате и суммы, согласно документа поступления.
  • ... до корректировки. Значение реквизита "... по документу поступления" до введения первого документа с хоз. операцией "Корректировка по согласованию сторон".  
  • ГТД / РНПТ. Ссылается на справочник «ГТД». Номер грузовой таможенной декларации (для товаров, поступивших по импорту) или регистрационный номер (РНПТ) для прослеживаемых товаров.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.
  • Форма документа. Основная форма документа. Используется для ввода и редактирования документов данного типа.

Способ вызова

Вызывается из меню "Сервис --> Все операции --> Документы --> Корректировка поступления".

Раскраска строк и ячеек табличной части товары

  • Строки:
    • Текст, позиции получение которых подтверждено покупателем (флаг "Подтверждение" установлен) выделяется обычным цветом, без видоизменения.
    • Текст, позиции получение которых не подтверждено покупателем (флаг "Подтверждение" снят) выделяется красным цветом с видоизменением - зачеркнутый.
    • Текст, позиции которых были добавлены в документ не на основании документа поступления(добавлены в ручную в табличную часть)  имеют начертание "Полужирный". Подтверждения покупателя о получении устанавливается автоматически, возможности изменить состояние флага не имеется.
  • Ячейки:
    • Текст в ячейках также выделяется цветом. Для положительных значений - зелёный, отрицательных значений - красный. Значение соответствующих ячеек устанавливается как разница между значением подтверждённым плательщиком к оплате и значением из документа поступления.
