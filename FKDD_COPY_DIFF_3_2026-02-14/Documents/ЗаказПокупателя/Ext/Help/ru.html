Документ "Заказ покупателя"

Назначение

Документ служит для отражения в учете заказов покупателей. В табличной части документа не могут фигурировать услуги, работы и т.п. При вводе покупателя из его реквизитов автоматически подставляются в документ договор, текущая скидка. Эти значения потом можно поменять.

При проведении документ формирует хозяйственные операции:

  • Заказ покупателя. Хозяйственная операция приходует заказы покупателя по списку заказанных товаров на регистр «Заказы покупателей».
  • Заказ и резервирование покупателя. Хозяйственная операция аналогична операции "Заказ покупателя", за тем исключением, что кроме приходования заказов, выполняется резервирование заказанных товаров по складу на регистре «Остатки товаров компании».
  • Резервирование покупателя.  Хозяйственная операция выполняется резервирование заказанных товаров по складу на регистре «Остатки товаров компании».

Относится к подсистеме: «Заказы на отгрузку».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «Счет на оплату», «Поступление товаров».

Объекты, вводимые на основании: «Заказ внутренний», «Корректировка заказа покупателя», «Приходный кассовый ордер», «Реализация товаров», «Резервирование заказов покупателя», «Корректировка», «Снятие резервов заказов покупателя», «Счет на оплату» , «Заказ поставщику», «Чек на оплату».

Реквизиты

  • Наличие замен. В колонке отображается иконка "Замены" в случае, когда данная номенклатура имеет замены или является заменой другой номенклатуры. По нажатию на колонку открывается обработка «Подбор замен». 
  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Контрагент, размещающий заказ.
  • Договор. Реквизит является обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор, по которому ведутся взаиморасчеты с контрагентом.
  • Процент предоплаты. Тип значения: «Число». Процент предоплаты по заказу.
  • Сумма предоплаты. Тип значения: «Число». Сумма предоплаты по заказу.  
  • Склад. Ссылается на справочник «Склады (места хранения) компании». Склад компании, на котором будет производиться резервирование товара по заказу.
  • Срок поставки. Реквизит является обязательным для заполнения. Тип значения: «Дата». Срок отгрузки товаров по заказу контрагенту. Срок подставляется по умолчанию из регистра «Сведения кампании» для выбранного контрагента. Если для контрагента значение не указано, то подставляется значение из права "Срок поставки покупателю по умолчанию".
  • Срок снятия резерва. Срок, после которого резерв товара может быть снят.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу. 
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ. 
  • Менеджер. Ссылается на справочник «Сотрудники». Сотрудник, заключивший сделку с контрагентом.
  • Документ основание. Ссылается на документы «Счет на оплату», «Поступление товаров». Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип Цен, по которым происходит заказ товаров.
  • Тип заказа. Ссылается на справочник «ТипыЗаказов». Один из возможных типов заказов. 
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа, на дату документа. 
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").
  • Вид оплаты. Реквизит является обязательным для заполнения. Ссылается на перечисление «Виды оплаты». Вид оплаты заказа контрагентом.
  • Карта. Ссылается на справочник «Карточки». Карточка покупателя, по которой заказали товар.
  • Тип скидки. Ссылается на справочник «Типы скидок и наценок». Тип скидки, предоставляемой контрагенту.
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция. 
  • Сумма. Тип значения: «Число». Сумма документа.
  • Проект. Ссылается на справочник «Проекты». Проект, к которому относится документ.

Табличная часть «Товары»

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар).
  • Количество. Реквизит является обязательным для заполнения. Количество единиц измерения номенклатуры.
  • Резерв. Количество единиц измерения номенклатуры, которое требуется зарезервировать на складе по заказу покупателя.
  • Остаток. Тип значения: «Число». Количество номенклатуры, которая есть в наличии на выбранном складе.
  • Единица измерения. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Реквизит является обязательным для заполнения. Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Цена. Тип значения: «Число». Цена номенклатурной позиции.
  • Сумма. Тип значения: «Число». Сумма всегда без скидки. НДС входит в сумму, если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Процент скидки. Тип значения: «Число». Процент скидки, предоставляемой контрагенту для данного товара.
  • Скидка. Тип значения: «Число». Итоговая сумма скидки по строке документа.
  • Всего. Тип значения: «Число». Итоговая сумма по строке документа со всеми налогами.
  • Процент НДС. Ссылается на справочник «Ставки НДС». Ставка НДС по номенклатурной позиции.
  • НДС. Тип значения: «Число». Сумма НДС по номенклатурной позиции.
  • Характеристика номенклатуры. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п.
  • Ячейка. Ссылается на справочник «Ячейки хранения». Ячейка хранения для номенклатуры. Значение по умолчанию хранится в номенклатуре в разрезе складов.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Заказ покупателя. Простая печатная форма. Выводит список заказанных товаров. 
  • Печать этикеток и ценников. Обработка, позволяющая получить печатную форму этикеток и ценников заказанных товаров.

Способ вызова

Вызывается из меню "Документы --> Заказы и резервы --> Заказ покупателя".

Журналы, отображающие документ: «Покупатели», «Заказы и резервы».

Движения

  • Остаточный регистр «Заказы покупателей».
  • Остаточный регистр «Остатки товаров компании».
