Документ "Перемещение автомобилей"

Назначение

Документ служит для отражения в учете факта перемещения автомобилей со склада на склад.

При проведении документ формирует хозяйственные операции:

  • Перемещение автомобилей. Хозяйственная операция списывает все автомобили со склада отправителя и приходует их склад получатель, с тем признаком партии (собственный или комиссионный), с которым автомобиль был оприходован, по регистру «Остатки автомобилей».
  • Перемещение автомобилей в филиал. Хозяйственная операция списывает все автомобили со склада отправителя по регистру «Остатки автомобилей». При этом на регистр «Доходы и расходы» поступает расход соответствующих денежных средств.
  • Перемещение автомобилей из филиала. Хозяйственная операция обратна по отношению к операции "Перемещение автомобилей в филиал".

Относится к подсистеме: «Автосалон».

Параметры

Длина номера: 10

Периодичность номера: Год

Тип номера: Строка

Вводится на основании: «Поступление автомобилей», «Таможенная декларация(импорт)»,  «Ввод остатков автомобилей», .

Объекты вводимые на основании: «Реализация автомобилей», «Поступление дополнительных расходов».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  • Комментарий. Произвольный комментарий к документу.
  • Склад компании. Реквизит является обязательным для заполнения для хозопераций "Перемещение автомобилей" и "Перемещение автомобилей в филиал". Ссылается на справочник «Склады (места хранения) компании». Склад компании, с которого списываются автомобили.
  • Склад получатель. Реквизит является обязательным для заполнения для хозопераций "Перемещение автомобилей" и "Перемещение автомобилей из филиала". Ссылается на справочник «Склады (места хранения) компании». Склад компании, на который поступают автомобили.
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип цен, в котором происходит перемещение товара.
  • Сумма. Сумма документа.
  • Хоз операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Курс валюты документа, на дату документа. 
  • Документ основание. Ссылается на документы «Поступление автомобилей». Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).

Табличная часть «Автомобили»

  • Автомобиль. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Автомобили». Перемещаемый автомобиль.
  • Количество. Реквизит является обязательным для заполнения. Количество перемещаемых автомобилей. Всегда равно 1.
  • Цена. Цена перемещаемого автомобиля.
  • Сумма. Сумма перемещаемого автомобиля. Сумма всегда без скидки. НДС входит в сумму если это указано в типе цен. Сумма всегда равна Цена*Количество
  • Сумма всего. Итоговая сумма по строке документа со всеми налогами.
  • Ставка НДС. Ссылается на справочник «Ставки НДС». Ставка НДС автомобиля.
  • Сумма НДС. Сумма НДС автомобиля.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • ТОРГ-13 (накладная на внутреннее перемещение). Унифицированная печатная форма №Торг-13, утвержденная постановлением Госкомстата России.
  • Перемещение автомобилей. Простая печатная форма, содержащая список перемещаемых автомобилей со склада на склад.

Способ вызова

Вызывается из меню "Документы --> Автосалон --> Складские операции --> Перемещение автомобилей".

Движения

  • Остаточный регистр «Доходы и расходы».
  • Остаточный регистр «Остатки автомобилей».
