Документ "Разукомплектация автомобилей"

Назначение

Документ "Разукомплектация автомобилей" предназначен для фиксации факта снятия с автомобиля оборудования, ранее установленного на него дополнительно. Дополнительные опции должны быть ранее установлены на автомобиль согласно заказ-нарядов с видом ремонта "Комплектация" или производителем автомобиля. Снимаемые с автомобиля опции при этом списываются с регистра комплектации автомобилей и приходуются на указанный склад компании как товарные позиции. Партии товаров при разукомплектации соответствуют партиям товаров, которые были использованы при первоначальной комплектации автомобиля или при поступлении автомобиля. При разукомплектации себестоимость автомобиля уменьшается на сумму снятых дополнительных опций.

Документ вводится на основании заказ-наряда с видом ремента "Комплектация". В таблице работ при этом указываются работы, необходимые для выполнения разукомплектации.

При проведении документ формирует хозяйственные операции:

  • Разукомплектация автомобилей. Хозяйственная операция списывает опции, ранее установленные на автомобиль по заказ-нарядам.

Относится к подсистеме: «Автосалон».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «Заказ-наряд».

Объекты, вводимые на основании: «Приходный складской ордер».

Реквизиты

  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад компании, на который поступают товары, списанные при разукомплектации автомобиля.  
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете. 
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Документ основание. Ссылается на документы «Заказ-наряд». Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен».
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа.
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Сумма. Тип значения: «Число». Сумма документа.
  • Проект. Ссылается на справочник «Проекты». Проект, к которому относится документ.

Табличная часть «Товары»

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Опции (номенклатура), снимаемые с автомобиля.
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество единиц измерения номенклатуры.
  • Единица измерения. Реквизит является обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.  
  • Цена. Тип значения: «Число». Цена номенклатурной позиции.
  • Процент наценки. Тип значения: «Число». Наценка на закупочную цену для вычисления розничной цены.
  • Цена розничная. Тип значения: «Число». Розничная цена для оприходования номенклатуры на розничный склад.
  • Сумма. Тип значения: «Число». НДС входит в сумму, если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Всего. Тип значения: «Число». Итоговая сумма по строке документа со всеми налогами.
  • Сумма розничная. Тип значения: «Число». Розничная сумма для оприходования номенклатуры на розничный склад. 
  • Процент НДС. Ссылается на справочник «Ставки НДС». Ставка НДС по номенклатурной позиции.
  • Сумма НДС. Тип значения: «Число». Сумма НДС по номенклатурной позиции.  
  • Ячейка. Ссылается на справочник «Ячейки хранения». Ячейка хранения для номенклатуры. Значение по умолчанию хранится в номенклатуре в разрезе складов.
  • Характеристика. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Приходная накладная. Простая печатная форма, содержит список приходуемых товаров поставщика или комитента.
  • Печать этикеток и ценников. Обработка, позволяющая получить печатную форму этикеток и ценников приходуемых товаров.

Способ вызова

Вызывается из меню "Документы --> Автосервис --> Разукомплектация автомобилей".

Журналы, отображающие документ: «Заказ-наряд».
