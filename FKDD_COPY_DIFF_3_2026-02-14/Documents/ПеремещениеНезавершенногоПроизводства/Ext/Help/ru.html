Документ "Перемещение незавершенного производства"

Назначение

Документ "Перемещение незавершенного производства" предназначен для перемещения деталей, уже находящиеся в произодстве, между заказ-нарядами или цехами одного заказ-наряда. Перемещено может быть не более чем ранее было перемещено в производство по заказ-наряду источнику. Для завершения работ по заказ-наряду в производство должно быть перемещено деталей в полном объеме для выполнения ремонтных работ.

Список хозяйственных операций:

  • Перемещение незавершенного производства. Хозяйственная операция производит перемещение деталей между заказ-нарядами или цехами одного заказ-наряда по регистру «Товары компании в производстве».

Относится к подсистеме: «Автосервис».

Параметры

Длина номера: 10

Периодичность номера: Год

Тип номера: Строка

Вводится на основании: «Заказ-наряд».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете. 
  • Комментарий. Произвольный комментарий к документу.
  • Сумма. Сумма документа.
  • Хоз операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Курс валюты документа.
  • Документ основание. Реквизит является обязательным для заполнения. Ссылается на документ «Заказ-наряд». Документ заказ-наряда, на основании которого введен данный. Заказ-наряд, детали которого перемещаются. 
  • Цех. Реквизит является обязательным для заполнения. Ссылается на справочник «Цеха». Цех, из которого производиться перемещение деталей.
  • Заказ-наряд. Реквизит является обязательным для заполнения. Ссылается на документ «Заказ-наряд». Заказ-наряд, в кторый осуществляется перемещение деталей. 
  • Цех получатель. Реквизит является обязательным для заполнения. Ссылается на справочник «Цеха». Цех, в который производиться перемещение деталей.

Табличная часть «Товары»

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Деталь, перемещаемая из производства.
  • Количество. Реквизит является обязательным для заполнения. Количество извлекаемых из производства деталей.
  • Единица измерения. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения детали.
  • Коэффициент. Реквизит является обязательным для заполнения. Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Характеристика номенклатуры. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры.
  • Партия. Ссылается на ПВХ «Типы партий». Партия перемещаемого товара.
  • Партия. Ссылается на справочник «ГТД». ГТД перемещаемого товара.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.
  • Форма документа. Основная форма документа. Используется для ввода и редактирования документов данного типа.

Печатные формы

  • Перемещение незавершенного производства. Накладная на перемещаемые детали между заказ-нарядами.

Способ вызова

Вызывается из меню "Документы --> Автосервис --> Перемещение незавершенного производства".

Движения
