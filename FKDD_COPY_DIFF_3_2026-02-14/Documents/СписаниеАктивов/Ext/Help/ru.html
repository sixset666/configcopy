Документ "Списание активов"

Назначение

Документ предназначен для отражения в учете факта списания нетоварных активов с баланса подразделения компании.

При проведении документ формирует хозяйственные операции:

  • Списание активов.  Хозяйственная операция списывает с баланса все позиции нетоварных активов, хранимые на регистре «Прочие активы в эксплуатации», с указанием статьи списания, отражает остаточную стоимость списанных активов на регистре «Доходы и расходы». 

Относится к подсистеме: «Нетоварные активы».

Параметры

Длина номера: 10.

Тип номера: Строка.

Периодичность номера: Год.

Вводится на основании: «Ввод в эксплуатацию», «Ввод остатков прочих активов», «Перемещение активов».

Объекты, вводимые на основании: «Корректировка», «Событие».

Реквизиты

  • Причина списания. Тип значения: «Строка». Текстовое описание причины списания активов.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Документ основание. Ссылается на документы: «Ввод в эксплуатацию», «Ввод остатков прочих активов»,  «Перемещение активов». Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа.
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Сумма. Тип значения: «Число». Сумма документа. 

Табличная часть «Активы»

  • Актив. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Прочие активы». Нетоварный актив.
  • Инвентарный номер. Тип значения: «Строка». Инвентарный номер актива.
  • Балансовая стоимость. Тип значения: «Число». Балансовая стоимость актива.
  • Начисленная амортизация. Тип значения: «Число». Начисленная стоимость актива.
  • Остаточная стоимость. Тип значения: «Число». Остаточная стоимость актива.
  • Тип эксплуатации. Ссылается на справочник «Типы эксплуатации».
  • Материально-ответственное лицо. Ссылается на справочник «Сотрудники». Материально ответственное за отправку активов лицо.
  • Сумма списания. Тип значения: «Число». Сумма списания актива.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Списание активов. Простая печатная форма, содержащая список списываемых нетоварных активов.
  • ОС-4 (Акт о списании объекта/группы объектов ОС). Унифицированная печатная форма №ОС-4, утвержденная постановлением Госкомстата России.
  • МБ-4 (Акт выбытия малоценных и быстроизнашивающихся предметов). Унифицированная печатная форма №МБ-4, утвержденная постановлением Госкомстата России.

Способ вызова

Вызывается из меню "Документы --> Прочие активы --> Списание активов"

Журналы, отображающие документ: «Прочие активы».

Движения

  • Остаточный регистр «Доходы и расходы».
  • Остаточный регистр «Прочие активы в эксплуатации».
