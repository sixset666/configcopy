Документ "Изменение ассортимента подразделения"

Назначение

Документ предназначен для "закрепления" товарного ассортимента за подразделением.

При проведении документ формирует хозяйственные операции:

  • Ввод товаров в ассортимент. Хозяйственная операция устанавливает, с каким товаром, и по какой цене может работать подразделение компании.
  • Вывод товаров из ассортимента. Хозяйственная операция позволяет вывести товар из ассортимента подразделения.
  • Отмена ассортимента. Позволяет временно вывести товар из ассортимента подразделения.

Относится к подсистеме: «Складские акты».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Объекты, вводимые на основании: «Корректировка».

Реквизиты

  • Дата ввода в ассортимент. Дата, начиная с которой, товары попадают или выходят из ассортимента подразделения.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип цен закупки. 
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа.
  • Курс валюты управленческого учета. Тип значения: «Число». Курс валюты управленческого учета.
  • Операция. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.

Табличная часть «Товары»

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар).
  • Цена. Тип значения: «Число». Цена номенклатурной позиции.
  • Процент наценки. Тип значения: «Число». Наценка на закупочную цену для вычисления розничной цены.
  • Цена розничная. Тип значения: «Число». Розничная цена номенклатурной позиции.
  • Ставка НДС. Ссылается на справочник «Ставки НДС». Ставка НДС по номенклатурной позиции.
  • Базовый ассортимент. Тип значения: «Булево». Флаг указания того, что данный товар вводится в базовый ассортимент. 

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Печать ассортимента товаров. Простая печатная форма, содержит список товаров поступивших или вышедших из ассортимента подразделения.

Способ вызова

Вызывается из меню "Документы --> Складские документы --> Изменение ассортимента подразделения".
