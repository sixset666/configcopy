Документ "Переоценка"

Назначение

Документ "Переоценка" предназначен для переоценки остатков товаров на розничных складах, находящихся на комиссии в компании или отданных на реализацию комиссионерам. В табличной части документа не могут находиться услуги,  работы. Если на розничном складе существуют остатки одного и того же товара по разным ценам, то в документе будет присутствовать строка с усредненной ценой данного товара.

При проведении документ формирует хозяйственные операции:

  • Переоценка товаров в рознице. Хозяйственная операция переоценивает партии собственных товаров, находящиеся на розничном складе по регистру «Остатки товаров компании. А так же сохраняет розничные цены, по которым была выполнена переоценка на регистре сведений «Цены».
  • Переоценка товаров принятых на комиссию. Хозяйственная операция переоценивает партии товаров, принятых на комиссию, по регистру «Партии товаров компании».
  • Переоценка товаров отданных на комиссию. Хозяйственная операция переоценивает партии товаров, отданные на комиссию, по регистру «Партии товаров отданные».

Относится к подсистеме: «Складские акты».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «Реализация товаров», «Изменение цен», «Поступление товаров», «Перемещение товаров», «Инвентаризация», «Пересортица товаров».

Объекты, вводимые на основании: «Корректировка», «Событие».

Реквизиты

  • Контрагент. Реквизит является обязательным для заполнения при переоценке комиссионных товаров. Ссылается на справочник «Контрагенты и контакты». Комитент/комиссионер, от которого мы получили, или которому отдали товар.
  • Договор. Реквизит является обязательным для заполнения при переоценке комиссионных товаров. Ссылается на справочник «Договоры взаиморасчетов». Договор, по которому ведутся взаиморасчеты с контрагентом.
  • Склад. Реквизит является обязательным для заполнения при переоценке товаров в рознице. Ссылается на справочник «Склады (места хранения) компании». Склад компании, на котором производится переоценка.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.  
  • Подразделение. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ. 
  • Организация. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  • Автор. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Документ основание. Ссылается на документы «Реализация товаров», «Изменение цен», «Поступление товаров», «Перемещение товаров», «Инвентаризация», «Пересортица товаров». Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип цен, в котором происходит перемещение товара.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа, на дату документа.
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах"). 
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Сумма. Тип значения: «Число». Сумма документа.
  • Проект. Ссылается на справочник «Проекты». Проект, к которому относится документ.

Табличная часть «Товары»

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар).
  • Количество. Тип значения: «Число». Количество единиц измерения номенклатуры.
  • Единица. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Цена старая. Тип значения: «Число». Цена старая. (Расчетная средняя цена партии).
  • Цена новая. Тип значения: «Число». Цена номенклатурной позиции.
  • Сумма. Тип значения: «Число». Сумма переоценки товаров. Рассчитывается как (Цена новая-Цена старая)*Количество.
  • Документ передачи. Реквизит является уникальным и обязательным для заполнения при переоценке комиссионных товаров. Ссылается на документ «Реализация товаров», «Поступление товаров», «Ввод остатков товаров». Документ передачи товара на комиссию.
  • Характеристика. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п. 

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Переоценка. Простая печатная форма, содержит список переоцениваемых товаров.
  • Печать этикеток и ценников. Обработка, позволяющая получить печатную форму этикеток и ценников переоцениваемых товаров.

Способ вызова

Вызывается из меню "Документы --> Складские документы --> Переоценка".

Движения

  • Регистр сведений «Допроведение по партиям».
  • Регистр сведений «Цены».
  • Остаточный регистр «Доходы и расходы».
  • Остаточный регистр «Партии товаров компании».
  • Остаточный регистр «Партии товаров отданные».
  • Остаточный регистр «Остатки товаров компании».

Последовательности, изменяемые документом: "Партии".
