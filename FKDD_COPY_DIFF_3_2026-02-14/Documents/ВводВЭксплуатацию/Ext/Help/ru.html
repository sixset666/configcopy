Документ "Ввод в эксплуатацию"

Назначение

Документ предназначен для отражения в учете факта ввода в эксплуатацию нетоварных активов с указанием склада, по которому происходит списание, по выбранной статье, и указанием подразделения компании, которое принимает активы на баланс. В табличной части документа не могут присутствовать услуги, работы и т.п.

При проведении документ формирует хозяйственные операции:

  • Ввод в эксплуатацию. Хозяйственная операция уменьшает материальные активы на складах предприятия, хранимые на регистрах «Остатки товаров компании» и «Партии товаров компании», с указанием статьи списания, отражает сумму списанных товаров на регистре «Доходы и расходы», приходует нетоварные активы в подразделение-получатель, хранимые на регистре «Прочие активы в эксплуатации». Разницу суммы списания материальных и балансовой стоимости введенных в эксплуатацию нетоварных активов отражает на регистре «Доходы и расходы».

Относится к подсистеме: «Нетоварные активы».

Параметры

Длина номера: 10.

Тип номера: Строка.

Периодичность номера: Год.

Вводится на основании: справочник «Прочие активы», документы «Поступление товаров», «Перемещение товаров».

Объекты, вводимые на основании: «Перемещение активов», «Обслуживание актива», «Реализация активов», «Списание активов», «Корректировка».

Реквизиты

  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад компании, на котором происходит списание.
  • Подразделение-получатель. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, принимающее на баланс нетоварные активы.
  • Материально ответственное лицо. Реквизит является обязательным для заполнения. Ссылается на справочник «Сотрудники». Материально ответственное за учёт нетоварных активов лицо, которое отображается в списке документов и по умолчанию подставляется в табличную часть "Активы".
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ. 
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Документ основание. Ссылается на документы «Перемещение товаров», «Поступление товаров». Документ, на основании которого введен данный.  
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа.
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").
  • Проект. Ссылается на справочник «Проекты». Проект, к которому относится документ.

Табличная часть «Активы»

  • Актив. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Прочие активы». Нетоварный актив.
  • Тип эксплуатации. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы эксплуатации». Тип эксплуатации.
  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар).
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество единиц измерения номенклатуры.
  • Остаток. Тип значения: «Число». Количество номенклатуры на складе отгрузки.
  • Единица измерения. Реквизит является обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Балансовая стоимость. Реквизит является обязательным для заполнения. Тип значения: «Число». Балансовая стоимость. По умолчанию рассчитывается по сумме списываемой номенклатуры по партиям.
  • Материально ответственное лицо. Ссылается на справочник «Сотрудники». Материально ответственное за учёт нетоварных активов лицо, принимающее актив в эксплуатацию.
  • Ячейка. Ссылается на справочник «Ячейки хранения». Ячейка хранения для номенклатуры. По умолчанию заполняется из номенклатуры.

    Формы
  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Ввод в эксплуатацию. Простая печатная форма, содержит список введенных в эксплуатацию нетоварных активов.
  • ОС-1 (Акт о приеме-передаче объекта ОС). Унифицированная печатная форма №ОС-1, утвержденная постановлением Госкомстата России.
  • М-11 (Требование-накладная). Унифицированная печатная форма №М-11, утвержденная постановлением Госкомстата России.

Способ вызова

Вызывается из меню "Документы --> Прочие активы --> Ввод в эксплуатацию".

Журналы, отображающие документ: «Прочие активы».

Движения

  • Остаточный регистр «Доходы и расходы».
  • Остаточный регистр «Остатки товаров компании».
  • Остаточный регистр «Партии товаров компании».
  • Остаточный регистр «Прочие активы в эксплуатации».
  • Регистр сведений  «Допроведение по партиям».

Последовательности, изменяемые документом: «Партии».
