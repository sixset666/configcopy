Документ "Инвентаризация"

Назначение

Документ предназначен для отражения в учете факта проведения инвентаризации товаров по выбранному складу. В табличной части документа не могут фигурировать услуги, работы и т.п. При проведении в случае оприходования излишков документ образует партию с признаком "Товар купленный". При проведении если в строке фактическое количество больше учетного, то документ приходует излишек (разницу между фактическим и учетным количеством) на склад, в противном случае, если фактическое количество меньше учетного, то документ списывает недостачу со склада по выбранной статье расходов. Новый (не проведенный) документ можно заполнить автоматически учетными остатками по выбранному складу (меню по кнопке  "Заполнение"). При вводе новых строк, автоматически подставляются учетные количество и стоимость.

При проведении документ формирует хозяйственные операции:

  • Инвентаризация товаров. Хозяйственная операция в зависимости от реальных остатков приходует или списывает материальные активы, хранимые на оптовых складах компании, в виде ТМЦ  с собственным признаком партии, на регистры «Остатки товаров компании» и «Партии товаров компании», отражает факт продажи товаров комитента, если они в процессе инвентаризации были утеряны или испорчены, на регистре «Реализованные товары». А так же регистрирует цены, которые были изменены в процессе инвентаризации, на регистре сведений «Цены».
  • Инвентаризация товаров, отданных на комиссию. Хозяйственная операция аналогична операции "Инвентаризации товаров" за тем исключением, что приходуются или списывается товары, отданные комиссионеру на регистре  «Партии товаров отданные», а также начисляются взаиморасчеты на сумму недостачи или излишка на регистре «Взаиморасчеты компании».

Относится к подсистеме: «Складские акты».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «Инвентаризация ордерного склада».

Объекты, вводимые на основании: «Приходный кассовый ордер», «Инвентаризация ордерного склада»,  «Корректировка», «Событие», «Переоценка», «Возврат от покупателя».

Реквизиты

  • Контрагент. Реквизит является обязательным для заполнения при инвентаризации товаров у комиссионера. Ссылается на справочник «Контрагенты и контакты». Комиссионер, которому мы отдавали товар на комиссию.
  • Договор. Реквизит является обязательным для заполнения при инвентаризации товаров у комиссионера. Ссылается на справочник «Договоры взаиморасчетов». Договор, по которому ведутся взаиморасчеты с контрагентом.
  • Склад. Реквизит является обязательным для заполнения при инвентаризации собственных товаров. Ссылается на справочник «Склады (места хранения) компании». Склад компании, на котором проводим инвентаризацию.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.  
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Документ основание. Ссылается на документ «Инвентаризация ордерного склада». Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип Цен, по которым инвентаризируется товар.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа, на дату документа.
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Сумма. Тип значения: «Число». Сумма документа.
  • Статья списания обнаруженной недостачи ТМЦ. Реквизит является обязательным для заполнения. Ссылается на справочник «Статьи доходов и расходов». Статья списания обнаруженной недостачи ТМЦ.
  • Статья оприходования обнаруженных излишков ТМЦ. Реквизит является обязательным для заполнения. Ссылается на справочник «Статьи доходов и расходов». Статья оприходования обнаруженных излишков ТМЦ.  
  • Основание для проведения инвентаризации. Тип значения: «Строка». Описание основания для проведения инвентаризации.
  • Режим корректировки книжного количества при проведении. Ссылается на перечисление "Режим корректировки количества в инвентаризации". Определяет возможность пользователя исправлять книжное количество в документе, что определяется специальным правом. 
  • Проект. Ссылается на справочник «Проекты». Проект, к которому относится документ.

Табличная часть «Товары»

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар).
  • Количество книжное. Тип значения: «Число». Расчетное (по базе данных) количество единиц измерения номенклатуры.
  • Количество фактическое. Тип значения: «Число». Фактическое количество товаров на складе.
  • Количество. Тип значения: «Число». По этому количеству будут производиться движения.
  • Единица. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Цена. Тип значения: «Число». Цена номенклатурной позиции.
  • Сумма книжная (нормативная). Тип значения: «Число». Оценочная нормативная сумма товара по данным из базы данных.
  • Сумма фактическая (нормативная). Тип значения: «Число». Фактическая нормативная сумма товара.
  • Сумма. Тип значения: «Число». Сумма разницы (нормативная). По этой сумме будут производиться движения.
  • Цена розничная. Тип значения: «Число». Цена отпускная номенклатурной позиции.
  • Сумма книжная (розничная). Тип значения: «Число». Оценочная отпускная сумма товара по данным из базы данных.
  • Сумма фактическая (розничная).  Тип значения: «Число». Фактическая отпускная сумма по данным инвентаризации.
  • Сумма (розничная). Тип значения: «Число». Отпускная сумма разницы. По этой сумме будут производиться движения.
  • ГТД / РНПТ. Ссылается на справочник «ГТД». Номер грузовой таможенной декларации (для автомобилей поступивших по импорту) или регистрационный номер (РНПТ) для прослеживаемых товаров.
  • Характеристика номенклатуры. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п.
  • Документ передачи. Ссылается на документы:  «Инвентаризация», «Реализация товаров». Документ передачи товаров на комиссию.
  • Партия. Ссылается ПВХ  «Типы партий». Партия, с которой будет списан товар. Если у пользователя установлено право "Выборочное списание партий", то этот реквизит является обязательным для заполнения.
  • Номер листа. Тип значения: «Число». Номер листа инвентаризационной ведомости.
  • Ячейка. Ссылается на справочник «Ячейки хранения». Ячейка хранения для номенклатуры. Значение по умолчанию хранится в номенклатуре в разрезе складов.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Инвентаризация товаров на комиссии. Простая печатная форма, содержит список инвентаризируемых товаров.
  • Инвентаризационная опись (ИНВ-3). Унифицированная печатная форма №ИНВ-3, утвержденная постановлением Госкомстата России.
  • Сличительная ведомость (ИНВ-19). Унифицированная печатная форма №ИНВ-19, утвержденная постановлением Госкомстата России.
  • Пустографка. Простая печатная форма, содержит список инвентаризируемых товаров без их фактического количества.
  • Итоги по листам. Печатная форма, содержит список инвентаризируемых товаров с указанием сумм закупки и сумм продажи.
  • Печать этикеток и ценников. Обработка, позволяющая получить печатную форму этикеток и ценников инвентаризируемых товаров.

Способ вызова

Вызывается из меню "Документы --> Складские документы --> Инвентаризация".

Журналы, отображающие документ: «Складские документы», «Складские акты».
