Документ «Опрос»

Назначение

В документе регистрируются ответы на вопросы анкеты, которая может быть указана в соответствующем реквизите документа. Документ можно распечатать в виде HTML страницы, нажав на кнопку "Просмотр HTML" в форме документа.

Важно!

Для работы HTML страницы необходимо, чтобы в веб-обозревателе было разрешено выполнение скриптов. В Internet Explorer по умолчанию выключена опция, разрешающая выполнение скриптов в файлах, открываемых локально. Поэтому когда страница открывается, скрипты блокируются, а в самом верху страницы отображается информационная панель (Information Bar), сообщающая о том, что активная часть содержимого страницы заблокирована. Чтобы разрешить выполнение заблокированной части страницы, необходимо щелкнуть на этой панели левой кнопкой мыши и в выпавшем меню выбрать соответствующий пункт (Allow Blocked Content). Кроме того, можно отключить блокирование скриптов для всех страниц, открываемых локально, используя настройки Internet Explorer. Для этого в меню Tools необходимо выбрать пункт Internet Options и в открывшемся окне выбрать закладку Advanced. На этой закладке, в разделе Security необходимо установить флаг Allow active content to run in files on My Computer.

 

Параметры

Длина номера: 10.

Периодичность номера: Непериодический.

Тип номера: Строка.

Вводится на основании: «Событие», «Контрагенты», «Рассылка», «Заказ-наряд», «Заявка на ремонт», «Реализация товаров», «Заказ покупателя», «Рабочий лист», «Типовые анкеты», «Тест-драйв», «Реализация автомобилей».

Объекты, вводимые на основании: «Событие», «Жалоба клиента», «Электронное письмо».

Реквизиты

  • Автор. Ссылается на справочник "Пользователи". Автор документа - пользователь, породивший документ.
  • Документ основание. Документ, на основании которого был введен опрос.
  • Комментарий. Тип значения: Строка. Произвольный комментарий к документу.
  • Менеджер. Ссылается на справочник "Сотрудники". Менеджер, создавший опрос.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник "Организации". Идентификатор юридического лица, от которого выписан документ.
  • Опрашиваемое лицо. Реквизит является обязательным для заполнения. Ссылается на справочник "Контрагенты". Опрашиваемое лицо.
  • Опрос завершен. Тип значения: Булево. Признак завершенности опроса.
  • Отказ. Тип значения: Булево. Признак отказа.
  • Подразделение компании. Реквизит является обязательным для заполнения. Ссылается на справочник "Подразделения компании". Подразделение компании, в котором был выписан документ.
  • Проект. Ссылается на справочник "Проекты". Проект документа.
  • Типовая анкета. Реквизит является обязательным для заполнения. Ссылается на справочник "Типовые анкеты". Типовая анкета.
  • Операция. Ссылается на справочник "Хоз. операции". Выполняемая документом операция.

Табличная часть «Вопросы»

  • Вопрос. Ссылается на план видов характеристик "Вопросы для анкетирования". Вопрос.
  • Ответ. Тип значения: "Строка". Ответ.
  • Типовой ответ. Ссылается на характеристику "Вопросы для анкетирования". Типовой ответ.
  • Раздел. Ссылается на справочник "Разделы анкеты". раздел анкеты. 

Табличная часть «Составной отчет»

  • Вопрос владелец. Ссылается на план видов характеристик "Вопросы для анкетирования". Владелец вопроса.
  • Вопрос. Ссылается на план видов характеристик "Вопросы для анкетирования". Вопрос.
  • Ответ. Тип значения: "Строка". Ответ.  
  • Типовой ответ. Ссылается на характеристику "Вопросы для анкетирования". Типовой ответ.  
  • Номер строки в табоице. Тип значения: "Число". Номер строки в таблице.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Документы --> Взаимоотношения с клиентами --> Опро".
