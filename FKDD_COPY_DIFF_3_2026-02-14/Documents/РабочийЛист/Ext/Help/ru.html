Документ «Рабочий лист»

Назначение

Используется для фиксирования всех параметров потенциальной продажи, для консолидирования истории взаимоотношений с клиентом и подразделениями компании до момента заключения соглашения (договора) с покупателем.

Примечание

После записи документа, если произошла смена его статуса, создаются документы, шаблоны которых присутствуют в табличной части "Шаблоны событий" элемента справочника "Статусы рабочего листа". Так же создать эти документы можно по кнопке "Создать события".

Параметры

Длина номера: 10.

Периодичность номера: Непериодический.

Тип номера: Строка.

Вводится на основании: «Событие».

Объекты, вводимые на основании: «Рабочий лист отдела страхования», «Рабочий лист кредитного отдела», «Заказ на автомобиль», «Событие», «Опрос».

Если телефон с данным видом контактной информации уже задан в регистре для данного клиента, то задается вопрос о замене телефона либо выборе другого вида контактной информации для записи.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.
  • Форма выбора состава событий. Форма, позволяющая выбрать состав создаваемых документов при переходе рабочего листа из одного статуса в другой, открывается, если право "Автоматическое создание документов Событие" в разделе "Управление взаимоотношениями с клиентами" установлено в значение "Ложь" (по умолчанию - "Истина").

Способ вызова

Вызывается из меню "Документы --> Взаимоотношения с клиентами --> Рабочий лист".
