Документ "Бюджет доходов и расходов"

Назначение

Документ «Бюджет доходов и расходов» содержит плановые данные по статьям доходов и расходов от основной и дополнительной деятельности.  Запланированные документом данные относятся к выбранному периоду планирования, который  соответствует периодичности сценария планирования (поддерживаются следующие периоды планирования: месяц, квартал, год). Планируемый период не может быть меньше даты документа.

При проведении документ формирует хозяйственные операции:

  • Бюджет доходов и расходов. Формируются движения по регистру накопления: «Бюджет доходов и расходов», в котором производится накопление запланированных данных. 

Относится к подсистеме: «Планирование».

Параметры

  • Длина номера: 10.
  • Периодичность номера: Год.
  • Тип номера: Строка.
  • Вводится на основании:«Бюджет доходов и расходов».
  • Объекты, вводимые на основании: «Бюджет доходов и расходов», «Корректировка»

Реквизиты

  • Сценарий планирования.  Реквизит является обязательным для заполнения. Ссылается на справочник «Сценарии планирования». Определяет периодичность планирования.
  • Период планирования. Реквизит является обязательным для заполнения. Тип значения: «Дата». Начальная дата периода, на который происходит планирование. Зависит от реквизита "Периодичность" выбранного сценария планирования. 
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу. 
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Использовать заполнение выручки. Тип значения: «Булево». Признак заполнения статей выручки.
  • Способ расчета выручки. Тип значения: «Число». Определяет способ заполнения статей дохода от основной деятельности. 
  • Количество периодов выручка. Тип значения: «Число». Количество периодов для анализа при автозаполнении статей дохода от основной деятельности.  
  • Коэффициент роста продаж. Тип значения: «Число». Используется при автозаполнении статей дохода от основной деятельности.
  • Коэффициент сезонности выручка. Тип значения: «Число». Используется при автозаполнении статей дохода от основной деятельности.
  • Использовать заполнение себестоимость. Тип значения: «Булево». Признак заполнения статей себестоимости.
  • Способ расчета себестоимости. Тип значения: «Число». Определяет способ заполнения статей расхода от основной деятельности.
  • Количество периодов себестоимость. Тип значения: «Число». Количество периодов для анализа при автозаполнении статей расхода от основной деятельности. 
  • Коэффициент роста себестоимости. Тип значения: «Число». Используется при автозаполнении статей расхода от основной деятельности.
  • Коэффициент сезонности себестоимость.  Тип значения: «Число». Используется при автозаполнении статей расхода от основной деятельности. 
  • Заполнить статьи накладных доходов и расходов. Тип значения: «Булево». Признак заполнения статей накладных доходов и расходов.  
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Не изменяется. Может принимать только одно значение - "Бюджет доходов и расходов". 
  • Сумма. Тип значения: «Число». Итоговое значение суммы планирования в валюте управленческого учета.   

Табличная часть «Статьи доходов и расходов»

Основная табличная часть документа, предназначенная для хранения запланированных данных.

  • Статья расхода\дохода. Ссылается на справочник «Статьи расходов и доходов».
  • Сумма доход. Тип значения: «Число». Прогнозируемый доход по статье.
  • Сумма расход. Тип значения: «Число». Прогнозируемый расход по статье.
  • Способ заполнения. Тип значения: «Строка». Используется для отображения способа, которым была заполнена текущая статья.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.
  • Форма списка для выбора. Основная форма выбора.
  • Форма автозаполнение. Произвольная форма. Включает в себя три независимых заполнений: статьями доходов от основной деятельности, статьями расходов от основной деятельности, статьями накладных расходов и доходов. 

Способ вызова

Вызывается из меню "Документы--> Бюджетирование --> Бюджет доходов и расходов".

Движения

  • Оборотный регистр «Бюджет доходов и расходов».
