Документ "SMS"

Назначение

Документ демонстрирует работу с SMS4B. В данном документе реализована отправка SMS нескольким получателям, данный документ создается при входящем SMS, в данном документе также хранится статус отправленного/полученного SMS.

Относится к подсистеме: «Коммуникатор».

Параметры

Длина номера: 9, тип номера: Строка.

Периодичность номера: Непериодический

Тип номера: Строка

Реквизиты

  • Входящее исходящее. Тип значения: «Строка». Краткое описание.
  • Текст сообщения. Тип значения: «Строка». Краткое описание.
  • Отчет о доставке. Тип значения: «Булево». Краткое описание.
  • Транслитерация. Тип значения: «Булево». Краткое описание.
  • Номер отправителя. Тип значения: «Строка». Краткое описание.
  • GUID SMS Сообщения. Тип значения: «Строка». Краткое описание.
  • Статус строкой. Тип значения: «Строка». Краткое описание.

Табличная часть «Получатели»

  • Статус. Тип значения: «Строка». Краткое описание.
  • Дата завершения. Тип значения: «Дата». Краткое описание.
  • Описание ошибки. Тип значения: «Строка». Краткое описание.
  • GUID SMS. Тип значения: «Строка». Краткое описание.

Формы

  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.

Движения
