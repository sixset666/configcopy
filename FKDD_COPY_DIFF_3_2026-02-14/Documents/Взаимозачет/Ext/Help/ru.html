Документ "Взаимозачет"

Назначение

Документ предназначен для отражения в учете операций по взаимозачетам между двумя контрагентами, имеющими дебиторскую и кредиторскую задолженность или для перераспределения задолженностей по различным договорам и сделкам. 

При проведении документ формирует хозяйственную операцию:

  • Взаимозачет. Хозяйственная операция отражает взаимозачеты между двумя контрагентами, имеющими дебиторскую и кредиторскую задолженность или перераспределения задолженностей на регистре «Взаиморасчеты компании», а также списывает суммовые разницы на регистре «Доходы и расходы» компании.

Относится к подсистеме: «Взаиморасчеты».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Реквизиты

  • Дебитор. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Контрагент, который имеет долги перед нами.
  • Кредитор. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Контрагент, которому мы должны.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ. 
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Тип значения: «Число». Реквизит является обязательным для заполнения. Курс валюты документа, на дату документа.
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").
  • Хоз операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Сумма дебет. Тип значения: «Число». Сумма дебет.
  • Сумма кредит. Тип значения: «Число». Сумма кредит.

Табличная часть «Состав»

  • Договор дебитора. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор взаиморасчетов дебитора.
  • Договор кредитора. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор взаиморасчетов кредитора.
  • Сделка дебитора. Реквизит является уникальным и обязательным для заполнения. Ссылается на план видов характеристик «Типы сделок». Сделка, по которой дебитор нам должен.
  • Сделка кредитора. Реквизит является уникальным и обязательным для заполнения. Ссылается на план видов характеристик «Типы сделок». Сделка, по которой мы должны кредитору.
  • Валюта договора дебитора. Ссылается на справочник «Валюты». Валюта договора дебитора.
  • Валюта договора кредитора. Ссылается на справочник «Валюты». Валюта договора кредитора.
  • Курс валюты договора дебитора. Курс валюты взаиморасчетов договора дебитора. Предназначен для ручного редактирования курса валюты. Для автоматического расчета курса необходимо оставить данное поле пустым.
  • Курс валюты договора кредитора. Курс валюты взаиморасчетов договора кредитора. Предназначен для ручного редактирования курса валюты. Для автоматического расчета курса необходимо оставить данное поле пустым.
  • Сумма. Реквизит является обязательным для заполнения. Тип значения: «Число». Сумма взаимозачета.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Взаимозачет. Простая печатная форма документа.

Способ вызова

Вызывается из меню "Документы --> Взаиморасчеты --> Взаимозачет".

Движения

  • Остаточный регистр «Взаиморасчеты компании».
  • Остаточный регистр «Доходы и расходы».

Последовательности, изменяемые документом: "Взаиморасчеты".
