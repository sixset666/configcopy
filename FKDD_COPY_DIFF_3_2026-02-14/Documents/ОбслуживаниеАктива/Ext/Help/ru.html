Документ "Обслуживание актива"

Назначение

Документ предназначен для отражения в учете факта обслуживания нетоварного актива.

При проведении документ формирует хозяйственные операции:

  • Обслуживание актива на расходы подразделения. Хозяйственная операция уменьшает материальные активы (расходные материалы для обслуживания) на складах предприятия, хранимые на регистрах «Остатки товаров компании» и «Партии товаров компании», с указанием статьи списания, отражает сумму списанных товаров на регистре «Доходы и расходы», начисляет кредиторскую задолженность компании перед контрагентом на регистре «Взаиморасчетов компании» на сумму оказанных услуг (выполненных работ), отражает доход по другому подразделению компании, оказывающему услуги (выполняющему работы) нашему подразделению на регистре «Доходы и расходы», отражает расход по нашему подразделению компании на регистре «Доходы и расходы». Начисляется сумма обслуживания на регистре «Прочие активы в эксплуатации».
  • Обслуживание актива на стоимость актива. Хозяйственная операция уменьшает материальные активы (расходные материалы для обслуживания) на складах предприятия, хранимые на регистрах «Остатки товаров компании» и «Партии товаров компании», с указанием статьи списания, отражает сумму списанных товаров на регистре «Доходы и расходы», начисляет кредиторскую задолженность компании перед контрагентом на регистре «Взаиморасчетов компании» на сумму оказанных услуг (выполненных работ), отражает доход по другому подразделению компании, оказывающему услуги (выполняющему работы) нашему подразделению на регистре «Доходы и расходы». Начисляется сумма обслуживания и балансовая стоимость (на общую сумму обслуживания) на регистре «Прочие активы в эксплуатации».

Относится к подсистеме: «Нетоварные активы».

Параметры

Длина номера: 10.

Тип номера: Строка.

Периодичность номера: Год.

Вводится на основании: «Ввод в эксплуатацию», «Ввод остатков прочих активов», «Перемещение активов».

Объекты, вводимые на основании: «Расходный кассовый ордер», «Корректировка», «Событие».

Реквизиты

  • Актив. Реквизит является обязательным для заполнения. Ссылается на справочник «Прочие активы». Нетоварный актив.
  • Статья расходов. Реквизит является обязательным для заполнения (в случае обслуживания на расходы подразделения). Ссылается на справочник  «Статьи доходов и расходов». Статья расходов по обслуживанию актива (по нашему подразделению).
  • Тип обслуживания. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы обслуживания». Тип обслуживания.
  • Подразделение, оказывающее услугу. Ссылается на справочник «Подразделения компании». Подразделение нашей компании, оказывающее услуги (выполняющее работы) по обслуживанию актива. 
  • Контрагент. Ссылается на справочник «Контрагенты и контакты». Контрагент, оказывающий услуги (выполняющий работы) по обслуживанию актива.
  • Статья доходов. Ссылается на справочник  «Статьи доходов и расходов». Статья доходов по обслуживанию актива (по подразделению, обслуживающему актив).
  • Договор. Ссылается на справочник «Договоры взаиморасчетов». Договор, по которому ведутся взаиморасчеты с контрагентом.
  • Стоимость собственных услуг. Тип значения: «Число». Стоимость оказанных услуг (выполненных работ) подразделения нашей компании, оказывающего услугу.  
  • Стоимость сторонних услуг. Тип значения: «Число». Стоимость оказанных услуг (выполненных работ) контрагента сторонней организации.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Документ основание. Ссылается на документы «Ввод в эксплуатацию», «Ввод остатков прочих активов», «Перемещение активов». Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).  
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа.
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Сумма. Тип значения: «Число». Сумма документа.
  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад компании, на котором происходит списание номенклатуры на обслуживание актива.
  • Проект. Ссылается на справочник «Проекты». Проект, к которому относится документ.

Табличная часть «Расходные материалы»

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество единиц измерения номенклатуры.
  • Остаток. Тип значения: «Число». Количество номенклатуры, которое есть на складе.
  • Единица измерения. Реквизит является обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Сумма. Тип значения: «Число». Сумма всегда без скидки. НДС не входит в сумму. Сумма всегда равна Цена*Количество.
  • Характеристика номенклатуры. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п.
  • Сумма списания. Тип значения: «Число». Сумма списанной партионной себестоимости.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Обслуживание актива. Простая печатная форма, содержит параметры обслуживания и список списываемой со склада номенклатуры для обслуживания.
  • ОС-3 (Акт о приемке-сдаче отремонтированных, реконструированных, модернизированных объектов основных средств). Унифицированная печатная форма №ОС-3, утвержденная постановлением Госкомстата России.  

Способ вызова

Вызывается из меню "Документы --> Прочие активы --> Обслуживание актива".

Журналы, отображающие документ: «Прочие активы».

Движения

  • Регистр сведений «Допроведение по партиям».
  • Остаточный регистр «Взаиморасчёты компании».
  • Остаточный регистр «Доходы и расходы».
  • Остаточный регистр «Остатки товаров компании».
  • Остаточный регистр «Партии товаров компании».
  • Остаточный регистр «Прочие активы в эксплуатации».

Последовательности, изменяемые документом: "Взаиморасчеты", "Партии".
