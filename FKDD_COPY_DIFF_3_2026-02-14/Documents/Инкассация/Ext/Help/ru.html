Документ "Инкассация"

Назначение

Документ фиксирует факт внесения или изъятия денежных средств из кассы ККМ. ККМ - контрольно кассовая машина.

При проведении документ формирует хозяйственные операции:

  • Внесение в кассу ККМ. Хозяйственная операция приходует денежные средства, вносимые в кассу ККМ на регистр «Кассы ККМ».
  • Изъятие из кассы ККМ. Хозяйственная операция обратная операции "Внесение в кассу ККМ".

Относится к подсистеме: «Движения денежных средств».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «Приходный кассовый ордер», «Расходный кассовый ордер».

Объекты, вводимые на основании: «Приходный кассовый ордер», «Корректировка», «Расходный кассовый ордер».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Документ основание. Ссылается на документы «Приходный кассовый ордер», «Расходный кассовый ордер». Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа, на дату документа.
  • Касса ККМ. Реквизит является обязательным для заполнения. Ссылается на справочник «Кассы ККМ». Касса ККМ, в которой проводят изъятие/внесение денег.
  • Хоз операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Номер чека. Реквизит является обязательным для заполнения. Тип значения: «Число». Номер чека ФР в пределах смены.
  • Дата ФР. Реквизит является обязательным для заполнения. Тип значения: «Дата». Дата чека, пробитого на ФР.
  • Номер документа. Реквизит является обязательным для заполнения. Тип значения: «Число». Сквозной номер чека ФР.
  • Номер смены. Реквизит является обязательным для заполнения. Тип значения: «Число». Номер смены ФР.
  • Инкассатор. Реквизит является обязательным для заполнения, если из кассы изымаются/вносятся наличные. Ссылается на справочник «Контрагенты и контакты». Контрагент, являющийся инкассатором.
  • Договор взаиморасчетов с инкассатором. Реквизит является обязательным для заполнения, если из кассы изымаются/вносятся наличные. Ссылается на справочник «Договоры взаиморасчетов». Договор взаиморасчетов с контрагентом, являющимся инкассатором.
  • Платежная система. Реквизит является обязательным для заполнения, если из кассы изымаются/вносятся безналичные. Ссылается на справочник «Контрагенты и контакты». Контрагент, являющийся процессинговым центром для безналичных расчетов.
  • Договор взаиморасчетов с платежной системой. Реквизит является обязательным для заполнения, если из кассы изымаются/вносятся безналичные. Ссылается на справочник «Договоры взаиморасчетов». Договор взаиморасчетов с контрагентом, являющимся процессинговым центром для безналичных расчетов.
  • Сумма. Реквизит является обязательным для заполнения. Тип значения: «Число». Сумма документа.
  • Сумма наличных. Тип значения: «Число». Сумма инкассации наличных денежных средств.
  • Сумма безналичными получено. Тип значения: «Число». Сумма инкассации безналичных денежных средств от реализации.
  • Сумма безнал возврат. Тип значения: «Число». Сумма инкассации безналичных денежных средств, на которую осуществлен возврат.
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").
  • Проект. Ссылается на справочник «Проекты». Проект, к которому относится документ.

Табличная часть «Оплаты»

  • Тип оплаты. Ссылается на справочник «Типы оплат». Тип оплаты.
  • Сумма. Тип значения: «Число». Сумма инкассации денежных средств.
  • Сумма возврат. Тип значения: «Число». Сумма инкассации денежных средств, на которую осуществлен возврат.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • КМ-6. Унифицированная печатная форма №KM-6, утвержденная постановлением Госкомстата России.

Способ вызова

Вызывается из меню "Документы --> Движение денежных средств --> Инкассация".

Движения

  • Остаточный регистр «Взаиморасчеты компании».
  • Остаточный регистр «Доходы и расходы».
  • Остаточный регистр «Кассы ККМ».

Последовательности, изменяемые документом: "Взаиморасчеты".
