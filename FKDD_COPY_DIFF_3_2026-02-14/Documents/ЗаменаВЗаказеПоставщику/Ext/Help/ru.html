Документ "Замена в заказе поставщику"

Назначение

Документ предназначен для оформления операций по замене заказанных товаров. В табличной части документа не должны фигурировать услуги, работы и т.п. В качестве табличной части используется дерево. На первом уровне дерева указываются заменяемые товары, на втором уровне товары заменители. Основным режимом ввода является ввод на основании документов "Заказ поставщику", однако, при необходимости, документ можно заполнить вручную.

При проведении документ формирует хозяйственные операции:

  • Замена в заказе поставщику. Хозяйственная операция формирует замены в заказе поставщику на регистре «Заказы поставщикам».

Относится к подсистеме: «Заказы на поставку».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «Заказ поставщику».

Объекты, вводимые на основании: «Корректировка», «Событие».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочники «Контрагенты и контакты». Поставщик, у которого происходит замена по заказу.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Вид оплаты. Реквизит является обязательным для заполнения. Ссылается на перечисление «Виды оплаты». Вид оплаты с контрагентом. 
  • Документ основание. Ссылается на документы: «Заказ поставщику». Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип цен, в которых заказывается товар.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Карточка. Ссылается на справочник «Карточки». Карточка покупателя, по которой продавали товар. 
  • Менеджер. Ссылается на справочник «Сотрудники». Сотрудник, заключивший сделку с контрагентом.
  • Сумма. Тип значения: «Число». Сумма документа.

Табличная часть «Товары»

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар), который мы заменяем или на который мы заменяем.
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество единиц измерения номенклатуры.
  • Единица. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Цена. Тип значения: «Число». Цена номенклатурной позиции.
  • Сумма. Тип значения: «Число». Сумма всегда без скидки. НДС входит в сумму если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Всего. Тип значения: «Число». Итоговая сумма по строке документа со всеми налогами.
  • Процент НДС. Ссылается на справочник «Ставки НДС». Ставка НДС по номенклатурной позиции.
  • НДС. Тип значения: «Число». Сумма НДС по номенклатурной позиции.
  • Характеристика номенклатуры. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п. 

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Замена в заказе поставщика. Простая печатная форма, содержит дерево замен товаров по текущему заказу.

Способ вызова

Вызывается из меню "Документы --> Заказы и резервы --> Замена в заказе поставщику".

Журналы, отображающие документ: «Заказы и резервы».

Движения

  • Остаточный регистр «Заказы поставщикам».
