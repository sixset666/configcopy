Форма выбора корзины

Назначение

Форма предназначена для просмотра списка корзин и выбора для загрузки товаров в документы или в АРМ Корзина с целью дальнейшего редактирования.

Форма состоит из следующих элементов:

  • Список корзин. Список записей регистра сведений "Корзина". При открытии формы из документа на список устанавливается отбор по контрагенту из документа.  Для упрощения поиска корзины в списке рекомендуется использовать отборы по номеру, дате и клиенту. Отборы вызываются кнопками командной панели.
  • Данные корзины. Содержит таблицу товаров корзины и значения реквизитов, которые сохраняются вместе с корзиной в регистре. Цены в таблице указываются те, которые сохранены в корзине.
  • Способ загрузки. Доступно для выбора при загрузке корзины из формы документов. 
    • "Номенклатура и количество" -  будут загружены товары и количество, которое указано в корзине. Если строка с таким товаром уже есть в документе, то количество будет увеличено на количество, которое указано в корзине. На добавленные строки будут подставлены действующие цены согласно указанному в документе тицу цен.
    • "Полная загрузка" - будут загружены все реквизиты из табличной части и шапки корзины, которые возможно загрузить в документ данного вида, в том числе контрагент, договор, склад компании, тип цен, валюта, курс, скидки. В табличную часть товары загружаются с ценами, скидками, ставкой НДС, количеством и др.реквизитами, которые указаны в корзине. Если строка с товаром уже есть в документе, то количество будет увеличено на количество, которое указано в корзине, а остальные реквизиты заменены на те, которые указаны в корзине.

Способ вызова

Открывается из «АРМ Корзина» по кнопке "Загрузить" или из товарных документов по кнопке "Заполнить из корзины".
