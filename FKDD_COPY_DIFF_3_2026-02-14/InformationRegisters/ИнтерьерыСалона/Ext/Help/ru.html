Регистр сведений "Интерьеры салона"

Назначение

Регистр предназначен для хранения списка возможных интерьеров салона для различных вариантов комплектации автомобилей. Состав интерьеров может быть задан в том числе с указанием возможных цветов кузова автомобиля.

Относится к подсистеме: «Автосалон».

Параметры

Периодичность: Непериодический

Режим записи: Независимый

Измерения

  • Вариант комплектации. Ссылается на справочник «Варианты комплектации». Вариант комплектации, для которой определяется интерьер салона.
  • Тип салона. Ссылается на справочник «Типы салона». Тип салона, допустимый для данного варианта комплектации.
  • Цвет. Ссылается на справочник «Цвета». Цвет автомобиля, для которого допустим данный интерьер салона. 

Способ вызова

Редактируется из формы элемента справочника «Варианты комплектации» в панеле "Опции варианта комплектации".
