Форма записи регистра сведений "Напоминания"

Назначение

Форма предназначена для вывода напоминания. При наступлении события выводится окно напоминания. В нем содержится:

  • Поле Тема события - предназначено для вывода темы напоминания
  • Поле Автор - предназначено для вывода автора напоминания
  • Поле Объект - предназначено вывода присоединенного к напоминанию объекта.
  • Поле Дата начала - предназначено для вывода даты и времени начала события.
  • Поле Дата напоминания - предназначено вывода следующей даты напоминания.
  • Поле Сообщение - предназначено для вывода сообщения.


В качестве реакции на вывод напоминания могут быть нажаты кнопки Завершить, Отложить либо Закрыть. Для периодического напоминания вместо кнопки Закрыть существует Перенести.
Кнопки Завершить и Закрыть, прекращают вывод напоминания безусловно. В случае периодических напоминаний, по кнопке Перенести происходит переход на следующее событие в соответствии с введенным периодом. Если дата следующего события не входит в интервал актуальности, напоминание будет завершено. Дальнейшие действия зависят от установок напоминания. Если при вводе напоминания был установлен флажок удаления, напоминание будет удалено из списка, иначе будет установлена пометка завершения напоминания.
По кнопке Отложить открывается дополнительное меню, содержащее время на которое следует отложить напоминание. По истечении выбранного данном меню срока окно напоминания будет выведено повторно.

Форма имеет следующие элементы:

  • Тема - тема напоминания; 
  • Автор - автор напоминания;
  • Объект - прикрепленный к напоминанию объект;
  • Дата начала - дата запланированного напоминания;
  • Дата напоминания - дата вывода напоминания;
  • Описание - текст напоминания.
