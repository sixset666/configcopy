Регистр сведений "Техническое диагностирование"

Назначение

Предназначен для отражения факта проведения технического диагностирования транспортных средств.

Относится к подсистеме: «Технический осмотр».

Параметры

Периодичность:Позиция регистратора

Режим записи: Подчинен регистратору

Измерения

  • Договор проведения технического осмотра. Измерение является обязательным для заполнения. Ссылается на документ «Договор проведения технического осмотра». Договор на проведение технического диагностирования, согласно которого оно выполнялось.
  • Автомобиль. Измерение является обязательным для заполнения. Ссылается на справочник «Автомобили». Автомобиль подлежащий техническому диагностированию.
  • Номенклатура. Измерение является обязательным для заполнения. Ссылается на справочник «Номенклатура». Услуга по проведению технического диагностирования.
  • Хозяйственная операция. Хозяйственная операция документа технического диагностирования. Ссылается на справочник «Хоз. операции».
  • Требование к транспортному средству. Измерение является обязательным для заполнения. Ссылается на справочник «Требования к транспортным средствам». Требование к транспортному средству, которое проверяется в рамках технического диагностирования. 

Ресурсы

  • Выполнено. Тип значения: «Булево». Признак соответствия транспортного средства требованию.
  • Значение параметра. Тип значения: «Число». Значение параметра требования, полученное в ходе проведения технического диагностирования.
  • Значение минимальное. Тип значения: «Число». Минимальное допустимое значение параметра требования.
  • Значение максимальное. Тип значения: «Число». Максимальное допустимое значение параметра требования.
