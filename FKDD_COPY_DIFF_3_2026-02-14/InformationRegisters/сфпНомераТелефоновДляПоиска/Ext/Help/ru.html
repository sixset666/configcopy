Регистр сведений "Номера телефонов для поиска"

Назначение

Регистр сведений предназначен для хранения данных для поиска контакта по его номеру телефона.

Создается в подсистемах:

«СофтФон Проф»

Измерения:

Объект.

Объект, владелец телефонного номера (Контактные лица клиента, Организации, Клиенты, Физические лица)

Вид.

Вид контактной информации (Виды контактной информации)

Порядковый номер.

Номер строки в табличной части контактной информации (Число)

Ресурсы:

Номер телефона.

Номер телефона (Строка)

Представление.

Представление номера телефона (Строка)

Пользователь.

Пользователь, на которого нужно перевести телефонный звонок (Пользователи)
