Регистр сведений "Заголовки полученных писем"

Назначение

Используется для хранения заголовков полученных писем учётной записи.

Относится к подсистеме: «Рарус».

Параметры

Периодичность: Непериодический.

Режим записи: Независимый.

Измерения

  • Учетная запись. Ссылается на справочник «Учетные записи электронной почты». Учётная запись, для которой хранятся заголовки писем.

Ресурсы

  • Заголовки писем. Тип значения: «Хранилище значений». Содержимое заголовков писем.
