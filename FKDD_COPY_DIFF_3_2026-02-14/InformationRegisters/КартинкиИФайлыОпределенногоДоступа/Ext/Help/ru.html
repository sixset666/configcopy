Регистр сведений "Картинки и файлы"

Назначение

Является универсальным хранилищем дополнительных данных (внешних объектов - файлов, картинок и т.п.).

Относится к подсистеме: «Сервис».

Параметры

Периодичность: Непериодический

Режим записи: Независимый

Измерения

  • Объект. Ссылается на справочники: «Номенклатура», «Контрагенты и контакты», «Подтверждающие документы», «Сотрудники», «Организации». Объект (справочник) для которого необходимо реализовать хранение некоторых данных (например картинок или электронных образов).
  • Идентификатор. Тип значения: «Строка». Идентификационная строка, разделяющая различные данные объекта. Например: Картинка 1, Картинка 2,...

Ресурсы

  • Имя файла. Строковое значение. Исходное имя файла для сохранения.
  • Дата сохранения. Тип значения: «Дата». Дата сохранения данных объекта.
  • Размер файла. Тип значения: «Число». Исходный размер файла.
  • Комментарий. Тип значения: «Строка». Произвольный строковый комментарий к файлу / картинке.
  • Данные. Хранилище данных картинки или файла.
  • Данные во внешнем файле. Признак того, что файл хранится не в базе данных, а в качестве внешнего файла на диске.
  • Картинка. Признак того, что файл является картинкой.

Способ вызова

Вызывается из форм списков и форм элементов справочников и документов, перечисленных в измерении "Объект" по кнопке "Перейти -> Картинки и файлы". Возможен вариант вызова из формы документа по кнопке (или подменю) "Картинки и файлы" на командной панели.
