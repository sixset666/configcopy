Регистр сведений "Ячейки хранения по умолчанию"

Назначение

Регистр сведений "Ячейки хранения по умолчанию" содержит список ячеек, которые используются для автоподстановки в документы для данной номенклатурной позиции в качестве ячейки хранения по умолчанию.

Относится к подсистеме: «Номенклатурные».

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Номенклатура. Ссылается на справочник «Номенклатура». Номенклатура, для которой задается ячейка хранения по умолчанию.
  • Склад компании. Ссылается на справочник «Склады компании». Склад, для которого назначается значение ячейки по умолчанию для номенклатуры.
  • Ячейка. Ссылается на справочник «Ячейки хранения». Ячейка, подставляемая по умолчанию в документы, для связки номенклатуры и склада компании.

Способ вызова

Редактируется из справочника «Номенклатура» на закладке "Ячейки хранения по умолчанию".
