Регистр сведений "Пользователи MicroCat"

Назначение

Регистр сведений "Пользователи MicroCat" предназначен для хранения списка пользователей и их параметров.

Относится к подсистеме: «Автосервис».

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Пользователь. Ссылается на справочник «Пользователи».

Ресурсы

  • Логин. Тип значения: «Строка». Логин пользователя в системе "Альфа-Авто".
  • Пароль. Тип значения: «Строка». Пароль пользователя в системе "Альфа-Авто".

Реквизиты

  • Параметры пользователя. Тип значения: «Хранилище значений». Параметры пользователя.

Способ вызова

Вызывается из обработки «Загрузка из файлов Microcat» из меню "Параметры" на панели формы.
