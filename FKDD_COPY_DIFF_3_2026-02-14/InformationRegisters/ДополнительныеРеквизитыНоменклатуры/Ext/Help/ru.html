Регистр сведений "Дополнительные реквизиты номенклатуры"

Назначение

Регистр предназначен для хранения некоторых дополнительных реквизитов справочника "Номенклатура". Состав дополнительных реквизитов может быть изменен в различных версиях и при сопровождении решения. Например, минимальный хранимый остаток на складе свой для каждого подразделения, изменяемый со временем.

Относится к подсистеме: «Номенклатурные».

Параметры

Периодичность: Секунда

Режим записи: Независимый

Измерения

  • Номенклатура. Ссылается на справочник «Номенклатура». Позиция номенклатуры, для которой определяется дополнительный реквизит.
  • Подразделение компании. Ссылается на справочники «Подразделения компании». Подразделение компании, для которой определяется дополнительный реквизит..
  • Дополнительный реквизит. Ссылается на перечисления «Дополнительные реквизиты номенклатуры». Определяемый реквизит.

Ресурсы

  • Значение. Тип значения: «Число». Значение дополнительного реквизита, которое необходимо сохранить.

Способ вызова

Вызывается из справочника «Номенклатура» по кнопке "Перейти -> Дополнительные реквизиты номенклатуры"
