Регистр сведений "Сведения о сотрудниках"

Назначение

Регистр предназначен для хранения сведений о базовом начислении зарплаты сотрудников, которые могут измениться со временем

Относится к подсистеме: «Структура компании».

Параметры

Периодичность: День

Режим записи: Независимый

Измерения

  • Сотрудник. Реквизит является обязательным для заполнения. Ссылается на справочник: «Сотрудники». Сотрудник.
  • Подразделение. Ссылается на справочник: «Подразделения компании». Подразделение.
  • Должность. Ссылается на справочник: «Должности». Должность.
  • Начисление / Удержание. Ссылается на план видов расчета «Начисления и удержания». Начисление / удержание.

Ресурсы

  • Базовая сумма. Тип значения: «Число». Базовая сумма (в валюте управленческого учета) для начисления / удержания.

Способ вызова

Вызывается из формы элемента справочника сотрудники по кнопке "Перейти -> Сведения о сотрудниках".
