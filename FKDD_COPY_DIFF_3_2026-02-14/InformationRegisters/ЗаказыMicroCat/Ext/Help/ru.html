Регистр сведений "Заказы MicroCat"

Назначение

Регистр сведений "Заказы MicroCat" предназначен для хранения списка выгруженных заказов из системы "MicroCat" и дальнейшей обработки.

Относится к подсистеме: «Автосервис».

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Дата. Тип значения: «Дата». Дата заказа.
  • Номер транзакции. Тип значения: «Строка». Номер транзакции.
  • Пользователь. Тип значения: «Пользователи». Пользователь системы "Альфа-Авто".

Ресурсы

  • Контрагент. Тип значения: «Контрагенты и контакты». Контрагент из заказа.
  • Автомобиль. Тип значения: «Автомобили». Автомобиль контрагента.
  • Заказ. Тип значения: «Хранилище значений». Состав заказа.
  • Дерево работ. Тип значения: «Хранилище значений». Дерево заказа.
