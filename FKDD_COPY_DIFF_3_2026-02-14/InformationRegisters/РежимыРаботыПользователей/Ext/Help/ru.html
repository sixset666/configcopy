Регистр сведений "Режимы работы пользователей"

Назначение

Регистр предназначен для хранения информации о необходимости завершения/разрешения работы пользователей (Режим=1,0), для хранения признака выполнения регламентного задания текущим сеансом (Режим=2), признака необходимости обновления прав и настроек пользователя после выполнения обмена по плану обмена "УдаленныеПодразделения".

Относится к подсистеме: «Рарус».

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Пользователь. Реквизит является обязательным для заполнения. Ссылается на справочник: «Пользователи». 
  • Номер соединения.  Тип значения: «Число». Номер соединения пользователя с информационной базой.
  • Режим. Тип значения: «Число». Режим работы пользователя. 1 - блокировка работы, 2 - выполнение регламентных действий, 0 - нормальный режим (как и отсутствие записи).

Ресурсы

  • Отключение. Тип значения: «Булево». Признак отключения соединения с информационной базой.
  • ОсталосьПредупреждений. Тип значения: «Число». Счетчик количества сообщений пользователю.
  • Конец. Тип значения: «Дата». Дата окончания действия установленного режима. Если не заполнен - дата окончания не определена.
  • Предупреждение. Тип значения: «Строка». Текст сообщения пользователю при его отключении.
  • Начало. Тип значения: «Дата». Начало действия режима. Если не заполнен - режим уже действует.
  • Текст. Тип значения: «Строка». Текст сообщения или дополнительной информации.

Способ вызова

Не предназначен для интерактивного вызова и редактирования.
