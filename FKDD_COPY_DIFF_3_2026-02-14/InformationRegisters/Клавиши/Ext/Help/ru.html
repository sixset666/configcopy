Регистр сведений "Клавиши"

Назначение

В регистре хранятся сведения о действиях, вызываемых по кнопке программируемой клавиатуры.

Относится к подсистеме: «Сервис».

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Клавиатура. Реквизит является обязательным для заполнения. Ссылается на справочники: «Настройки клавиатур». Клавиатура для которой настраиваются клавиши.
  • Код клавиши. Реквизит является обязательным для заполнения. Тип значения: «Строка». Код клавиши, формируется в виде: PPNNN, где: 

     PP - префикс клавиатуры,  NNN - номер клавиши.

Ресурсы

  • Объект. Ссылается на справочники: «Номенклатура». Связанный с клавишей объект (номенклатурная позиция).
  • Действие. Тип значения: «Строка».  Строковой код выполняемого действия .
  • Наименование кнопки. Тип значения: «Строка». Печатаемое наименование клавиши.

Способ вызова

Доступен для просмотра и редактирования: из формы элемента справочника «Настройки клавиатур».
