Регистр сведений "Расширение прав доступа"

Назначение

Используется для расширения прав доступа в разрезе видов документов.

Относится к подсистеме: «Сервис».

Параметры

Периодичность: Непериодический.

Режим записи: Независимый.

Измерения

  • Пользователь. Ссылается на справочник «Пользователи», «Подразделения компании», «Организации». Пользователь, подразделение или организация, для которых устанавливается значение права доступа, отличное от значения по умолчанию.
  • Право. Ссылка на план видов характеристик «Права и настройки». Право, для которого устанавливается расширение права.
  • Объект. Строка. Имя документа, для которого устанавливается право, отличное от значения по умолчанию.

Ресурсы

  • Значение. ПВХ «Права и настройки». Задает значение расширения прав доступа, отличные от значений по умолчанию.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка расширений прав доступа.
