Регистр сведений "Регистрация звонков и контактов"

Назначение

Предназначен для регистрации обращений клиентов специалистами ресепшен или call-центром. Может заполняться вручную или загружаться из табличного документа.

Относится к подсистеме: «Управление взаимоотношениями с клиентами».

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Пользователь, зарегистрировавший звонок/визит.
  • Дата день. Тип значения: «Дата». День входящего звонка или контакта.
  • Дата время. Тип значения: «Дата». Время входящего звонка или контакта.
  • Сотрудник. Реквизит является обязательным для заполнения. Ссылается на справочник «Сотрудники». Сотрудник, связанный с автором.
  • Клиент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенрты» или строка. Клиент, обратившийся к сотруднику.

Ресурсы

  • Организация. Ссылается на справочник «Организации». Организация. 
  • Подразделение. Ссылается на справочник «Подразделения компании». Подразделение. 
  • Модель. Ссылается на справочник «Модели». Модель.
  • Вид контакта. Ссылается на перечисление «Виды событий». Указывает на вид контакта (телефонный звонок/личная встреча).
  • Источник информации. Тип значения: «Число». Ссылается на справочник «Источники информации». Источник информации (Реклама).
  • Вторичный контакт. Тип значения: «Булево». Признак, что данный контакт является вторичным.
  • Юридическое лицо. Тип значения: «Булево». Признак, что клиент является юридическим лицом.
  • Примечание. Тип значения: «Строка». Примечание в произвольной форме к контакту.
  • Участник.  Ссылается на справочник «Сотрудники». Сотрудник, которому предназначено напоминание для зарегистрированного события.
  • Группа ресурсов для хранения номера телефона клиента.
  • Документ событие. Ссылается на документ «Событие».

Способ вызова

Вызывается из меню "Справочники --> Взаимоотношения с клиентами --> Регистрация звонков и контактов" или из обработки «Функциональная панель» - «АРМ Ресепшен».
