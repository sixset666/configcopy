Регистр сведений "Объекты для замены"

Назначение

Регистр сведений "Объекты для замены" предназначен для хранения заменяемых объектов для обработки «Поиск и замена значений».

Относится к подсистеме: «Сервис».

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Идентификатор заменяемого. Реквизит является обязательным для заполнения. Тип значения: «Строка». Реквизит предназначен для хранения уникального идентификатора ссылки заменяемого объекта.

Ресурсы

  • Идентификатор заменителя. Тип значения: «Строка». Реквизит предназначен для хранения уникального идентификатора ссылки объекта заменителя.
  • Автор. Ссылается на справочник «Пользователи». Реквизит предназначен для хранения ссылки на автора записи - пользователя, который создал запись.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.

Колонка "Заменяемый объект" отображает ссылку на справочник, подлежащую замене с помощью обработки «Поиск и замена значений». Колонка "Объект заменителя" отображает ссылку на справочник, на которую будет произведена замена.

Действия подменю "Поиск ссылок" предназначены для поиска ссылок на заменяемый объект или заменитель в других объектах информационной базы. Если такие ссылки будут найдены, внизу появится список найденных ссылок с указанием объекта, где присутствует искомая ссылка и типа данного объекта. При клике по строке открывается форма элемента объекта.

Внизу на панели списка отображены типы заменяемой ссылки и заменителя.

Кнопка "Удалить устаревшие" удаляет строки регистра сведений, в которых заменяемая ссылка или заменитель были удалены в информационной базе.

  • Форма записи. Основная форма для редактирования записи регистра.

Кнопки "Поиск ссылок" предназначены для поиска ссылок на заменяемый объект или заменитель. По аналогии с формой списка внизу появится список найденных ссылок на объект.

Гиперссылки "Владелец: ХХХХ" отображаются в том случае, если заменяемая ссылка или заменитель относятся к подчиненным справочникам. Если заменяемая ссылка или заменитель заполнены, то по клику по данной гиперссылке откроется форма владельца, иначе форма выбора справочника-владельца.  При выборе владельца из формы выбора его представление отобразится над подчиненным элементом. При этом при открытии формы подчиненного элемента будет установлен отбор по владельцу. 

При вызове данной формы из меню "Действия -> Заменить на другой объект" верхней командной панели форм элементов подчиненных справочников гиперссылки "Владелец: ХХХХ" будут сразу заполнены владельцем заменяемой ссылки.

Способ вызова

Вызывается из меню "Действия -> Заменить на другой объект" верхней командной панели форм элементов справочников.

Дополнительная информация

В качестве примера использования данного регистра можно привести следующий.

Предположим, необходимо выполнить замену договора взаморасчетов контрагента "Торговый дом"  "Поставка 555 от 01.01.07" как введенный ошибочно на договор "Поставка 555 от 01.09.06".

  • В форме списка справочника Договоры взаиморасчетов находим "Поставка 555 от 01.01.07" и выбираем пункт меню "Действия -> Заменить на другой объект".
  • В форме записи регистра указываем объект заменителя "Поставка 555 от 01.09.06".
  • При необходимости можем открыть форма элемента контрагента "Торговый дом"
  • Нажимаем кнопку "Поиск ссылок" справа от заменяемого объекта. Если ссылки будут найдены, внизу появится список. Можно открыть любую из найденных ссылок для сверок.
  • Кликом по кнопке "Поиск и замена" можем открыть форму обработки «Поиск и замена значений» с уже заполненными значениями реквизитом. Далее можем выполнить замену значений.
