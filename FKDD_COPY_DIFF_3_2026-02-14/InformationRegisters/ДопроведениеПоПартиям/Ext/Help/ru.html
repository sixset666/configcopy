Регистр сведений "Допроведение по партиям"

Назначение

Регистр предназначен для хранения списка документов, которые необходимо допровести по партиям.

Относится к подсистеме: «Рарус».

Параметры

Периодичность: Позиция регистратора.

Режим записи: Подчинение регистратору.

Измерения

  • Подразделение компании. Ссылается на справочник «ПодразделенияКомпании». Подразделение компании, для которого проводится допроведение по партиям.
