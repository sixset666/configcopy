Регистр сведений "Границы последовательности партий"

Назначение

Используется для восстановления последовательности ввода и проведения документов. Предназначен для хранения информации о нарушениях ввода товарных документов в разрезе складов компании. Применяется в обработке «Восстановление последовательностей».

Относится к подсистеме: «Поступление ТМЦ», «Отгрузка ТМЦ».

Параметры

Периодичность: Непериодический.

Режим записи: Независимый.

Измерения

  • Склад компании. Ссылается на справочник «Склады (места хранения) компании». Склад компании, ввод поступления/расхода товаров по которому нарушена.

Ресурсы

  • Момент времени. Тип значения: «Дата и время». Момент времени, на который нарушена последовательность ввода документов.
  • Документ ссылка. Тип значения: «ДокументСсылка». Документ, нарушивший последовательность ввода документов.

Формы

  • Форма списка. Основная форма списка и редактирования. Используется для отображения списка нарушений ввода документов.

Дополнительная информация

Регистр сведений "Границы последовательности партий" является служебным и не предназначен для ручного редактирования. Сведения из данного регистра используются для восстановления последовательности проведения товарных документов.
