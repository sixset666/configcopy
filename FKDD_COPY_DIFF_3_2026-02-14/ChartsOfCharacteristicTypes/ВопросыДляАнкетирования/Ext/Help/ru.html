План видов характеристик "Вопросы для анкетирования"

Назначение

 

Вопросы для анкетирования представляют собой фактически базу знаний - вопросы и варианты ответов на них, используемые при анкетировании соискателей работы или опросах работников предприятия.

Для вопроса можно указать его весовой коэффициент, который в дальнейшем можно использовать для оценки ответов.

Если вопрос может иметь несколько известных заранее ответов, тогда эти ответы можно вписать в таблицу типовых ответов. Каждому такому ответу можно присвоить некоторую оценку, а также указать, что к этому ответу требуется развернутое пояснение опрашиваемого лица. 


Относится к подсистеме: «Учет взаимоотношений с клиентами».

Параметры

Длина кода: 9.

Длина наименования: 100.

Иерархический.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка и выбора элементов.
  • Форма элемента. Форма для ввода или редактирования вопроса как элемента плана видов характеристик "Вопросы для анкетирования". В форме имеются следующие реквизиты:
    • Наименование - наименование вопроса;
    • Код - код вопроса, недоступен для редактирования;
    • Формулировка - полное описание вопроса;
    • Группа - принадлежность данного вопроса к группе вопросов;
    • Ответ в виде - выбор вида ответа на вопрос из предложенного списка (да/нет, даты, строки и другие).

В зависимости от выбора вида вопроса данная форма дополняется реквизитами (в виде полей или таблиц) для определения формата ответа под заданный вид ответа. 

  • Форма группы. Форма для работы с группами вопросов. Содержит следующие реквизиты:
    • Группа -  заполняется тогда, когда данная группа находится в группе;
    • Наименование - наименование данной группы;
    • Код - код данной группы, недоступен для редактирования.

Способ вызова

Вызывается из элемента справочника «Типовые анкеты».
