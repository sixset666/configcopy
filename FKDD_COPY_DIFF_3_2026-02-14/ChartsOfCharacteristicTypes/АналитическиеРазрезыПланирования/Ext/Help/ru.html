План видов характеристик "Аналитические разрезы планирования"

Назначение

 

Содержит описание всех возможных аналитических разрезов планов и фактов по образу и подобию видов субконто. Имеет подчиненный справочник для создания дополнительных разрезов планирования. Не имеет реквизитов. Каждый элемент указывает на несоставной ссылочный тип, применяемый для типизации реквизитов документов подсистемы планирования.

Относится к подсистеме: «Статьи и свойства».

Типы возможных значений: Все ссылки, которые могут быть использованы для планирования.

Параметры

Длина кода: 5.

Длина наименования: 25.

Не иерархический.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка и выбора элементов.

Способ вызова

Вызывается из элемента справочника «ВидыПлановКомпании», из меню "Планы видов характеристик --> Виды планов компании".

Дополнительная информация

Используется для фильтрации состава аналитических объектов в обработках заполнения.
