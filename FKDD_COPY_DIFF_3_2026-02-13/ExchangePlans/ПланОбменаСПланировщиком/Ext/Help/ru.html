План обмена "План обмена с планировщиком"

Назначение

План обмена предназначен для осуществления взаимодействия (обмена данными) решений Альфа-Авто с системой Time Planner.

Относится к подсистеме: «Обмен данными».

Параметры

Длина кода: 8.

Длина наименования: 100.

Настройка    

  • Вызовете план обмена Сервис->Все операции..->План обмена->План обмена с планировщиком .

  • Измените план обмена по умолчанию установив значения реквизитов Код = "ОС", Наименование = "ОС".

  • Добавьте новый план обмена со значениями реквизитов Код = "TP", Наименование = "TP".

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Сервис->Все операции..->План обмена".
