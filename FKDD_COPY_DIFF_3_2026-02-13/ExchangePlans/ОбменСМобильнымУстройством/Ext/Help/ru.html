План обмена "Обмен с мобильным устройством"

Назначение

Предназначен для обмена данными между информационными базами ААА 5 и Мобильным мастером-приемщиком. Обмен производится данными о свойствах, характеристиках, стоимости, повреждениях и т.д.

Относится к подсистеме: «Сервис».

Параметры

Длина кода: 3.

Длина наименования: 100.

Реквизиты

  • ID устройства Тип значения: «Строка». imei код и наименование устройства.
  • Принтер Тип значения: «Строка». Имя принтера в системе на который отправляются документы для этого узла обмена.
  • Тип цен Тип значения: Справочник Типы цен. Тип цен для расчета доппродаж.
  • Цех Тип значения: Справочник Цеха. Цех для расчета доппродаж.
  • Вид ремонта Тип значения: Справочник Виды ремонта. Вид ремонта для расчета доппродаж. 
  • Подразделения компании Тип значения: «Список значений». Спиоск подразделений документы которых будут выгружаться на мобильное устройство через этот план обмена.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка узлов.
  • Форма узла. Форма, используемая для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню  "Справочники--> Структура компании --> Обмен с мобильным устройством".
