Редактирование списка значений

Для поля ввода, предназначенного для ввода данных, имеющих тип "СписокЗначений", по нажатии кнопки выбора открывается диалог.

Диалог имеет табличное поле, которое предназначено для отображения списка значений и редактирования этого списка.

Данные могут быть предварительно подготовлены в модуле формы, а также могут вводиться с помощью данного диалога.
