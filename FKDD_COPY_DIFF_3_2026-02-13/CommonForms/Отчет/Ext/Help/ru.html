Общая форма "Отчет"

Назначение

Данная общая форма используется для представления большинства отчетов конфигурации и управления частью параметров данных отчетов.

Способ вызова

Форма вызывается при формировании отчетов автоматически.

Виды отчета:

  •  Сводная таблица. 

         Отчет представляется в виде сводной таблицы, пользователь может изменить состав отображаемых измерений и показателей без переформирования отчета.

  •  Табличный документ.

         Отчет представляется в виде стандартного табличного документа.

  •  Сводная диаграмма.

             Отчет представляется в виде диаграммы, пользователь может изменить состав отображаемых измерений и показателей без переформирования отчета.

  •  Диаграмма. 

             Отчет представляется в виде диаграммы.

    Настройка табличного документа:

  •  Кнопка Действия. Разворачивает список настроек табличного документа со следующими значениями:

      • Обновить. Обновляет сформированный табличный документ, учитывая настройки текущего отчета, такие как Период, Отбор и т.д.

      • Печать. Выводит текущий табличный документ на принтер.

      • Сохранить как веб-страницу. Создает "html" страничку с текущим табличным документом

      • Сохранить как PDF-файл. Создает "PDF"-файл с текущим табличным документом

      • Открыть в Excel. Создает "Excel" страничку, с текущим табличным документом

      • Отправить по e-mail. Позволяет отправить текущий табличный документ по электронной почте.

      • Добавить в избранное. Разворачивает список со следующими возможными вариантами: 

        • Форму отчета. Позволяет добавить форму отчета в список избранного "панели АРМ".

        • Печатную форму. Позволяет добавить печатную форму отчета в список избранного "панели АРМ".

      • Уровни группировок - Разворачивает список вариантов просмотра группировок до заданного уровня

  •  Обновить. Обновляет сформированный табличный документ, учитывая измененные в текущей форме настройки табличного документа, такие как Период, Отбор и т.д.
  •  Печать. Выводит табличный документ на принтер
  •  Настройка. При помощи этой кнопки в любой момент можно перейти к текущей форме настройки отчета.
  •  Отбор. Эта кнопка показывает/скрывает панель быстрого отбора. Быстрый отбор разрешает настраивать первых три отбора формы настройки отчета,
    это следует учитывать, если есть необходимость использовать определенные фильтры не обращаясь, каждый раз, к форме настроек, если строки отбора формы настройки отсутствуют, то кнопка отбор будет не доступна.
  •  Сводная диаграмма. Формирует данный отчет в виде сводной диаграммы.
    Эта возможность поддерживается не во всех отчетах, в веду специфики каждого отчета, для отчета не поддерживающего
    этот режим кнопка "Сводная диаграмма" будет недоступна
  •  Диаграмма. Формирует данный отчет в виде диаграммы.
    Эта возможность поддерживается не во всех отчетах, ввиду специфики каждого отчета, для отчета не поддерживающего
    этот режим кнопка "Диаграмма" будет недоступна
  •  Сводная таблица. Формирует данный отчет в виде сводной таблицы.
    Эта возможность поддерживается не во всех отчетах, ввиду специфики каждого отчета, для отчета не поддерживающего
    этот режим кнопка "Сводная таблица" будет недоступна
  •  Стиль. Разворачивает список с вариантами стилей оформления табличного документа и флагом "С использованием автоотступа ",
     который включает/отключает Г-образную группировку уровней и включает автоотступ (отступ на один символ больше, чем у предыдущего уровня)


    Дополнительная информация

    Отчеты любого вида изменить основные параметры отчета  как (дата, период) и управлять стилем отображения отчета. При помощи кнопки  "Настройка" в любой момент можно перейти к текущей форме настройки отчета.
