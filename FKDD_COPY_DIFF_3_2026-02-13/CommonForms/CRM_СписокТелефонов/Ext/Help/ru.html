Форма списка телефонов обработки "СофтФон"

Назначение

Форма предназначена для быстрого поиска контактной информации и набора телефонного номера. Открытие формы происходит при нажатии на кнопку "Набрать" или автоматически при переводе звонка на другой номер. Так же можно выполнить набор произвольного номера телефона, не сохраненного в базе данных.

Окно выбора телефонного номера содержит закладки (отображение и порядок закладок зависит от модуля коммуникатора):

  • Выбор телефона - Служит для выбора телефонов из справочников программы. В списках показываются только те контакты, у которых есть телефонные номера в контактной информации. Контактная информация сгруппирована в три списка:
    • Контрагенты - список содержит контрагентов и подчиненных ему контактных лиц
    • Контактные лица - список содержит всех контактных лиц
    • Сотрудники - список содержит пользователей программы и сотрудников компании. Напротив каждого из сотрудников, есть графа, содержащая картинку телефонного аппарата. Цвет телефонного аппарата показывает оператору, занят или свободен внутренний номер сотрудника. Так же при двойном щелчке мыши на графе с телефоном можно позвонить по свободному номеру.
  • Записная книга - Служит для формирования индивидуального для каждого пользователя списка самых популярных контактов. Контакты выбираются из справочников программы.
  • Звонки - Содержит информацию об истории звонков. Звонки сгруппированы следующим образом:
    • Пропущенные
    • Набранные
    • Принятые

ВНИМАНИЕ!!! Информация об истории звонков берется из внешней таблицы MS SQL Server-а. Если таблица не используется, т.е. не настроена на сервере "СофтФона, Проф", данная закладка будет недоступна, и получить информацию о звонках будет невозможно.

Для "СофтФона, Лайт" информация берется из регистра сведений "CRM Телефонные звонки и номера"

  • SMS Сообщения. Содержит список sms сообщений.
    • Входящие
    • Исходящие
  • Факс Сообщения. Содержит список факс сообщений.
    • Входящие
    • Исходящие

Для быстрого поиска информации существует несколько способов:

  • по первым символам  - при наборе первых символов объекта поиска курсор позиционируется на строке содержащей эти символы (в начале строки).
  • через поле "Поиск" - поиск осуществляется по вхождению символов введенных в поле "Поиск". Курсор позиционируется на первом найденном вхождении. Для продолжения поиска необходимо нажать кнопки со стрелочками: поиск вперёд и поиск назад.
  • через поле "Отбор" - формирует список объектов, которые содержат символы, введенные в поле "Отбор".
