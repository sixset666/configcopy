Отчет "Показатели работы менеджеров"

Назначение

Этот отчет служит для анализа эффективности работы менеджеров автосалона.

Относится к подсистеме: «Автосалон».

Способ вызова

Вызывается из меню "Отчеты --> Автосалон --> Показатели работы менеджеров"

Показатели отчета

  • Карточки клиентов - Количество элементов справочника “Контрагенты”, созданных менеджером за период формирования отчета.
  • Входящий трафик, общий - Количество записей в регистре звонков и контактов с указанием менеджера в качестве сотрудника за период формирования отчета.
  • Входящий трафик, звонки - Количество записей в регистре звонков и контактов с указанием менеджера в качестве сотрудника и видом контакта “Телефонный звонок” за период формирования отчета.
  • Входящий трафик, визиты - Количество записей в регистре звонков и контактов с указанием менеджера в качестве сотрудника и видом контакта “Личная встреча” за период формирования отчета.
  • Входящие звонки - Количество документов “Событие” с видом “Телефонный звонок”, типом “Входящее” и состоянием “Завершено”, по менеджеру за период формирования отчета.
  • Входящие визиты - Количество документов “Событие” с видом “Личная встреча”, типом “Входящее” и состоянием “Завершено”, по менеджеру за период формирования отчета.
  • Исходящие звонки - Аналогично входящим звонкам, но тип событий “Исходящее”.
  • Исходящие визиты - Аналогично входящим визитам, но тип событий “Исходящее”.
  • Рабочие листы - Количество созданных рабочих листов по менеджеру за период формирования отчета.
  • Тест-драйвы - Количество документов “Тест-драйв” в состоянии “Завершено” по менеджеру за период формирования отчета.
  • Контракты - Количество документов “Заказ на автомобиль” по которым (или по документам “Счет на оплату за автомобиль” введенным на основании заказа) получена предоплата от клиента, по менеджеру за период формирования отчета. Предоплатой считается первое поступление денежных средств от клиента по заказу на автомобиль. Датой предоплаты считается дата первого поступления денежных средств от клиента по заказу на автомобиль.
  • Оформления (Выписки) - Количество проведенных документов “Реализация автомобилей” по менеджеру за период формирования отчета. Количество проданных автомобилей определяется по числу строк в документе реализации.
  • Выдачи - Количество документов “Событие” с видом “Выдача автомобиля” и состоянием “Завершено” по менеджеру за период формирования отчета. Учитываются только те выдачи, у которых заполнено поле “Фактическая дата выдачи” и значение этого поля попадает в период формирования отчета.
  • Несостоявшиеся (Сделки) - Количество документов “Рабочий лист”, которые достигли последнего по порядку статуса, но по ним так и не был создан документ “Заказ на автомобиль”, по менеджеру за период формирования отчета.
  • Отказы - Количество документов “Заказ на автомобиль”, на основании которых введен документ “Отмена заказа на автомобиль”, по менеджеру за период формирования отчета.
  • Trade-in (Трейд-ин) - Количество документов “Поступление автомобилей”, которыми был зарегистрирован выкуп клиентского автомобиля по документу “Заказ на автомобиль”, по менеджеру за период формирования отчета. Учитывается число строк в документе поступления автомобилей по трейд-ин.
