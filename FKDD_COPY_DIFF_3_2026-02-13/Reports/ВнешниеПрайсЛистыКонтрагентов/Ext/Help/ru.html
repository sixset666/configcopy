Отчет "Внешние прайс-листы контрагентов"

Назначение

Отчет служит для получения упрощенной печатной формы прайс-листа (список товаров с их ценами), по контрагентам из внешних источников. Внешние источники могут представлять собой различные форматы: базы данных, обращение к которым может осуществляться посредством ODBC, файлы в формате Microsoft Excel, Open office calc, текстовый файл.

Относится к подсистеме: «Поступление ТМЦ».

Способ вызова

Вызывается из меню "Отчеты-->Поступление ТМЦ-->Внешние прайс-листы контрагентов"

Дополнительная информация

Отчет может формироваться только в виде табличного документа.

При помощи включения флага "Выбирать валюту отчета" можно получить цены прайс-листов для выбранной валюты на заданную дату, если же этот флаг будет отключен, то цены будут выведены в валюте указанной в самих прайс-листах и появится дополнительный показатель "Валюта" для отображения наименования валюты текущего прайс-листа.
