Отчет "План-фактный анализ бюджета закупок"

Назначение

Этот отчет служит для сравнения запланированных и фактических данных о закупках в разрезе номенклатуры, либо в разрезе категорий номенклатуры.

 

Относится к подсистеме: «Планирование».

Формы

  • План-фактный анализ.  Позволяет сформировать отчет "План-фактный анализ закупок" с возможностью гибкой настройки группировок строк, показателей, дополнительных полей и т.д.  

Назначение некоторых реквизитов формы

  • Аналитика отчета. По номенклатуре, либо по категориям номенклатуры.
  • Сценарий планирования. Обязательный для заполнения. Справочник «Сценария планирования». Определяет сценария планирования, по которому будет производиться сравнение данных.
  • Разворачивать по периодам. Булево. Истина - данные разворачиваются по периодам (соответствующим периодичности выбранного сценария планирования).
  • Выводить незапланированные данные. Булево. Управляет отображением в отчете данных по номенклатуре\категориям номенклатуры,  по которым есть фактические данные закупок, но нет запланированных данных.

Способ вызова

Вызывается из меню "Отчеты--> Бюджетирование -->План фактный анализ бюджета закупок"

Дополнительная информация

Отчет может формироваться только в виде табличного документа
