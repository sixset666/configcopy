План видов характеристик "Типы сделок"

Назначение

Предназначен для описания видов документов, применимых в качестве сделок взаиморасчетов. Используется в качестве типизатора измерения "Сделка" регистра «Взаиморасчеты компании» и для выбора вида сделки в некоторых документах. Все возможные виды сделок введены в виде предопределенных элементов.

Относится к подсистеме: «Сервис».

Типы возможных значений: Ссылки на все документы, которые могут выступать в роли сделок.

Параметры

Длина кода: 5.

Длина наименования: 50.

Не иерархический.

Реквизиты

  • Запретить переплату по сделкам. Тип значения: «Булево». Если флажок установлен, то в документе данного типа запрещено превышать сумму сделки.
  • Дебетовые. Тип значения: «Булево». Признак того, что данный тип может закрывать сделки, приводящие к увеличению долга контрагента. Используется для отбора разрешенных типов сделок в некоторых документах.
  • Кредитовые. Тип значения: «Булево». Признак того, что данный тип может закрывать сделки, приводящие к уменьшение долга контрагента. Используется для отбора разрешенных типов сделок в некоторых документах.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Не вызывается и не редактируется интерактивно.

Дополнительная информация

Введение этого объекта позволяет упростить поддержку типового решения в случае изменения состава взаиморасчетных документов. В этом случае следует изменять тип значения характеристик а не измерения "Сделка".
