План видов характеристик "Свойства объектов"

Назначение

Предназначен для описания видов дополнительных свойств объектов базы данных. Механизм свойств позволяет произвольно расширять аналитические реквизиты объектов нормативно-справочной подсистемы и документов. В процессе работы с типовой конфигурацией пользователь может добавлять свои свойства, имеющие различные типы значений и функциональное назначение.

Относится к подсистеме: «Статьи и свойства».

Типы возможных значений: «Булево», «Число», «Строка», «Дата», «Валюты», «Номенклатура», «Контрагенты», «Пользователи», «Классификатор стран мира», «Значения свойств», «Классификатор единиц измерения», «Сотрудники», «Склады (места хранения) компании», «Подразделения компании», «Организации».

Параметры

Длина кода: 5.

Длина наименования: 50.

Не иерархический.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма выбора. Основная форма выбора. Используется для отображения списка выбора элементов.

Способ вызова

Вызывается из меню "Справочники --> Статьи и свойства --> Свойства объектов".

Дополнительная информация

Назначение конкретных видов свойств объектов разного вида хранятся в плане видов характеристик «Назначения свойств объектов». Для каждого объекта могут быть назначены значения видов свойств различного типа. Значения свойств объектов заносятся с помощью обработки «Значения свойств объекта».
