План видов характеристик "Назначения свойств объектов"

Назначение

Предназначен для привязки необходимых видов дополнительных свойств к различным объектам информационной базы. Предопределенные элементы этого плана видов характеристик содержат список ВСЕХ объектов конфигурации, которые могут быть расширены дополнительными свойствами. Для отдельных групп справочников и характеристик по типам номенклатуры можно создать дополнительные объекты назначения свойств, для задания уникальной конфигурации свойств. При этом создать новое назначение можно только путем ввода на основании существующего предопределенного назначения, определяющего тип и становящегося базовым по отношению к создаваемому.

Относится к подсистеме: «Статьи и свойства».

Типы возможных значений: «Счет на оплату», «Расходный кассовый ордер», «Номенклатура», «Контрагенты», «Договоры взаиморасчетов», «Подтверждающие документы», «Реализация товаров», «Сотрудники», «Поступление товаров», «Склады (места хранения) компании», «Типы номенклатуры», «Подразделения компании», «Организации», «Приходный кассовый ордер» и т.д. (все объекты, поддерживающие расширение свойствами).

Параметры

Длина кода: 7.

Длина наименования: 50.

Иерархический.

Вводится на основании: «Назначения свойств объектов».

Объекты, вводимые на основании: «Назначения свойств объектов».

Реквизиты

  • Базовый объект. Ссылается на план видов характеристик «Назначения свойств объектов». Ссылка на предопределенное назначение, определяющее набор свойств вида объекта. Используется для свойств конкретных объектов.
  • Объект. Ссылается на документы: «Счет на оплату», «Расходный кассовый ордер», «Реализация товаров», «Поступление товаров»,  «Приходный кассовый ордер», на справочники: «Номенклатура», «Контрагенты», «Договоры взаиморасчетов», «Подтверждающие документы»,  «Сотрудники»,  «Склады (места хранения) компании», «Типы номенклатуры», «Подразделения компании», «Организации». Объект, для которого назначаются виды свойств. Если пусто - то для всех объектов данного вида.

Табличная часть «Виды свойств»

  • Свойство. Реквизит является уникальным и обязательным для заполнения. Ссылается на план видов характеристик «Свойства объектов». Вид свойства, назначаемый объекту.

Формы

  • Форма списка. Основная форма выбора группы. Используется для выбора групп.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма группы. Основная форма группы. Используется для заполнения обязательных реквизитов группы.

Способ вызова

Вызывается из меню "Справочники --> Статьи и свойства --> Назначения свойств объектов".
