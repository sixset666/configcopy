План видов характеристик "Права и настройки"

Назначение

План видов характеристик предназначен для хранения списка прав и настроек для всей конфигурации. Т.е. в данном плане содержатся предопределенные виды прав доступа ко всем объектам конфигурации, дополнительные настройки, используемые при автоматическом заполнении новых элементов справочников, документов и других объектов. Настраивать права можно не только на конкретного пользователя, но и на подразделение и организацию на закладке "Права и настройки" панели  «АРМ Администратора».

Относится к подсистеме: «Классификаторы».

Типы возможных значений: «Булево», «Число», «Дата», «Валюты», «Варианты ответов», «Режим префиксации», «Номенклатура», «Ставки НДС», «Контрагенты», «Режимы списания при возврате товаров», «Скидки способы выбора», «Цеха», «Режимы регламентных операций», «Типы цен», «Свойства объектов», «Кассы компании», «Склады (места хранения) компании», «Подразделения компании», «Организации».

Параметры

Длина кода: 5.

Длина наименования: 70.

Иерархический.

Реквизиты

  • Назначение. Реквизит является обязательным для заполнения. Ссылается на перечисление «Назначение прав и настроек». Признак доступности права/настройки для компании/организации/подразделения/пользователя.
  • Это настройка. Тип значения: «Булево». Настройки могут изменяться пользователем, права изменяются только администратором.
  • Значение по умолчанию. Тип значения: «Булево», «Число», «Дата», ссылка на справочники: «Валюты», «Номенклатура», «Ставки НДС», «Контрагенты», «Цеха», «Типы цен», «Кассы компании», «Склады (места хранения) компании», «Подразделения компании», «Организации»; ссылка на перечисления: «Варианты ответов», «Режимы списания при возврате товаров», «Режимы регламентных операций», «Скидки способы выбора», «Режим префиксации»; ссылка на план видов характеристик «Свойства объектов». Значение права / настройки по умолчанию.

Формы

  • Форма списка. Основная форма просмотра и выбора элементов из списка.

Печатные формы

  • Настройки по умолчанию. Не предназначена для печати - используется для начального заполнения реквизитов плана при первом запуске и обновлении конфигурации.

Способ вызова

Вызывается из элемента справочника «Пользователи», «Организации»,«Подразделение»путем нажатия на кнопку "Права"(«АРМ Администратора»открывается автоматически).
