Регистр сведений "Обмен с AudaPad Web таблица соответствий"

Назначение

Регистр сведений "Обмен с AudaPad Web таблица соответствий" предназначен для хранения, заданных пользователем соответствий, между деталями и работами из AudaPad Web и информационной базы ААА 5.

Относится к подсистеме: «Обмен данными».

Параметры

Периодичность:Периодический

Режим записи: Независимый

Измерения

  • Артикул. Реквизит является обязательным для заполнения. Тип значения: «Строка».  Реквизит предназначен для хранения артикула из AudaPad Web.
  • Наименование. Реквизит является обязательным для заполнения. Тип значения: «Строка». Реквизит предназначен для хранения наименования из AudaPad Web.
  • Наименование полное. Тип значения: «Строка». Реквизит предназначен для хранения полного наименования из AudaPad Web.
  • Модель. Тип значения: «Модели автомобилей». Реквизит предназначен для хранения модели автомобиля для которой устанавливается соответствие.

Ресурсы

  • Номенклатура. Тип значения: «Хранилище значения». Реквизит содержит данные калькуляции, загруженные из Audatex.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма записи. Основная форма для редактирования записи регистра.

Способ вызова

Вызывается из меню "Сервис => Все операции -> Регистр сведений -> Обмен с AudaPad Web таблица соответствий".
