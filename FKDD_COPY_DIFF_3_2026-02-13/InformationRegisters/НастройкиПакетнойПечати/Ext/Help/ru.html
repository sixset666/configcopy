Регистр сведений "Настройки пакетной печати"

Назначение

Используется для хранения пользовательских настроек пакетной печати.

Относится к подсистеме: «Сервис».

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Подразделение. Ссылается на справочник «Подразделения компании».  Подразделение компании, для которого используется настройка.
  • Сотрудник.  Ссылается на справочник «Сотрудники». Сотрудник, для которого используется настройка.
  • ТипДокумента. Тип значения: «Строка». Вид документа, для которого предназначена настройка. 
  • ИмяМакета. Тип значения: «Строка». Макет, для которого предназначена настройка.
  • ИмяНастройки. Тип значения: «Строка».  Имя настройки (набора печатных форм).

Ресурсы

  • КоличествоКопий. Тип значения: «Число». Число копий макета для печати.

Способ вызова

Не предназначен для интерактивного вызова и редактирования.
