Форма загрузки адресного классификатора регистра сведений "Адресный классификатор"

Назначение

Форма загрузки адресного классификатора регистра сведений "Адресный классификатор" предназначена для загрузки адресных элементов из файлов с классификаторами адресов в регистр сведений "Адресный классификатор".

Закладка "Настройки"

Поля ввода группы "Настройка доступа к данным" предназначены для указания пути к файлам с классификаторами адресов.

  • Файл с адресным классификатором. Обязательно для заполнения.
  • Файл с классификатором улиц. Обязательно для заполнения.
  • Файл с классификатором домов. Заполняется, если формат файлов с классификаторами 2003 года(информация о домах в отдельной таблице).
  • Файл с классификатором сокращений. Обязательно для заполнения.

Поля ввода группы "Настройка загрузки данных" предназначены для указания формата файлов с классификаторами адресов.

  • Кодировка поставляемых таблиц. DOS(866) или Windows(1251).
  • Формат поставляемых таблиц. 2002 года (информация о домах и улицах совмещена) или 2003 года (информация о домах в отдельной таблице).

Закладка "Фильтр по регионам"

На закладке "Фильтр по регионам" находится список регионов. Список регионов необходимо заполнить перед началом загрузки адресных классификаторов из файлов, нажав кнопку "Заполнить регионы".

Кнопки

  • Очистить классификатор. Удаляет записи из регистра "Адресный классификатор". Перед очисткой необходимо выбрать в списке регионов регионы, информацию о которых необходимо удалить.
  • Загрузить регионы. Загружает список регионов.
  • Загрузить. Загружает данные в регистр "Адресный классификатор". Перед загрузкой необходимо выполнить настройки на закладке "Настройки", а так же выбрать в списке регионов регионы, информацию о которых необходимо загрузить.
