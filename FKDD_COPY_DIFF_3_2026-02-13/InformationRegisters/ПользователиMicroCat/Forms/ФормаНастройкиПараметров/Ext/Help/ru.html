Общая форма "Настройка параметров"

Назначение

Форма предназначена для единого заполнения констант конфигурации. При первом запуске форма вызывается автоматически, предоставляя пользователю определить один раз принцип поведения всей системы в целом. После начала работы с системой дальнейшее изменение значений некоторых констант может быть запрещено для поддержки целостности учета.

Значения констант

  • Номер релиза конфигурации. Текущий номер релиза конфигурации, не подлежит изменению пользователем. Заполняется при старте системы при условии, что он отличен от внутреннего релиза конфигурации. 
  • Общий каталог. Основной каталог, который содержит внешние компоненту, настройки системы защиты и управления оборудованием. Если компонента не будет найдена в этом каталоге, или значение каталога будет незаполнено, система потребует выполнить настройку системы защиты, и не будет корректно работать до завершения настройки.
  • Валюта управленческого учета компании. Ссылается на справочник «Валюты». Валюта, в которой будет строиться управленчеcкий баланс, и оцениваться вся деятельность компании. Валюта устанавливается только один раз при первом запуске конфигурации. Менять управленческую валюту в дальнейшем (при наличии проведенных документов) запрещается.
  • Валюта регламентированного учета организаций. Ссылается на справочник «Валюты». Национальная валюта страны, в которой будет происходить выгрузка в бухгалтерию и проводиться выписка большинства первичных документов. Должна иметь единичный курс. Валюта устанавливается только один раз при первом запуске конфигурации. Менять регламентированную валюту в дальнейшем не рекомендуется.
  • Стратегия списания партий товаров по датам. Ссылается на перечисление "Стратегия списания партий товаров по датам". Определяет последовательность списания партий товаров со склада по времени. Стратегию также можно определить для каждого склада в отдельности.
  • Стратегия списания партий товаров по статусам. Ссылается на перечисление "Стратегия списания партий товаров по статусам". Определяет приоритет собственных и комиссионных товаров при списании партий со склада.
  • Документ партии товаров среднего. Задает партию товаров для партии товаров отрицательных остатков.
  • Основная единица измерения массы. Ссылается на справочник «Классификатор единиц измерения». Обычно килограмм. Используется для расчета массовых показателей в товарных документах.
  • Основная единица измерения количества. Ссылается на справочник «Классификатор единиц измерения». Обычно штука. Используется для помощи при вводе новых товаров.
  • Основная единица размерности ордерного склада. Ссылается на справочник «Классификатор единиц измерения». Обычно метр. Используется для автоматической подстановки в качестве единицы размерности ячеистого склада и его ячеек.
  • Префикс штучного ШК. Префикс штучного штрих-кода. Т.е. это число, с которого будет начинаться штучный штрих-код. По умолчанию 20. 
  • Префикс весового ШК. Префикс весового штрих-кода. Т.е. это число, с которого будет начинаться весовой штрих-код. По умолчанию 22. 
  • Валюта накопительных сумм. Ссылается на справочник «Валюты». Валюта, в которой будет накапливаться суммы для расчета накопительных скидок.
  • Период накопительных скидок. Ссылается на перечисление "Периодичность анализа накопительных скидок". Период, за который будут накапливаться скидки.

Способ вызова

Вызывается из меню "Сервис --> Настройка параметров"
