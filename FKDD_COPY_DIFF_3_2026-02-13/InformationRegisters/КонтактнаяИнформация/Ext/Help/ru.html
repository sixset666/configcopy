Регистр сведений "Контактная информация"

Назначение

Регистр сведений предназначен для хранения контактной информации (телефон, адрес, адрес эл.почты и пр.) по объектам информационной базы «Контрагенты и контакты», «Организации», «Сотрудники».

Относится к подсистеме: «Идентификационные».

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Объект. Является обязательным для заполнения. Ссылается на справочники «Контрагенты и контакты», «Организации», «Сотрудники». Объект-владелец контактной информации.
  • Тип. Является обязательным для заполнения. Ссылается на перечисление «Типы контактной информации». Тип контактной информации.
  • Вид. Является обязательным для заполнения. Ссылается на справочник «Виды контактной информации». Вид контактной информации.

Ресурсы

  • Представление. Тип значения: «Строка». Строковое представление контактной информации.
  • Поле1. Тип значения: «Строка». Вспомогательное поле: "Код страны" или "Индекс".
  • Поле2. Тип значения: «Строка». Вспомогательное поле: "Код города" или "Регион".
  • Поле3. Тип значения: «Строка». Вспомогательное поле: "Номер телефона" или "Район".
  • Поле4. Тип значения: «Строка». Вспомогательное поле: "Внутренний (или добавочный) номер" или "Город".
  • Поле5. Тип значения: «Строка». Вспомогательное поле: "Населенный пункт".
  • Поле6. Тип значения: «Строка». Вспомогательное поле: "Улица".
  • Поле7. Тип значения: «Строка». Вспомогательное поле: "Дом".
  • Поле8. Тип значения: «Строка». Вспомогательное поле: "Корпус".
  • Поле9. Тип значения: «Строка». Вспомогательное поле: "Квартира".
  • Поле10. Тип значения: «Строка». Вспомогательное поле: "Подъезд".
  • Поле11. Тип значения: «Строка». Вспомогательное поле: "Этаж".
  • Поле12. Тип значения: «Строка». Вспомогательное поле: "Код двери/домофон".
  • Значение по умолчанию. Тип значения: «Булево». Значение по умолчанию.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий.

Реквизиты

  • Краткое представление. Тип значения: «Булево». Признак того, что данная запись представляет собой краткое представление контактной информации.

Формы

  • Форма списка. Служит для просмотра и редактирования контактной информации. В левой части расположено поле списка элементов справочникам, для которых можно указать контактную информацию. При помощи поля выбора можно выбрать тип справочника, элементы которого будут выводиться. В правой части находится собственно список контактной информации, принадлежащий выбранному элементу-владельцу. Можно установить отбор по конкретному типу контактной информации.

Способ вызова

Вызывается из меню "Справочники  --> Идентификационные --> Виды контактной информации".

Дополнительная информация

Контактная информация хранится по типам, которые жестко задаются в конфигурации и не могут быть изменены пользователем. Пользователь может ввести произвольное количество видов контактной информации, принадлежащих одному из типов. Для этого служит справочник «Виды контактной информации».
