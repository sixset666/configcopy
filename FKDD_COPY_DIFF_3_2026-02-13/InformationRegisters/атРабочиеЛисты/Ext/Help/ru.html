Регистр сведений "Помощник продаж. Рабочие листы"

Назначение

Регистр сведений предназначен для хранения рабочих листов клиентов, автомобилей или моделей автомобилей.

Относится к подсистеме: «Сервис».

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Автор. Тип значения: «Строка» или ссылка на справочник «Пользователи». Автор рабочего листа, если не заполнен, то данный рабочий лист содержит список каталогов для объекта.
  • Объект. Является обязательным для заполнения. Тип значения: «Строка», ссылка на справочник «Контрагенты», ссылка на справочник «Автомобили», ссылка на справочник «Модели». Объект, для которого подобран рабочий лист.

Ресурсы

  • Дата. Тип значения: «Дата». Дата обновления рабочего листа.
  • Значение. Тип значения: «ХранилищеЗначения». Содержит структуру с реквизитами рабочего листа и его составом.
  • Клиент. Тип значения: «Строка», ссылка на справочник «Контрагенты», ссылка на справочник «Автомобили», ссылка на справочник «Модели». Сопутствующий объекту рабочего листа элемент из корзины для которого производился подбор в рабочий лист.

Формы

  • Форма выбора. Служит для просмотра списка рабочих листов с возможностью наложения отборов. В верхней части расположен список рабочих листов. В нижней части расположены данные рабочего листа: реквизиты и состав.

Способ вызова

Вызывается из из АРМ "Корзина" при выборе автомобиля - пункт меню "Найти в рабочих листах".
