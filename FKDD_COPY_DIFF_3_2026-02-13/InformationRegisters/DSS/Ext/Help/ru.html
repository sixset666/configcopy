Регистр сведений «DSS»

Регистр сведений предназначен для хранения настроек "DSS Консоли" по объектам информационной базы:

  • Пользователи

Измерения :

  • Пользователь - владелец настройки
  • Номер - внутренний телефонный номер (или комбинация символов для набора в линию)
  • Порядок - порядок, в котором будет отображаться кнопка (отсчет начинается с левого верхнего угла)

Ресурсы:

  • Название- Название кнопки
