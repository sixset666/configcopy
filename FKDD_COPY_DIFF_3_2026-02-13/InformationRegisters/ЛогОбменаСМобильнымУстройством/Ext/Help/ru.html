Регистр сведений "Лог обмена с мобильным устройством"

Назначение

В регистре хранится история сообщений обмена с мобильными устройствами. Позволяет просматривать сообщения.

Относится к подсистеме: «Обмен с мобильным устройством».

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Узел. Реквизит является обязательным для заполнения. Ссылается на план обмена «Обмены с мобильным устройством». Узел обмена.
  • Ответ. Реквизит является обязательным для заполнения. Тип значения: «Булево». Признак ответа мобильному устройства.

Ресурсы

  • Текст. Тип значения: «Строка». Текст сообщения.

Реквизиты

  • Команда. Тип значения: «Строка». Команда запроса.

Способ вызова

Вызывается из пункта меню "Сервис-> Все операции-> Регистры сведений-> Лог обмена с мобильным устройством".
