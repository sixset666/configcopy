Регистр сведений "Напоминание о записи на ремонт"

Назначение

Регистр сведений предназначен для хранения информации о записях на ремонт и дальнейшей отправки сообщений.

Относится к подсистеме: «Автосервис».

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Заявка на ремонт. Является обязательным для заполнения. Ссылка на документ «ЗаявкаНаРемонт». Заявка на ремонт, по которой будет сформировано и отправлено смс сообщение.  

Ресурсы

  • Документ SMS. Ссылка на документ «SMS». Сообщение, которое будет отправлено заказчику.
  • Время отправки. Является обязательным для заполнения. Тип значения: «Дата». Дата отправки смс.
  • Дата начала ремонта. Является обязательным для заполнения. Тип значения: «Дата». Дата начала работ.
  • Дата реальной отправки. Тип значения: «Дата». Дата реальной отправки смс сообщения.

Формы

  • Форма списка. Служит для просмотра и редактирования записей регистра.

Способ вызова

Вызывается из "Сервис --> Все операции --> Регистр сведений --> Напоминание о записи на ремонт".

Дополнительная информация

Признаком отправки смс сообщения служит заполненое измерение «ДокументSMS». При отмене заявки на ремонт, запись из регистра удаляется, а если документ «SMS», уже создан, то у него возводится «ПометкаУдаления».
