Регистр сведений "Задачи планировщика"

Назначение

Регистр сведений предназначен для автоматизации выполнения определенным пользователем периодических задач в определенное время. К таким задачам могут относиться: восстановление последовательности документов, завершение работы системы, открытие различных форм, запуск процедур встроенного языка.

Относится к подсистеме: «Сервис».

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Пользователь. Ссылается на справочник «Пользователи». Пользователь, для которого назначается задача.
  • Вид задачи. Ссылается на перечисление «Вид задач планировщика». Вид выполняемой задачи.
  • Объект. Тип значения: «Строка». Имя формы или имя запускаемой процедуры.
  • Период запуска задачи. Ссылается на перечисление «Периоды запуска задач планировщика». Период запуска задачи.
  • Момент запуска задачи. Тип значения: «Дата». Момент запуска задачи.
  • Действие при ошибке. Ссылается на перечисление «Виды действий при ошибке выполнения задачи». Действие при ошибке выполнения задачи.
  • Периодическая. Тип значения: «Булево». Признак периодичности задачи.
  • Дата и время однократного запуска. Тип значения: «Дата». Дата и время однократного запуска задачи.
  • Удалять задачу после запуска. Тип значения: «Булево». Признак того, что задачу после первого запуска необходимо удалить из очереди запускаемых задач.

Формы

  • Дерево форм. Форма предоставляет возможность выборы задачи планировщика.
  • Форма списка. Основная форма списка и редактирования. Используется для отображения списка или редактирования элементов.
  • Форма записи. Форма записи регистра.
