Регистр сведений "Запрещенные склады компании"

Назначение

Регистр предназначен для хранения прав доступа пользователей к складам компании. В нем для каждого пользователя хранятся только те склады, доступ к которым у пользователя отсутствует.

Относится к подсистеме: «Структура компании».

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Пользователь. Реквизит является обязательным для заполнения. Ссылается на справочники: «Пользователи». Пользователь, у которого запрещен доступ к складу.
  • Склад компании. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад, к которому запрещен доступ пользователя.

Способ вызова

Не предназначен для интерактивного вызова и редактирования.
