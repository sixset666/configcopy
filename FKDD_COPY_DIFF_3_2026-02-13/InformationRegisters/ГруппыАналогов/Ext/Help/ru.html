Регистр сведений "Группы аналогов"

Назначение

Регистр сведений "Группы аналогов" предназначен для хранения списка аналогов номенклатуры, замена одного на другой которого допустима.

Относится к подсистеме: «Номенклатурные».

Параметры

Периодичность: Непериодический

Режим записи: Независимый

Измерения

  • Идентификатор группы аналогов. Уникальный идентификатор, объединяющий номенклатуру в одну группу.
  • № артикула для поиска. Номер номенклатуры по каталогу для поиска, входящий в группу аналогов.
  • Производитель. Ссылается на справочник «Производители». Содержит производителя номенклатуры.
  • Заменитель. Ссылается на справочник «Номенклатура». Номенклатура, на которую надо заменять.
  • Заменитель. Ссылается на справочник «Номенклатура». Номенклатура, на которую надо заменять.
  • Заменитель. Ссылается на справочник «Номенклатура». Номенклатура, на которую надо заменять.

Реквизиты

  • Главный по группе. Признак того, что номенклатура с данным номером по каталогу является главной (предпочтительной) в группе аналогов.
  • Главный по производителю. Признак того, что номенклатура с данным номером по каталогу является главной (предпочтительной) у поставщика.
  • № по каталогу. Номер номенклатуры по каталогу, входящий в группу аналогов.
