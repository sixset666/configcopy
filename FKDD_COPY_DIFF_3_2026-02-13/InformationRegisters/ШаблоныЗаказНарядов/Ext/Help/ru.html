Регистр сведений "Шаблоны заказ-нарядов"

Назначение

Регистр предназначен для хранения списка шаблонов, по которым могут формироваться новые заказ-наряды. Сами шаблоны являются документами заказ-нарядов, реальными или введенными в качестве примера перечня деталей и работ, необходимых для выполнения типовых ремонтов. Используя шаблоны можно формировать новые документы заказ-нарядов на выполнение типовых авторемонтных работ.

Относится к подсистеме: «Автосервис».

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Наименование шаблона. Тип значения: «Строка». Наименование шаблона заказ-наряда.
  • Заказ-наряд шаблона. Ссылается на документ «Заказ-наряд». Заказ-наряд, используемый в качестве шаблона.

Способ вызова

Вызывается из формы списка документов «Заказ-наряд» из меню "Перейти" на панели формы. Редактирование списка шаблонов доступно из меню "Документы --> Автосервис --> Шаблоны заказ-нарядов".
