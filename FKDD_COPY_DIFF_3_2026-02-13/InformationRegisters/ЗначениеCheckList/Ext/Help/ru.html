Регистр сведений "Значение списка проверки т\с"

Назначение

В регистре хранится состав списка проверки т/с для документов. Доступен просмотр и редактирование.

Относится к подсистеме: «Обмен с мобильным устройством».

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Объект. Реквизит является обязательным для заполнения. Ссылается на документы «Заявка на ремонт», «Заказ-наряд». Объект для которого составлен список.
  • Пункт проверки. Реквизит является обязательным для заполнения. Ссылается на справочник «Список проверки транспортного средства (т\с)». Пункт проверки.
  • GUID. Реквизит является обязательным для заполнения. Тип значения: «Строка». Уникальный идентификатор для выгрузки на мобильное устройство.

Ресурсы

  • Значение. Тип значения: «Строка», «Булево», «Число», «Дата». Значение пункта проверки.
  • Checked. Тип значения: «Булево». Признак выполнения пункта проверки.
  • Порядок. Тип значения: «Число». Порядок пункта на мобльном устройстве.

Способ вызова

Вызывается из пункта меню "Сервис-> Все операции-> Регистры сведений-> Значение списка проверки т\с".
