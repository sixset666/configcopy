Регистр сведений "Границы последовательности взаиморасчетов"

Назначение

Используется для восстановления последовательности ввода и проведения документов. Предназначен для хранения информации о нарушениях ввода взаиморасчетных документов в разрезе договоров взаиморасчетов с контрагентами. Применяется в обработке «Восстановление последовательностей».

Относится к подсистеме: «Взаиморасчеты».

Параметры

Периодичность: Непериодический.

Режим записи: Независимый.

Измерения

  • Договор взаиморасчетов. Ссылается на справочник «Договоры взаиморасчетов». Договор взаиморасчетов с контрагентом, последовательность взаиморасчетов по которому нарушена.

Ресурсы

  • Момент времени. Тип значения: «Дата и время». Момент времени, на который нарушена последовательность ввода документов.
  • Документ ссылка. Тип значения: «ДокументСсылка». Документ, нарушивший последовательность ввода документов.

Формы

  • Форма списка. Основная форма списка и редактирования. Используется для отображения списка нарушений ввода документов.

Дополнительная информация

Регистр сведений "Границы последовательности взаиморасчетов" является служебным и не предназначен для ручного редактирования. Сведения из данного регистра используются для восстановления последовательности проведения взаиморасчетных документов.
