Регистр сведений "Параметры обмена данными"

Назначение

Предназначен для хранения результатов обмена по плану обмена "Удаленные подразделения". Значения ресурса "Имя файла последней загрузки" позволяют точно определить имя последнего загружаемого файла при выполнении дочитывания сообщения обмена от главного узла плана обмена после чтения изменений конфигурации.

Относится к подсистеме: «Рарус».

Параметры

Периодичность:Непериодический

Режим записи: Независимый

Измерения

  • Узел ИБ. Ссылается на справочник «Структура информационных баз». Узел, с которым производился обмен.
  • Настройка обмена данными. Ссылается на справочник «Настройки доставки сообщений». Настройка доставки, по которой был произведен обмен.

Ресурсы

  • Дата последней загрузки. Тип значения: «Дата». Дата последней выгрузки сообщения обмена в узел ИБ.
  • Дата последней выгрузки. Тип значения: «Дата». Дата последней загрузки сообщения обмена в узел ИБ.
  • Результат последней загрузки. Тип значения: «Булево». Истина - удачная загрузка.
  • Результат последней выгрузки. Тип значения: «Булево». Истина - удачная выгрузка.
  • Дата последней удачной загрузки. Тип значения: «Дата». Дата последней удачной загрузки сообщения обмена в узел ИБ. 
  • Дата последней удачной выгрузки. Тип значения: «Дата». Дата последней удачной выгрузки сообщения обмена в узел ИБ. 
  • Имя файла последней загрузки. Тип значения: «Строка». Имя файла последней загрузки сообщения обмена в узел ИБ.
  • Имя файла последней выгрузки. Тип значения: «Строка». Имя файла последней выгрузки сообщения обмена в узел ИБ. 

Реквизиты

  • Комментарий выгрузки. Тип значения: «Строка». Пояснение к выгрузке сообщения обмена. При неудачной выгрузке содержит текст ошибки.
  • Комментарий загрузки. Тип значения: «Строка». Пояснение к загрузке сообщения обмена. При неудачной загрузке содержит текст ошибки.

Способ вызова

Вызывается из меню "Обработки->Параметры обмена данными". Не предназначен для редактирования.
