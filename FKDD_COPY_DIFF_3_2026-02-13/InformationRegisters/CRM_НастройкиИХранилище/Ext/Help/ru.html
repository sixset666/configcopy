Регистр сведений "CRM коммуникатор настройки и хранилище"

Назначение

Регистр сведений используется для хранения произвольной информации в разрезе пользователей и уникального идентификатора. В этом регистре хранятся записные книги каждого пользователя.

Относится к подсистеме: «SMS Коммуникатор», «СофтФонПроф», «СофтФонЛайт».

Параметры

Периодичность: Непериодический

Режим записи: Независимый

Измерения

  • Пользователь. Ссылается на справочник «Пользователи». Пользователь, для которого храниться информация.
  • Идентификационная строка, разделяющая данные объекта. Тип значения: «Строка». Идентификационная строка, разделяющая данные объекта.

Ресурсы

  • Данные хранилище значений. Тип значения: «Хранилище значения». Данные хранилище значений.
  • Значение. Тип значения: «Булево», Тип значения: «Дата», Тип значения: «Строка», Тип значения: «Число», Ссылается на документы «SMS-сообщение», «Факс сообщение», Ссылается на справочник «CRM оборудование». Хранимое значение.
