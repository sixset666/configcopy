Обработка "АРМ Автосалон" (бета-тестирование)

Назначение

Обработка представляет собой рабочее место менеджера кредитно-страхового отдела, осуществляющего оформление страховок и кредитов клиентов.

Относится к подсистеме: «Учет взаимоотношений с клиентами».

Формы

Интерфейс кредитно-страхового отдела.

Является основной формой АРМ. Форма состоит из следующих элементов:

  • Закладка Рабочие листы

Содержит список рабочих листов.

  • Закладка "Рабочие листы кредитного отдела"

Содержит список рабочих листов кредитного отдела.

  • Закладка "Рабочие листы отдела страхования"

Содержит список рабочих листов отдела страхования.

  • Закладка "Страховые полисы"

Содержит список страховых полисов.

  • Панель основных действий.

Содержит кнопки основных действий.

Основные действия:

    • Кредиты - создать новый документ Рабочий лист кредитного отдела на основании Рабочего листа в текущей строке списка.
    • Страхование - создать новый документ Рабочий лист отдела страхования на основании Рабочего листа в текущей строке списка.
    • Закрыть - закрыть форму

Способ вызова

Открывается из обработки "Функциональная панель". Отдельно не используется.
