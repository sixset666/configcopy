Обработки "Проверка номенклатуры по типу"

Назначение

Обработка "Проверка номенклатуры по типу" предназначена для исправления ошибок в логической структуре данных, связанных с типизацией номенклатуры, а именно: использование единиц измерений, использование характеристик и параметры штрихкодирования. Данная обработка предназначена для специалистов по внедрению и опытных пользователей.

Относится к подсистеме: «Служебные».

Способ вызова

Вызывается из меню "Сервис --> Все операции --> Обработки --> Проверка номенклатуры по типу".
