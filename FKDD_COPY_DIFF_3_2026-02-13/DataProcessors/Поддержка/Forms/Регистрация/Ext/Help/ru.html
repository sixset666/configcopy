Форма "Регистрация"

Назначение

Форма "Регистрация" предназначена для формирования и отправки заявки на регистрацию программного продукта на сервер поддержки 1С-Раруса Альфа-Авто. Регистрация на сервере поддержки позволяет оперативно получать  обновления для типового решения. Для отправки заявки на регистрацию необходимо заполнить обязательные реквизиты. Это можно сделать как в ручную, так и воспользовавшись автоматическим заполнением (Заполнить -> Заполнить по текущему пользователю). По мимо сведений об вашей организации и продавце программного продукта необходимо указать информацию о компьютере, на котором установлена данная программа. Для этого рекомендуется использовать автоматическое заполнение сведений о компьютере (Заполнить -> Заполнить сведения о компьютере), так как этот способ позволяет получить наиболее полную информацию о системе.

Внимание! Если вы уже были зарегистрированы на сервере поддержки, то вам необходимо указать свой логин и пароль. В противном случае эти поля необходимо оставить пустыми.

После заполнения всех обязательных реквизитов необходимо нажать на кнопку "Регистрация". По введенным вами данным будет сформирована заявка на регистрацию и отправлена на сервер поддержки. После этого заявка будет проверена специалистом службы техподдержки, и вам будет выслано письмо с результатом регистрации. При этом, если вы регистрируетесь первый раз, то для вас будет создана новая учетная запись (Логин и Пароль), которую вы будете использовать для дальнейшей работы с сервером поддержки. В случае, если учетная запись у вас уже есть, то ее права будут расширены.

 

Относится к подсистеме: «Сервис».

Способ вызова

Вызывается из меню "Сервис --> Поддержка--> Регистрация"

Реквизиты

  • Серийный номер. Строка. Реквизит является обязательным для заполнения. Серийный номер регистрируемого программного продукта (он напечатан на вашей регистрационной анкете)
  • E-mail. Строка. Реквизит является обязательным для заполнения. Адрес электронной почты, на которую вам будет выслано письмо с результатом регистрации.
  • Полное наименование организации. Строка. Реквизит является обязательным для заполнения. Полное наименование организации, на которую осуществляется регистрация текущего программного продукта.
  • Руководитель. Строка. Реквизит является обязательным для заполнения. Руководитель организации.
  • Ответственный за работу. Строка. Реквизит является обязательным для заполнения. Сотрудник вашей организации, ответственный за работу с программой.
  • Почтовый адрес. Строка. Реквизит является обязательным для заполнения. Почтовый адрес организации, на которую осуществляется регистрация текущего программного продукта
  • Почтовый индекс. Строка. Реквизит является обязательным для заполнения. Почтовый индекс.
  • Телефон. Строка. Реквизит является обязательным для заполнения. Телефон для связи с вами службой техподдержки.
  • Факс. Строка. Реквизит является обязательным для заполнения. Факс для связи с вами службой техподдержки.
  • Полное наименование продавца программы. Строка. Реквизит является обязательным для заполнения. Название фирмы-продавца программного продукта.
  • Код партнера 1С. Строка. Код партнера 1С фирмы-продавца.
  • Дата приобретения программы. Строка. Дата покупки программного продукта.
  • Сведения о компьютере. Строка. Реквизит является обязательным для заполнения. Содержит информацию в произвольной форме о компьютере, на котором установлен текущий программный продукт. Рекомендуется использовать автозаполнение этого поля.
  • Логин. Строка. Логин учетной записи, полученной на сервере поддержки. Заполняется, если вы уже проходили регистрацию ранее.
  • Пароль. Строка. Пароль учетной записи, полученной на сервере поддержки. Заполняется, если вы уже проходили регистрацию ранее
