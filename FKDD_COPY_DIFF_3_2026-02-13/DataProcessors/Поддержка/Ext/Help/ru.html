Обработки "Поддержка"

Назначение

Обработка "Поддержка" предназначена для работы блока "Поддержка". Содержит формы и некоторые методы для отправки регистрационных анкет, формирования писем в службу техподдержки программного продукта.

Относится к подсистеме: «Сервис».

Формы

  • Написать писмо в техподдержку. Вспомогательная форма для обработки команды меню  "Сервис --> Поддержка--> Написать письмо в службу техподдержки". Визуального отображения не имеет.
  • Ожидание создания файла с информацией. Вспомогательная форма. Используется при ожидании формирования утилитой SU.exe файла с информацией о системе.
  • Регистрация. Основная форма обработки. Предназначена для создания и отправки заявки на регистрацию типового решения.
