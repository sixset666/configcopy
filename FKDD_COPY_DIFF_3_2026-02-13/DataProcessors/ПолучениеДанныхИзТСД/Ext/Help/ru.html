Обработка "Получение данных из ТСД"

Назначение

Получает данные из Терминала Сбора Данных (ТСД). Полученные данные: выводятся в отчет, сохраняются в файл или на основании этих данных может быть заполнена таблица документа.

Относится к подсистеме: «Работа с оборудованием».

Формы

  • Форма. Основная форма. В поле ТСД указывается элемент справочника Оборудование, который является терминалом сбора данных. На закладке Источник данных ТСД указываются параметры для получения данных непосредственно из ТСД:
    - Сохранить данные в файл - если флажок установлен, то данные из ТСД получаются как текстовый файл и сохраняются на диске. Этот способ наиболее быстрый, но рекомендуется им пользоваться только в случае возникновения проблем с получением данных непосредственно из ТСД. Если флажок снят, то данные из ТСД будут получаться построчно.
    - Очищать терминал после получения данных - если флажок установлен, то после получения данных, информация о считанных в ТСД товарах будет стерта
    Если активна закладка Источник данных файл, то выбирается файл, который был ранее получен из ТСД (см. флаг Сохранить данные в файл), и таблица товаров формируется на основании информации указанной в этом файле, наличие ТСД в этом случае не обязательно.

Способ вызова

Вызывается из меню "Обработки-->Прием данных из ТСД"

Дополнительная информация

Запись данных в файл возможна только в том случае, если в ТСД модели Cipher используется прошивка, поставляемая компанией ООО "1С-Рарус Альфа-Авто".
