Обработка "Выгрузка загрузка шаблонов документов"

Назначение

Обработка "Выгрузка загрузка шаблонов документов" предоставляет возможность сохранять документы различных видов в виде шаблона, сохраняемого в регистре "Картинки и файлы". Выгрузить документ в шаблон можно из подменю "Действия" формы списка документов, либо из самого документа. Способ загрузки аналогичен. При загрузки из списков и журналов документов происходит создание нового документа по выбранному шаблону.

Относится к подсистеме: «Сервис».

Способ вызова

Открывается из обработки меню Действие соответствующих форм.

Формы

  • Выгрузка/загрузка шаблонов документов. Основная форма списка для создания, удаления и редактирования шаблонов (планов) документов. Состоит из таблицы "Шаблоны документов", где отображается наименование шаблона и его владелец (пользователь). С помощью комендной панели можно сделать шаблон общим, изменить пользователя.
  • Выбор имени шаблона. Для созданного шаблона можно создать собственное наименование или выбрать из уже имеющихся шаблонов.
  • Пометка реквизитов шаблонов. Данная форма позволяет выбрать реквизиты шаблона из выбранного документа.
