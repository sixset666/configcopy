Скидки.

Порядок работы со скидками.

В программе реализована достаточно гибкая система по работе со скидками. Работа со скидками производится в несколько этапов: 

    1. Определение аналитики используемых скидок. Для этого создается необходимое количество элементов справочника «Типы скидок». Справочник не задает конкретные значения скидок, а лишь описывает типы скидок, которые будут использоваться при работе. Справочник содержит следующую информацию о параметрах скидок:
      • Способ назначения: Автоматическая (тип скидки в документе подбирается программой) или ручная (тип скидки в документе задается вручную). 
      • Вариант применения: На строку документа или на документ целиком.
      • Способ вычисления: Относительная (процент от суммы документа)  или абсолютная (сумма в рег. валюте).
    2. Задание прав пользователей (см. "Описание значений права  «Способ выбора скидки»"). 
    3. Задание значений скидок. Для задания значений скидок предназначены документы «Назначение общих скидок» (назначение скидок на документ в целом) и  «Назначение товарных скидок» (назначение скидок на номенклатурные позиции или типы номенклатуры). 
      • Документ «Назначение общих скидок» - значения скидок задаются в табличной части "Скидки" документа. Указанные значения скидок будут действовать только для подразделения компании, указанного в шапке документа, и начиная с даты, указанной в реквизите "Действует С".  В табличной части помимо значений скидок указываются и условия их применения:
        • Скидка - Справочник «Типы скидок». Реквизит обязательный для заполнения. Задает тип скидки, для которой устанавливается значение. Можно выбрать только тип скидки, у которой "Вид начисления скидки" равен "На документ".
        • Значение - значение скидки. Если выбран тип скидки абсолютный, то задается сумма скидки в валюте рег. учета, иначе - процент от суммы документа.
        • Сумма чека - задает минимальную сумму документа, при которой скидка начинает действовать. Если сумма равна нулю, то данное условие не анализируется.
        • Дисконтная карта - справочник «Карточки». Если выбрана карточка, то скидка будет действовать только для данной карты. Если поле пустое, то условие не анализируется.
        • На сумму покупок - задает минимальную накопительную сумму дисконтной карты, при которой скидка начинает действовать. Если дисконтная карта не выбрана, то данное поле не анализируется.
        • С - время суток, начиная с которого действует скидка.
        • По - время суток, до которого действует скидка.
        • Пн, Вт, Ср, Чт, Пт, Сб, Вс - задаются дни недели, в которые действует скидка.
        • Склад - справочник «Склады компании». Склад компании, для которой действует скидка. Если склад не указан, то скидка действует для всех складов компании.
      • Документ «Назначение товарых скидок» - значения скидок задаются в табличной части "Скидки" документа. Указанные значения скидок будут действовать только для подразделения компании, указанного в шапке документа, и начиная с даты, указанной в реквизите "Действует С".  В табличной части помимо значений скидок указываются и условия их применения:
        • Скидка - Справочник «Типы скидок». Реквизит обязательный для заполнения. Задает тип скидки, для которой устанавливается значение. Можно выбрать только тип скидки, у которой "Вид начисления скидки" равен "На строку документа".
        • Значение - значение скидки. Если выбран тип скидки абсолютный, то задается сумма скидки в валюте рег. учета, иначе - процент от суммы документа.
        • Номенклатура - справочник «Номенклатура». Номенклатура, на которую распространяется скидка.
        • Тип номенклатуры - справочник «Типы номенклатуры». Тип номенклатуры, для которой действует скидка. Для скидки можно задать либо только номенклатуру, либо только тип номенклатуры.
        • Количество - задает минимальное значение количества номенклатуры (в базовой ед. измерения) в строке табличной части документа отгрузки, при которой скидка начинает действовать.
        • Правило - правило расчета скидки по количеству. Может принимать значения: Не используется - скидка предоставляется на всю позицию, если количество превысило заданное; Больше - скидка предоставляется только на количество товара превышающее заданное; Меньше - скидка предоставляется только на количество товара не превышающее заданное; Упаковка - Скидка предоставляется только на количество кратное заданному.
        • Сумма строки- задает минимальное значение суммы строки табличной части документа отгрузки, при которой скидка начинает действовать.
        • Дисконтная карта - справочник «Карточки». Если выбрана карточка, то скидка будет действовать только для данной карты. Если поле пустое, то условие не анализируется.
        • На сумму покупок - задает минимальную накопительную сумму дисконтной карты, при которой скидка начинает действовать. Если дисконтная карта не выбрана, то данное поле не анализируется.
        • С - время суток, начиная с которого действует скидка.
        • По - время суток, до которого действует скидка.
        • Пн, Вт, Ср, Чт, Пт, Сб, Вс - задаются дни недели, в которые действует скидка.
        • Склад - справочник «Склады компании». Склад компании, для которой действует скидка. Если склад не указан, то скидка действует для всех складов компании.
    4. Подбор скидки в документах отгрузки. В документе могут применятся одновременно и товарные скидки, и скидки на документ в целом:
      • Порядок работы со скидками на документ ("шапочные" скидки): Если необходимо подобрать "ручную" скидку, то в форме "Цены и валюты" в поле "Тип скидки" нужно вручную выбрать тип скидки. Далее программа автоматически подберет максимальное значение скидки в соответствии с выбранным типом скидки (при этом будет произведена проверка всех условий применения скидки). Если необходимо подобрать "автоматическую" скидку, то в таком случае необходимо очистить поле "Тип скидки" на форме "Цены и валюта". После этого программа автоматически подберет максимальное значение скидки, соответствующее всевозможным "автоматическим" типам скидок, удовлетворяющим условиям.
        Применение скидки. Способ вычисления "шапочной" скидки зависит от значения реквизита "Способ вычисления" типа подобранной скидки. Если "Способ вычисления" - "Относительная (проценты)", то значение скидки (а в данном случае, это будет значение процента) подставляется в колонку "% скидки". После чего рассчитывается сумма скидки для каждой строки. Эти суммы заносятся в колонку "Скидка" (по умолчанию не отображается). Если "Способ вычисления" - "Абсолютная (сумма)", то сначала значение скидки пересчитывается в валюту документа. Затем высчитывается процент скидки относительно суммы документа (итог по колонке "Сумма"). Процент подставляется в колонку "% скидки", и далее вычисляется сумма скидки, приходящаяся на каждую строку. Сумма заносится в колонку "Скидка". В результате, итог по колонке "Скидка" должен совпадать со значением подобранной абсолютной скидки.
        При редактировании реквизитов документа, которые влияют на подбор скидки, например, выбора склад компании, выбора дисконтной карты, изменение суммы строки (что приводит к изменению итоговой суммы документа, а значит базы для расчета скидки),  производится попытка подбора подходящей скидки. Если тип или значение новой найденной скидки отличается от типа или значения уже примененной скидки, то производится автоматический пересчет всей табличной части.
        ВАЖНАЯ ОСОБЕННОСТЬ: Расчет суммы скидки производится всегда от колонки "Сумма", вне зависимости от того, включает эта сумма НДС или нет. Это означает, что, если в документе выбран тип цен, включающий в себя НДС, сумма скидки также будет включать в себя НДС.
      • Порядок работы с товарными скидками ("строчные" скидки). Для работы со строчными скидками в табличной части документа предусмотрены три реквизита "Скидка на товар", "Процент скидки строки", "Сумма скидки строки". По аналогии с работой со скидками на документ, для задания скидки необходимо выбрать тип скидки, либо, если оставить поле "Скидка на товар" пустым, тип скидки подберется автоматически. Возможность работы с "ручными" или "автоматическими" скидками определяется правом "Способ выбора скидки". Если пользователю запрещена работа с ручными скидками, то поле "Скидка на товар" будет недоступно для редактирования, и тип скидки будет подбираться только "автоматический". Если же пользователю запрещена работа с "автоматическими" скидками, то для получение товарной скидки необходимо в обязательном порядке задать тип скидки вручную. После того, как тип скидки выбран, программа автоматически подбирает значение скидки, дающее максимальную скидку на строку и удовлетворяющее условиям применения скидки. Как видно, работа с "ручными" и "автоматическими" скидками имеет существенное различие. В случае "ручной" скидки, указывается конкретный тип скидки (но только "ручной" тип скидки), и уже в пределах данного типа подбирается наиболее выгодное значение скидки. В случае же "автоматической" скидки (поле "Скидка на товар" очищено), программа осуществляет поиск максимальной скидки среди всех "автоматических" типов скидок. И уже после выбора максимального значения определяется конкретный тип скидки.
        Товарные скидки работают независимо от "шапочных" скидок. Итоговая скидка по строке складывается из товарной скидки и части суммы общей скидки, приходящейся на текущую строку.

Описание значений права  «Способ выбора скидки»

Данное право определяет какие типы скидок доступны для подбора в документы отгрузки пользователем.

    1. Скидки запрещены - пользователю не доступна возможность назначить скидку в документе.
    2. Автоматические скидки - скидки подбираются в документ (как общие, так и товарные) автоматически. Пользователь не имеет возможности вручную задать тип скидки. Но, если для выбранного в документе договора взаиморасчетов установлена ручная скидка, то она автоматически будет подставлена в документ.
    3. Ручные скидки - скидки автоматически в документ не подбираются. Для применения скидки пользователю необходимо выбрать тип скидки вручную.
    4. Автоматические и ручные скидки - пользователь может вручную указать тип скидки, иначе тип скидки будет подобран автоматически.
    5. Ручные скидки и редактирование процента - пользователь может вручную указать тип скидки, а также отредактировать значение процента скидки (как для у товарной скидки, так и у скидки на документ в целом, но только если способ вычисления типа скидки - "Относительная (Процент)". В остальных случаях, значение скидки подбирается в пределах выбранного типа скидки, по принципу максимальной суммы скидки.
