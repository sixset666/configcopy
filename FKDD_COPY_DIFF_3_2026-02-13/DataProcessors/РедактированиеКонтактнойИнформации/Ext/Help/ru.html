Обработка "Редактирование контактной информации"

Назначение

Обработка служит для ввода контактной информации. Для каждого типа контактной информации используется своя, специальная форма ввода данные.

Относится к подсистеме: «Сервис».

Способ вызова

Вызывается при вводе контактной информации.

Формы

  • Форма дерева видов. Форма выбора вида вводимой информации. Данная форма всегда вызывается первой при вводе новой контактной информации. На форме расположено дерево, в котором иерархически выведены виды контактной информации, сгруппированные по типам. При выборе конкретного значения будет открыта форма ввода, предназначенная для данного вида контактной информации. При установке флажка "Только предопределенные" в нижней командной панели в дереве будут выводиться только предопределенные виды контактной информации, введенные пользователями виды будут скрыты (это удобно когда введено много дополнительных видов, а для текущей работы достаточно предопределенных). Прямо в форме можно ввести новый вид контактной информации, для этого служит кнопка "Добавить новый вид". После ввода нового вида он сразу отобразится в дереве выбора, при этом автоматически будет снят флажок "Только предопределенные".
  • Форма ввода адреса. Служит для ввода адреса. Возможен ввод адреса в произвольном формате, а также при помощи Адресного классификатора (при условии, что он загружен).
  • Форма ввода телефона. Служит для ввода телефонного номера. Позволяет указать код страны, код города, номер в международном формате, а также внутренний добавочный номер.
  • Форма ввода адреса электронной почты. Служит для ввода адреса e-mail. Из поля ввода адреса можно создать новое письмо, нажав на "лупу" (при условии, что в установлен и настроен почтовый клиент).
  • Форма ввода адреса веб-страницы. Служит для ввода адреса веб-страницы. Из поля ввода адреса можно открыть данную веб-страницу, нажав на "лупу" (при условии подключения к интернет и, если установлен и настроен веб-браузер).
  • Форма ввода номера ICQ. Служит для ввода номера ICQ. Из поля ввода номера можно открыть окно данного пользователя ICQ , нажав на "лупу" (при условии, что в установлен и настроен ICQ клиент).
  • Форма ввода дополнительной информации. Служит для ввода произвольной информации.

Дополнительная информация

Сменить тип введенной контактной информации нельзя, можно лишь открыть запись и изменить вид контактной информации.
