Обработка "АРМ отчеты"

Назначение

Обработка используется для быстрого доступа пользователя к отчетной информации.

Относится к подсистеме: «Автоматизированные рабочие места».

Формы

  • Интерфейс. Является основной формой АРМ. Форма состоит из следующих областей:
    • Панель "Заголовок". Содержит поле выбора настройки АРМ отчеты, панель "Действия", которая содержит ряд стандартных действий с отчетами: печать, сохранение в форматах PDF, HTML, открытия в пакетах Open Office Calc, Microsoft Excel, отправку отчета в электронном письме, добавление в Избранное. Содержит кнопку открытия формы настроек текущего отчета, выведенного в области вывода отчета.
    • Список отчетов. Состоит из настраиваемого списка отчетов и панели управления. Каждый пользователь может составить индивидуальный список необходимых для него отчетов. Кроме изменения состава списка есть возможность настройки и сохранения параметров каждого отчета из списка. Пользователь может сохранять сколько угодно таких списков. Текущий список выбирается в верхнем левом поле формы. Если у пользователя нет настроенного списка, то по-умолчанию будет показан список из всех отчетов.
      Если состав списка был изменен, его можно обновить (кнопка  "Обновить список"). Любой отчет из списка может быть сформирован и непосредственно выведен в области вывода отчета. Для этого достаточно нажать на название отчета. При нажатии на иконку  рядом с названием откроется стандартная форма настроек выбранного отчета. Если после этого сформировать отчет, то его результат будет показан в новом окне.
    • Область вывода отчета. В правую часть формы выводится результат отчета. Если для текущего отчета были сохранены настройки параметров, то они будут использоваться при формировании отчета. Отдельно указывается период отчета на форме в правом верхнем углу. Кнопка "Обновить" для переформирования текущего отчета. 

Способ вызова

Открывается из обработки "Функциональная панель". Отдельно не используется.
