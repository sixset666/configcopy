Обработка "СофтФон, Проф"

Назначение

Обработка «СофтФон, Проф» далее «СофтФон» предназначена для управления внутренними и внешними, входящими и исходящими телефонными звонками. «СофтФон» дублирует основные функции телефонного аппарата. Обработка интегрируется в систему «1С:Предприятие 8» и напрямую взаимодействует с базой данных.

Относится к подсистеме: «СофтФонПроф».

Как работает обработка «СофтФон»

  • При поступлении входящего звонка, по телефонному номеру клиента, указанному в системе «1С:Предприятие 8» происходит идентификация клиента и в окне обработки «СофтФон» одновременно со звонком появляется информация о клиенте из CRM-системы «1С:Предприятие 8».
  • Оператор обрабатывает звонок. Существует возможность управлять звонком через окно обработки «СофтФон»: ответить на звонок, повесить на Hold / снять с Hold, перевести звонок, завершить разговор (положить трубку).
  • Из окна «СофтФона» с информацией о клиенте можно быстро открывать нужные объекты CRM-системы «1С:Предприятия 8» связанные с этим клиентом:
    • Открыть карточку клиента, контактного лица с историей предыдущих контактов и характеристиками клиента.
    • Создать новый документ «Событие» с автоматическим заполнением информации о клиенте и возможностью просмотра карточки клиента. «Событие» позволяет зарегистрировать информацию о звонке, запланировать следующие контакты с клиентом, обеспечивает хранение истории контактов. Зарегистрированную информацию о звонке можно передать другому сотруднику при переадресации звонка во время разговора, а если сотрудник отсутствует можно оставить напоминание со ссылкой на зарегистрированный звонок.
  • Если клиент обращается впервые и информация о клиенте в CRM-системе отсутствует, тогда также есть возможность зарегистрировать нового клиента. При регистрации нового клиента телефонный номер заполняется автоматически.
  • Автоматический набор телефонного номера из CRM-системы облегчает выполнение исходящих звонков. Автоматический набор телефонного номера клиента можно выполнить из карточки клиента и контактного лица, документа «Событие», «Телемаркетинг» и окна «СофтФона».
  • «СофтФон» обрабатывает и внутренние звонки между сотрудниками. В окне «СофтФона» появляется информация о сотруднике и его внутреннем номере.
  • Для облегчения поиска и набора телефонных номеров в специальном окне «СофтФона» существует возможность быстрого поиска по списку контрагентов, контактных лиц и сотрудников.
  • Для облегчения отбора, часто используемых телефонных номеров существует возможность каждому пользователю формировать «Записную книгу» со списком наиболее часто набираемых телефонных номеров клиентов и сотрудников.
  • «СофтФон» хранит историю звонков. Формирует список пропущенных, набранных и принятых звонков. При возникновении пропущенных звонков в окне «СофтФона» появляется надпись красного цвета «Пропущенные звонки» со ссылкой на список пропущенных звонков. Ни один звонок клиента или сотрудника не останется без внимания.

 

Кнопки управления звонком:

  • Ответ на звонок
  • Повесить на HOLD, снять с HOLD-а

  • Перевод звонка на другой номер

  • Перевод звонка в режиме "консультационный звонок", с последующим подтверждением перевода

  • Отбой звонка

  • Набор произвольного номера

  • Набор номера сохраненного в базе данных

 

Формы

  • Форма списка телефонов. Используется для произвольного набора номера, а так же выбора номера телефона из базы данных.
  • Форма настройки. Используется для настройки работы "СофтФона". Без настройки данной формы - работа невозможна. 
  • Форма выбора контакта. Если разным клиентам принадлежит одинаковый телефонный номер, то при поступлении входящего звонка появляется список клиентов, которым принадлежит этот телефонный номер. Оператору вручную нужно выбрать клиента и выполнить стандартные операции по регистрации звонка в CRM системе "1С:Предприятия 8".
  • Записная книга выбор. Используется для выбора объекта в записную книгу.
  • Форма заполнения телефона. Используется для уточнения номера телефона оператором. Используется на форме "Списка телефонов", на закладке "Звонки". Вызывается автоматически, когда оператор пытается позвонить по внешнему номеру.

 

Способ вызова

Обработка «СофтФон» вызывается автоматически при старте системы «1С:Предприятие 8», в виде прикрепленного окна к правой части экрана и активизируется при возникновении входящего или исходящего звонка, при условии, что в настройках для данного пользователя установлен флажок "Использовать СофтФон".

Настройка работы "СофтФона"

Для правильной работы "СофтФона" указать настройки для подключения к серверу и префиксы, используемые в телефонной станции. А именно, префикс выхода в город и межгород. Так же указать код страны и код города. Это поможет в дальнейшем автоматически заполнять соответствующие поля при вводе нового номера, и автоматического набора префикса при исходящем звонке.

Настройка "СофтФона" разделяется на два этапа. Настройка основных значений, определенных для всей системы в целом. И настройка локальных значений для отдельно взятого пользователя (настройка значений для пользователя используется по желанию).

Для указания основных значений используется форма "Настройка параметров учета" на закладке "СофтФон". Эти значения будут использоваться для любого оператора системы, при условии, что оператор не настраивал локальные значения специально для себя. Обычно, эти значения устанавливает администратор, и изменять их уже не следует.

Работа "СофтФона" защищена ключом защиты, поэтому при первом запуске системы, необходимо настроить общий каталог работы с компонентой защиты. Загрузка компоненты из общего каталога и синхронизация осуществляется с помощью обработки "Старт системы", которая представляет из себя удобный мастер по подключению компоненты, но, как правило, если нет никаких ошибок, работа этой обработки не визуальна.
