Обработка "Поиск контрагентов и контактных лиц"

Назначение

Служит для поиска контрагентов и их контактных лиц по наименованию и/или контактной информации.

Так же может использоватеся для подбора значение в документы.

Относится к подсистеме: «Сервис».

Формы

  • Форма. Основная форма поиска. В поле "Искать" вводится текст поиска. Автоматически запоминаются 10 последних введенных строк поиска, для просмотра истории поиска воспользуйтесь  ниспадающим списком при нажатии на соответствующую кнопку поля ввода. При выборе значения из списка оно помещается в начало списка. Поиск в этом случае срабатывает автоматически.
    - Список найденных элементов: в этот список помещаются все найденные ссылки. Выводится дополнительная информация:тип клиенты, пол, день рождения.  
    В контекстном меню списка присутствуют следующие пункты:
    • Открыть элемент - открывает форму выбранного элемента.
    • Перейти в список - перейти в список элементов данного типа . При этом открывается форма списка элементов по умолчанию и курсор позиционируется на выбранном элементе.
    • Добавить нового - Открывает форму нового элемента справочника контрагенты и контакты, после его записи добавляет стору в список результатов поиска.

Флаг "Закрывать после выбора" определяет поведение формы: либо форма поиска будет закрыта после выбора элемента, либо останется на экране (состояние формы поиска "Свободное", поэтому оно находится поверх других окон).

 По кнопке "Настройка" открывается формы настройки поиска. В данном форме указываются варианты поиска: искать по наименованию, полному наименованию, всем видам контактной информации и по комментариям.

Способ вызова

Вызывается из форм документов и форм списков справочников: контрагенты и контакты при нажатии на кнопку "Найти клиента". Вызов также возможен при нажатии комбинации клавиш [Shift + Alt + F] из форм документов.
