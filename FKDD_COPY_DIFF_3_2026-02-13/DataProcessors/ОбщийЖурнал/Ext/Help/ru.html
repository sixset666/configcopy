Обработки "Общий журнал"

Назначение

Воспроизводит общий журнал документов. Список документов показываемых в этом журнале формируется динамически.

Относится к подсистеме: «Сервис».

Формы

  • Форма. Основная форма. Содержит список документов и основные действия списка документов (в верхней командной панели).
  • Отборы. Форма, в которой настраиваются отборы документов в общем журнале.
  • Поиск по номеру. Повторяет диалоговое окно "Поиск по номеру" в стандартных списках документов. Предназначена для поиска документов по номеру в общем журнале.
  • Вспомогательная. Вспомогательная форма для представления некоторых списочных данных.

Способ вызова

Вызывается из меню "Документы-->Общий журнал"
