Обработки "Печать этикеток и ценников"

Назначение

Эта обработка позволяет напечатать произвольное количество этикеток или ценников на выбранные товарные позиции. При такой распечатке можно использовать стандартный принтер Windows или специальное оборудование для печати (если такой специализированный принтер подключен к системе). 

Относится к подсистеме: «Работа с оборудованием».

Формы

  • Форма. Основная форма. Позволяет напечатать произвольное количество этикеток или ценников на выбранные товарные позиции.
    Товары можно добавлять вручную, с помощью кнопки "Подбор" или воспользоваться одним из вариантов заполнения с помощью кнопки "Заполнить".
    Выбрав закладку "Этикетки", можно напечатать этикетки для выбранных товаров. Количество отпечатанных этикеток будет соответствовать количеству, указанному в соответствующей колонке таблицы. При печати ценников (закладка "Ценники"), будет отпечатано по одному ценнику для каждой товарной позиции.
    Выбрав закладку "Коды", вы сможете распечатать список товаров, на которые необходимо наклеить этикетки.
    В случае использования специального принтера этикеток, вам необходимо выбрать один из заранее созданных шаблонов для печати. Шаблоны можно создавать и редактировать с помощью отдельной утилиты "Редактор шаблонов" (LP_Editor.exe)
  • Выбор варианта печати. Дополнительная форма. Позволяет выбрать вариант заполнения табличной части обработки номенклатурным составом товарного документа. 

Способ вызова

Вызывается по кнопке "Печать" формы товарных документов.
