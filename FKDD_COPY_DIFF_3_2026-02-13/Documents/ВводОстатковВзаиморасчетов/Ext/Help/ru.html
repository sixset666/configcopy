Документ "Ввод остатков взаиморасчетов"

Назначение

Документ предназначен для ввода начальных остатков по взаиморасчетам с контрагентами, имеющими дебиторскую или кредиторскую задолженность по различным договорам взаиморасчетов. 

При проведении документ формирует хозяйственную операцию:

  • Ввод остатков взаиморасчетов. Хозяйственная операция отражает начальные остатки взаимозачетов с контрагентом, имеющим дебиторскую или кредиторскую задолженность.

Относится к подсистеме: «Взаиморасчеты».

Параметры

Длина номера: 10.

Тип номера: Строка.

Периодичность номера: Год.

Реквизиты

  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Тип значения: «Число». Реквизит является обязательным для заполнения. Курс валюты документа, на дату документа.  
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Сумма дебет. Тип значения: «Число». Сумма дебиторской задолженности по документу.
  • Сумма кредит. Тип значения: «Число». Сумма кредиторской задолженности по документу.
  • Сумма документа. Тип значения: «Число». Сумма документа.

Табличная часть «Состав»

  • Контрагент. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Контрагент, для которого вводятся остатки по взаиморасчетам.
  • Договор взаиморасчетов. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор взаиморасчетов с контрагентом.
  • Курс взаиморасчетов. Тип значения: «Число». Курс взаиморасчетов, на дату документа. 
  • Сумма дебет. Тип значения: «Число». Сумма задолженности контрагента перед фирмой по данному договору.
  • Сумма кредит. Тип значения: «Число». Сумма задолженности фирмы перед контрагентом по данному договору.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Остатки взаимозачетов. Простая печатная форма документа.

Способ вызова

Вызывается из меню "Документы --> Взаиморасчеты --> Ввод остатков взаиморасчетов".

Движения

  • Остаточный регистр «Взаиморасчеты компании».
  • Остаточный регистр «Доходы и расходы».

Последовательности, изменяемые документом: "Взаиморасчеты".
