Произвольная форма "Заполнение по документу основанию"

Назначение

Произвольная форма документа «Бюджет оказания услуг по автоработам», предназначена для заполнения документа данными, на основании  документа «Бюджет оказания услуг по автоработам». 

Способы вызова

Форма вызывается из меню командной панели табличной части документа «Бюджет оказания услуг по автоработам»: "Заполнение по документу основанию", а также в случае ввода на основании штатными средствами платформы 1С.

Принцип работы с формой

В случае если заполняемый документ имеет вид бюджета "Бюджет оказания услуг по автоработам", то вся работа с формой сводится к выбору документа основания. В этом случае в качестве документа основания может выступать документ «Бюджет оказания услуг по автоработам»  с аналитикой по работам, причем периодичности сценариев планирования документов должны совпадать. Также должно выполняться условие, что период планирования документа основания должен быть не больше периода планирования заполняемого документа. Далее таблица формы заполнится табличной частью документа основания и будет недоступна для редактирования. При нажатии "ОК" данные таблицы скопируются в табличную часть заполняемого документа.
