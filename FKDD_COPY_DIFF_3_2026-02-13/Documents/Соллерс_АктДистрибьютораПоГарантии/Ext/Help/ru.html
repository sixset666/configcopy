Документ "Акт разногласий"

Назначение

Документ "Акт разногласий" предназначен для ввода корректировок в закрытый заказ-наряд. При этом документ заполняется выполненными работами и использованными при этом деталями. Возможно изменение количества, цен, скидок, нормочасов с перерасчетом суммы заказ-наряда. Документ может быть введен только на основании закрытого заказ-наряда с платным видом ремонта.

Список хозяйственных операций:

  • Акт разногласий. Хозяйственная операция начисляет дебиторскую задолженность плательщика по заказ-наряду или сторнирует его задолженость на регистре «Взаиморасчетов компании». Так же осуществляется увеличение проданных деталей/работ по заказ-наряду или сторнирование проданных деталей/работ по заказ-наряду на регистре «Продажи».

Относится к подсистеме: «Автосервис».

Параметры

Длина номера: 10

Периодичность номера: Год

Тип номера: Строка

Вводится на основании: «Заказ-наряд».

Объекты, вводимые на основании: «Счет на оплату», «Счет фактура выданный».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Менеджер. Ссылается на справочник «Сотрудники». Сотрудник, заключивший сделку с контрагентом.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  • Комментарий. Произвольный комментарий к документу.
  • Документ основание. Ссылается на документ «Заказ-наряд». Документ, на основании которого введен данный , необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Курс валюты документа.
  • Курс валюты упр. Реквизит является обязательным для заполнения. Курс валюты управленческого учета.
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип Цен, в которых отгружаются детали для выполнения работ.
  • Вид ремонта. Реквизит является обязательным для заполнения. Ссылается на справочник «Виды ремонта». Вид ремонта, производимого по заказ-наряду.
  • Заказчик. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты». Заказчик выполнения работ по заказ-наряду.
  • Автомобиль. Реквизит является обязательным для заполнения. Ссылается на справочник «Автомобили». Ремонтируемый автомобиль.
  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты». Плательщик по заказ-наряду.
  • Договор. Реквизит является обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор по которому ведутся взаиморасчеты с контрагентом.
  • Автоматическое закрытие сделок. Если флажок установлен, то при проведении документа сначала будут оплачены все незакрытые сделки.
  • Вид оплаты. Вид оплаты по акту разногласий.
  • Сумма. Сумма документа.
  • Сумма номенклатуры документа. Сумма запчастей по заказ-наряду.
  • Сумма работ документа. Сумма работ по заказ-наряду. 

Табличная часть «Товары»

  • Подтверждение. Флаг подтверждения оплаты позиции плательщиком.
  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Деталь для выполнения работ по заказ-наряду.
  • Количество. Количество деталей, подтвержденных плательщиком к оплате.
  • Количество по заказ-наряду. Количество деталей согласно заказ-наряда.
  • Количество разница. Разница количества деталей, подтвержденных плательщиком к оплате и количеством согласно заказ-наряда.
  • Единица измерения. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения деталей, подтвержденных плательщиком к оплате.
  • Единица измерения по заказ наряду. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения деталей, согласно заказ-наряда.
  • Коэффициент. Реквизит является обязательным для заполнения. Коэффициент пересчета по отношению к базовой единице номенклатуры, подтвержденных плательщиком к оплате.
  • Коэффициент по заказ наряду. Реквизит является обязательным для заполнения. Коэффициент пересчета по отношению к базовой единице номенклатуры согласно заказ-наряда.
  • Цена. Цена детали в валюте документа, подтвержденная плательщиком к оплате.
  • Сумма. Сумма деталей, подтвержденная плательщиком к оплате. Сумма всегда без скидки. НДС входит в сумму если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Ставка НДС. Ставка НДС, подтвержденная плательщиком к оплате. Ссылается на справочник «Ставки НДС».
  • Ставка НДС по заказ-наряду. Ставка НДС, согласно заказ-наряда. Ссылается на справочник «Ставки НДС».
  • Сумма НДС. Сумма НДС по номенклатурной позиции, подтвержденная плательщиком к оплате.
  • Сумма НДС по заказ-наряду. Сумма НДС по номенклатурной позиции, согласно заказ-наряда.
  • Разница сумм НДС. Разница сумм НДС, подтвержденной плательщиком к оплате и суммы НДС, согласно заказ-наряда.
  • Процент скидки. Процент скидки на деталь, подтвержденная плательщиком к оплате.
  • Сумма скидки. Сумма скидки на деталь, подтвержденная плательщиком к оплате.
  • Сумма скидки по заказ-наряду. Сумма скидки на деталь, согласно заказ-наряда.
  • Разница сумм скидки. Разница сумм скидки на деталь, подтвержденной плательщиком к оплате и суммы скидки, согласно заказ-наряда. 
  • Характеристика номенклатуры. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры.
  • Всего. Итоговая сумма по строке документа со всеми налогами, подтвержденная плательщиком к оплате.
  • Всего по закза-наряду. Итоговая сумма по строке документа со всеми налогами, согласно заказ-наряду.
  • Разница всего. Разница итоговых сумм по строке документа со всеми налогами, подтвержденная плательщиком к оплате и суммы, согласно заказ-наряда. 

Табличная часть «Работы»

  • Подтверждение. Флаг подтверждения оплаты позиции плательщиком.
  • Работа. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Автоработы». Работа для выполнения заказ-наряда.
  • Количество. Количество работ, подтвержденных плательщиком к оплате.
  • Количество по заказ-наряду. Количество работ, согласно заказ-наряда.
  • Количество разница. Разница количеств работ, подтвержденных плательщиком к оплате и количества, согласно заказ-наряда.
  • Нормочас. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Нормочасы». Нормочас работы, подтвержденный плательщиком к оплате.
  • Коэффициент. Норма времени выполнения одной работы (в часах), подтвержденная плательщиком к оплате.
  • Коэффициент по заказ наряду. Реквизит является обязательным для заполнения. Норма времени выполнения одной работы (в часах), согласно заказ-наряда.
  • Коэффициент разница. Разница норм времени выполнения одной работы, подтвержденной плательщиком к оплате и нормы времени, согласно заказ-наряда. 
  • Цена. Цена работы в валюте документа, подтвержденная плательщиком к оплате.
  • Сумма. Сумма работ, подтвержденная плательщиком к оплате. Сумма всегда без скидки. НДС входит в сумму если это указано в типе цен. Сумма всегда равна Цена*Количество*Коэффициент.
  • Ставка НДС. Ссылается на справочник «Ставки НДС». Ставка НДС по работе, подтвержденная плательщиком к оплате.
  • Ставка НДС по заказ-наряду. Ссылается на справочник «Ставки НДС». Ставка НДС по работе, согласно заказ-наряда. 
  • Сумма НДС. Сумма НДС по работе, подтвержденная плательщиком к оплате.
  • Сумма НДС по заказ-наряду. Сумма НДС по работе, согласно заказ-наряда.
  • Разница сумм НДС. Разница сумм НДС по работе, подтвержденной плательщиком к оплате и суммы НДС, согласно заказ-наряда.
  • Процент скидки. Процент скидки на работу, подтвержденный плательщиком к оплате.
  • Сумма скидки. Сумма скидки на работу, подтвержденная плательщиком к оплате.
  • Сумма скидки по заказ-наряду. Сумма скидки на работу, согласно заказ-наряда.
  • Разница сумм скидки. Разница сумм скидки на работу, подтвержденной плательщиком к оплате и суммы скидки, согласно заказ-наряда. 
  • Всего. Итоговая сумма по строке документа со всеми налогами, подтвержденная плательщиком к оплате.
  • Всего по заказ-наряду. Итоговая сумма по строке документа со всеми налогами, согласно заказ-наряда.
  • Разница всего. Разница итоговых сумм по строке документа со всеми налогами, подтвержденной плательщиком к оплате и всего, согласно заказ-наряда. 

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.
  • Форма документа. Основная форма документа. Используется для ввода и редактирования документов данного типа.

Печатные формы

  • Печатная форма. Описание печатной формы.

Способ вызова

Вызывается из меню "Документы --> Автосервис --> Акт разногласий"

Движения

  • «Взаиморасчеты компании». Взаиморасчеты компании с контрагентами.
  • «Доходы и расходы». Доходы и расходы компании.
  • «Продажи». Учет продаж товаров и оказания услуг.  

Дополнительная информация

Если после ввода акта разногласий заказ-наряд будет изменен - требуется перезаполнение акта разногласий.
