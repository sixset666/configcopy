Документ "Реализация активов"

Назначение

Документ предназначен для отражения в учете факта реализации нетоварных активов подразделением компании.

При проведении документ формирует хозяйственные операции:

  • Реализация активов.  Хозяйственная операция списывает с баланса все позиции нетоварных активов, хранимые на регистре «Прочие активы в эксплуатации, отражает остаточную стоимость списанных активов на регистре «Доходы и расходы». Разность суммы реализации и остаточной стоимости активов записывается по соответствующей статье в регистр «Доходы и расходы». Приходует сумму и себестоимость проданных товаров на регистре «Продажи». Если среди реализуемых активов есть автомобили, то их сумма и себестоимость приходуются на регистре «Продажи автомобилей». Начисляет дебиторскую задолженность покупателя перед компанией на регистре «Взаиморасчетов компании».

Относится к подсистеме: «Нетоварные активы».

Параметры

Длина номера: 10.

Тип номера: Строка.

Периодичность номера: Год.

Вводится на основании: «Ввод в эксплуатацию», «Перемещение активов».

Объекты, вводимые на основании: «Приходный кассовый ордер», «Корректировка», «Событие».

Реквизиты

  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Покупатель, которому реализуются активы.
  • Договор. Реквизит является обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор, по которому ведутся взаиморасчеты с покупателем.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Менеджер. Ссылается на справочник «Сотрудники». Менеджер - лицо, заключившее сделку.  
  • Документ основание. Ссылается на документы «Ввод в эксплуатацию», «Перемещение активов». Документ, на основании которого введен данный.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа.
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Сумма. Тип значения: «Число». Сумма документа.
  • Проект. Ссылается на справочник «Проекты». Проект, к которому относится документ.

Табличная часть «Активы»

  • Актив. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Прочие активы». Нетоварный актив.
  • Инвентарный номер. Тип значения: «Строка». Инвентарный номер актива.
  • Стоимость реализации. Тип значения: «Число». Стоимость реализации актива.
  • Процент НДС. Ссылается на справочник «Ставки НДС». Ставка НДС по позиции нетоварного актива.
  • НДС. Тип значения: «Число». Сумма НДС по позиции нетоварного актива.
  • Балансовая стоимость. Тип значения: «Число». Балансовая стоимость актива.
  • Начисленная амортизация. Тип значения: «Число». Начисленная стоимость актива.
  • Остаточная стоимость. Тип значения: «Число». Остаточная стоимость актива.
  • Сумма списания. Тип значения: «Число». Сумма списания актива.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Реализация активов. Простая печатная форма, содержащая список реализуемых нетоварных активов.
  • М-15 (Накладная на отпуск материалов на сторону). Унифицированная печатная форма №М-15, утвержденная постановлением Госкомстата России.

Способ вызова

Вызывается из меню "Документы --> Прочие активы --> Реализация активов"

Журналы, отображающие документ: «Прочие активы».

Движения

  • Остаточный регистр «Взаиморасчеты компании».
  • Остаточный регистр «Доходы и расходы».
  • Оборотный регистр «Продажи».
  • Оборотный регистр «Продажи автомобилей».
  • Остаточный регистр «Прочие активы в эксплуатации».

Последовательности, изменяемые документом: "Взаиморасчеты".
