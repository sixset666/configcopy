Документ "Акт сверки взаиморасчетов"

Назначение

Документ "Акт сверки взаиморасчетов" предназначен для согласования сумм задолженности контрагента или организации, по документам взаиморасчетов, при возникновении расхождения данных о задолженности между двумя сторонами.

Документ имеет две закладки "По данным организации" и  "По данным контрагента". Документ может быть заполнен по данным регистра «Взаиморасчеты компании» для организации.

В документе указывается контрагент, договор, документ, по которому образовалась задолженность, период получения данных о задолженности и сделка.

На закладке "По данным организации" графа "Увеличение долга" означает сумму долга контрагента перед организацией, а графа "Уменьшение долга" сумма долга организации перед контрагентом.

На закладке "По данным контрагента" графа "Увеличение долга" означает сумму долга организации перед контрагентом, а графа "Уменьшение долга" сумма долга контрагента перед организацией.

В одном документе может быть введено несколько строк по различным договорам, оформленным с контрагентом для этого договор в шапке документа должен быть незаполненным. Суммы вводятся в той валюте, которая определена, как валюта взаиморасчетов по договору с контрагентом.

Документ не выполняет никаких движений, он может быть распечатан и передан контрагенту для согласования.

После согласования данных с контрагентом, при необходимости, можно произвести корректировку задолженности с помощью документа "Корректировка долга"

 

Относится к подсистеме: «Взаиморасчеты».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Объекты, вводимые на основании: «Событие».

Реквизиты

  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Контрагент, долг которому мы корректируем.
  • Сверка согласована. Тип значения: «Булево». флаг означающий окончательное решение обеих сторон по процессу урегулирования суммы задолженности за текущий период или момент.
  • Договор. Если не заполнен, то необходимо ввести для каждой строки табличных частей. Ссылается на справочник «Договоры взаиморасчетов». Договор, по которому ведутся взаиморасчеты с контрагентом.
  • Период с. Тип значения: «Дата». Начальная дата периода получения данных о задолженности контрагента.
  • Период по. Реквизит является обязательным для заполнения. Тип значения: «Дата». Конечная дата периода получения данных о задолженности контрагента.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.  
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа, на дату документа.
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").
  • Приход. Тип значения: «Число». Сумма увеличения долга за период, по данным организации.
  • Расход. Тип значения: «Число». Сумма уменьшения долга за период, по данным организации.
  • Расхождение. Тип значения: «Число». Сумма расхождения данных организации и контрагента, за выбранный период.
  • Остаток на начало. Тип значения: «Число». Сумма долга на начальную дату.
  • Остаток на конец. Тип значения: «Число». Сумма долга на конечную дату.
  • Сумма приход по контрагенту. Тип значения: «Число». Сумма увеличения долга за период, по данным контрагента.
  • Сумма расход по контрагенту. Тип значения: «Число». Сумма уменьшения долга за период, по данным контрагента.
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Проект. Ссылается на справочник «Проекты». Проект, к которому относится документ.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Акт сверки. Простая печатная форма, содержит акт сверки по данным организации и контрагента и если сверка согласована то выводится результат сверки.

Способ вызова

Вызывается из меню "Документы --> Взаиморасчеты --> Акт сверки взаиморасчетов".
