Документ "Списание автомобилей"

Назначение

Документ предназначен для отражения в учете факта списания автомобилей с указанием склада, по которому происходит списание, по выбранной статье.

При проведении документ формирует хозяйственную операцию:

  • Списание автомобилей. Хозяйственная операция уменьшает остатки автомобилей, на складах предприятия в виде автомобилей с собственным или комиссионным признаком партии хранимые на регистре «Остатки автомобилей», с указанием статьи списания, и отражает сумму списанных автомобилей на регистре «Доходы и расходы».
  • Списание автомобилей отданных на комиссию. Хозяйственная операция аналогична предыдущей с той разницей, что уменьшает партии автомобилей, отданных на реализацию хранимые на регистре «Автомобили отданные» и список автомобилей, за которые комиссионер должен перед нами отчитаться на регистре «Реализованные автомобили». Сумма списанных автомобилей на доходах и расходах компании не отражается.

Относится к подсистеме: «Автосалон».

Параметры

Длина номера: 10

Периодичность номера: Год

Тип номера: Строка

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  • Комментарий. Произвольный комментарий к документу.
  • Контрагент. Ссылается на справочник «Контрагенты и контакты». Комиссионер, которому мы передали автомобили на комиссию.
  • Договор взаиморасчетов. Ссылается на справочник «Договоры взаиморасчетов». Договор, по которому ведутся взаиморасчеты с комиссионером.
  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад компании, на котором происходит списание.
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип цен, по которым происходит списание автомобилей.  
  • Хозяйственная операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Курс валюты документа.
  • Сумма. Сумма документа.
  • Подразделение затрат. Реквизит является обязательным для заполнения при списании автомобилей со склада. Ссылается на справочник «Подразделения компании». Подразделение компании, на которое списываются затраты. 
  • Статья списания ТМЦ. Реквизит является обязательным для заполнения. Ссылается на справочник «Статьи доходов и расходов». Статья списания товарно - материальных ценностей.

Табличная часть «Автомобили»

  • Автомобиль. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Автомобили». Автомобили, подлежащие списанию.
  • Количество. Реквизит является обязательным для заполнения. Количество автомобилей. Всегда равно 1.
  • Документ передачи. Ссылается на документы «Реализация автомобилей», «Ввод остатков автомобилей», «Инвентаризация автомобилей». Документ передачи автомобиля на комиссию.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Акт списания автомобилей. Печатная форма, содержащая список автомобилей списанных со склада, с указанием ответственных лиц.
  • ТОРГ-16. Унифицированная печатная форма №Торг-16, утвержденная постановлением Госкомстата России.

Способ вызова

Вызывается из меню "Документы --> Автосалон --> Складские операции --> Списание автомобилей".
