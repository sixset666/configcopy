Документ "Возврат автомобилей поставщику"

Назначение

Документ предназначен для отражения в оперативном учете операции по возврату поставщику или комитенту ранее купленных у него автомобилей. При вводе контрагента из его реквизитов автоматически подставляется в документ договор. Это значение потом можно поменять.

При проведении документ формирует хозяйственные операции:

  • Возврат автомобилей поставщику. Хозяйственная операция списывает кредиторские задолженности поставщика по другим сделкам, если таковые были и начисляет дебиторскую задолженность поставщика перед компанией на регистре «Взаиморасчеты компании», сторнируем материальные активы, со склада предприятия с собственным признакам партии, хранимые на регистре «Остатки автомобилей».
  • Возврат автомобилей поставщику (комиссия). Хозяйственная операция аналогична операции "Возврат автомобилей поставщику" за тем исключением, что дебиторская задолженность не начисляется, а сторнируется товарная задолженность нашей компании перед комитентом по регистру «Остатки автомобилей» с комиссионным признаком партии.

Относится к подсистеме: «Автосалон».

Параметры

Длина номера: 10

Периодичность номера: Год

Тип номера: Строка

Вводится на основании: «Поступление автомобилей».

Объекты, вводимые на основании: «Приходный кассовый ордер», «Счет фактура выданный».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании в котором был выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  • Комментарий. Произвольный комментарий к документу.
  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Поставщик или комитент, которому мы возвращаем товар.
  • Договор взаиморасчетов. Реквизит является обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор, по которому ведутся взаиморасчеты с контрагентом.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Курс валюты документа, на дату документа.
  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад компании, с которого списываются возвращаемые поставщику товары.
  • Документ основание. Ссылается на документ «Поступление товаров». Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).
  • Тип Цен. Ссылается на справочник «Типы цен». Тип Цен, по которым возвращаем товары.
  • Сумма. Сумма документа.
  • Хозяйственная операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.

Табличная часть «Автомобили»

  • Автомобиль. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Автомобили». Возвращаемый автомобиль.
  • Количество. Реквизит является обязательным для заполнения. Количество возвращаемых автомобилей. Всегда равно 1.
  • Цена. Цена автомобиля.
  • Сумма. Сумма всегда без скидки. НДС входит в сумму если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Сумма всего. Итоговая сумма по строке документа со всеми налогами.
  • Ставка НДС. Ссылается на справочник «Ставки НДС». Ставка НДС автомобиля.
  • Сумма НДС. Сумма НДС автомобиля.
  • Партия. Реквизит является уникальным и обязательным для заполнения. Ссылается на документ «Поступление автомобилей». Документ, согласно которому был приобретен автомобиль.

Табличная часть «Товары» (Оборудование)

  • Автомобиль. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Автомобили». Возвращаемый автомобиль.
  • Номенклатура.  Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (оборудование).
  • Количество. Реквизит является обязательным для заполнения. Количество оборудования установленного в автомобиль.
  • Единица измерения. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения оборудования.
  • Коэффициент. Реквизит является обязательным для заполнения. Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Цена. Цена оборудования.
  • Сумма. Сумма всегда без скидки. НДС входит в сумму если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Процент скидки. Процент скидки.
  • Сумма скидки. Сумма скидки по строке документа.
  • Сумма всего. Итоговая сумма по строке документа со всеми налогами.
  • Ставка НДС. Ссылается на справочник «Ставки НДС». Ставка НДС оборудования.
  • Сумма НДС. Сумма НДС оборудования.
  • Характеристика номенклатуры. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Накладная. Простая печатная форма, содержит список возвращаемых автомобилей поставщику или комитенту.
  • ТОРГ-12 (Товарная накладная за поставщика). Унифицированная печатная форма №Торг-12, утвержденная постановлением Госкомстата России.

Способ вызова

Вызывается из меню "Документы --> Автосалон --> Поступление автомобилей --> Возврат автомобилей поставщику".

Движения

  • Остаточный регистр «Взаиморасчеты компании».
  • Остаточный регистр «Остатки автомобилей».
  • Остаточный регистр «Доходы и расходы».
  • Остаточный  регистр «Комплектация автомобилей».
  • Оборотный  регистр «Продажи». 
  • Остаточный  регистр «Реализованные товары».
