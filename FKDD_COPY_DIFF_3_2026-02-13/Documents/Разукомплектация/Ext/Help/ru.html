Документ "Разукомплектация"

Назначение

Документ предназначен для отражения в учете операций по разукомплектации (разборке) комплекта - списанию комплектов и оприходованию на склад комплектующих. В табличной части документа не могут фигурировать услуги, работы и т.п. Признак товара должен быть только комплект. 

При проведении документ формирует хозяйственную операцию:

  • Разукомплектация. Хозяйственная операция списывает комплект со склада и затем приходует комплектующие на тот же склад с собственным признаком партии на регистрах «Остатки товаров компании» и «Партии товаров компании», приходует сумму отклонения себестоимости при отприходовании комплектующих, если таковая была выявлена, на регистр «Доходы и расходы». Если разукомплектация происходит на розничном складе, то устанавливаются розничные цены комплектующих на регистре сведений «Цены».

Относится к подсистеме: «Производство».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «Комплектация».

Объекты, вводимые на основании: «Поступление дополнительных расходов», «Расходный складской ордер», «Корректировка», «Приходный складской ордер», «Событие».

Реквизиты

  • Комплект. Реквизит является обязательным для заполнения. Ссылается на справочник «Номенклатура». Разбираемый комплект. Вид товара должен быть только комплект.
  • Количество. Тип значения: «Число». Количество разбираемых комплектов.
  • Цена комплекта. Тип значения: «Число». Цена разбираемого комплекта.
  • Цена комплекта розничная. Тип значения: «Число». Розничная цена разбираемого комплекта.
  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад компании, на котором происходит разукомплектация.
  • Характеристика. Реквизит является обязательным для заполнение, если для комплекта ведётся учёт по характеристикам. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия комплекта, например, цвет, размер и т.п. 
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ. 
  • Организация. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  • Автор. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Тип Цен. Ссылается на справочник «Типы цен». Тип цен.
  • Валюта. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Тип значения: «Число». Курс валюты документа.
  • Операция. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Сумма. Тип значения: «Число». Сумма документа.
  • Проект. Ссылается на справочник «Проекты». Проект, к которому относится документ.
  • Ячейка. Ссылается на справочник «Ячейки хранения». Ячейка хранения для комплекта. Значение по умолчанию хранится в номенклатуре в разрезе складов.

Табличная часть «Товары»

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар).
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество единиц измерения номенклатуры.
  • Количество всего. Тип значения: «Число». Общее количество номенклатуры.
  • Единица измерения. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Цена. Тип значения: «Число». Цена номенклатурной позиции.
  • Процент наценки. Тип значения: «Число». Наценка для вычисления розничной цены.
  • Цена розничная. Тип значения: «Число». Розничная цена для оприходования комплектующей на розничный склад.
  • Сумма. Тип значения: «Число». Сумма всегда без скидки. НДС входит в сумму, если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Сумма розничная. Тип значения: «Число». Розничная сумма для оприходвания Комплектующей на розничный склад.
  • Характеристика номенклатуры. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п.
  • Ячейка. Ссылается на справочник «Ячейки хранения». Ячейка хранения для номенклатуры. Значение по умолчанию хранится в номенклатуре в разрезе складов. 

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Разукомплектация. Простая печатная форма, содержит комплект и список комплектующих.
  • Печать этикеток и ценников. Обработка, позволяющая получить печатную форму этикеток и ценников списка комплектующих товаров.

Способ вызова

Вызывается из меню "Документы --> Производство--> Разукомплектация"

Движения

  • Регистр сведений «Цены».
  • Регистр сведений «Допроведение по партиям».
  • Остаточный регистр «Доходы и расходы».
  • Остаточный регистр «Остатки товаров компании».
  • Остаточный регистр «Партии товаров компании».

Последовательности, изменяемые документом: "Партии".
