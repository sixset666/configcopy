Документ "Заказ поставщику"

Назначение

Документ служит для отражения в учете заказов поставщикам. В табличной части документа не могут фигурировать услуги, работы и т.п. При вводе контрагента из его реквизитов автоматически подставляется в документ договор. Это значение потом можно поменять.

При проведении документ формирует хозяйственную операцию:

  • Заказ поставщику. Хозяйственная операция приходует заказы поставщику, по списку заказанных товаров, на регистр «Заказы поставщикам», если есть распределение товарных позиций, то они приходуются или списываются на регистре «Заказы распределение».

Относится к подсистеме: «Заказы на поставку».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «Счет на оплату», «Поступление товаров», «Заказ покупателя», «Заказ внутренний», «Бюджет закупок».

Объекты, вводимые на основании: «Корректировка заказа поставщику», «Платежное поручение», «Поступление товаров», «Расходный кассовый ордер», «Корректировка», «Счет на оплату», «Подтверждающие документы».

Реквизиты

  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Контрагент, у которого производится заказ номенклатуры.
  • Договор. Реквизит является обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор, по которому ведутся взаиморасчеты с контрагентом.
  • Срок поставки. Реквизит является обязательным для заполнения. Тип значения: «Дата». Срок получения товаров по заказу.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.  
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Документ основание. Ссылается на документы «Счет на оплату», «Поступление товаров», «Заказ покупателя», «Заказ внутренний». Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип цен, по которым заказываются товары.
  • Тип заказа. Ссылается на справочник «ТипыЗаказов». Один из возможных типов заказов.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Строка». Курс валюты документа, на дату документа.
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Сумма. Тип значения: «Строка». Сумма документа.
  • Проект. Ссылается на справочник «Проекты». Проект, к которому относится документ.
  • Учитывать кратность поставки. Тип значения: «Число». Если флаг установлен, то для номенклатурных позиций, для которых задана кратность поставки, заказываемое количество будет округлено вверх до ближайшего кратного количества.

Табличная часть «Товары»

  • Наличие замен. В колонке отображается иконка "Замены" в случае, когда данная номенклатура имеет замены или является заменой другой номенклатуры. По нажатию на колонку открывается обработка «Подбор замен».
  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар, услуга и т.п.).
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество единиц измерения номенклатуры.
  • Распределено. Тип значения: «Число». Количество товара, распределенного в заказ поставщику из заказов покупателей.
  • Единица измерения. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Цена. Тип значения: «Число». Цена номенклатурной позиции.
  • Сумма. Тип значения: «Число». Сумма всегда без скидки. НДС входит в сумму, если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Всего. Тип значения: «Число». Итоговая сумма по строке документа со всеми налогами.
  • Процент НДС. Ссылается на справочник «Ставки НДС». Ставка НДС по номенклатурной позиции.
  • НДС. Тип значения: «Число». Сумма НДС по номенклатурной позиции.
  • Характеристика номенклатуры. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п.  

Табличная часть «Распределение заказа»

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар).
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество единиц измерения номенклатуры.
  • Единица измерения. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Цена. Тип значения: «Число». Цена номенклатурной позиции.
  • Заказ. Реквизит является уникальным и обязательным для заполнения. Ссылается на документы: «Заказ внутренний», «Заказ покупателя». Заказ покупателя или внутренний заказ, под который осуществляется заказ данной номенклатуры.
  • Характеристика номенклатуры. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Заказ поставщику. Простая печатная форма, которая содержит список заказанных товаров.
  • Печать этикеток и ценников. Обработка, позволяющая получить печатную форму этикеток и ценников заказанных товаров.

Способ вызова

Вызывается из меню "Документы --> Заказы и резервы --> Заказ поставщику".

Журналы, отображаемые документ: «Поставщики», «Заказы и резервы».

Движения

  • Остаточный регистр «Заказы поставщикам».
  • Остаточный регистр «Заказы распределение».
