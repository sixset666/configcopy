Документ "Кассовый чек коррекции"

Назначение

Документ "Кассовый чек коррекции" служит для коррекции суммы в оборудовании по средством пробтия чека коррекции.

Параметры

Длина номера: 10

Периодичность номера: Год

Тип номера: Строка

Реквизиты

  • Ответственный. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Пользователь, оформивший документ  и ответственный за его обработку.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение компании. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в которой был выписан документ.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.   
  • Тип расчета. Реквизит является обязательным для заполнения. Тип значения: «ПеречислениеСсылка.ТипыРасчетаДенежнымиСредствами». Коррекция прихода, операция, при которой пользователь вносит денежные средства коррекции. Коррекция расхода, операция, при которой пользователь изымает денежные средства коррекции.
  • Тип коррекции. Тип значения: «Число». Самостоятельная операция, или операция по предписанию.
  • Номер чека ККМ. Тип значения: «Число». Номер чека, пробитого на фискальном регистраторе.
  • Предписание. Тип значения: «Строка». Текст предписания.
  • Сумма документа. Тип значения: «Число». Итоговая сумма документа.
  • Пробит чек. Тип значения: «Булево». Признак пробития чека ККМ.
  • Основания для коррекции. Реквизит является обязательным для заполнения. Тип значения: «Строка». Документ основание для коррекции.
  • Дата документа основания. Реквизит является обязательным для заполнения. Тип значения: «Дата». Дата документа основания для коррекции.
  • Номер документа основания. Реквизит является обязательным для заполнения. Тип значения: «Строка». Номер документа основания для коррекции.
  • ККТ. Ссылается на справочник «Оборудование». Фискальный регистратор, на котором пробивается чек.
  • Представление основания. Тип значения: «Строка». Представление документа основания.
  • Система налогообложения. Тип значения: «ПеречислениеСсылка.ТипыСистемНалогообложенияККТ». Тип системы налооблажения.
  • Сумма расчета без НДС. Тип значения: «Число». Cумма расчета по чеку без НДС.
  • Сумма расчета НДС 0%. Тип значения: «Число».  Cумма расчета по чеку с НДС по ставке 0%
  • Сумма НДС 10%. Тип значения: «Число». Cумма НДС чека по ставке 10%
  • Сумма НДС 18%(20%). Тип значения: «Число». Cумма НДС чека по ставке 18%(20%)
  • Сумма НДС 10/110. Тип значения: «Число».  Cумма НДС чека по расч. ставке 10/110
  • Сумма НДС 18/118. Тип значения: «Число». Cумма НДС чека по расч. ставке 18/118
  • Хозяйственная операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.

Табличная часть «Оплата»

  • Тип оплаты. Реквизит является уникальным и обязательным для заполнения. Ссылается на перечисление «Типы оплаты ККТ». Вид оплаты.
  • Сумма. Реквизит является обязательным для заполнения. Тип значения: «Число». Сумма по типу оплаты.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Документы --> Движения денежных средств --> Кассовый чек коррекции".
