Документ "Списание товаров"

Назначение

Документ предназначен для отражения в учете факта списания товаров с указанием склада, по которому происходит списание, по выбранной статье. В табличной части документа не могут присутствовать услуги, работы и т.п.

При проведении документ формирует хозяйственную операцию:

  • Списание товаров. Хозяйственная операция уменьшает материальные активы на складах предприятия в виде ТМЦ с собственным или комиссионным признаком партии, хранимые на регистрах «Остатки товаров компании» и «Партии товаров компании», с указанием статьи списания, и отражает сумму списанных товаров на регистре «Доходы и расходы», а также, если списываемые товары были приняты нами ранее на комиссию, то они приходуются на регистр  «Реализованные товары», как товары проданные, но за которые мы ещё не отчитались перед комитентом.
  • Списание товаров отданных на комиссию. Хозяйственная операция аналогична предыдущей с той разницей, что уменьшает партии товаров, отданных на реализацию, хранимых на регистре «Партии товаров отданные» . Сумма списанных товаров отражается на регистре «Доходы и расходы».
  • Списание товаров в производство. Хозяйственная операция аналогична предыдущей с той разницей, что уменьшает партии товаров, отданных на реализацию, хранимых на регистре «Партии товаров отданные» . Сумма списанных товаров отражается на регистре «Доходы и расходы».

Относится к подсистеме: «Складские акты».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «Ввод остатков товаров», «Заказ-наряд», «Постулпение товаров», «Расходный складской ордер».

Объекты, вводимые на основании: «Расходный складской ордер», «Корректировка», «Событие», «Электронное письмо».

Реквизиты

  • Статья списания. Реквизит является обязательным для заполнения. Ссылается на справочник «Статьи доходов и расходов». Статья списания товарно-материальных ценностей.
  • Контрагент. Ссылается на справочник «Контрагенты и контакты». Комиссионер, которому мы передали товары на комиссию.
  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад компании, на котором происходит списание.
  • Договор взаиморасчетов. Ссылается на справочник «Договоры взаиморасчетов». Договор, по которому ведутся взаиморасчеты с комиссионером.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.  
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Документ основание. Ссылается на документы «Расходный складской ордер». Документ, на основании которого введен данный.
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип цен, по которым происходит списание товара.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа.
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Подразделение затрат. Реквизит является обязательным для заполнения при списании товаров со склада. Ссылается на справочник «Подразделения компании». Подразделение компании, на затраты которого ложаться списанные товары. 
  • Статья списания ТМЦ. Реквизит является обязательным для заполнения. Ссылается на справочник «Статьи доходов и расходов». Статья списания товарно - материальных ценностей.
  • Сумма. Тип значения: «Число». Сумма документа.
  • Проект. Ссылается на справочник «Проекты». Проект, к которому относится документ.

Табличная часть «Товары»

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар, услуга и т.п.).
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество единиц измерения номенклатуры.
  • Остаток. Тип значения: «Число». Количество номенклатуры, имеющейся на складе.
  • Единица. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Цена розничная. Тип значения: «Число». Розничная цена для переоценки остатков перед списанием.
  • Сумма розничная. Тип значения: «Число». Розничная сумма.
  • Документ передачи. Ссылается на документы: «Инвентаризация», «Реализация товаров». Документ передачи товаров на комиссию.
  • Характеристика номенклатуры. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п.
  • Сумма списания. Тип значения: «Число». Сумма всегда без скидки. НДС входит в сумму, если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Партия. Партия, с которой списывается товар.
  • ГТД / РНПТ. Ссылается на справочник «ГТД». Номер грузовой таможенной декларации (для автомобилей поступивших по импорту) или регистрационный номер (РНПТ) для прослеживаемых товаров.
  • Ячейка. Ссылается на справочник «Ячейки хранения». Ячейка хранения для номенклатуры. Значение по умолчанию хранится в номенклатуре в разрезе складов. 

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Акт списания. Печатная форма, содержащая список товаров списанных со склада, с указанием ответственных лиц.
  • ТОРГ-16. Унифицированная печатная форма №Торг-16, утвержденная постановлением Госкомстата России.
  • Печать этикеток и ценников. Обработка, позволяющая получить печатную форму этикеток и ценников списываемых товаров.

Способ вызова

Вызывается из меню "Документы --> Складские документы --> Списание товаров".

Журналы, отображающие документ: «Складские документы», «Складские акты».

Движения

  • Регистр сведений «Допроведение по партиям».
  • Остаточный регистр «Доходы и расходы».
  • Остаточный регистр «Заказы покупателей». 
  • Остаточный регистр «Остатки товаров компании».
  • Остаточный регистр «Партии товаров компании».
  • Остаточный регистр «Партии товаров отданные».
  • Остаточный регистр «Реализованные товары».

Последовательности, изменяемые документом: "Партии".
