Документ "Комплектация"

Назначение

Документ предназначен для отражения в учете операций по комплектации - оприходованию на склад комплектов, списанию комплектующих. В табличной части документа не могут фигурировать услуги, работы и т.п. При проведении документ образует партию с признаком "Собственный товар".

При проведении документ формирует хозяйственную операцию:

  • Комплектация. Хозяйственная операция списывает комплектующие со склада и затем приходует комплект на тот же склад с собственным признаком партии на регистр «Остатки товаров компании» и «Партии товаров компании», приходует сумму отклонения себестоимости при оприходовании комплекта, если таковая была выявлена, на регистр «Доходы и расходы».
  • Комплектация в производство. Хозяйственная операция списывает комплектующие со склада и затем приходует комплект в производство по соответствующему заказ-наряду с собственным признаком партии на регистр  «Товары в производстве», приходует сумму отклонения себестоимости при оприходовании комплекта, если таковая была выявлена, на регистр «Доходы и расходы».

Относится к подсистеме: «Производство».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Объекты, вводимые на основании: «Разукомплектация», «Приходный складской ордер», «Корректировка», «Поступление дополнительных расходов», «Расходный складской ордер», , «Событие».

Реквизиты

  • Комплект. Реквизит является обязательным для заполнения. Ссылается на справочник «Номенклатура». Собираемый комплект. Вид товара должен быть комплект.
  • Склад компании. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад компании, на котором происходит операция комплектации.
  • Заказ-наряд. Реквизит является обязательным для заполнения при хоз.операции "Комплектация в производство". Ссылается на документ «Заказ-наряд». Заказ-наряд, для которого происходит комплектация в производство.
  • Цех. Реквизит является обязательным для заполнения при хоз.операции "Комплектация в производство". . Ссылается на справочник «Цеха». Цех, для которого происходит операция комплектации в производство.
  • Количество. Тип значения: «Число». Количество собираемых комплектов.
  • Цена комплекта. Тип значения: «Число». Цена комплекта.
  • Характеристика. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия собираемого комплекта, например, цвет, размер и т.п.
  • Розничная цена комплекта. Тип значения: «Число». Розничная цена собираемого комплекта.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ. 
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.   
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип Цен, в которых отгружается товар.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа, на дату документа.
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").
  • Сумма. Тип значения: «Число». Сумма документа.
  • Проект. Ссылается на справочник «Проекты». Проект, к которому относится документ.
  • Ячейка. Ссылается на справочник «Ячейки хранения». Ячейка хранения для комплекта. Значение по умолчанию хранится в номенклатуре в разрезе складов.

Табличная часть «Товары»

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (составляющие комплекта).
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество единиц измерения номенклатуры в комплекте.
  • Количество всего. Тип значения: «Число». Общее количество единиц измерения номенклатуры.
  • Единица. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Цена. Тип значения: «Число». Цена номенклатурной позиции.
  • Сумма. Тип значения: «Число». Сумма равна Цена*Количество.
  • Цена розничная. Тип значения: «Число». Розничная цена комплектующей.
  • Сумма розничная. Тип значения: «Число». Сумма розничная для комплектующей, равна Цена розничная*Количество.
  • Характеристика номенклатуры. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п.
  • Ячейка. Ссылается на справочник «Ячейки хранения». Ячейка хранения для номенклатуры. Значение по умолчанию хранится в номенклатуре в разрезе складов. 

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Комплектация. Простая печатная форма, содержит комплект и список комплектующих.
  • Печать этикеток и ценников. Обработка, позволяющая получить печатную форму этикеток и ценников списка комплектующих товаров.

Способ вызова

Вызывается из меню "Документы --> Производство --> Комплектация".
