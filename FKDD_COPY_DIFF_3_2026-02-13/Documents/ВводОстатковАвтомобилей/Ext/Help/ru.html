Документ "Ввод остатков автомобилей"

Назначение

Документ предназначен для ввода начальных остатков автомобилей собственных, принятых на комиссию и переданных на комиссию.

При проведении документ формирует хозяйственную операцию:

  • Ввод остатков автомобилей. Хозяйственная операция отражает начальные остатки автомобилей.
  • Ввод остатков автомобилей переданных на комиссию. Хозяйственная операция отражает начальные остатки автомобилей переданных на комиссию.
  • Ввод остатков автомобилей принятых на комиссию. Хозяйственная операция отражает начальные остатки автомобилей принятых на комиссию.

Относится к подсистеме: «Автосалон».

Параметры

Длина номера: 10, тип номера: Строка.

Периодичность номера: Год

Тип номера: Строка

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании в котором был выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Контрагент, который поставил или принял автомобили.
  • Договор взаиморасчетов. Реквизит является обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор взаиморасчетов с контрагентом.
  • Склад компании. Склад компании при поступлении автомобилей, при передачи автомобилей на комиссию реквизит склад компании не используется. Ссылается на справочник «Склады компании».
  • Тип цен. Тип цен для автомобилей. Ссылается на справочник «Типы цен».
  • Комментарий. Произвольный комментарий к документу.
  • Сумма документа. Сумма документа.
  • Хоз операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Курс валюты документа, на дату документа.
  • Статья оприходования ТМЦ. Ссылается на справочник «Статьи доходов и расходов». Статья дохода по которой приходуются автомобили.

Табличная часть «Автомобили»

  • Автомобиль. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Автомобили». Автомобиль, для которого вводятся остатки.
  • Количество. Реквизит является обязательным для заполнения. Количество остатков автомобиля. Всегда равняется 1 и используется для расчета итогов по количеству автомобилей.
  • Цена. Цена автомобиля.
  • Сумма. Сумма. НДС входит в сумму, если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Сумма всего. Итоговая сумма по строке документа со всеми налогами.
  • Ставка НДС. Ссылается на справочник «Ставки НДС». Ставка НДС для автомобиля.
  • Сумма НДС. Сумма НДС автомобиля.
  • ГТД / РНПТ. Ссылается на справочник «ГТД». Номер грузовой таможенной декларации (для автомобилей поступивших по импорту) или регистрационный номер (РНПТ) для прослеживаемых товаров.
  • Автомобиль Б\У. Признак подержанного автомобиля.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Остатки автомобилей. Простая печатная форма документа.

Способ вызова

Вызывается из меню "Документы --> Автосалон --> Поступление автомобилей --> Ввод остатков автомобилей".

Движения

  • Регистр накопления «Автомобили отданные».
  • Регистр накопления «Доходы и расходы».
  • Регистр накопления «Остатки автомобилей».
