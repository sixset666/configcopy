Документ "Счет-фактура выданный"

Назначение

Документ "Счет-фактура выданный" всегда вводится на основании другого документа. Информация в документе "Счет-фактура выданный" заполняется в соответствии с информацией, введенной в документе-основании. Для ввода документа "Счет-фактура выданный" в документах, на основании которых он вводится, предусмотрена гиперссылка "Ввести счет-фактуру". В случае оформления авансового платежа, документ счет-фактура вводится на основании платежных документов «Приходный кассовый ордер» или «Банковская выписка».

Документ содержит хозяйственные операции:

  • Счет-фактура выданный. Не создает движений. 
  • Корректировочный счет-фактура выданный. Не создает движений.  

Относится к подсистеме: «Отгрузка ТМЦ».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «Реализация товаров», «Реализация автомобилей», «Заказ-наряд», «Приходный кассовый ордер», «Отчет комиссионера», «Отчет комиссионера за автомобили» «Возврат поставщику», «Возврат поставщику автомобилей», «Выписка», «Отчет комитенту», «Отчет комитенту за автомобили», «Акт разногласий»,  «Закрытие смены», «Корректировка реализации», «Корректировка реализации автомобилей», «Чек», «Чек на оплату».

Объекты, вводимые на основании: «Корректировка», «Событие», «Электронное письмо».

Реквизиты

  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Контрагент, которому выдается счет-фактура.
  • Договор. Реквизит является обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор, по которому ведутся взаиморасчеты с контрагентом.
  • Документ основание. Ссылается на документы: «Возврат поставщику», «Выписка», «Реализация товаров», «Отчет комиссионера», «Отчет комитенту», «Приходный кассовый ордер». Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).  
  • Грузополучатель. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Контрагент, фактический грузополучатель.
  • Платежные документы. Тип значения: «Строка». Строковое описание платежно-расчетных документов.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.  
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип цен, в которых был продан товар.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа.
  • Операция. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Сумма. Тип значения: «Число». Сумма документа.
  • Проект. Ссылается на справочник «Проекты». Проект, к которому относится документ.
  • Не выставляется – флажок устанавливается в случае, если счет-фактура подлежит регистрации в Журнале регистрации, но при этом не выставляется покупателям.
  • Выставлен - флажок устанавливается в случае, если счет-фактура передан покупателю и подлежит регистрации в Журнале регистрации
    • Если флажок установлен, то необходимо указать дату передачи счета-фактуры покупателю,
    • Способ выставления – выбирается значение На бумажном носителе или В электронном виде.
  • Исправление номер, дата – указать номер исправления. Устанавливается автоматически при вводе на основании документа Корректировка реализации с операцией Исправление в первичных документах
    • Если флажок установлен, то необходимо указать номер и дату корректировочного счета-фактуры, для которого регистрируется исправление. Данные заполняются автоматически при вводе на основании документа Корректировка реализации,
  • К счету фактуре номер, дата - необходимо указать номер и дату счета-фактуры, для которого ранее был зарегистрирован корректировочный счет-фактура. Данные заполняются автоматически при вводе на основании документа Корректировка реализации,
  • Флажок С учетом исправления устанавливается в случае, если регистрируется корректировочный счет-фактура на исправленный счет-фактуру. Устанавливается автоматически при вводе на основании документа Корректировка реализации
    • Если флажок установлен, то необходимо указать номер и дату исправления счета-фактуры, для которого регистрируется корректировочный счет-фактура.

     

Табличная часть «Товары»

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар, услуга и т.п.).
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество единиц измерения номенклатуры.
  • Единица. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Цена. Тип значения: «Число». Цена номенклатурной позиции.
  • Сумма. Тип значения: «Число». Сумма всегда без скидки. НДС входит в сумму, если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Всего. Тип значения: «Число». Итоговая сумма по строке документа со всеми налогами.
  • Процент НДС. Ссылается на справочник «Ставки НДС». Ставка НДС по номенклатурной позиции.
  • НДС. Тип значения: «Число». Сумма НДС по номенклатурной позиции.
  • ГТД / РНПТ. Реквизит является уникальным. Ссылается на справочник «ГТД». Номер грузовой таможенной декларации (для товаров, поступивших по импорту) или регистрационный номер (РНПТ) для прослеживаемых товаров.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Счет-фактура.  Печатная форма счета-фактуры, утвержденная постановлением Правительства Российской Федерации.
  • Корректировочный счет-фактура. Печатная форма счета-фактуры, утвержденная постановлением Правительства Российской Федерации.

Способ вызова

Вызывается из меню "Документы --> Отгрузка ТМЦ --> Счет-фактура выданный".
