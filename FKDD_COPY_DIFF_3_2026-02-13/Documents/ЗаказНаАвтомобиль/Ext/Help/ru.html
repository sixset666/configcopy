Документ "Заказ на автомобиль"

Назначение

Документ служит для отражения в учете заказов покупателей на автомобили.

Есть возможность формирования заказа на автомобиль с одновременной фиксацией приема автомобиля от покупателя в зачет нового автомобиля (Trade-In). Для этого на закладке "Автомобиль" требуется установить флажок "Trade-In". На появившейся закладке можно ввести предварительные данные об автомобилей, который будет поставлен покупателем в зачет нового автомобиля. Эти данные носят информативный характер и могут быть отражены в отчетах по заказам. Вводом на основании заказа на автомобиль документа поступления автомобилей через пункт панели управления "Поступление автомобиля по Trade-In" может быть создан связанный документ поступления автомобилей от покупателя в зачет нового автомобиля. Данная связка так же может быть организованна вручную путем задания дополнительных свойств документа поступления автомобилей "Trade-In". Для зачета суммы поступившего автомобиля на основании заказа на автомобиль или связанного с ним документа поступления автомобилей вводиться документ «Взаимозачет», переносящий нашу задолженность за поставленный автомобиль на заказ на автомобиль, тем самым закрывая задолженность покупателя за приобретаемый им автомобиль.

При проведении документ формирует хозяйственные операции:

  • Заказ клиента на автомобиль. Хозяйственная операция приходует заказы покупателя на автомобили по списку заказанных товаров на регистр «Заказы автомобилей». Если автомобиль имеется в свободном остатке на складе, то происходит автоматическое его резервирование.
  • Изменение автомобиля.  Хозяйственная операция позволяет оформить заказ на автомобиль для изменения заказываемого автомобиля.
  • Переуступка заказа на автомобиль.  Хозяйственная операция позволяет оформить заказ на автомобиль для изменения заказываемого автомобиля. 
  • Отмена заказа на автомобиль.  Хозяйственная операция выполняется отмену заказа на автомобиль, на основании которого введен на регистре «Заказы автомобилей». Если автомобиль был зарезервированы под меняемый заказ на автомобиль, то происходит автоматическое снятие резерва.

Относится к подсистеме: «Автосалон».

Параметры

Длина номера: 10

Периодичность номера: Год

Тип номера: Строка

Объекты, вводимые на основании: «Жалоба клиента», «Приходный кассовый ордер», «Поступление автомобилей», «Снятие резервов автомобилей», «Реализация автомобилей», «Заявка на ремонт», «Заказ-наряд», «Счет на оплату за автомобили», «Взаимозачет», «Платежное поручение», «План поступления ДС», «Чек на оплату», «Заказ поставщику на автомобиль», справочник «Автомобили».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Комментарий. Произвольный комментарий к документу.
  • Документ основание. Ссылается на документы «Рабочий лист» и "Заказ на автомобиль". Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).
  • Покупатель. Реквизит является обязательным для заполнения в случае заказа автомобиля контрагенту. Ссылается на справочник «Контрагенты и контакты». Контрагент, оплачивающий заказ.
  • Договор с покупателем. Реквизит является обязательным для заполнения в случае заказа автомобиля контраненту. Ссылается на справочник «Договоры взаиморасчетов». Договор по которому ведутся взаиморасчеты с покупателем.
  • Заказчик. Реквизит является обязательным для заполнения в случае заказа автомобиля контраненту. Ссылается на справочник «Контрагенты и контакты». Контрагент, размещающий заказ.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Курс валюты документа, на дату документа. 
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип Цен, по которым происходит заказ товаров.
  • Сумма. Сумма документа.
  • Хоз операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Скидка наценка. Ссылается на справочник «Типы скидок и наценок». Тип скидки, предоставляемой контрагенту на дополнительное оборудование.
  • Процент предоплаты. Процент предоплаты по заказу.
  • Сумма предоплаты. Сумма предоплаты по заказу.
  • Срок поставки. Реквизит является обязательным для заполнения. Срок отгрузки автомобиля по заказу контрагенту. Срок подставляется по умолчанию из права "Срок поставки покупателю по умолчанию".
  • Срок снятия резерва. Срок, после которого резерв автомобиля может быть снят.
  • Вид оплаты. Реквизит является обязательным для заполнения. Ссылается на перечисление «Виды оплаты». Вид оплаты заказа контрагентом.
  • Менеджер. Ссылается на справочник «Сотрудники». Сотрудник, заключивший сделку с контрагентом.
  • Статус автомобиля. Ссылается на справочник «Статусы автомобилей». Текущее состояние выполнение заказа.
  • Тип салона. Ссылается на справочник «Типы салона». Тип салона заказываемого автомобиля.
  • Автомобиль. Реквизит является обязательным для заполнения. Ссылается на справочник «Автомобили». Автомобиль, на который оформляется заказ. Если автомобиль не выбран, то будет создан новый элемент справочника автомобилей с параметрами, указанными в заказе. 
  • Модель автомобиля. Реквизит является обязательным для заполнения. Ссылается на справочник «Модели автомобилей». Модель заказываемого автомобиля.
  • Вариант комплектации автомобиля. Ссылается на справочник «Варианты комплектации». Вариант комплектации модели автомобиля.
  • Цвет автомобиля. Ссылается на справочник «Цвета». Цвет заказываемого автомобиля.
  • Код лакокрасочного покрытия. Код лакокрасочного покрытия согласно спецификации производителя.  
  • Внутренний номер заказа. Внутренний номер заказа в компании. Например, внешний номер заказа поставщику, введенного на основании заказа покупателя.
  • Цена автомобиля продажная. Цена продажи автомобиля покупателю.
  • Маркетинговая программа. ёСсылается на справочник «Маркетинговые программы». Маркетинговая программа, согласно которой осуществляется продажа автомобиля.
  • Скидка/наценка на автомобиль. Ссылается на справочник «Типы скидок и наценок». Скидка предоставляемая клиенту на автомобиль.
  • Процент скидки на автомобиль. Процент скидки, предоставляемой клиенту на автомобиль.
  • Сумма скидки на автомобиль. Сумма скидки, предоставляемой клиенту на автомобиль.
  • Ставка НДС на автомобиль. Ссылается на справочник «Ставки НДС». Ставка НДС на автомобиль.
  • Сумма НДС на автомобиль. Сумма НДС на автомобиль.
  • Сумма всего на автомобиль. Сумма автомобиля со скидкой и налогами.
  • Скидка наценка на опции. Ссылается на справочник «Типы скидок и наценок». Скидка покупателю на опции производителя.
  • Банк. Имеет составной тип. Ссылается на справочник "Банки" или "Контрагенты". Банк, связанный с рабочим листом.
  • Рабочий лист. Ссылается на документ "Рабочий лист". Рабочий лист, по которому используется заказ на автмобиль.
  • Рабочий лист кредитного отдела. Ссылается на документ "Рабочий лист кредитного отдела".
  • Себестоимость автомобиля. Себестоимость автомобиля, купленного у физ. лица. Формируется при выборе автомобиля, который находится в наличии. Имеется возможность редактирования себестоимости если была она изменена. При этом важен контроль себестоимости для корректного рассчета суммовых показателей при вводе реализации автомобиля.

Табличная часть «Опции»

  • Опция. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Опции автомобилей». Опции, которые будут установлены на автомобиль поставщиком.
  • Количество. Реквизит является обязательным для заполнения. Количество устанавливаемых опций.
  • Цена. Цена единицы опции.
  • Сумма. Сумма всегда без скидки. НДС входит в сумму если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Процент скидки. Процент скидки, предоставляемой контрагенту на данную опцию.
  • Сумма скидки. Итоговая сумма скидки по строке документа.
  • Ставка НДС. Ссылается на справочник «Ставки НДС». Ставка НДС на опцию.
  • Сумма НДС. Сумма НДС на опцию.
  • Сумма всего. Итоговая сумма по строке документа со всеми налогами.

Табличная часть «Товары» (Оборудование автомобиля)

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Оборудование, устанавливаемое на автомобиль нашей организацией.
  • Количество. Реквизит является обязательным для заполнения. Количество единиц измерения оборудования.
  • Единица измерения. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения оборудования.
  • Коэффициент. Реквизит является обязательным для заполнения. Коэффициент пересчета оборудования по отношению к базовой единице измерения.
  • Цена. Цена продажи оборудования.
  • Сумма. Сумма всегда без скидки. НДС входит в сумму если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Процент скидки. Процент скидки, предоставляемой контрагенту на данное оборудование.
  • Сумма скидки. Итоговая сумма скидки по строке документа.
  • Ставка НДС. Ссылается на справочник «Ставки НДС». Ставка НДС на оборудование.
  • Сумма НДС. Сумма НДС на оборудование.
  • Характеристика номенклатуры. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия дополнительного оборудования, например, цвет, размер и т.п.
  • Сумма всего. Итоговая сумма по строке документа со всеми налогами.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Заказ. Простая печатная форма. Выводит информацию о заказаном автомобиле и его оборудовании.
  • Договор продажи. Печатная форма договора о продаже автомобиля.
  • Акт осмотра автомобиля с пробегом. Акт осмотра автомобиля при поступлении по Trade-In. Доступен для печати только если установлен флаг "Trade-In" и заполнена модель автомобиля для поступления.

Способ вызова

Вызывается из меню "Документы --> Автосалон --> Заказы на автомобили --> Заказ на автомобиль".
