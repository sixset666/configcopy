Документ "Корректировка бонусов"

Назначение

Документ служит для ручного корректирования бонусных баллов контрагента в регистре «Бонусные баллы».

 

Документ содержит хозяйственную операцию:

  • Корректировка бонусов. Создает движения по регистру «Бонусные баллы».

Относится к подсистеме: «Бонусные программы».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: нет.

Объекты, вводимые на основании: «SMS сообщение», «Электронное письмо».

Реквизиты

  • Бонусная программа. Ссылается на справочник «Бонусные программы». Бонусная программа используется для заполнения документа.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Проект. Ссылается на справочник «Проекты». Проект, к которому относится документ.

Табличная часть «Бонусы к начислению»

  • Карточка. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Карточки». Бонусная карта для которой происходит корректирование.
  • Дата списания. Реквизит является обязательным для заполнения. Тип значения: «Дата». Дата списания бонусов в регистре.
  • Количество. Тип значения: «Число». Активные бонусные баллы.
  • Количество к начислению. Тип значения: «Число». Неактивные бонусные баллы.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Нет.

Способ вызова

Вызывается из меню "Все операции--> Документы --> Корректировка бонусов".
