Документ "Пересортица товаров"

Назначение

Документ предназначен для устранения возникающей пересортицы. Пересортицу можно учесть как на оптовом, так и на розничном складе. В случае розничного склада, имеется возможность установить розничные цены приходуемого товара, для этого появляются две дополнительные колонки цена и сумма розничная.

При проведении документ формирует хозяйственную операцию:

  • Пересортица товаров. Хозяйственная операция списывает недостающий товар и приходует имеющийся на выбранный склад, в том случае, если была обнаружена пересортица, на регистрах «Остатки товаров компании» и «Партии товаров компании», приходуем или списываем суммовую разницу, выявленную при пересортице, на регистр «Доходы и расходы», а так же установка розничных цен, в случае изменения цен на розничном складе на регистре «Цены».

Относится к подсистеме: «Ценообразование».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Объекты, вводимые на основании: «Корректировка», «Приходный складской ордер», «Расходный складской ордер», «Событие», «Переоценка».

Реквизиты

  • Склад компании. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад компании, на котором произошла пересортица.
  • Причина. Тип значения: «Строка». Предназначен для указания причины пересортицы.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип цен.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа. 
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа, на дату документа.  
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция. 
  • Сумма. Тип значения: «Число». Сумма документа.
  • Проект. Ссылается на справочник «Проекты». Проект, к которому относится документ.

Табличная часть «Товары»

  • Расходуем. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура, которую расходуем.
  • Характеристика расход. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия расходуемой номенклатуры, например, цвет, размер и т.п.
  • Партия. Реквизит является обязательным для заполнения, если у пользователя установлено право "Выборочное списание партий". Ссылается на ПВХ «Типы партий». Партия, с которой будет списан товар.
  • Единица измерения расход. Реквизит является обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения расходуемого товара.
  • Коэффициент расход. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Остаток расход. Тип значения: «Число». Количество расходуемого товара на складе.
  • Количество расход. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество списываемого товара.
  • Понижение сортности. Тип значения: «Булево». Если реквизит установлен, то имеем операцию понижения сортности товара и при проведении партии списанных товаров станут партиями оприходованных товаров.
  • Цена розничная расход. Тип значения: «Число». Цена розничная расходуемого товара.
  • Сумма розничная расход. Тип значения: «Число». Сумма розничная расходуемого товара.
  • Сумма списания. Тип значения: «Число». Сумма списания товара.
  • Приходуем. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура, которую приходуем.
  • Характеристика приход. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия приходуемой номенклатуры, например, цвет, размер и т.п.
  • Единица измерения приход. Реквизит является обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения приходуемого товара.
  • Коэффициент приход. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Остаток приход. Тип значения: «Число». Количество приходуемого товара на складе.
  • Количество приход. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество приходуемого товара.
  • Цена розничная приход. Тип значения: «Число». Цена розничная приходуемого товара.
  • Сумма розничная приход. Тип значения: «Число». Сумма розничная приходуемого товара.
  • Сумма переоценки. Тип значения: «Число». Сумма переоценки приходуемого товара.
  • Цена. Тип значения: «Число». Цена приходуемого товара.
  • Сумма. Тип значения: «Число».  Сумма приходуемого товара.
  • ГТД / РНПТ. Ссылается на справочник «ГТД». Номер грузовой таможенной декларации (для автомобилей поступивших по импорту) или регистрационный номер (РНПТ) для прослеживаемых товаров.
  • Ячейка. Ссылается на справочник «Ячейки хранения». Ячейка хранения для номенклатуры. Значение по умолчанию хранится в номенклатуре в разрезе складов.
  • ЯчейкаРасход. Ссылается на справочник «Ячейки хранения». Ячейка хранения для номенклатуры. Значение по умолчанию хранится в номенклатуре в разрезе складов. 

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы 

  • Переоценка оприходованных товаров. Простая печатная форма, содержит список переоцененных товаров.
  • Пересортица. Простая печатная форма, содержит список приходуемых и расходуемых при пересортице товаров.

Способ вызова

Вызывается из меню "Документы --> Складские документы --> Пересортица товаров".

Журналы, отображающие документ: «Складские документы», «Складские акты».

Движения

  • Регистр сведений «Допроведение по партиям».
  • Регистр сведений «Цены».
  • Остаточный регистр «Доходы и расходы».
  • Остаточный регистр «Заказы покупателей».
  • Остаточный регистр «Остатки товаров компании».
  • Остаточный регистр «Партии товаров компании».

Последовательности, изменяемые документом: "Партии".
