Документ "План / факт"

Назначение

Документ "План / факт" содержит все плановые или фактические данные системы. Может быть введен интерактивно в списке или автоматически при помощи обработки или бизнес процесса «мастер планирования». Вид документа определяется хозяйственной операцией.

При проведении документ формирует хозяйственные операции:

  • План. Хозяйственная операция двигает регистр накопления: «Планирование», в котором производится накопление данных по всем планам. Проведением документа заполняются ресурсы: показатели плана. 
  • Факт. Хозяйственная операция двигает регистр накопления: «Планирование», в котором производится накопление данных по всем фактам. Проведением документа заполняются ресурсы: показатели факта.

Относится к подсистеме: «Планирование».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «План факт».

Объекты, вводимые на основании: «План факт», «Корректировка», «Событие».

Реквизиты

  • Вид плана. Реквизит является обязательным для заполнения. Вид плана компании, отражаемого данным документом. Заполнение остальных реквизитов и табличных частей возможно только после выбора этого реквизита.
  • Объект планирования. Основной объект планирования. Ссылается на  характеристику «Аналитические разрезы планирования». Используется для фильтрации состава аналитических объектов в плановой таблице и обработках заполнения.
  • Дата начала. Реквизит является обязательным для заполнения. Тип значения: «Дата». Дата начала периода планирования по документу.
  • Дата конца. Реквизит является обязательным для заполнения. Тип значения: «Дата». Дата конца периода планирования по документу.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ. 
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Документ основание. Ссылается на документ «План факт». Документ, на основании которого введен данный.
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.

Табличная часть «Показатели»

Основная табличная часть документа, предназначенная для хранения показателей плана в разрезе аналитик. Аналитики и показатели определяются (получают названия и типизируются) в соответствии с настройками выбранного вида плана. В случае отсутствия второй аналитики и/или второго, третьего показателей, соответствующие колонки не отображаются на экране.

  • Вид аналитики планирования 1. Ссылается на характеристики «Аналитические разрезы планирования». Содержит описание всех возможных аналитических разрезов планов и фактов.

  • Вид аналитики планирования 2. Ссылается на характеристики «Аналитические разрезы планирования». Содержит описание всех возможных аналитических разрезов планов и фактов.

  • Показатель плана 1. Тип значения: «Число». Значение первого числового показателя документа планирования.
  • Показатель плана 2. Тип значения: «Число». Значение второго числового показателя документа планирования.
  • Показатель плана 3. Тип значения: «Число». Значение третьего числового показателя документа планирования.

Табличная часть «Параметры плана»

Содержит значения параметров планового документа.

  • Вид параметра. Ссылается на план вида характеристики «Параметры планирования». Автоматически заполняется при выборе вида плана.
  • Значение параметра. Ссылается на характеристику «Параметры планирования». При автозаполнении от выбранного плана, копируются значения предыдущего периода.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
    • План. Закладка списка и выбора документов с «ХозОперацией» План
    • Факты. Закладка списка и выбора документов с «ХозОперацией» Факт
    • Планы + Факты. Закладка списка и выбора документов с «ХозОперацией» План и Факт
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы 

  • План / Факт. Простая печатная форма, содержит все значения план / факта.

Способ вызова

Вызывается из меню "Документы --> Регламент --> План / факт".

Движения

  • Оборотный регистр «Планирование».

Дополнительная информация

 

Документ план формирует проводки, аналогичные своей структуре по регистру «Планирование» Показатели 1-3 записываются в ресурсы ПоказательПлан1 – ПоказательПлан3.

Структура документа факт аналогична плану, с добавлением ссылки на соответствующий плану факт.
