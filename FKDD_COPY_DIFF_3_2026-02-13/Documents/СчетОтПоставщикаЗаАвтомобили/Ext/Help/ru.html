Документ "Счет от поставщика за автомобили"

Назначение

Документ служит для фиксации предварительной договоренности о приобретении автомобилей у поставщика, и может являться документом, на основании которого производится поступление автомобилей. При вводе контрагента из его реквизитов автоматически подставляются в документ договор. Эти значения потом можно поменять.

Документ содержит хозяйственную операцию:

  • Счет от поставщика за автомобили. Не создает движений.

Относится к подсистеме: «Автосалон».

Параметры

Длина номера: 10

Периодичность номера: Год

Тип номера: Строка

Вводится на основании: «Заказ на автомобиль», «Поступление автомобилей».

Объекты, вводимые на основании: «Поступление автомобилей».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Комментарий. Произвольный комментарий к документу.
  • Поставщик. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Поставщик, выставивший счет.
  • Договор с поставщиком. Реквизит является обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор, по которому ведутся взаиморасчеты с поставщиком.  
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Курс валюты документа.
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип цен, в которых выписывается счет.
  • Сумма. Сумма документа.
  • Хозяйственная операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Документ основание. Ссылается на документы: «Заказ на автомобиль», «Поступление автомобилей». Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).  
  • Расчетный счет контрагента. Реквизит является обязательным для заполнения. Ссылается на справочник «Банковские счета». Номер расчетного счета поставщика.
  • Номер входящего документа. Тип значения: «Строка». Номер входящего документа, используется для отражения номера бумажного документа, т.к. номер полученного документа может отличаться от нашего внутреннего номера системы, а также при формировании печатной формы.
  • Дата входящего документа. Тип значения: «Дата». Дата входящего документа, используется для отражения даты бумажного документа, т.к. дата полученного документа может отличаться от даты регистрации документа в нашей системе, а также при формировании печатной формы.

Табличная часть «Автомобили»

  • Автомобиль. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Автомобили» или «Модели автомобилей». Автомобиль или модель автомобиля, счет на который выписывается.
  • Вариант комплектации автомобиля. Ссылается на справочник «Варианты комплектации». Вариант комплектации модели автомобиля.
  • Количество. Реквизит является обязательным для заполнения. Количество автомобилей. Всегда равно 1.
  • Цена продажная. Цена автомобиля.
  • Сумма продажи. Сумма всегда без скидки. НДС входит в сумму если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Ставка НДС. Ссылается на справочник «Ставки НДС». Ставка НДС автомобиля.
  • Сумма НДС. Сумма НДС автомобиля.
  • Сумма всего. Итоговая сумма по строке документа со всеми налогами.  

Табличная часть «Опции»  

  • Дополнительные опции.  Реквизит является обязательным для заполнения. Ссылается на справочник «Опции автомобилей».  Опции, установленные в автомобиле производителем.
  • Количество. Реквизит является обязательным для заполнения. Количество опций, установленных в автомобиль.
  • Цена. Цена опции.
  • Сумма. Сумма всегда без скидки. НДС входит в сумму если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Сумма всего. Итоговая сумма по строке документа со всеми налогами.
  • Ставка НДС. Ссылается на справочник «Ставки НДС». Ставка НДС опции.
  • Сумма НДС. Сумма НДС опции.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Счет на оплату. Печатная форма, выставляемого счета от поставщика.

Способ вызова

Вызывается из меню "Документы --> Автосалон --> Поступление автомобилей --> Счет от поставщика за автомобили".
