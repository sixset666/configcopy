Документ "Закрытие кассовой смены"

Назначение

Документ предназначен для свертки чеков розничных продаж в рамках кассовой смены. При формировании документа табличные части заполняется товарами, реализованными за смену, предоставленными скидками и их суммами и списком контрагентов, осуществившие покупки. При проведении документа все свернутые чеки удаляются. Документ является регламентным и формируется в конце каждой кассовой смены. Формирование документа осуществляется при помощи регламентной обработки «Закрытие смены». Также документ может быть введен вручную для отражения розничной продажи конечному покупателю.

При проведении документ формирует хозяйственные операции:

  • Закрытие кассовой смены. Хозяйственная операция сворачивает чеки в пределах кассовой смены.
  • Акт о реализации. Хозяйственная операция отражает розничную продажу конечному покупателю.

  • Относится к подсистеме: «Регламент».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Объекты, вводимые на основании: «Возврат от покупателя», «Корректировка».

Реквизиты

  • Касса ККМ. Реквизит является обязательным для заполнения. Ссылается на справочник «Кассы ККМ». Касса ККМ, по которой осуществляется закрытие кассовой смены.
  • Склад компании. Ссылается на справочник «Склады (места хранения) компании». Склад компании.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип Цен, по которым происходит продажа товаров.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа, на дату документа.
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах"). 
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.  
  • Сумма. Тип значения: «Число». Сумма документа.
  • Получено нал. Тип значения: «Число». Сумма документа, оплаченная наличными.
  • Получено безнал. Тип значения: «Число». Сумма документа, оплаченная по безналичному расчету.
  • Возврат безнал. Тип значения: «Число». Сумма возвратов документа по безналичному расчету. 
  • Проект. Ссылается на справочник «Проекты». Проект, к которому относится документ.

Табличная часть «Товары»

  • Номенклатура. Ссылается на справочник «Номенклатура». Номенклатура (товар, услуга и т.п.).
  • Количество. Тип значения: «Число». Количество единиц измерения номенклатуры. Количество номенклатуры, проданной за кассовую смену.
  • Остаток. Тип значения: «Число». Количество номенклатуры, имеющейся в наличии на выбранном складе.
  • Единица измерения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Цена. Тип значения: «Число». Средняя цена продажи номенклатурной позиции за кассовую смену.
  • Сумма. Тип значения: «Число». Сумма всегда без скидки. НДС входит в сумму, если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Процент НДС. Ссылается на справочник «Ставки НДС». Ставка НДС по номенклатурной позиции.
  • Сумма НДС. Тип значения: «Число». Сумма НДС по номенклатурной позиции.
  • Скидка. Тип значения: «Число». Сумма скидки по строке документа.
  • Всего. Тип значения: «Число». Итоговая сумма по строке документа со всеми налогами.
  • Отдел. Ссылается на справочник «Склады (места хранения) компании». Отдел, в котором был куплен товар.
  • Чеков. Тип значения: «Число». Количество чеков, по которым был реализован товар.
  • Характеристика номенклатуры. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п.
