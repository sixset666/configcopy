Документ "Бюджет движения денежных средств"

Назначение

Документ «Бюджет движения денежных средств» предназначен для планирования движения денежных средств подразделений в разрезе статей ДДС. Запланированные документом данные относятся к выбранному периоду планирования, который  соответствует периодичности сценария планирования (поддерживаются следующие периоды планирования: месяц, квартал, год). Планируемый период не может быть меньше даты документа.

Предусмотрена возможность отражения факта отсрочки платежа - смещение оплаты относительно планируемого периода. Для этого в документе заполняются не только оборотные суммы по каждой статье, но и процентное распределение этих сумм по периодам - график погашения по периодам. График может быть общим для всех статей, либо индивидуальным для каждой статьи.

При проведении документ формирует хозяйственные операции:

  • Бюджет движения денежных средств. Формируются движения по регистру накопления: «Бюджет ДДС», в котором производится накопление запланированных данных. 

Относится к подсистеме: «Планирование».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании:«Бюджет ДДС».

Объекты, вводимые на основании: «Бюджет ДДС»,  «Корректировка»,   

Реквизиты

  • Сценарий планирования.  Реквизит является обязательным для заполнения. Ссылается на справочник «Сценарии планирования». Определяет периодичность планирования.
  • Использовать раздельные графики погашения. Тип значения: «Булево». Ложь - используется один график распределения платежей для всех статей документа. Истина - используются раздельные графики для каждой статьи.
  • Период планирования. Реквизит является обязательным для заполнения. Тип значения: «Дата». Начальная дата периода, на который происходит планирование. Зависит от реквизита "Периодичность" выбранного сценария планирования.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу. 
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.  
  • Документ основание. Ссылается на документ «Бюджет ДДС». Документ, на основании которого введен данный.
  • Способ автозаполнения. Тип значения: «Булево». Используется для автозаполнения.
  • Начальное смещение. Тип значения: «Число». Используется для автозаполнения. Задает первый период, в котором начинается погашение долга. 
  • Количество периодов. Тип значения: «Число». Используется для автозаполнения. Значение процента погашения вычисляется в зависимости от количества периодов.  

Табличная часть «База распределения ДДС»

Основная табличная часть документа, предназначенная для хранения запланированных данных.

  • Статья ДДС. Ссылается на справочник «Статьи ДДС».
  • Базовая сумма упр. Тип значения: «Число». Прогнозируемый оборот по статье.
  • Базовый бюджет. Ссылается на документы вида: «Бюджет продаж», «Бюджет закупок». Определяет бюджет, на основании которого заполнена сумма предполагаемого движения по статье.

Табличная часть «График погашения по периодам»

Подчиненная табличная часть документа, предназначенная для хранения данных о распределении платежей по периодам. Связь с табличной частью "Статьи ДДС" осуществляется через реквизит "Статья ДДС".

  • Смещение. Тип значения: «Число». Смещение периода относительно базового.
  • Период. Тип значения: «Число». Смещение периода относительно прогнозируемого. Может принимать отрицательные значения (предоплата), Если равно нулю, то период равен планируемому периоду. 
  • Процент. Тип значения: «Число». Процент платежа в текущем периоде от общей суммы по статье.
  • Сумма погашения упр. Тип значения: «Число». Сумма платежа в текущем периоде в валюте управленческого учета.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.
  • Форма списка для выбора. Основная форма выбора.
  • Форма автозаполнения. Произвольная форма. Предназначена для заполнения табличной части "График распределения платежей" по всем статьям табличной части "Статьи ДДС". Позволяет использовать два способа заполнения: от количества периодов (равномерное распределение сумм по периодам), от суммы процента (равномерное распределение сумм по процентам)

 

Способ вызова

Вызывается из меню командной панели "Документы--> Бюджетирование --> Бюджет ДДС".

Движения

  • Оборотный регистр «Бюджет ДДС».
