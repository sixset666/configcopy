Форма заполнения по МОЛ документа "Ввод в эксплуатацию"

Назначение

Предназначена для заполнения табличной части "Товары" документа "Ввод в эксплуатацию" списком товаром и материально ответственных лиц.
При нажатии на кнопку "ОК", табличная часть документа "Ввод в эксплуатацию" будет заполнена всеми товарами из списка соответственно для каждого МОЛа. Для этого заполняются список "Товары" (для каждой номенклатуры можно указать характеристику) и список "Материально ответственные лица". Для заполнения номенклатуры и сотрудников можно воспользоваться подбором. Ниже заполняется тип эксплуатации.
