Документ "Заказ внутренний"

Назначение

Документ "Внутренний заказ" оформляется в том случае, если необходимо, исходя из потребностей «склада», пополнить запас товаров с других складов. В качестве других складов могут выступать удаленные торговые точки или магазины, относящиеся к одной организации. При этом номенклатурные позиции могут быть зарезервированы, как из текущего остатка на любом из складов, входящих в состав компании, так и размещены в «заказах поставщиков».

При проведении документ формирует хозяйственные операции:

  • Заказ внутренний. Хозяйственная операция приходует заказы покупателя по списку заказанных товаров на регистр «Заказы покупателей», если есть распределение товаров, то оно регистрируется на регистре «Заказы распределение».
  • Заказ и резервирование внутренние. Хозяйственная операция аналогична операции "Заказ внутренний" за тем исключением, что помимо заказов происходит резервирование товаров на регистре «Остатки товаров компании», по выбранному складу.
  • Резервирование внутреннее. Хозяйственная операция резервирует заказанные товары на регистре «Остатки товаров компании», по выбранному складу.

Относится к подсистеме: «Заказы на отгрузку».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «Заказ внутренний», «Заказ покупателя».

Объекты, вводимые на основании: «Заказ внутренний», «Снятие резервов заказов покупателя», «Резервирование заказов покупателя», «Корректировка», «Корректировка заказа покупателя», «Перемещение товаров», «Реализация товаров», «Заказ поставщику».

Реквизиты

  • Получатель. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, размещающее заказ.
  • Склад. Ссылается на справочник «Склады (места хранения) компании». Склад компании, на котором будет производиться резервирование товара по заказу.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ. 
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ. 
  • Документ основание. Ссылается на документы «Заказ внутренний», «Заказ покупателя». Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип Цен, по которым заказываются товары.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа, на дату документа.
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Сумма. Тип значения: «Число». Сумма документа. 

Табличная часть «Товары»

  • Наличие замен. В колонке отображается иконка "Замены" в случае, когда данная номенклатура имеет замены или является заменой другой номенклатуры. По нажатию на колонку открывается обработка «Подбор замен».
  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар).
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество единиц измерения номенклатуры.
  • Резерв. Тип значения: «Число». Количество товара, резервируемое на складе по этому заказу сразу по проведению документа.
  • Распределено. Тип значения: «Число». Количество товара, распределенного в заказ поставщику из заказов покупателей.
  • Остаток. Тип значения: «Число». Количество товара, который имеется в наличии на выбранном складе.
  • Единица измерения. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Цена. Тип значения: «Число». Цена номенклатурной позиции.
  • Сумма. Тип значения: «Число». Сумма всегда без скидки. НДС входит в сумму, если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Всего. Тип значения: «Число». Итоговая сумма по строке документа со всеми налогами.
  • Характеристика номенклатуры. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п..

Табличная часть «Распределение заказа»

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар).
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество единиц измерения номенклатуры.
  • Единица измерения. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Цена. Тип значения: «Число». Цена номенклатурной позиции.
  • Заказ покупателя. Реквизит является уникальным и обязательным для заполнения. Ссылается на документы: «Заказ внутренний», «Заказ покупателя». Заказ покупателя или внутренний заказ, под который осуществляется заказ данной номенклатуры.
  • Характеристика номенклатуры. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Внутренний заказ. Простая печатная форма. Выводит список заказанных товаров.
  • Печать этикеток и ценников. Обработка, позволяющая получить печатную форму этикеток и ценников заказанных товаров.

Способ вызова

Вызывается из меню "Документы --> Заказы и резервы --> Заказ внутренний".

Журналы, отображающие документ: «Заказы и резервы».

Движения

  • Остаточный регистр «Заказы покупателей».
  • Остаточный регистр «Заказы распределение».
  • Остаточный регистр «Остатки товаров компании».
