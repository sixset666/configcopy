Документ "Закрытие накопительных сумм"

Назначение

Документ предназначен для свертки накопительных сумм для расчета автоматических скидок. При формировании документа табличная часть заполняется накоплениями за период превышающий период накопительных скидок. В регистр накопления сумм записываются свернутые суммы накопления в разрезе контрагентов, дисконтных карт и подразделения. Для каждой свернутой суммы в качестве документа накопления подставляется документ свертки, и дата накопления устанавливается в значение последней даты сворачиваемой серии. Документ сворачивает накопительные суммы только для подразделения, от имени которого вводится документ закрытия накопительных сумм. Документ является регламентным и формируется с периодичностью периода накопительных скидок для каждого подразделения, осуществляющего проведение документов отгрузки (реализация товаров, чек и возврат от покупателя).

При проведении документ формирует хозяйственные операции:

  • Закрытие накопительных сумм. Хозяйственная операция сворачивает накопленные суммы в разрезе контрагентов, дисконтных карт и подразделения на регистре сведений «Накопление cумм».

Относится к подсистеме: «Регламент».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Объекты, вводимые на основании: «Корректировка».

Реквизиты

  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу. 
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа, на дату документа.
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Сумма. Тип значения: «Число». Сумма документа.
  • Очистка сумм накопления. Тип значения: «Булево». Флаг, задающий режим проведения документа, свертку или очистку сумм накопления. 

Табличная часть «Накопления сумм контрагентов»

  • Контрагент. Ссылается на справочник «Контрагенты и контакты». Владелец накопительной скидки.
  • Карточка. Ссылается на справочник «Карточки». Дисконтная карта или карта покупателя.
  • Период накопления. Тип значения: «Дата». Дата суммы накопления.
  • Подразделение. Ссылается на справочник «Подразделения компании». Подразделение компании, сформировавшее накопление.
  • Документ. Тип значения: «Строка». Строковое представление документа, сформировавшего накопительную сумму  
  • Сумма. Тип значения: «Число». Сумма накопления.
  • Количество чеков. Тип значения: «Число». Количество чеков в накоплении.
  • Количество номенклатуры. Тип значения: «Число». Количество единиц номенклатуры в накоплении.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Документы --> Регламент --> Закрытие накопительных сумм".

Движения

  • Регистр сведений «Накопление cумм».
