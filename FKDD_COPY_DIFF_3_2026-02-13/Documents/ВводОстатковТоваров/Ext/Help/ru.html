Документ "Ввод остатков товаров"

Назначение

Документ предназначен для ввода начальных остатков по товарам.

При проведении документ формирует хозяйственную операцию:

  • Ввод остатков товаров. Хозяйственная операция отражает начальные остатки товаров.
  • Ввод остатков товаров принятых на комиссию. Хозяйственная операция отражает начальные остатки товаров принятых на комиссию.
  • Ввод принятых на комиссию и проданных товаров. Хозяйственная операция отражает принятые на комиссию и проданные товары.
  • Ввод остатков товаров переданных на комиссию. Хозяйственная операция отражает начальные остатки товаров переданных на комиссию.

Относится к подсистеме: «Поступление ТМЦ».

Параметры

Длина номера: 10, тип номера: Строка.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «Приходный складской ордер».

Объекты, вводимые на основании: «Отчет комитенту», «Событие», «Возврат поставщику», «Отчет комиссионера», «Списание товаров», «Возврат от покупателя», «Изменение цен», «Корректировка», «Электронное письмо», «Реализация товаров», «Приходный складской ордер».

Реквизиты

  • Склад компании. Ссылается на справочник «Склады компании». Склад компании при поступлении товара, при передачи товара на комиссию реквизит склад компании не используется.
  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Контрагент, который поставил или принял товар.
  • Договор. Реквизит является обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор взаиморасчетов с контрагентом.
  • Статья приходования ТМЦ. Ссылается на справочник «Статьи доходов и расходов». Статья дохода, по которой приходуется товар.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Документ основание. Ссылается на документы «Приходный складской ордер». Документ, на основании которого введен данный.  
  • Тип цен. Ссылается на справочник «Типы цен». Тип цен для товаров. 
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа, на дату документа.
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").
  • Время. Тип значения: «Дата». Время создания документа.
  • Сумма. Тип значения: «Число». Сумма документа.

Табличная часть «Товары»

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура, для которой вводятся остатки.
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество остатков номенклатуры.
  • ЕдиницаИзмерения. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатуры.
  • Коэффициент. Реквизит является обязательным для заполнения. Коэффициент пересчета количества номенклатуры
  • Цена. Тип значения: «Число». Цена номенклатуры.
  • Процент наценки. Тип значения: «Число». Наценка на закупочную цену для вычисления розничной цены.
  • Цена розничная. Тип значения: «Число». Розничная цена для оприходования номенклатуры на розничный склад.
  • Сумма. Тип значения: «Число». Сумма по номенклатуре.
  • Всего. Тип значения: «Число». Сумма по номенклатуре.
  • Сумма розничная. Тип значения: «Число». Розничная сумма для оприходования номенклатуры на розничный склад.
  • Процент НДС. Ссылается на справочник «Ставки НДС». Ставка НДС для номенклатуры.
  • НДС. Тип значения: «Число». Сумма НДС по номенклатуры.  
  • ГТД / РНПТ. Ссылается на справочник «ГТД». Номер грузовой таможенной декларации (для автомобилей поступивших по импорту) или регистрационный номер (РНПТ) для прослеживаемых товаров.
  • Характеристика. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика / серия номенклатуры. 
  • Ячейка. Ссылается на справочник «Ячейки хранения». Ячейка хранения для номенклатуры. Значение по умолчанию хранится в номенклатуре в разрезе складов. 

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Остатки товаров. Простая печатная форма документа.

Способ вызова

Вызывается из меню "Документы --> Поступление ТМЦ --> Ввод остатков товаров".
