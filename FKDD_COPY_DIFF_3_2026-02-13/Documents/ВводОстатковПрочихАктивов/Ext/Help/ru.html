Документ "Ввод остатков прочих активов"

Назначение

Документ предназначен для ввода начальных остатков по прочим активам.

При проведении документ формирует хозяйственную операцию:

  • Ввод остатков прочих активов. Хозяйственная операция отражает начальные остатки прочих активов.

Относится к подсистеме: «Нетоварные активы».

Параметры

Длина номера: 10, тип номера: Строка.

Периодичность номера: Год..

Тип номера: Строка.

Объекты, вводимые на основании: «Реализация активов», «Списание активов», «Обслуживание актива», «Перемещение активов», «Корректировка», «Событие».

Реквизиты

  • Подразделение-получатель. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в которое вводим остатки прочих активов.
  • Материально Ответственное Лицо. Ссылается на справочник «Сотрудники». Материально ответственное лицо.
  • Статья приходования активов. Ссылается на справочник «Статьи доходов и расходов». Статья дохода, по которой приходуются прочие активы.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение компании. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  •  Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа. 
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа, на дату документа.
  • Курс валюты упр. учета. Тип значения: «Число». Курс валюты управленческого учета. (Возможность редактирования задается правом  "Редактирование курса управленческой валюты в документах").
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Сумма документа. Тип значения: «Число». Сумма документа.

Табличная часть «Активы»

  • Прочий актив. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Прочие активы». Нетоварный актив.
  • Тип эксплуатации. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы эксплуатации». Тип эксплуатации.
  • Текущая стоимость. Тип значения: «Число». Остаточная (текущая) стоимость прочего актива на дату ввода остатков.
  • Амортизация. Тип значения: «Число». Сумма начисленной на прочий актив амортизации на дату ввода остатков.
  • Сумма обслуживания. Тип значения: «Число». Сумма, на которую был обслужен прочий актив на дату ввода остатков.
  • Срок эксплуатации. Тип значения: «Число». Срок эксплуатации прочего актива на дату ввода остатков, в мес.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Остатки прочих активов. Простая печатная форма документа.

Способ вызова

Вызывается из меню "Документы --> Прочие активы --> Ввод остатков прочих активов".

Движения

  • Регистр накопления «Доходы и расходы».
  • Регистр накопления «Прочие активы в эксплуатации».
