Документ "Возврат товаров поставщику"

Назначение

Документ предназначен для отражения в оперативном учете операции по возврату поставщику или комитенту  ранее купленных у него товаров. В табличной части документа не могут фигурировать услуги, работы и т.п. При вводе контрагента из его реквизитов автоматически подставляется в документ договор. Это значение потом можно поменять.

При проведении документ формирует хозяйственные операции:

  • Возврат товаров поставщику. Хозяйственная операция списывает кредиторские задолженности поставщика по другим сделкам, если таковые были и начисляет дебеторскую задолженность поставщика перед компанией на регистре «Взаиморасчеты компании», сторнируем материальные активы, с оптового склада предприятия с собственным признаком партии, хранимые на регистрах «Остатки товаров компании» и «Партии товаров компании».
  • Возврат товаров поставщику комиссия. Хозяйственная операция аналогична операции "Возврат поставщику" за тем исключением, что дебиторская задолженность не начисляется, а сторнируется товарная задолженность нашей компании перед комитентом по регистру «Партии товаров компании» с  комиссионным признаком партии.

Относится к подсистеме: «Поступление ТМЦ».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «Поступление товаров», «Возврат от покупателя», «Ввод остатков товаров», «Ввод остатков товаров в производстве».

Объекты, вводимые на основании: «Приходный кассовый ордер», «Счет фактура выданный», «Расходный складской ордер», «Корректировка».

Реквизиты

  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад компании, с которого списываются возвращаемые поставщику товары.
  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Поставщик или комитент, которому мы возвращаем товар.
  • Договор взаиморасчетов. Реквизит является обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор, по которому ведутся взаиморасчеты с контрагентом.
  • Документ основание. Ссылается на документ «Поступление товаров», «Возврат от покупателя». Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу. 
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.  
  • Тип Цен. Ссылается на справочник «Типы цен». Тип Цен, по которым возвращаем товары.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Тип значения: «Число». Реквизит является обязательным для заполнения. Курс валюты документа.
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция. 
  • Дата. Тип значения: «Дата». Дата создания документа.
  • Время. Тип значения: «Дата». Время создания документа.
  • Сумма. Тип значения: «Число». Сумма документа.
  • Проект. Ссылается на справочник «Проекты». Проект, к которому относится документ.

Табличная часть «Товары»

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар).
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество единиц измерения номенклатуры.
  • Остаток. Тип значения: «Число». Количество номенклатуры, которая хранится на выбранном складе компании.
  • Единица. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Цена. Тип значения: «Число». Цена номенклатурной позиции.
  • Сумма. Тип значения: «Число». Сумма всегда без скидки. НДС входит в сумму, если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Всего. Тип значения: «Число». Итоговая сумма по строке документа со всеми налогами.
  • Процент НДС. Ссылается на справочник «Ставки НДС». Ставка НДС по номенклатурной позиции.
  • НДС. Тип значения: «Число». Сумма НДС по номенклатурной позиции.
  • Характеристика. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры, например, цвет, размер и т.п. 
  • Партия. Реквизит является уникальным и обязательным для заполнения. Ссылается ПВХ «Типы партий». Партия, по которой возвращаем товар.
  • Ячейка. Ссылается на справочник «Ячейки хранения». Ячейка хранения для номенклатуры. Значение по умолчанию хранится в номенклатуре в разрезе складов. 

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Печать этикеток и ценников. Обработка, позволяющая получить печатную форму этикеток и ценников возвращаемых товаров.
  • Возвратная накладная. Простая печатная форма, содержит список возвращаемых товаров поставщику или комитенту.
  • ТОРГ-12 (Товарная накладная за поставщика). Унифицированная печатная форма №Торг-12, утвержденная постановлением Госкомстата России.

Способ вызова

Вызывается из меню "Документы --> Поступление ТМЦ --> Возврат поставщику".

Журналы, отображаемые документ: «Поставщики», «Складские документы».

Движения

  • Регистр сведений  «Допроведение по партиям».
  • Остаточный регистр «Взаиморасчеты компании».
  • Остаточный регистр «Доходы и расходы».
  • Остаточный регистр «Остатки товаров компании».
  • Остаточный регистр «Партии товаров компании».

Последовательности, изменяемые документом: "Взаиморасчеты", "Партии".
