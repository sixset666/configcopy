Документ "Договор проведения технического осмотра"

Назначение

Документ "Договор проведения технического осмотра" предназначен для оформления выполнения технического осмотра транспортных средств. 

Относится к подсистеме: «Технический осмотр».

При проведении документ формирует хозяйственные операции:

  • Договор проведения технического осмотра.
  • Повторный договор проведения технического осмотра.

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Контрагент, предоставляющий автомобиль для проведения диагностирования.
  • Представитель. Реквизит является обязательным для заполнения если контрагент имеет форму собственности юредическое лицо. Ссылается на справочник «Контрагенты и контакты». Представитель контрагента, предоставляющий автомобиль для проведения диагностирования.
  • Документ основание. Ссылается на документ «Договор проведения технического осмотра». Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).
  • Договор. Реквизит является обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор, по которому ведутся взаиморасчеты с контрагентом.
  • Курс валюты взаиморасчетов. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты взаиморасчетов, на дату документа.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Тип значения: «Число». Курс валюты документа, на дату документа.
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип цен, в которых оказывается услуга технического осмотра.
  • Сумма. Тип значения: «Число». Сумма документа.
  • Номенклатура. Услуга по проведению технического осмотра. Ссылается на справочник «Номенклатура».
  • Ставка НДС. Ссылается на справочник «Ставки НДС». Ставка НДС по услуге проведения технического осмотра.
  • НДС. Тип значения: «Число». Сумма НДС по услуге проведения технического осмотра.
  • Автомобиль. Реквизит является обязательным для заполнения. Ссылается на справочник «Автомобили». Автомобиль подлежащий диагностированию.
  • Цена. Тип значения: «Число». Цена проведения технического осмотра.
  • Дата проведения. Тип значения: «Дата». Дата проведения технического осмотра.
  • Тормоза. Реквизит является обязательным для заполнения. Ссылается на перечисление «Тип привода тормозной системы». Тип привода тормозной системы автомобиля.
  • Шины. Реквизит является обязательным для заполнения. Тип "Строка". Марка шин автомобиля.
  • Тип топлива. Реквизит является обязательным для заполнения. Ссылается на перечисление «Типы топлива». Тип топлива используемый автомобилем.
  • Масса. Реквизит является обязательным для заполнения. Тип "Число". Масса автомобиля без нагрузки.
  • Макс. масса. Реквизит является обязательным для заполнения. Тип "Число". масимально. Максимально разрешенная масса автомобиля.

  • Формы

    • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
    • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

    Печатные формы

    • Договор проведения техосмотра. Печатная форма договора с контрагентом о проведении технического осмотра. 
    • Повтроный договор проведения техосмотра. Печатная форма повторного договора с контрагентом о проведении технического осмотра.

    Способ вызова

    Вызывается из меню "Документы --> Автосервис --> Договор проведения технического осмотра".
