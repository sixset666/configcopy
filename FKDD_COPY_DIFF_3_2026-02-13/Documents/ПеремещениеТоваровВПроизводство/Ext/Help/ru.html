Документ "Перемещение товаров в производство"

Назначение

Документ "Перемещение товаров в производство" предназначен для перемещения деталей, требуемых для выполнения работ по заказ-наряду со склада компании или от заказчика в производство. В производство может быть перемещено деталей не более чем требуется для выполнения работ. Для завершения работ по заказ-наряду в производство должно быть перемещено деталей в полном объеме для выполнения ремонтных работ.

Список хозяйственных операций:

  • Перемещение товаров в производство. Хозяйственная операция производит расход деталей для выполнения работ по заказ-наряду со склада компании (регистр «Остатков товаров компании») и приходование их и деталей заказчика на регистр «ТоварыВПроизводстве». Для деталей заказанных у поставщиков для выполнения работ по заказ-наряду производится закрытие выполненных заказов покупателей.

Относится к подсистеме: «Автосервис».

Параметры

Длина номера: 10

Периодичность номера: Год

Тип номера: Строка

Вводится на основании: «Заказ-наряд».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете. 
  • Комментарий. Произвольный комментарий к документу.
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип Цен, в которых отгружаются детали для выполнения работ по заказ-наряду.
  • Сумма. Сумма документа.
  • Хоз операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Курс валюты документа.
  • Документ основание. Реквизит является обязательным для заполнения. Ссылается на документ «Заказ-наряд». Документ заказ-наряда, на основании которого введен данный.
  • Склад компании. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад компании, с которого производиться перемещение деталей в производство по заказ-наряду.
  • Цех. Реквизит является обязательным для заполнения. Ссылается на справочник «Цеха». Цех, в который производиться перемещение деталей для выполнения работ.
  • Заказ-наряд. Реквизит является обязательным для заполнения. Ссылается на документ «Заказ-наряд». Заказ-наряд, для которого осуществляется перемещение деталей.
  • Сотрудник выдавший детали. Ссылается на справочник «Сотрудники». Сотрудник, выдавший детали со склада в производство.
  • Сотрудник получивший детали. Ссылается на справочник «Сотрудники». Сотрудник, получивший детали со склада в производство.

Табличная часть «Товары»

  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Деталь, перемещаемая в производство.
  • Количество. Реквизит является обязательным для заполнения. Количество перемещаемых деталей.
  • Остаток. Тип значения: «Число». Количество номенклатурны имеющейся на складе компании.
  • Единица измерения. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения детали.
  • Коэффициент. Реквизит является обязательным для заполнения. Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Цена розничная. Тип значения: «Число». Розничная цена для оприходования номенклатуры на розничный склад.
  • Сумма розничная. Тип значения: «Число». Розничная сумма для оприходования номенклатуры на розничный склад.
  • Сумма списания. Тип значения: «Число». Сумма списания товара.
  • Характеристика номенклатуры. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры.
  • Партия. Ссылается на план видов характеристик «Типы партий». Партия, с которой списывается товар.
  • ГТД/ РНПТ. Ссылается на справочник «ГТД». Номер грузовой таможенной декларации (для товаров, поступивших по импорту) или регистрационный номер (РНПТ) для прослеживаемых товаров.
  • Ячейка. Ссылается на справочник «Ячейки хранения». Ячейка хранения для номенклатуры. Значение по умолчанию хранится в номенклатуре в разрезе складов.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.
  • Форма документа. Основная форма документа. Используется для ввода и редактирования документов данного типа.

Печатные формы

  • Перемещение товаров в производство. Накладная на детали, перемещенные в производство.
  • ТОРГ-13 (накладная на внутреннее перемещение). Унифицированная печатная форма №Торг-13, утвержденная постановлением Госкомстата России.
  • 1-Т (Товарно-транспортная накладная). Унифицированная печатная форма №1-Т, утвержденная постановлением Госкомстата России.
  • Приложение 4 (Транспортная накладная). Унифицированная печатная форма, утвержденная постановлением Госкомстата России.
  • Печать этикеток и ценников. Обработка, позволяющая получить печатную форму этикеток и ценников переоцениваемых товаров.
  • М-11 (требование-накладная). Унифицированная форма № М-11, утвержденная постановлением Госкомстата России.

Способ вызова

Вызывается из меню "Документы --> Автосервис --> Перемещение товаров в производство".

Движения

  • «Заказы покупателей». Заказы покупателей.
  • «Заказы распределение». Хранит состояние распределения ожидаемых заказов поставщиков, покупателей и подразделений.
  • «Остатки товаров компании». Остатки товаров компании на складах.
  • «Партии товаров компании». Партии товаров компании.
  • «Товары компании в производстве». Товары компании в производстве.
