Документ "Заявка на ремонт"

Назначение

Предварительная заявка на ремонт автомобиля. Предназначена планирования загрузки ремзон и предварительного расчета стоимости ремонта.

Список хозяйственных операций:

  • Заявка на ремонт. Хозяйственная операция не производит движения каких либо регистров, и предназначена для планирования работ по ремзонам.

Относится к подсистеме: «Автосервис».

Параметры

Длина номера: 10

Периодичность номера: Год

Тип номера: Строка

Объекты, вводимые на основании: «Заказ-Наряд».

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании в котором был выписан документ.
  • Комментарий. Произвольный комментарий к документу.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Курс валюты документа.
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип Цен, в которых отгружаются детали для выполнения работ.
  • Маркетинговая программа. Ссылается на справочник «Маркетинговые программы». Маркетинговая программа, в рамках которой планируется ремонт.
  • Скидка/наценка. Ссылается на справочник «Типы скидок и наценок». Скидка/наценка на запчасти.
  • Скидка/наценка на работы. Ссылается на справочник «Типы скидок и наценок». Скидка/наценка на работы.
  • Вид оплаты. Вид оплаты работ и запчастей.
  • Сумма. Сумма документа.
  • Сумма номенклатуры документа. Сумма запчастей по заявке на ремонт.
  • Сумма работ документа. Сумма работ по заявке на ремонт.
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Автомобиль. Реквизит является обязательным для заполнения. Ссылается на справочник «Автомобили». Ремонтируемый автомобиль.
  • Государственный номер. Государственный номер автомобиля. Подставляется из регистра сведений. Может быть изменен в форме. При изменении старое значение в регистре будет перезаписано на новое.
  • Вид ремонта. Реквизит является обязательным для заполнения. Ссылается на справочник «Виды ремонта». Вид ремонта, который требуется произвести. При выборе вида ремонта задается вопрос пользователю о добавлении работ, назначененных по умолчанию для данного вида ремонта, в табличные части документа.
  • Дата начала. Планируемая дата начала работ.
  • Дата окончания. Планируемая дата окончания работ.
  • Причина обращения. Причина обращения заказчика в ремонт.
  • Мастер. Ссылается на справочник «Сотрудники». Мастер, ответственный за выполнение ремонта.
  • Диспетчер. Ссылается на справочник «Сотрудники». Диспетчер, принявший заказ на выполнение ремонта. Заполняется автоматически при добавлении предопределенных работ "Приемка автомобиля" и "Выдача автомобиля" в табличную часть "Планирование" на сотрудника, являющегося диспетчером.
  • Заказчик. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Заказчик на выполнение работ.
  • Цех. Реквизит является обязательным для заполнения. Ссылается на справочник «Цеха». Основной цех проведения работ по заказу.
  • Исполнители для всех работ. Признак одинаковых исполнителей для всех работ по заказу.
  • Контактная информация. Текстовое поле для указания контактной информации для заказчиков типа "Строка".
  • Телефон. Телефон клиента, по которому он обратился в автосервис. Может быть выбран из списка, отредактирован или введен новый. Редактируется в диалоге. Для нового номера по умолчанию подставляется вид контактной информации "мобильный телефон". Если телефон с данным видом контактной информации уже задан в регистре для данного клиента, то задается вопрос о замене телефона либо выборе другого вида контактной информации для записи.
  • Состояние. Состояние заявки на ремонт. 
  • Причина отказа. Причина отказа от обслуживания.  

Табличная часть «Товары»

  • Наличие замен. В колонке отображается иконка "Замены" в случае, когда данная номенклатура имеет замены или является заменой другой номенклатуры. По нажатию на колонку открывается обработка «Подбор замен».
  • Номенклатура. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Номенклатура». Деталь для выполнения работ по заказу.
  • Количество. Реквизит является обязательным для заполнения. Количество деталей.
  • Единица измерения. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения детали.
  • Коэффициент. Реквизит является обязательным для заполнения. Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Цена. Цена детали в валюте документа.
  • Сумма. Сумма всегда без скидки. НДС входит в сумму если это указано в типе цен. Сумма всегда равна Цена*Количество.
  • Ставка НДС. Ссылается на справочник «Ставки НДС». Ставка НДС по номенклатурной позиции.
  • Сумма НДС. Сумма НДС по номенклатурной позиции.
  • Процент скидки. Процент скидки на деталь.
  • Сумма скидки. Сумма скидки на деталь.
  • Характеристика номенклатуры. Реквизит является уникальным. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры.
  • Всего. Итоговая сумма по строке документа со всеми налогами.
  • Источник. Реквизит является уникальным. Источник поступления детали (со склада/заказывается у поставщика/предоставляется клиентом).
  • Склад компании. Ссылается на справочник «Склады (места хранения) компании». Склад компании, откуда поступает деталь.

Табличная часть «Работы»

  • Работа. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Автоработы». Работа для выполнения заказа.
  • Количество. Реквизит является обязательным для заполнения. Количество работ.
  • Нормочас. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Нормочасы». Нормочас работы.
  • Коэффициент. Реквизит является обязательным для заполнения. Цена нормочаса.
  • Цена. Цена работы в валюте документа.
  • Сумма. Сумма всегда без скидки. НДС входит в сумму если это указано в типе цен. Сумма всегда равна Цена*Количество*Коэффициент.
  • Ставка НДС. Ссылается на справочник «Ставки НДС». Ставка НДС по работе.
  • Сумма НДС. Сумма НДС по работе.
  • Процент скидки. Процент скидки на работу.
  • Сумма скидки. Сумма скидки на работу.
  • Всего. Итоговая сумма по строке документа со всеми налогами.

Табличная часть «Исполнители»

  • Работа. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Автоработы». Работа, для которой назначен исполнитель.
  • Исполнитель. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Сотрудники». Исполнитель работы.
  • Цех. Реквизит является обязательным для заполнения. Ссылается на справочник «Цеха». Цех исполнения работы.
  • Процент. Реквизит является обязательным для заполнения. Процент участия исполнителя при выполнении данной работы.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.
  • Форма документа. Основная форма документа. Используется для ввода и редактирования документов данного типа.
  • Форма ввода дат документа. Форма используется для ввода дат начала и окончания работ по заявке на ремонт. Содержит раздельный ввод даты и времени соответствующих реквизитов.

Печатные формы

  • Бланк заявки. Бланк заявки на проведение ремонтных работ с указанием заказчика и ремонтируемого автомобиля, а так же причины обращения заказчика. Для вывода в печатной формы схемы транспортного средства, отличной от по умолчанию, в модели автомобиля (или в группе моделей) необходимо добавить картинку с данной схемой. Идентификатор картинки при этом должен быть "СхемаТС".
  • Рабочая заявка. Рабочая заявка на ремонт с перечнем работ и требующихся для их выполнения деталей.

Способ вызова

Вызывается из меню "Документы --> Автосервис --> Заявка на ремонт".

Движения

Движение регистров не производиться.

Дополнительная информация

Кнопка «Подбор Рекомендаций» вызывает обработку «Подбор рекомендаций» и служит для подбора рекомендуемой автоработы или номенклатуры для данного автомобиля.
Отображение иконки  возле реквизита «Автомобиль» показывает наличие рекомендаций для выбранного автомобиля.

При записи на ремонт или при отмене заявки, пользователю будет выведено окно отправки смс («Документ SMS»), которое будет заполнено на основании текущей заявки. Такое поведение возможно лишь при включенном поле "Отправка уведомлений" в справочнике «ВидыРемонта». Шаблон смс будет выбран в соответствии с полем «Шаблон при записи на ремонт» или «Шаблон при отмене заявки» справочника «ВидыРемонта». Если шаблон не найден, то текст сообщения формироваться не будет, но остальные параметры отправки заполнятся.

При записи на ремонт, помимо интерактивной отправки смс, формируется смс уведомление, с датой отправки равной дате начало ремонта с учетом поля «Отправлять за» справочника «ВидыРемонта», которое будет отправлено автоматически. Шаблон смс будет выбран в соответствии с полем «Шаблон при записи на ремонт» справочника «ВидыРемонта».

Автоматическое сообщение смс не будет формироваться в случае, если:

  • дата начала ремонта, меньше текущей даты;
  • заявка на ремонт имеет статус "Отклонено";
  • при перезаписи документа, время ремонта не изменилось;
  • документ не проведен.

 

Автоматическое сообщение смс не будет отправляться в случае, если:

  • поле «Шаблон при записи на ремонт» справочника «ВидыРемонта» не заполнено;
  • не указаны обязательные реквизиты для отправки смс («Документ SMS»);
  • не настроен сервис отправки смс (SMS4B).

 

В ситуации. когда дата отправки смс меньше текущей даты, но время заявки на ремонт больше текущей даты, смс будет сформировано на текущую дату.
