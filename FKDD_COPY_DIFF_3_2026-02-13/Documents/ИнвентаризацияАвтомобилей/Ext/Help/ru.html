Документ "Инвентаризация автомобилей"

Назначение

Документ предназначен для отражения в учете факта проведения инвентаризации автомобилей по выбранному складу. При проведении в случае оприходывания излишков документ образует партию с признаком "Товар купленный". При проведении если в строке фактическое количество больше учетного, то документ приходует излишек (разницу между фактическим и учетным количеством) на склад, в противном случае, если фактическое количество меньше учетного, то документ списывает недостачу со склада по выбранной статье расходов. Новый (непроведенный) документ можно заполнить автоматически учетными остатками по выбранному складу (меню по кнопке  "Заполнение"). При вводе новых строк автоматически подставляются учетные количество и стоимость.

При проведении документ формирует хозяйственные операции:

  • Инвентаризация автомобилей. Хозяйственная операция в зависимости от реальных остатков приходует или списывает автомобили, хранимые на складах компании, с собственным признаком партии, на регистр «Остатки автомобилей», отражает факт продажи автомобилей комитента, если они в процессе инвентаризации были утеряны или испорчены, на регистре «Реализованные автомобили».
  • Инвентаризация автомобилей у комиссионера. Хозяйственная операция аналогична операции "Инвентаризации автомобилей" за тем исключение, что приходуются или списываются автомобили, с комиссионным признаком партии, на регистре «Остатки автомобилей», и не отражает факт продажи автомобилей комитента.

Относится к подсистеме: «Автосалон».

Параметры

Длина номера: 10

Периодичность номера: Год

Тип номера: Строка

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Регламентированный учет. Тип значения: «Булево». Флаг отражения документа в регламентированном учете.
  • Комментарий. Произвольный комментарий к документу.
  • Контрагент. Ссылается на справочник «Контрагенты и контакты». Комиссионер, которому мы отдавали автомобили на комиссию.
  • Договор. Ссылается на справочник «Договоры взаиморасчетов». Договор, по которому ведутся взаиморасчеты с контрагентом.
  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад компании, на котором проводим инвентаризацию.
  • Тип Цен. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы цен». Тип Цен, по которым инвентаризируются автомобили.
  • Хозяйственная операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Валюта. Реквизит является обязательным для заполнения. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Реквизит является обязательным для заполнения. Курс валюты документа, на дату документа.
  • Сумма. Сумма документа.
  • Статья списания обнаруженной недостачи ТМЦ. Реквизит является обязательным для заполнения. Ссылается на справочник «Статьи доходов и расходов». Статья списания обнаруженной недостачи автомобилей.
  • Статья оприходования обнаруженных излишков ТМЦ. Реквизит является обязательным для заполнения. Ссылается на справочник «Статьи доходов и расходов». Статья оприходования обнаруженных излишков автомобилей.

Табличная часть «Автомобили»

  • Автомобиль. Реквизит является уникальным и обязательным для заполнения. Ссылается на справочник «Автомобили». Инвентаризируемый автомобиль.
  • Количество. Расчетное (по базе данных) количество автомобилей. Всегда равно 1.
  • Количество по данным учета. Фактическое количество автомобилей на складе.
  • Количество разница. Количество расчетное - Количество фактическое.
  • Цена. Цена автомобиля.
  • Сумма. Оценочная сумма автомобиля по данным из базы данных.
  • Сумма по данным учета. Фактическая сумма автомобиля.
  • Сумма разницы. Сумма расчетная - Сумма фактическая.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Инвентаризация автомобилей на комиссии. Простая печатная форма, содержит список инвентаризируемых автомобилей.
  • Сличительная ведомость (ИНВ-19). Унифицированная печатная форма №ИНВ-19, утвержденная постановлением Госкомстата России.
  • Инвентаризационная опись (ИНВ-3). Унифицированная печатная форма №ИНВ-3, утвержденная постановлением Госкомстата России.
  • Пустографка. Простая печатная форма, содержит список инвентаризируемых автомобилей без их фактического количества.

Способ вызова

Вызывается из меню "Документы --> Автосалон--> Складские операции --> Инвентаризация автомобилей".

Движения

  • Остаточный регистр «Доходы и расходы».
  • Остаточный регистр «Остатки автомобилей».
  • Остаточный регистр «Автомобили отданные».
  • Остаточный регистр «Реализованные автомобили».
  • Остаточный регистр «Взаиморасчеты компании».
