Документ "Переразмещение товаров"

Назначение

Документ "Переразмещение товаров" предназначен для автоматизации процесса перемещения товаров внутри ячеистого склада. Есть возможности автоматического заполнения документа:

  • "Погашение минусов" - из ячеек с положительным остатком товары записываются в ячейку с отрицательным остатком так, чтобы в новой ячейке количество стало нулевым.
  • "Оптимизация" - собираются одинаковые товары из разных ячеек в одну с изначальным максимальным количеством.

При проведении документ формирует хозяйственные операции:

  • Переразмещение товаров. Хозяйственная операция списывает товары из одной ячейки (Старая ячейка) склада по регистру «Остатки товаров на ордерном складе» и приходует в другую (Новая ячейка).

Относится к подсистеме «Ордерный склад».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «Приходный складской ордер», "Переразмещение товаров".

Объекты, вводимые на основании: «Расходный складской ордер», «Корректировка», "Переразмещение товаров", «Событие».

Реквизиты

  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад компании, на который поступают товары.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Юридическое лицо, от которого выписан документ.
  •  Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, создавший документ.
  • Документ основание. Ссылается на документы «Приходный складской ордер», "Переразмещение товаров". Документ, на основании которого введен данный.
  • Операция. Ссылается на справочник «Хоз. операции». Хозяйственная операция.  

Табличная часть «Товары»

  • Номенклатура. Реквизит является обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар, услуга и т.п.).
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество единиц измерения номенклатуры.
  • Остаток. Тип значения: «Число». Количество номенклатуры имеющейся в наличии на складе компании.
  • Количество базовое старое. Тип значения: «Число». Количество базовое старое.
  • Количество базовое новое. Тип значения: «Число». Количество базовое новое.
  • Единица измерения старая. Реквизит является обязательным для заполнения. Ссылается на справочник «Единицы измерения». Старая единица измерения номенклатурной позиции.
  • Единица измерения новая. Реквизит является обязательным для заполнения. Ссылается на справочник «Единицы измерения». Новая единица измерения номенклатурной позиции.
  • Ячейка старая. Реквизит является обязательным для заполнения. Ссылается на справочник «Ячейки хранения». Старое место хранения ячеистого склада.
  • Ячейка новая. Реквизит является обязательным для заполнения. Ссылается на справочник «Ячейки хранения». Новое место хранения ячеистого склада.
  • Характеристика. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Переразмещение товаров. Простая печатная форма документа.

Способ вызова

Вызывается из меню "Документы --> Складские документы --> Переразмещение товаров"

Журналы, отображающие документ: «Складские ордера».

Движения

  • «Остатки товаров на ордерном складе».
