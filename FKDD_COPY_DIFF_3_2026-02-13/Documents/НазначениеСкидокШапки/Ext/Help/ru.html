Документ "Назначение общих скидок"

Назначение

Предназначен для установки или отмены действующих скидок/наценок на документ в целом.

При проведении документ формирует хозяйственные операции:

  • Установка скидки. Хозяйственная операция регистрирует установку скидки в регистре сведений «Скидки шапки».
  • Отмена скидки. Хозяйственная операция регистрирует отмену скидки в регистре сведений «Скидки шапки».

Относится к подсистеме: «Ценообразование».

Параметры

Длина номера: 10, тип номера: Строка.

Периодичность номера: Год.

Тип номера: Строка.

Реквизиты

  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Идентификатор юридического лица, от которого выписан документ.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Действует с. Реквизит является обязательным для заполнения. Тип значения: «Дата». Дата и время с которого начинает действовать скидка.
  • Комментарий. Произвольный комментарий к документу.

Табличная часть «Скидки»

  • Скидка. Реквизит является обязательным для заполнения. Ссылается на справочник «Типы скидок». Определяет тип назначаемой скидки.
  • Вытесняющая. Тип значения: «Булево». Признак возможность использования данной скидки совместно с другими скидками.
  • Распространяется на товары. Тип значения: «Булево». Признак, что скидка действует для табличной части "Товары".
  • Распространяется на работы. Тип значения: «Булево». Признак, что скидка действует для табличной части "Работы".
  • Значение. Тип значения: «Число». Значение скидки. Для абсолютных скидок - сумма, для относительных - процент.
  • Процент. Единица скидки. Для абсолютных скидок - валюта, для относительных - знак "%".
  • Свойство. Ссылается на план видов характеристик «Свойства объектов». Значение свойства документа при котором скидка может быть назначена. Вид свойства определяется в «Типе скидки».
  • Сумма чека. Тип значения: «Число». Минимальный порог суммы документа покупки, начиная с которого, скидка может быть назначена.
  • Дисконтная карта. Ссылается на справочник «Карточки». Дисконтная карта или группа карт, на которые скидка может быть назначена.
  • На сумму покупок. Тип значения: «Число». Минимальный порог накопительной суммы покупок по дисконтной карте за период, начиная с которого, скидка может быть назначена.
  • Время С. Тип значения: «Дата». Время начала действия скидки.
  • Время ПО. Тип значения: «Дата». Время окончания действия скидки.
  • Дни недели. Тип значения: «Строка». Дни недели в которые действует скидка (Маска дней недели 0/1).
  • Прайс-лист. Ссылается на справочник «Виды прайс-листов». Вид прайс-листа из которого будет выбираться подарок.
  • Сумма подарка. Тип значения: «Число». Максимальная сумма на которую можно выбрать подарок.
  • Склад. Ссылается на справочник «Склады компании». Склад компании (торговый зал или отдел) или группа складов, на которые скидка может быть назначена.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Назначение скидок шапки. Простая печатная форма, содержит список устанавливаемых скидок.

Способ вызова

Вызывается из меню "Документы --> Ценообразование --> Назначение общих скидок".

Движения

  • Регистр сведений «Скидки шапки».
