Документ "Платежное поручение"

Назначение

Документ предназначен для оформления и печати платежных поручений. При создании нового документа «организация», от имени которой оформляется платежное поручение, устанавливается в соответствии с настройками по умолчанию для «пользователя». Расчетный счет ("Счет организации") из справочника «Банковские счета» устанавливается в соответствии с выбранным основным расчетным счетом для собственного юридического лица организации. При необходимости, в документе можно выбрать другую организацию и другой расчетный счет.
В качестве счета по умолчанию для получателя, используется тот расчетный счет, который установлен для юридического лица «контрагента», как основной банковский счет. Вид платежа выбирается из списка («почтой», «телеграфом», «электронно»).

Документ содержит хозяйственную операцию:

  • Платежное поручение. Не создает движение.

Относится к подсистеме: «Движения денежных средств».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «Заказ поставщику», «Поступление товаров», «Возврат от покупателя», «Отчет комитенту».

Объекты, вводимые на основании: «Корректировка», «Событие».

Реквизиты

  • Счет организации. Ссылается на справочник «Банковские счета». Расчетный счет организации.
  • Перечисление налога, сбора или иного платежа. Тип значения: «Булево». Истина - это не просто платеж, а перечисление налогов. Ложь - обычный платеж.
  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты и контакты». Получатель денежных средств.
  • Договор. Реквизит является обязательным для заполнения. Ссылается на справочник «Договоры взаиморасчетов». Договор, по которому ведутся взаиморасчеты с контрагентом.
  • Сумма. Тип значения: «Число». Сумма документа.
  • Ставка НДС. Ссылается на справочник «Ставки НДС». Ставка НДС.
  • Сумма НДС. Тип значения: «Число». Сумма НДС.
  • Счет контрагента. Ссылается на справочник «Банковские счета». Расчетный счет контрагента.
  • Очередность платежа. Тип значения: «Число». Очередность платежа.
  • Вид платежа. Вид платежа: электронно, почтой, телеграфом.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • ИНН плательщика. Тип значения: «Строка». ИНН нашей компании.
  • КПП плательщика. Тип значения: «Строка». КПП нашей компании.
  • Текст плательщика. Тип значения: «Строка». Текст плательщика заполняется автоматически в соответствии с информацией, введенной в форму расчетного счета, выбранного для организации.
  • ИНН получателя. Тип значения: «Строка». ИНН контрагента.
  • КПП получателя. Тип значения: «Строка». КПП контрагента.
  • Текст получателя. Тип значения: «Строка». Текст получателя заполняется автоматически в соответствии с информацией, введенной в форму расчетного счета, выбранного для контрагента-покупателя.
  • Текст назначение платежа. Тип значения: «Строка». Назначение платежа. 
  • Статус составителя платежного документа. Тип значения: «Строка». Статус составителя платежного документа на перечисление налогов, сборов и иных обязательных платежей.
  • Основания. Тип значения: «Строка». Условное обозначение основания налогового платежа.
  • Дата. Тип значения: «Дата». Дата документа - основания перечисления налога, сбора или иного платежа.
  • Код БК. Тип значения: «Строка». Код бюджетной классификации доходов РФ.
  • Период. Тип значения: «Строка». Условное обозначение периода, за который перечисляется налог.
  • Типа платежа. Тип значения: «Строка». Условное обозначение типа налогового платежа.
  • Код ОКТМО. Тип значения: «Строка». Код ОКТМО сборщика платежей.
  • Код УИН. Тип значения: «Строка».  Код уникальный идентификатор начислений.
  • Номер. Тип значения: «Строка». Номер документа - основания перечисления налога, сбора или иного платежа.
  • Подразделение. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация.  Ссылается на справочник «Организации». Юридическое лицо, от которого выписан документ.
  • Автор. Ссылается на справочник «Пользователи». Автор документа - пользователь, породивший документ.
  • Документ основание. Ссылается на документы «Заказ поставщику», «Поступление товаров», «Возврат от покупателя», «Отчет комитенту». Документ, на основании которого введен данный, необходим для привязки подчиненных документов к документу основанию, с помощью этого реквизита, затем строится дерево подчиненных документов (по кнопке ).
  • Валюта. Ссылается на справочник «Валюты». Валюта документа.
  • Курс. Тип значения: «Число». Курс валюты документа, на дату документа.
  • Хоз операция. Реквизит является обязательным для заполнения. Ссылается на справочник «Хоз. операции». Выполняемая документом операция.
  • Проект. Ссылается на справочник «Проекты». Проект, к которому относится документ.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма документа. Используется для заполнения обязательных реквизитов.
  • Форма периода. Дополнительная форма документа. Используется для удобного ввода значения реквизита "Показатель периода". 
  • Перечисление налога. Дополнительная форма документа. Используется для заполнения дополнительных реквизитов в случае использования платежного поручения для перечисления налога.
  • Форма подбора платёжных поручений. Используется для выбора платежных поручений при заполнении документа «Банковская выписка».

    Печатные формы

    • Платежное поручение. Печатная форма платежного поручения, которая выписывается покупателю или поставщику.

    Способ вызова

    Вызывается из меню "Документы --> Движения денежных средств --> Платежное поручение".
