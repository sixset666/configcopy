Документ "Расходный складской ордер"

Назначение

Отгрузка номенклатурных позиций с ордерного или ячеистого склада. В табличной части документа не могут фигурировать услуги, работы и т.п.  Для ячеистого склада возможно автоматическое заполнение колонки "Ячейка". При этом подставляется ближайшая ко входу ячейка, в которой имеется требуемое количество товара (в автозаполнении не участвуют ячейки, у которых установлено свойство "Недоступна").

При проведении документ формирует хозяйственные операции:

  •  Расходный складской ордер. Хозяйственная операция расходует со складов предприятия, имеющие вид "Ордерный" или "Ордерный и ячеистый", материальные активы, хранимые в регистре «Остатки товаров на ордерном складе».

Относится к подсистеме «Ордерный склад».

Параметры

Длина номера: 10.

Периодичность номера: Год.

Тип номера: Строка.

Вводится на основании: «Возврат поставщику», «Перемещение товаров», «Списание товаров», «Реализация товаров», «Переразмещение товаров», «Приходный складской ордер», «Комплектация», «Разукомплектация», «Пересортица товаров», «ВВод в эксплуатацию».

Объекты, вводимые на основании: «Перемещение товаров», «Реализация товаров», «Списание товаров», «Корректировка», «Приходный складской ордер», «Событие».

Реквизиты

  • Склад. Реквизит является обязательным для заполнения. Ссылается на справочник «Склады (места хранения) компании». Склад компании, с которого отгружаются товары.
  • Владелец товара. Ссылается на справочники «Контрагенты и контакты», «Склады (места хранения) компании». Контрагент/склад, получающий товар.
  • Документ основание. Ссылается на документы «Возврат поставщику», «Инвентаризация», «Перемещение товаров», «Переразмещение товаров», «Реализация товаров», «Списание товаров». Документ, на основании которого введен данный.
  • Комментарий. Тип значения: «Строка». Произвольный комментарий к документу.
  • Подразделение. Реквизит является обязательным для заполнения. Ссылается на справочник «Подразделения компании». Подразделение компании, в котором был выписан документ.
  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Юридическое лицо, от которого выписан документ.
  • Автор. Реквизит является обязательным для заполнения. Ссылается на справочник «Пользователи». Автор документа - пользователь, создавший документ.
  • Операция. Ссылается на справочник «Хоз. операции». Хозяйственная операция.

Табличная часть «Товары»

  • Номенклатура. Реквизит является обязательным для заполнения. Ссылается на справочник «Номенклатура». Номенклатура (товар, услуга и т.п.).
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество единиц измерения номенклатуры.
  • Остаток. Тип значения: «Число». Количество номенклатуры, имеющееся на складе.
  • Единица. Реквизит является обязательным для заполнения. Ссылается на справочник «Единицы измерения». Единица измерения номенклатурной позиции.
  • Коэффициент. Реквизит является обязательным для заполнения. Тип значения: «Число». Коэффициент пересчета по отношению к базовой единице номенклатуры.
  • Количество базовое. Тип значения: «Число». Количество номенклатуры в базовых единицах (коэффициент=1).
  • Ячейка. Ссылается на справочник «Ячейки хранения». Место хранения ячеистого склада.
  • Характеристика. Ссылается на справочник «Характеристики номенклатуры». Характеристика/серия номенклатуры.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Печатные формы

  • Расходный ордер.  Простая печатная форма, содержит список расходуемых со склада товаров. Если склад  ячеистый, содержит колонку, отображающую ячейки, из которых расходуются товары.
  • Карта склада. Выводит графическое представление склада с отображением маршрута движения. Маршрут начинается в точке входа склада и заканчивается в точке выхода. Вывод карты склада возможен только для ячеистого склада.  

Способ вызова

Вызывается из меню "Документы --> Складские документы --> Расходный складской ордер"

Журналы, отображающие документ: «Складские ордера».

Движения

  • Остаточный регистр «Остатки товаров на ордерном складе».
