Справочник "Классификатор товарной номенклатуры внешнеэкономической деятельности"

Назначение

В справочнике кодов товарной номенклатуры внешнеэкономической деятельности хранятся коды, которые используются при заполнении реквизитов номенклатуры.

Данный справочник может быть заполнен с помощью подбора из классификатора кодов ТН ВЭД, в котором хранятся все коды товарной номенклатуры внешнеэкономической деятельности, а также вручную пользователем поэлементно. При использовании подбора из классификатора кодов существенно снижается трудоемкость ввода информации, ввод информации о коде сводится к выбору из классификатора, соответственно сводится к минимуму количество ошибок в вводе.

Коды, которые уже были выбраны из классификатора подбора, в списке отмечаются синим цветом.

Относится к подсистеме: «Классификаторы».

Параметры

Длина кода: 10, тип кода: Строка.

Длина наименования: 150.

Не иерархический.

Реквизиты

  • Полное наименование. Тип значения: «Строка». Полное наименование кода ТН ВЭД. 
  • Сырьевой товар. Тип значения: «Булево». Признак того, что это код сырьевого товара.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма выбора. Основная форма выбора. Используется для отображения выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма подбора из классификатора. Дополнительная форма подбора. Используется для подбора элементов из классификатора кодов ТН ВЭД. 

Печатные формы

  • Классификатор товарной номенклатуры внешнеэкономической деятельности. Используется не для печати, а для подбора новых кодов ТН ВЭД.

Способ вызова

Вызывается из меню "Справочники --> Классификаторы --> Классификатор ТН ВЭД".
