Справочник "Варианты отчетов"

Назначение

Редактирование, открытие и выбор вариантов отчетов.

Относится к подсистеме: «Отчеты».

Параметры

Длина кода: 0.

Длина наименования: 150.

Не иерархический.

Реквизиты

  • Имя отчета. Тип значения: «Строка». Имя отчета.
  • Ключ отчета. Тип значения: «Строка». Уникальный идентификатор отчета.
  • Представление отчета. Тип значения: «Строка». Представление отчета.
  • Ключ варианта. Тип значения: «Строка». Уникальный идентификатор варианта отчета.
  • Автор. Ссылается на справочник «Пользователи». Автор варианта отчета.
  • Настройки. Тип значения: «Хранилище значения». Настройки варианта отчета.
  • Описание. Тип значения: «Строка». Описание варианта отчета.
  • Назначение доступа. Ссылается на перечисление «Назначение прав и настроек». Определяет тип объекта доступа.
  • Объект доступа. Объект по которому производится отображение настраиваемых вариантов отчетов.
  • Это предопределенный. Тип значения: «Булево». Признак, что вариант настраивается в конфигураторе.
  • Пользовательская настройка. Тип значения: «Булево». Признак пользовательской настройки.
  • Вариант периода. Тип значения: «Строка». Вариант начального заполнения периода отчета.

Формы

  • Форма списка. Основная форма списка. Используется для отображения списка элементов.
  • Форма элемента. Основная форма элемента. Используется для заполнения реквизитов, просмотра и редактирования варианта отчета.
  • Форма Выбора. Основная форма выбора. Используется для отображения выбора элементов.

Способ вызова

Вызывается из меню "Справочники --> Структура компании --> Варианты отчетов".
