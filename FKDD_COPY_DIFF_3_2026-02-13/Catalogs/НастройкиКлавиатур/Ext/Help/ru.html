Справочник "Настройки клавиатур"

Назначение

В справочник заносятся список используемых наборов клавиатур.

Относится к подсистеме: «Сервис».

Параметры

Длина кода: 3, тип кода: Число.

Длина наименования: 25.

Не иерархический.

Реквизиты

  • Префикс. Реквизит является обязательным для заполнения. Тип значения: «Строка». Префикс клавиатуры.
  • Количество клавиш. Тип значения: «Число». Определяет количество клавиш для варианта раскладки клавиатуры.
  • Количество строк. Тип значения: «Число». Определяет количество строк в программируемой клавиатуре.
  • Количество колонок. Тип значения: «Число». Определяет количество колонок в программируемой клавиатуре
  • Шаблон подписей. Тип значения: «Строка». Указывает расположение файла с таблицей-рисунком клавиш.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов. 
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Справочники --> Розница и оборудование --> Настройки клавиатур"
