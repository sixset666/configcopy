Справочник «Разделы анкеты»

Назначение

Справочник содержит варианты разделов анкет, которые используются при заполнении типовых анкет.

Параметры

Длина кода: 8, тип кода: Строка.

Длина наименования: 150.

Не иерархический.

Реквизиты

  • Комментарий. Тип значения: «Строка». Комментарий раздела.

Формы

  • Форма выбора. Основная форма выбора. Используется для выбора элементов справочника.
  • Форма списка. Основная форма списка. Используется для отображения списка.

Способ вызова

Вызывается из меню "Справочники --> Взаимоотношения с клиентами --> Разделы анкеты".
