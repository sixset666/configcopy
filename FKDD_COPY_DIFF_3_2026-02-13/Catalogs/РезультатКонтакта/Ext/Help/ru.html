Справочник "Результат контакта"

Назначение

Справочник предназначен для хранения обобщенных вариантов результатов контактов. Значения из этого справочника используются при выборе результата события «Тест-драйв», а также в отчетах по анализу контактов и оценке удовлетворенности клиентов.

Относится к подсистеме: «Сервис».

Параметры

Длина кода: 10, тип кода:Строка.

Длина наименования: 100.

Иерархический

Реквизиты

  • Комментарий. Любая дополнительная информация. 

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма группы. Основная форма группы. Используется для заполнения обязательных реквизитов группы.  

Способ вызова

Вызывается из меню "Справочники --> Взаимоотношения с клиентами --> Результат контакта".
