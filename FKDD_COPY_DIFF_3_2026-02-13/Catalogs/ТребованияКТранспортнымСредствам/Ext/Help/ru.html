Справочник "Требования к транспортным средствам"

Назначение

Справочник "Требования к транспортным средствам" содержит перечень параметров, которым долны удовлетворять транспортные средства про прохождении государственного технического осмотра.

Относится к подсистеме: «Технический осмотр».

Параметры

Длина кода: 10, тип кода:Строка.

Длина наименования: 150.

Иерархия групп и элементов

Реквизиты

  • Номер. Число. Номер требования, согласно диагностической карты.
  • Полное наименование. Строка. Полное наименование требования.
  • Описание. Строка. Подробное описание требования.
  • Значение минимальное. Число. Минимальное значение данного требования вся всех категорий транспортных средств (если такое необходимо).
  • Значение максимальное. Число. Максимальное значение данного требования вся всех категорий транспортных средств (если такое необходимо).

Табличные части

  • Диапазон значений. Диапазон допустимых значений требования для различных категорий транспортных средств (если такое необходимо).

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Справочники --> Автосервис --> Требования к транспортным средствам".

Дополнительная информация

Загрузка требований может быть выполнена из макета при первом запуске системы или из формы списка по кнопке "Загрузить". Для загрузке используется макет справочника "Приложение1".
