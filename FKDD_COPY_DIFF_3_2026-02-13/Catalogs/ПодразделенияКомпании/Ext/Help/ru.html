Справочник "Подразделения компании"

Назначение

Справочник содержит фактическую организационную структуру автоматизируемой информационной базы (ИБ) нашей компании. При этом справочник представляет собой иерархию элементов, описывающую основные точки центров ответственности компании. Различные подразделения могут располагаться в разных узлах ИБ компании. Каждый документ, оформляемый в ИБ, обязательно привязывается к какому-либо подразделению.

Для каждого подразделения может быть назначен префикс номера документов. Это позволяет вести раздельную нумерацию для документов, оформленных от имени разных подразделений, а также для распределенных ИБ.

Подразделение заполняется во всех документах, оформляемых в информационной базе. Подразделение используется в качестве дополнительного аналитического признака во многих отчетах.

Относится к подсистеме: «Структура компании».

Предопределенные элементы

  • Вся компания целиком (измените на ваше название). Элемент, зарезервированный для основного подразделения.

Параметры

Длина кода: 8, тип кода: Строка.

Длина наименования: 50.

Иерархический.

Реквизиты

  • Организация. Реквизит является обязательным для заполнения. Ссылается на справочник «Организации». Организация, к которой относится подразделение. 
  • Префикс. Тип значения: «Строка». Префикс автонумерации для документов (до 3-х букв). Подстановка префикса в документ зависит от параметров префиксации, указанных в правах и настройках.
    Доступ осуществляется через «АРМ администратора».
  • График работы. Ссылается на справочник «Графики работы». График работы данного подразделения.
  • Балансовое. Тип значения: «Булево». Признак ведения балансового учета в подразделении.
  • КПП. Тип значения: «Строка». Код причины постановки на учет. 
  • Формировать цену розничной торговли. Тип значения: «Булево». При установлении флага цена товара в документах поступления будет фиксироваться как розничная цена товара.
  • Формировать закупочную цену. Тип значения: «Булево». При установлении флага цена товара в документах поступления будет фиксироваться как закупочная цена товара.
  • Формировать нормативную цену. Тип значения: «Булево». При установлении флага цена товара в документах поступления будет фиксироваться как нормативная цена товара.

    Информация об изменениях цен сохраняется с учетом истории изменения. Для просмотра ранее установленных значений используется список регистра сведений «Цены».
    Вызов: подменю формы "Перейти --> Цены".

  • Переоценка розничных остатков при оприходовании товаров. Тип значения: «Булево». Установленный флаг означает, что при оприходовании товаров на  предприятии будет происходить переоценка розничных остатков.
  • Переоценка розничных остатков при списании товаров. Тип значения: «Булево». Установленный флаг означает, что переоценка розничных остатков будет происходить при списании товаров на  предприятии.
  • Ответственный за тест-драйв. Ссылается на справочник «Пользователи». Пользователь, которому будут приходить напоминания о необходимости заказать автомобиль для тест-драйва, вывести автомобиль из тест-драйва. Используется только для автосалона.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка подразделений компании. На форме имеется кнопка "Права и настройки", при нажатии на которую открывается «АРМ администратора» на закладке "Права и настройки". На этой закладе администратор может установить права и настройки для выбранного подразделения.

    Информация об изменениях прав и настроек подразделения сохраняется с учетом истории изменения. Для просмотра ранее установленных значений используется список регистра сведений «Права и настройки».
    Вызов: подменю формы "Перейти --> Права и настройки".

  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов. Также форма элемента используется для заполнения параметров подразделения компании. Форма содержит две закладки, которые устанавливают соответствующие параметры подразделения.
    • Закладка "Основные данные". Устанавливаются основные данные о предприятии: организация, к которой относится подразделение, график работы, формирование цены, переоценка розничных товаров, партионный учет. 
    • Закладка "Контактная информация". Предоставляет возможность просмотра и редактирования регистра сведений «Контактная информация» связанного с подразделением компании.

Способ вызова

Вызывается из меню "Справочники --> Структура компании --> Подразделения компании".

Дополнительная информация

Во время работы с формой элемента доступны следующие сочетания клавиш для быстрого ввода:

  • [Ctrl+T] - Добавить новый телефон;
  • [Ctrl+W] - Добавить новый адрес Web-страницы;
  • [Ctrl+E] - Добавить новый адрес электронной почты;
  • [Ctrl+R] - Добавить новый адрес.
