Справочник "Команды мобильного приложения"

Назначение

Справочник используется для создания и хранения команд для обмена с мобильным приложением.

Относится к подсистеме: «Обмен с мобильным устройством».

Параметры

Длина кода: 10, тип кода: Строка.

Длина наименования: 100.

Не иерархический.

Реквизиты

  • Представление на мобильном устройстве. Тип значения: «Строка». Длина: 50. Хранит представление, которое будет отображаться на мобильном устройстве.  
  • Порядок. Тип значения: «Число». Порядок сортировки.
  • Активность. Тип значения: «Булево». Активность команды.
  • Синхронизация. Тип значения: «Число». Синхронизация.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка и выбора элементов.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.
