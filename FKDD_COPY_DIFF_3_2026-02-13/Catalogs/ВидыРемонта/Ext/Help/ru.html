Справочник "Виды ремонта"

Назначение

Справочник "Виды ремонта" предназначен для классификации работ по заказ-нарядам и контроль оплаты по выполненным работам и примененным деталям для выполнения работ по заказ-нарядам. Также в справочнике существует возможность настройки отправки смс уведомлений, по каждому из видов ремонта.

Относится к подсистеме: «Автосервис».

Параметры

Длина кода: «8», тип кода: «Строка».

Длина наименования: «100».

Не иерархический.

Реквизиты

  • Тип ремонта. Реквизит является обязательным для заполнения. Ссылается на перечисление «ТипыРемонта». Типы ремонта (платный/бесплатный).
  • На себестоимость. Тип значения: «Булево».  Признак, по которому стоимость ремонта распределяется на себестоимость автомобиля.
  • Статья расхода деталей. Ссылается на справочник «СтатьиДоходовИРасходов». Статья, по которой расходуются детали. 
  • Статья дохода деталей. Ссылается на справочник «СтатьиДоходовИРасходов». Статья, по которой приходят детали.
  • Статья дохода работ. Ссылается на справочник «СтатьиДоходовИРасходов». Статья, по которой выполняются работы.
  • Техобслуживание. Тип значения: «Булево».  Признак технического обслуживания.
  • Выработка сотрудников по выполнению заказ наряда. Тип значения: «Булево».  Признак того, как выставляется выработка.
  • Работы без НДС. Тип значения: «Булево».  Признак указывающий на то, что работы выполняется без учета НДС.
  • Товары без НДС. Тип значения: «Булево».  Признак указывающий на то, что детали испоьзуются без учета НДС.
  • Отправлять уведомления. Тип значения: «Булево».  Признак, указывающий на необходимость отправки смс уведомлений. Если галочка установлена, то при отклонении заявки на ремонт и изменения состояния заказ наряда, пользователю будут выводится «ДокументSMS» в интерактивном режиме, а также будут формироватся сообщения смс, с напоминанием о том, что заказчик записан на ремонт, и отправлятся в автоматическом режиме.
  • Шаблон при записи на ремонт. Ссылается на справочник «ШаблоныТекстовыхСообщений». Шаблон, по которому будет формироваться сообщение при записи на ремонт. Если шаблон не указан, то сообщения формироватся не будут.
  • Шаблон при отмене заявки. Ссылается на справочник «ШаблоныТекстовыхСообщений». Шаблон, по которому будет формироваться сообщение при отмене заявки.
  • Время начала отправки. Тип значения: «Дата». Начало промежутка времени, в которое необходимо отправлять смс.
  • Время окончания отправки. Тип значения: «Дата». Конец промежутка времени, в которое необходимо отправлять смс.
  • Отправлять за время. Тип значения: «Число». Количество часов, за которое необходимо отправлять смс сообщение. Реквизит является обязательным для заполнения, если шаблон при записи на ремонт не пустой. Минимально возможное значение составляет 1 час.

Табличная часть «Гарантия»

  • Модель. Реквизит является обязательным для заполнения. Ссылается на справочник «Модели». Модель автомобиля.
  • Контрагент. Реквизит является обязательным для заполнения. Ссылается на справочник «Контрагенты». Гарантийный плательщик.
  • Договор взаиморасчетов. Реквизит является обязательным для заполнения. Ссылается на справочник «ДоговорыВзаиморасчетов». Договор взаиморасчетов. 

Табличная часть «Работы»

  • Авторабота. Реквизит является обязательным для заполнения. Ссылается на справочник «Автоработы». Авторабота по умолчанию.
  • Количество. Реквизит является обязательным для заполнения. Тип значения: «Число». Количество авторабот.

Табличная часть «Настройка уведомлений»

  • Состояние заказ наряда. Реквизит является обязательным для заполнения. Ссылается на справочник «ВидыСостоянийЗаказНарядов». Состояние заказ-наряда.
  • Шаблон сообщения. Реквизит является обязательным для заполнения. Ссылается на справочник «ШаблоныТекстовыхСообщений». Шаблон сообщения для текущего состояния.

Формы

  • Форма списка. Основная форма списка, выбора и редактирования. Используется для отображения списка, выбора элементов и редактирования справочника.
  • Форма элемента. Основная форма редактирования элемента. Используется для отображения редактирования элементов справочника.

Способ вызова

Вызывается из меню "Справочники --> Автосервис --> Виды ремонта".
