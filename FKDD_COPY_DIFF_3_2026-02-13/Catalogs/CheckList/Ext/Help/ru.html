Справочник "Список проверки транспортного средства (т\с)"

Назначение

Справочник "Список проверки транспортного средства(т\с)" предназначен для ведения перечня пунктов проверки автомобиля при его приемке.

Каждый элемент справочника описывает пункт который должен проверить мастер приемки при приеме автомобиля от клиента.

Относится к подсистеме: «Обмен с мобильным устройством».

Параметры

Длина кода: 10, тип кода: Строка.

Длина наименования: 50.

Не иерархический.

Владельцы справочника: нет.

Реквизиты

  • Представление. Тип значения: «Строка». Данное описание будет предоставлено при выводе пункта проверки пользователю.
  • Тип значения. Тип значения: «Число». Определяет какого типа должно быть пояснение к пункту проверки.
  • Доступные варианты. Тип значения: «Список значений». Определяет список возможных знаений у пункта проверки. доступен только для типа значения "Выбор из списка значений".

Формы

  • Форма списка. Основная форма выбора группы. Используется для выбора групп.
  • Форма элемента. Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из меню "Справочники -- Структура компании -- Список проверки транспортного средства(т\с)".
