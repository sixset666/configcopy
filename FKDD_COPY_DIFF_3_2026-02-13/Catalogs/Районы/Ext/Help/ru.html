Справочник "Районы"

Назначение

Справочник "Районы" предназначен для ведения списка городских районов.

Относится к подсистеме: «Классификаторы».

Параметры

Длина кода: 8, тип кода: Строка.

Длина наименования: 50.

Иерархический.

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора элементов. На нижней командной панели находится кнопка "Загрузить", при нажатии которой осуществляется загрузка списка городских районов из макета.

Способ вызова

Вызывается из меню "Справочники --> Классификаторы --> Районы"

Дополнительная информация

Загрузка районов из макета выполняется при нажатии кнопки "Загрузить" на форме списка справочника и при первом старте системы. Перед загрузкой пользователю будет предложено выбрать из списка макет, список районов из которого будет загружен.
